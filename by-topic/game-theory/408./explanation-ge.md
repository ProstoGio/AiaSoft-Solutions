# განმარტება — 408. სამი გოჭი

## მიდგომა

ეს ამოცანა ჩვეულებრივი ჯეირანია (ქვა-მაკრატელი-ქაღალდი), სადაც უბრალოდ უნდა დავითვალოთ გამარჯვებები. ციკლში ვკითხულობთ თითოეულ რაუნდს და ვამოწმებთ ვის აქვს უკეთესი არჩევანი მოცემული წესების მიხედვით. თუ ფრეა, არცერთს ვუმატებთ ქულას, ხოლო ბოლოს უბრალოდ ვადარებთ ერთმანეთს დაგროვილ ქულებს. თუ ტოლია, მაშინ ნუფნუფი იგებს.

## სირთულე

| | |
|-|--|
| დრო | O(N) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- თუ ქულები თანაბარია, გამარჯვებული ნუფნუფია.
- არ გვჭირდება ქულების შენახვა მასივში, საკმარისია ორი ცვლადი.

## ხშირი შეცდომები

- იმის დავიწყება, რომ ფრის შემთხვევაში არავის ემატება ქულა.
- პირობების არასწორად დაწერა, სადაც 1, 2 და 3 მაკრატელ-ქვა-ბადეს შეესაბამება.

## ნაბიჯ-ნაბიჯ ახსნა

შემოგვაქვს N, შემდეგ ვტრიალებთ ციკლით N-ჯერ. ყოველ ჯერზე ვკითხულობთ ორ რიცხვს, ვამოწმებთ if-else პირობებით ვინ მოიგო რაუნდი და ვუმატებთ ქულას შესაბამის გოჭს. ბოლოს შედარებით ვადგენთ გამარჯვებულის სახელს.
