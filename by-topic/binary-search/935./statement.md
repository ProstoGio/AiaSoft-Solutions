# Problem Statement — 935. ეშმაკური იდეა

> Copied from [AiaSoft.ge #935](https://aiasoft.ge/problem/935)

**Time limit:** 3.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მრავალსართულიან კორპუსში, რომელშიც ცოტნე ცხოვრობს, სულ N რაოდენობის ბინაა. ბინები გადანომრილია 1-დან N-მდე ნატურალური რიცხვებით და თითოეულ ბინას თავისი ნომერი კარებზე მიკრულ ფირფიტაზე აწერია. ცოტნემ, რომელიც არითმეტიკაში უკვე საკმაოდ ერკვევა და თანაც სიცუღლუტითაც გამოირჩევა, გადაწყვიტა ფირფიტების ნაწილი შეცვალოს ისე, რომ ყველა ბინის ნომერთა ჯამმა ზუსტად S რიცხვი შეადგინოს. მისი ჩანაფიქრის განხორციელებას ხელს ისიც უწყობს, რომ მას შეუძლია მაღაზიაში შეიძინოს ფირფიტები ნებისმიერი ნატურალური რიცხვებით. თუმცა, ცოტნეს სულაც არ უნდა ზედმეტი თანხა დახარჯოს და აინტერესებს, რა მინიმალური თანხა დასჭირდება თავისი იდეის რეალიზაციისათვის. აქ კი მას ცოტა გაუჭირდა, რადგან ასეთი გამოთვლების განსახორციელებლად ჯერ კიდევ საკმაოდ პატარაა. დაეხმარეთ ცოტნეს და დაწერეთ პროგრამა, რომელიც დაადგენს ფირფიტების იმ მინიმალურ რაოდენობას, რომელთა ყიდვაც დასჭირდება მას თავისი ეშმაკური ჩანაფიქრის განსახორციელებლად.

## Input

ფაილის ერთადერთ სტრიქონში მოცემულია ერთი ჰარით გამოყოფილი ორი მთელი N და S რიცხვი.
1 ≤ N ≤ 10⁹
1 ≤ S ≤ 10¹⁸

## Output

ფაილის ერთადერთ სტრიქონში უნდა ჩაიწეროს ერთი მთელი რიცხვი - ფირფიტების ის მინიმალური რაოდენობა, რომელთა ყიდვაც დასჭირდება ცოტნეს, რომ მისთვის სასურველი S ჯამი მიიღოს. პასუხის არსებობა გარანტირებულია.
