# განმარტება — 145. სამკუთხედის ფართობი (კოორდინატებით)

## მიდგომა

თავიდანვე ჰერონის ფორმულა გამახსენდა, რადგან წვეროების კოორდინატები გვაქვს და მანძილების პოვნა მარტივია. შევქმენი Point სტრუქტურა კოორდინატებისთვის და Triangle, სადაც გვერდების სიგრძეებს ვიანგარიშებ მანძილის ფორმულით. ბოლოს უბრალოდ ჰერონის ფორმულაში ჩავსვი პერიმეტრის ნახევარი და ფართობიც ამოვიდა. Shoelace ფორმულა უფრო მოკლეა, მაგრამ ეს მეთოდი ბევრად უფრო მკაფიოა და ნაკლებად დამაბნეველი, თუ გეომეტრიის საფუძვლები იცი.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- ჰერონის ფორმულა მშვენივრად მუშაობს, რადგან სამივე გვერდის სიგრძის დათვლა პითაგორას თეორემით ადვილად გამოდის.
- აუცილებლად double ტიპი უნდა გამოვიყენოთ, რომ კვადრატული ფესვების დროს სიზუსტე არ დავკარგოთ.

## ხშირი შეცდომები

- fixed-ით და setprecision-ით დაფორმატება თუ დაგავიწყდა, პასუხს არასწორად ჩაგითვლის, რადგან ნაგულისხმევი სიზუსტე არასაკმარისია.
