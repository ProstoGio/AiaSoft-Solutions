# Problem Statement — 145. სამკუთხედის ფართობი (კოორდინატებით)

> Copied from [AiaSoft.ge #145](https://aiasoft.ge/problem/145)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია სამკუთხედი, რომლის სამივე წვეროს კოორდინატები მთელი რიცხვებია. იპოვეთ და გამობეჭდეთ მისი ფართობი.

## Input

პირველ ხაზზე შემოდის სამუკუთხედის A წვეროს კოორდინატები: x1 და y1
მეორე ხაზზე შემოდის სამუკუთხედის B წვეროს კოორდინატები: x2 და y2
მესამე ხაზზე შემოდის სამუკუთხედის C წვეროს კოორდინატები: x3 და y3

გარანტირებულია, რომ სამივე წერტილი ერთ წრფეზე არ მდებარეობს და ყველა შემოსული რიცხვი მოდულით ნაკლებია ან ტოლია 1000 - ზე.

## Output

გამოიტანეთ ერთი რიცხვი - სამკუთხედის ფართობი, რომლის სიზუსტე უნდა იყოს არანაკლებ 5 ციფრი მძიმის შემდეგ.
