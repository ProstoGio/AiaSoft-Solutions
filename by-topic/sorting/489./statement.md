# Problem Statement — 489. დალაგება სიხშირით

> Copied from [AiaSoft.ge #489](https://aiasoft.ge/problem/489)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია N ელემენტიანი მიმდევრობა. დაალაგეთ მოცემული მიმდევრობა რიცხვთა სიხშირის ზრდადობის მიხედვით. თუ რამდენიმე რიცხვს აქვს თანაბარი სიხშირე, მაშინ პირველად გამობეჭდეთ მათგან უდიდესი მნიშვნელობის მქონე.

## Input

პირველ ხაზზე მოცემულია ერთი მთელი რიცხვი: N - მასივში ელემენტების რაოდენობა.
მეორე ხაზზე მოცემულია N ცალი მთელი რიცხივი: a1, a2, ..., an

1<=N<=100
-100<=ai<=100

## Output

ერთ ხაზზე გამობეჭდეთ ზემოთ მოცემული პირობით დალაგებული მიმდევრობა.
რიცხვები გამოყავით თითო ჰარით.
