# განმარტება — 297. უმოკლესი გზა

## მიდგომა

რადგან გრაფი არაწონიანია და გვინდა უმოკლესი გზა, BFS იდეალური არჩევანია. ვიწყებთ საწყისი წვეროდან, ვათვალიერებთ მეზობლებს ფენა-ფენა და როგორც კი მივადგებით სამიზნეს, ეგრევე ვბეჭდავთ მანძილს. `d` მასივში ვინახავთ რამდენი ნაბიჯი დაგვჭირდა, ხოლო `used` გვჭირდება იმისთვის, რომ წრეში არ ვიტრიალოთ.

## სირთულე

| | |
|-|--|
| დრო | O(N + M) |
| მეხსიერება | O(N) |

## მთავარი დაკვირვებები

- BFS-ის თავისებურება ისაა, რომ პირველივე ჯერზე, როცა სამიზნე წვეროს მივაგნებთ, ეგ არის გარანტირებულად უმოკლესი მანძილი.
- ადჯასენსი ლისტი (adjacency list) უფრო მოსახერხებელია, ვიდრე მატრიცა, მით უმეტეს როცა გრაფი გათხელებულია.

## ხშირი შეცდომები

- თუ გრაფი დაშლილია და A-დან B-მდე გზა არ არსებობს, ბევრს ავიწყდება -1-ის დაბეჭდვა.

## ნაბიჯ-ნაბიჯ ახსნა

დავუშვათ, გვაქვს 1-დან 3-მდე გზა. ვიწყებთ 1-ით, ვამატებთ ქიუში. ვიღებთ 1-ს, მის მეზობლებს (ვთქვათ 2-ს) ვანიჭებთ მანძილს 1. შემდეგ ვიღებთ 2-ს, ვპოულობთ 3-ს, ვანიჭებთ მანძილს 2 და როგორც კი 3 ამოვა ქიუდან, ვამთავრებთ მუშაობას.
