# Problem Statement — 382. საგუშაგოები

> Copied from [AiaSoft.ge #382](https://aiasoft.ge/problem/382)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

თქვენი ქალაქი N უბნისგან შედგება, რომლებიც ერთმანეთთან დაკავშირებულია M `bცალმხრივი` გზით. თქვენი, როგორც ქალაქის მერის, პასუხისმგებლობაა ყველა უბნის უსაფრთხოება.

უსაფრთხოების უზრუნველყოფისთვის რამდენიმე პოლიციის საგუშაგოს აშენება გადაწყვიტეთ. საგუშაგოს აშენება მხოლოდ უბნებშია დასაშვები. i საგუშაგოს პოლიციას შეუძლია უსაფრთხოების უზრულველყოფა ყველა ისეთ j უბანში, რომელშიც i უბნიდან მისვლა და უკან დაბრუნება შესაძლებელია.

საგუშაგოების აშენება თავის მხრივ ხარჯებთანაა დაკავშირებული. ზოგიერთი უბანი ქალაქში უფრო პოპულარულია, ამიტომ ზოგ უბანში საგუშაგოს აშენება შესაძლოა უფრო ძვირი დაჯდეს ვიდრე სხვებში.

თქვენი მიზანია მინიმალური ხარჯებით უზრუნველყოთ ყველა უბნის უსაფრთხოება. ასევე გაინტერესებთ საგუშაგოების აშენების რამდენი განსხვავებული ვარიანტი არსებობს, ისე, რომ მინიმალური თანხა დაიხარჯოს და მინიმალური რაოდენობის საგუშაგოები აშენდეს.

## Input

პირველ ხაზზე მოცემულია ერთი მთელი რიცხვი: N - უბნების რაოდენობა ქალაქში.
მომდევნო ხაზზე მოცემულია N რაოდენობის არაუარყოფითი მთელი რიცხვი: c1, c2, ..., cn. ci აღნიშნავს i უბანში საგუშაგოს აშენების ფასს.
მესამე ხაზზე მოცემულია ქალაქში გზების რაოდენობა: M.
მომდევნო M ხაზი შეიცავს ორ-ორ რიცხვს. კერძოდ i-ურ ხაზზე მოცემულია vi და ui,  რაც ნიშნავს, რომ მოცემული ცალმხრივი გზა მიმართულია vi უბნიდან ui უბნისკენ.

1 <= N <= 10⁵
0 <= ci <= 10⁹
0 <= M <= 3*10⁵
1 <= vi, ui <= N, (vi != ui)

## Output

ერთ ხაზზე გამობეჭდეთ  ჰარით გამოყოფილი ორი მთელი რიცხვი, რომელთაგან პირველი იქნება საგუშაგოების აშენების მინიმალური ღირებულება, ხოლო მეორე - საგუშაგოების მინიმალურ ფასად აშენების სხვადასხვა ვარიანტების რაოდენობა, ეს უკანასკნელი გამობეჭდეთ 10⁹+7-ზე დანაშთული სახით.
