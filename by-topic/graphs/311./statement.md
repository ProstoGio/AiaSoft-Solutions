# Problem Statement — 311. ძლიერი სამეფო

> Copied from [AiaSoft.ge #311](https://aiasoft.ge/problem/311)

**Time limit:** 5.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

გაგიგიათ ხალხური ანდაზა: სახელმწიფო მაშინაა ძლიერი, თუ მისი ნებისმიერი ქალაქიდან ნებისმიერ სხვა ქალაქში მისვლა შესაძლებელია და თანაც მაქსიმუმ T საათშიო?

ეს ძველი ბალმონდელი ხალხის გამონათქვამია და მათ ამის სჯეროდათ. ბალმონდის ახლანდელი მეფე, თემური, ძველი ტრადიციების, წეს-ჩვეულებებისა და თქმულებების დიდი მოყვარულია. მას აინტერესებს, ბალმონდის ქალაქების ახლანდელი განლაგება რამდენად აკმაყოფილებს ზემოთ თქმულ მოთხოვნებს.

მან ამ საქმის მოგვარება თქვენ, როგორც მის უერთგულეს პროგრამისტ ვეზირს, დაგავალათ და თან ქვეყნის რუკა გადმოგცათ, სადაც დატანილია ყველა საჭირო მონაცემი ამ ამოცანის ამოსახსნელად.

ირკვევა, რომ ბალმონდი სულ N ქალაქისგან შედგება და ამ ქალაქებს შორის სულ M ორმხრივი გზა გადის (რა, თქმა უნდა დაზუსტებით ვიცით რომელი გზა რომელ ორ ქალაქს აკავშირებს და რა დრო სჭირდება ამ გზის დაფარვას). ზოგ ქალაქს შორის შესაძლოა ერთზე მეტი პირდაპირი გზაც არსებობდეს.

დაადგინეთ, არის თუ არა ქვეყანა ბალმონდი ძლირი სამეფო.

## Input

პირველ ხაზზე მოცემულია სამი მთელი რიცხვი: N, M და T
მომდევნო M ხაზზე მოცემულია გზების აღწერა. კერძოდ i-ურ ხაზზე მოცემული იქნება სამი რიცხვი: vi, ui, ti - რაც ნიშნავს, რომ i-ური გზა აკავშირებს vi და ui ქალაქებს, და ამ გზის გავლას ti საათი სჭირდება.

1 <= N <= 500
0 <= M <= 10⁵
1 <= T, ti <= 10⁹
1 <= vi, ui <= N,   vi != ui

## Output

გამოიტანეთ `byes` თუ ბალმონდის ნებისმიერი ქალაქიდან ნებისმიერ ქალაქში მისვლა შესაძლებელია T საათში, წინააღმდეგ შემთხვევაში - `bno`.
