# განმარტება — 662. ტყე

## მიდგომა

ამ ამოცანის მთავარი მუღამი ისაა, რომ თუ ხის რომელიმე ქვეხეს ლუწი რაოდენობის წვერო აქვს, მისი მოჭრით ორივე მხარეს (მოჭრილ კომპონენტსაც და დარჩენილსაც) ლუწი წვეროები დარჩება. ანუ, უბრალოდ DFS-ით უნდა დავითვალოთ თითოეული წვეროს ქვეშ არსებული წვეროების რაოდენობა. როგორც კი ქვეხეში წვეროების რაოდენობა ლუწი აღმოჩნდება, იმ წიბოს თამამად ვჭრით.

## სირთულე

| | |
|-|--|
| დრო | O(n + m) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- თუ subtree-ს წვეროების რაოდენობა ლუწია, იმ წიბოს მოჭრა, რომელიც ამ subtree-ს თავის მშობელთან აკავშირებს, პრობლემას არ ქმნის.
- DFS-ის ბოლოში, როცა რეკურსია ბრუნდება, მშობელს ვუმატებთ შვილის 'leaves' რაოდენობას და ზომა ავტომატურად ითვლება.

## ხშირი შეცდომები

- გრაფის არასწორი წარმოდგენა ან 0-დან იდექსირების არევა, როცა წვეროები 1-დანაა დანომრილი.

## ნაბიჯ-ნაბიჯ ახსნა

DFS-ს ვიწყებთ 1-დან. თითოეულ წვეროს თავიდანვე ვანიჭებთ 1-ს (თავისი თავი). როცა რეკურსია ბრუნდება, შვილის ქვეხის ზომას ვუმატებთ მშობლისას. ბოლოს უბრალოდ ვამოწმებთ 2-დან n-მდე ყველა წვეროს, და თუ მისი ქვეხის ზომა ლუწია, ვთვლით, რომ ეგ წიბო მოსაჭრელია.
