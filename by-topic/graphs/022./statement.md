# Problem Statement — 022. ღრმად ძებნაში შესვლის და გამოსვლის დრო

> Copied from [AiaSoft.ge #22](https://aiasoft.ge/problem/22)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია ბმული, არაორიენტირებული გრაფი. გრაფი შეიცავს n წვეროს და m წიბოს. ვთქვათ, ღრმად ძებნის ალგორითმი გავუშვით რაიმე x წვეროდან. ნებისმიერი k წვეროსთვის განვიხილოთ რიცხვი, რომელიც უდრის რიგით მერამდენე იყო ეს წვერო რომელიც DFS-მა მოინახულა. ამ რიცხვს ეწოდება DFS-ში შესვლის დრო k წვეროსთვის და აღინიშნება როგორც t_in[k]. ასევე, ნებისმიერი k წვეროსთვის განვიხილოთ რიცხვი, რომელიც უდრის რიგით მერამდენე იყო ეს წვერო, რომელშიც DFS-ის ალგორითმი დასრულდა (ამ წვეროდან გასულმა ყველა რეკურსიამ მუშაობა დაასრულა) ამ რიცხვს ეწოდება DFS-დან გამოსვლის დრო k წვეროსთვის და აღინიშნება როგორც t_out[k]. ცხადია, DFS-ში შესვლისა და DFS-დან გამოსვლის დროები დამოკიდებულია იმაზე თუ საიდან დაიწყებს DFS მოძრაობას და ასევე, იმაზეც თუ რა რიგით გადავალთ ერთი წვეროდან მის მეზობლებთან. 

ვთქვათ, ერთი წვეროდან მის მეზობლებთან გადავდივართ მათი ნომრების ზრდის მიხედვით.

მოცემული გრაფისათვის და მოცემული საწყისი წვეროსათვის იპოვეთ DFS-ში შესვლის და DFS-დან გამოსვლის დროები ყველა წვეროსათვის.

## Input

პირველ ხაზზე შემოდის n და m შესაბამისად, გრაფის წვეროების და წიბოების რაოდენობა.
მომდევნო m ხაზზე შემოდის ორი რიცხვი u და v რაც ნიშნავს, რომ მათ შორის არსებობს წიბო.
ხოლო ბოლო ხაზზე შემოდის ერთადერთი რიცხვი x, საიდანაც DFS იწყებს მოძრაობას.
1 ≤ x ≤ n ≤ 10⁴
1 ≤ m ≤ 10⁶

## Output

პირველ ხაზზე გამოიტანეთ t_in მასივი, რომელშიც i-ურ ადგილას იქნება i წვეროს DFS-ში შესვლის დრო, ხოლო მეორე ხაზზე t_out მასივი, რომელშიც i-ურ ადგილას იქნება i წვეროს DFS-დან გამოსვლის დრო.
