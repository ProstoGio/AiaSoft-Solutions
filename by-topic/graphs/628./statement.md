# Problem Statement — 628. კლონების არმია

> Copied from [AiaSoft.ge #628](https://aiasoft.ge/problem/628)

**Time limit:** 2.0s &nbsp;|&nbsp; **Memory limit:** 512 MB

## Task

x კლონისგან შემდგარი არმია შეიჭრა საჰაერო ხომალდ "სიკვდილის ვარსკვლავში" რათა დაეხმარონ ლუკა სკაიუოკერს დარს ვეიდერის დამარცხებაში. საჰაერო ხომალდი შეიცავს n ოთახს და m ორმხრივ გასასვლელს მათ შორის. კლონები თავიდან არიან ნომერ 1 ოთახში და უნდათ მივიდნენ მე-n ოთახში, იქ სადაც არის ლუკა.

თუმცა ყველა ოთახი დაცულია დროიდებით. i ნომრის ოთახს იცავს ai ცალი დროიდი. როდესაც კლონები შედიან ოთახში, ბრძოლა მათსა და დროიდებს შორის იწყება. თუ კლონების რაოდენობა უფრო მეტია ვიდრე დროიდების, კლონები მოკლავენ ყველა დროიდს და ყველა მათგანი დარჩება ცოცხალი. თუ არადა, კლონები მაინც მოკლავენ ყველა დროიდს მაგრამ მათი რაოდენობა განახევრდება: თუ არის x კლონი ბრძოლამდე, ბრძოლის შემდეგ იქნება ⌊ fracx2 ⌋ კლონი. კლონებმა უნდა იბრძოლონ ყველა ოთახში, რომელშიც მოხვდებიან, 1-ის და n-ის ჩათვლით.

დაეხმარეთ კაპიტანს დაადგინოს მაქსიმუმ რამდენი კლონი მიაღწევს 1-ლი ოთახიდან მე-n ოთახში.

*პირობაში ⌊ x ⌋ აღნიშნავს უდიდეს მთელ რიცხვს, რომელიც ნაკლები ან ტოლია x-ზე.

## Input

პირველი ხაზი შეიცავს ორ მთელ რიცხვს n,m - ოთახების და გასასვლელების რაოდენობას "სიკვდილის ვარსკვლავში" (1<=n,m<=2 · 10⁵).

შემდეგი m ხაზი აღწერს გასასვლელებს: i გასასვლელი აღიწერება ორი მთელი რიცხვით vi და ui - ოთახების ნომრები რომლებიც დაკავშირებულნი არიან i გასასვლელით. (1 <= ui,vi <=n , ui≠vi) გარანტირებულია, რომ ყოველი წყვილი ოთახი დაკავშირებულია მაქსიმუმ ერთი გასასვლელის მეშვეობით.

შემდეგ ხაზზე შემოდის ერთი მთელი რიცხვი x - კლონების რაოდენობა არმიაში (1<=x<=10⁹).

უკანასკნელი ხაზი შეიცავს n ცალ მთელ რიცხვს a1,a2,...,an - დროიდების რაოდენაბას ოთახებში (1<=ai<=10⁹).

## Output

დაბეჭდეთ ერთადერთი მთელი რიცხვი - კლონების მაქსიმალური რაოდენობა, რომლებსაც შეუძლიათ გავიდნენ 1-ლი ოთახიდან მე-n ოთახში. თუ არ არის ისეთი გზა, რომლის მეშვეობითაც 1 კლონი მაინც გადარჩება, მაშინ დაბეჭდეთ 0.
