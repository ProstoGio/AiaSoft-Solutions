# განმარტება — 062. პირველკლასელი (ოღონდ მეორე)

## მიდგომა

ამ ამოცანის მთავარი მუღამი ისაა, რომ სტრიქონი სწორად დავჭრათ და რიცხვებად გადავაქციოთ. `stringstream`-ს არ ვიყენებ, არც `stoi`-ს, არამედ ათის ხარისხებს ვინახავ წინასწარ ვექტორში. ასე უფრო კონტროლირებადია პროცესი. როგორც კი ოპერატორის სიმბოლოს ვიპოვი, მარცხნივ და მარჯვნივ დარჩენილ ნაწილებს რიცხვებად ვაწყობ და ბოლოს უბრალოდ ვუყრი თავს არითმეტიკულ მოქმედებას.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- სიმბოლოსგან ციფრის მისაღებად `s[i] - '0'` (ან `s[i] - 48`) კლასიკაა და ყოველთვის მუშაობს.
- რადგან 8 ნიშნა რიცხვებზეა ლაპარაკი, `long long`-ის გამოყენება იდეალურია, რამე რომ არ გაიჭედოს.

## ხშირი შეცდომები

- ოპერატორის პოვნაში შეცდომის დაშვება, რამაც შეიძლება ნომერი არასწორად დაჭრას.
- სტრიქონის ბოლო ინდექსების არევა ციკლის დროს.
