# განმარტება — 076. ეგოისტი ბავშვები

## მიდგომა

სინამდვილეში, აქ მთავარია დაინახო, რომ თუ ორივე ყუთის სიმბოლოებს ერთად შევყრით, ყველა ასო წყვილი უნდა იყოს. თუ რომელიმე სიმბოლო კენტი რაოდენობით გვაქვს, მაშინ ორ თანაბარ ნაწილად მათ ვერ გავყოფთ. ამიტომ, ავიღე მარტივი მასივი, სადაც თითოეული ასოს სიხშირეს ვითვლი ASCII კოდების მიხედვით. თუ ბოლოს რომელიმე პოზიციაზე კენტი რიცხვი დაგვხვდა, ესე იგი ლუკა და აკაკი კმაყოფილები ვერ დარჩებიან.

## სირთულე

| | |
|-|--|
| დრო | O(n + m) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- თუ ჯამური რაოდენობა სიმბოლოსი არის კენტი, თანაბარ კომპლექტებს ვერ მივიღებთ.
- სორტირების ნაცვლად სიხშირეების მასივის გამოყენება ამ პრობლემას ბევრად ამარტივებს და მუშაობს O(N)-ში.

## ხშირი შეცდომები

- ASCII ცხრილის საზღვრების არასწორად დათვლა ან ზედმეტი სორტირების გაკეთება, რაც დროს კარგავს.
