# Problem Statement — 346. შეტყობინება

> Copied from [AiaSoft.ge #346](https://aiasoft.ge/problem/346)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 128 MB

## Task

თამარმა შეტყობინება დაწერა, რომელიც მხოლოდ ქართული ანბანის სიმბოლოებისა და ჰარებისგან შედგებოდა. შემდეგ დაწერილი შეტყობინება შემდეგნაირად დაშიფრა: თითოეული ქართული სიმბოლო ჩაანაცვლა მისი რიგითი ნომრით ანბანში ( ა - 1, ბ - 2, გ - 3, ..., ჯ - 32, ჰ - 33), ხოლო ყველა ჰარი ჩაანაცვლა 0-ით.

მაგალითად, თუ თავდაპირველი შეტყობინება იყო "აი ია", მაშინ დაშიფრვის შემდეგ გვექნებოდა "19091", ან თუ გვქონდა "ერთი", მივიღებდით "51789"-ს.

დაშიფრვის შედეგად მიღებულ კოდებზე დაკვირვების შემდეგ, თამარი მიხვდა, რომ შესაძლოა ერთი და იგივე კოდი რამდენიმე სხვადასხვა ტექსტით მიიღებოდეს. მაგალითად, კოდი "19091" შეიძლება მივიღოთ ორი ტექსტით "აი ია" და "ტ ია" (ტ ანბანში მე-19 ადგილზე დგას).

თქვენი დავალებაა მოცემული კოდისთვის დაადგინოთ, რამდენი განსხვავებული საწყისი ტექსტით შეიძლება მოცემული კოდის მიღება.

## Input

ერთ ხაზზე მოცემულია კოდი, რომელიც მხოლოდ ციფრებისგან შედგება. კოდის სიგრძე არ აღემატება 80-ს.

## Output

გამოიტანეთ ერთი რიცხვი, საწყისი ტექსტების რაოდენობა, რომელთა დაშიფრვის შედეგადაც მოცემული კოდი მიიღება.
