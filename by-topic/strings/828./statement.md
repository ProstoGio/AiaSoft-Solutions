# Problem Statement — 828. კატო და საყვარელი სიმბოლო

> Copied from [AiaSoft.ge #828](https://aiasoft.ge/problem/828)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

კატოს ძალიან უყვარს ავტობუსით მგზავრობა და მგზავრობისას მაღაზიების აბრებზე დაკვირვება. კატო სკოლაში წასვლამდე ირჩევს საყვარელ სიმბოლოს და აბრებზე დავირვებისას, მათში ითვლის არჩეული სიმბოლოს რაოდენობას.
მალე სკოლაში არდადეგები იწყება დაკატოს იშვიათად მოუწევს ხოლმე ავტობუსით მგზავრობა, რის გამოც ის დამწუხრებულია. თქვენი დავალებაა კატოს გამხიარულება შემდეგი თამაშით: კატო გეუბნებათ აბრაზე გამოკრულ სტრიქონს და ასევე მის საყვარელ სიმბოლოს. თქვენ უნდა დათვალოთ სტრიქონში კატოს საყვარელი სიმბოლოს რაოდენობა.

## Input

ერთ ხაზზე მოცემულია S და K - შესაბამისად აბრაზე გამოკრული სტრიქონი და კატოს საყვარელი სიმბოლო. 
გარანტირებულია, რომ S შეიცავს მხოლოდ ლათინური ანბანის პატარა და დიდ სიმბოლოებს და 1<=|S|<=500

## Output

გამობეჭდეთ აბრაზე კატოს საყვარელი სიმბოლოების რაოდენობა.
