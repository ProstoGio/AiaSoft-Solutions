# განმარტება — 822. ნულების ამოგდება

## მიდგომა

რადგან რიცხვი მაქსიმუმ 9-ნიშნაა, რიცხვად შენახვას აზრი არ აქვს, ჯობია პირდაპირ სტრინგად წავიკითხოთ. ასე მარტივად გავივლით თითოეულ სიმბოლოს ციკლით და შევამოწმებთ, ნულია თუ არა. თუ ნოლი არ არის, ახალ სტრინგს მივაწერ და ეგაა, შედეგიც მზად არის.

## სირთულე

| | |
|-|--|
| დრო | O(N) |
| მეხსიერება | O(N) |

## მთავარი დაკვირვებები

- სტრინგებთან მუშაობა ბევრად მარტივია, ვიდრე ციფრების ამოჭრა მათემატიკური ოპერაციებით.
- O(N) სირთულე სრულიად საკმარისია ასეთი პატარა შეზღუდვისთვის.

## ხშირი შეცდომები

- integer ტიპის ცვლადების გამოყენება, რაც ართულებს ლოგიკას.
- რიგითობის არევა, თუ რამეს არასწორად დავაპრინტავთ.

## ნაბიჯ-ნაბიჯ ახსნა

შემომყავს რიცხვი string-ად. შემდეგ ვაკეთებ ციკლს, რომელიც გადის სტრინგის ყველა სიმბოლოს. თუ სიმბოლო არ უდრის '0'-ს, ვამატებ მას ახალ სტრინგში (n2). ბოლოს უბრალოდ ვბეჭდავ n2-ს.
