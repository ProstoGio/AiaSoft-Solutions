# განმარტება — 214. შეცვალეთ სიმბოლოები

## მიდგომა

ეს პრობლემა ძაან მარტივია, პირდაპირ ASCII კოდებზე დაყრდნობით ვაკეთებ. მთავარი იდეა ისაა, რომ შევამოწმოთ სიმბოლო 97-ზე ნაკლებია თუ არა; თუ ნაკლებია, ესე იგი დიდი ასოა და პატარად ვაქცევ, ხოლო თუ არა - პირიქით. ვექტორში ვინახავ სიმბოლოებს და რიგ-რიგობით ვამუშავებ თითოეულს `tolower` და `toupper` ფუნქციების გამოყენებით.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- ASCII ცხრილში დიდი ლათინური ასოები 97-ზე ნაკლებია, რაც პირდაპირ გვაძლევს პირობის შემოწმების საშუალებას.
- არ არის საჭირო ხელით 32-ის მიმატება-გამოკლება, როცა C++-ის სტანდარტული ბიბლიოთეკა გვთავაზობს მზა ფუნქციებს.

## ხშირი შეცდომები

- ზოგჯერ ავიწყდებათ, რომ input-ში შესაძლოა იყოს ისეთი სიმბოლო, რომელიც არ არის ლათინური ასო, თუმცა ამ კონკრეტულ ამოცანაში მხოლოდ ასოებზეა საუბარი.
