# განმარტება — 086. სტრიქონის სიგრძე

## მიდგომა

ამ ამოცანაში მთავარი ისაა, რომ სტრიქონში შეიძლება სფეისები (spaces) შეგვხვდეს. თუ ჩვეულებრივ cin-ს გამოვიყენებდი, პირველ სფეისზე გაჩერდებოდა და მთლიან სტრიქონს ვერ წაიკითხავდა, ამიტომ getline გამოვიყენე. შემდეგ უკვე სტრიქონის .size() მეთოდი პირდაპირ გვაძლევს რაოდენობას და მეტი არაფერია საჭირო.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- getline-ის გამოყენება სავალდებულოა, თუ გვინდა, რომ სტრიქონში არსებული სფეისებიც გავითვალისწინოთ.
- .size() მეთოდი C++-ის სტრიქონებისთვის ოპტიმიზებულია და პირდაპირ გვიბრუნებს სიმბოლოების რაოდენობას.

## ხშირი შეცდომები

- cin >> s-ის გამოყენება, რაც სტრიქონს მხოლოდ პირველ სფეისამდე კითხულობს.
