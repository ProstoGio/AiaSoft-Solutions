# Problem Statement — 821. ნულების გადასმა

> Copied from [AiaSoft.ge #821](https://aiasoft.ge/problem/821)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია ნატურალური რიცხვი, რომლის ციფრების რაოდენობა არ აღემატება 9-ს. 
დაწერეთ პროგრამა, რომელიც ამ რიცხვის ნულის ტოლ ციფრებს გადასვამს რიცხვის ბოლოში, დანარჩენი ციფრები უნდა მოთავსდნენ რიცხვის დასაწყისში თავდაპირველი რიგითობის შეუცვლელად.

## Input

ერთადერთ ხაზზე მოცემულია ერთი ნატურალური რიცხვი N, რომელშიც ციფრთა რაოდენობა არ აღემატება 9-ს.

## Output

ერთადერთ ხაზზე გამობეჭდეთ მიღებული რიცხვი.
