# Problem Statement — 199. სეგმენტის შემობრუნება

> Copied from [AiaSoft.ge #199](https://aiasoft.ge/problem/199)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

მოცემულია ინგლისური სიმბოლოებით შედგენილი სტრიქონი S და სეგმენტის საზღვრები l და r. 
S-ში შეაბრუნეთ [l, r] სეგმენტი და გამოიტანეთ მიღებული სტრიქონი.

## Input

პირველ ხაზზე შემოდის სტრიქონი S (1 <= |S| <= 100)
მეორე ხაზზე შემოდის ორი მთელი რიცხვი l და r  (1 <= l <= r <= |S|)

## Output

გამოიტანეთ [l, r] სეგმენტის შებრუნების შედეგად მიღებული სტრიქონი
