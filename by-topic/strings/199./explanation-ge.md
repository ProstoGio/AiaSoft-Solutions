# განმარტება — 199. სეგმენტის შემობრუნება

## მიდგომა

ეს არის კლასიკური სტრიქონის მანიპულაციის დავალება, სადაც კონკრეტული სეგმენტი უნდა დავატრიალოთ. რადგან C++-ში სტრიქონის ინდექსაცია ნულიდან იწყება, პირველ რიგში l და r მნიშვნელობებს ვაკლებ ერთს. შემდეგ ვიყენებ ორი პოინტერის მეთოდს: ერთი იწყება l-დან, მეორე r-დან, და სანამ არ შეხვდებიან, უბრალოდ ვცვლი მათ ადგილებს swap-ით. ბოლოს ვბეჭდავ შეცვლილ სტრიქონს.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- ინდექსები აუცილებლად უნდა გავასწოროთ, რადგან დავალებაში 1-დან იწყება, C++-ში კი 0-დან
- two-pointer მეთოდი ოპტიმალურია და დამატებითი მეხსიერება არ გვჭირდება
- swap ფუნქცია ამ საქმისთვის საუკეთესოა

## ხშირი შეცდომები

- ინდექსების დაკლება თუ დაგავიწყდა, არასწორ სეგმენტს შეატრიალებ
- while ციკლის პირობაში თუ შეცდი, შესაძლოა სტრიქონის საზღვრებიდან გახვიდე

## ნაბიჯ-ნაბიჯ ახსნა

შემოგვაქვს სტრიქონი და ინდექსები, ვაკლებთ მათ ერთს, ვატრიალებთ სეგმენტს swap-ით, ვბეჭდავთ შედეგს.
