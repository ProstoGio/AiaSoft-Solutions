# განმარტება — 212. სიმბოლოთა ჯაჭვი

## მიდგომა

C++-ში ასოები რეალურად ASCII კოდებად ინახება, ამიტომ მათი ინკრემენტირება ან დეკრემენტირება ჩვეულებრივი რიცხვებივით შეგვიძლია. თავიდან ცოტა ზედმეტი ვიჩალიჩე მასივის შექმნაზე, მაგრამ საბოლოო ჯამში ლოგიკა მარტივია: ვამოწმებ პირველი ასო მეორეზე ნაკლებია თუ მეტი და იმის მიხედვით ვუშვებ ციკლს წინ ან უკან. მეტი არაფერი სჭირდება, უბრალოდ `for` ციკლში უნდა მივუთითოთ სწორი მიმართულება.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- ASCII ცხრილის წყალობით, char ტიპის ცვლადებზე არითმეტიკული მოქმედებები პირდაპირ მუშაობს.
- თუ input-ში პირველი ასო მეორეზე დიდია, ციკლი `i--` პრინციპით უნდა დავატრიალოთ, რომ არაფერი გამოგვრჩეს.

## ხშირი შეცდომები

- მხოლოდ ერთი მიმართულების გათვალისწინება, როცა პირობა არ აზუსტებს, რომ ყოველთვის ზრდადობით უნდა დავბეჭდოთ.
