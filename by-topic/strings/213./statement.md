# Problem Statement — 213. სიმბოლოს გარდაქმნა

> Copied from [AiaSoft.ge #213](https://aiasoft.ge/problem/213)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია ლათინური ანბანის მთავრული სიმბოლოების მიმდევრობა. საჭიროა თითოეული მათგანის ნაცვლად ჩავწეროთ მისი შესაბამისი პატარა სიმბოლო.

## Input

პირველ სტრიქონზე შემოდის ნატურალური რიცხვი n, სიმბოლოთა რაოდენობა.
მეორე სტრიქონზე შემოდის ლათინური ანბანის მთავრული ასოების მიმდევრობა. სიმბოლოები ერთმანეთისაგან თითო ჰარითაა გამოყოფილი.

## Output

გამოიტანეთ იგივე მიმდევრობა, რომელშიც სიმბოლოები შეცვლილია ამოცანის პირობის შესაბამისად.
