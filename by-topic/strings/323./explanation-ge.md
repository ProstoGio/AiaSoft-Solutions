# განმარტება — 323. სიმბოლოს რიგითი ნომერი

## მიდგომა

ამ ამოცანის ამოხსნის ყველაზე მარტივი გზა არის ანბანის სტრიქონად შენახვა. უბრალოდ ჩავწერე ყველა ასო 'a'-დან 'z'-მდე და 'for' ციკლით ვამოწმებ, რომელ ინდექსზეა ჩვენი შემოსული სიმბოლო. რადგან ციკლი 0-დან იწყება, პასუხად 'i+1'-ს ვბეჭდავთ და მორჩა, ეგაა მთელი ფილოსოფია. რა თქმა უნდა, შეიძლებოდა ASCII კოდებითაც დაგვეთვალა, მაგრამ ეს მეთოდი გაცილებით უფრო გასაგებია.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- ინგლისური ანბანის სიგრძე ფიქსირებულია, ამიტომ ციკლი მაქსიმუმ 26-ჯერ ტრიალებს, რაც რეალურად O(1)-ია.
- მთავარია გვახსოვდეს, რომ C++-ში ინდექსები 0-დან იწყება, ანბანში კი პირველი ასო 1-ლია.

## ხშირი შეცდომები

- ინდექსაციის არევა და პასუხად 1-ის ნაცვლად 0-ის დაბეჭდვა პირველი ასოსთვის.
