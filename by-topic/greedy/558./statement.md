# Problem Statement — 558. ზრდადი მიმდევრობა

> Copied from [AiaSoft.ge #558](https://aiasoft.ge/problem/558)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია n ელემენტიანი S მიმდევრობა. თქვენი დავალებაა შეცვალოთ ეს მიმდევრობა ისე, რომ ყოველი ელემენტი არ იყოს ნაკლები მის წინა ელემენტზე. ყოველ ჯერზე თქვენ შეგიძლიათ მიმდევრობის ნებისმიერი ელემენტი გაზარდოთ 1-ით. დაადგინეთ მინიმუმ რამდენჯერ მოგიწევს ამ მოქმედების გამოყენება.

## Input

პირველ ხაზზე მოცემულია ერთი მთელი რიცხვი: n - მიმდევრობაში ელემენტების რაოდენობა.
მომდევნო ხაზზე მოცემულია n მთელი რიცხვი s1,s2...sn.
1<=n<=2*10⁵
1<=si<=10⁹

## Output

ერთ ხაზზე გამობეჭდეთ მინიმუმ რამდენჯერ დაგვჭირდება ზემოთ მოცემული მოქმედების გამოყენება, რათა მიმდევრობაში ყოველი ელემენტი არ იყოს ნაკლები მის წინა ელემენტზე.
