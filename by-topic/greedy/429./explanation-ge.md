# განმარტება — 429. სამი ფერის ბურთულა

## მიდგომა

ეს ტიპური 'Worst Case' ამოცანაა. იმისთვის, რომ გარანტირებულად ამოვიღოთ წითელი ბურთულა, უნდა წარმოვიდგინოთ, რომ ყველაზე მეტად არ გაგვიმართლა. ეს ნიშნავს, რომ ყუთიდან ჯერ ამოვიღეთ ყველა თეთრი და შავი ბურთულა. ამიტომ, მათ რაოდენობას ვუმატებთ ერთს, რაც ზუსტად ის ბოლო ნაბიჯია, რომელიც წითელ ბურთულას გარანტირებულს ხდის.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- ყველაზე ცუდი სცენარის გათვალისწინება ყველაზე მნიშვნელოვანია.

## ხშირი შეცდომები

- ერთიანის დამატების დავიწყება.
- წითელი ბურთულების რაოდენობის შეცდომით გათვალისწინება.

## ნაბიჯ-ნაბიჯ ახსნა

პროგრამა კითხულობს A, B და C მნიშვნელობებს. ჩვენ გვაინტერესებს B და C, რადგან ისინი არ არის წითელი. ფორმულა B + C + 1 პირდაპირ გვაძლევს პასუხს, რადგან B + C რაოდენობის ბურთულის ამოღების შემდეგ ყუთში მხოლოდ წითლები რჩება, შესაბამისად, მომდევნო ამოღება აუცილებლად წითელი იქნება.
