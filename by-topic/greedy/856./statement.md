# Problem Statement — 856. მოლაშქრე კენგურუ

> Copied from [AiaSoft.ge #856](https://aiasoft.ge/problem/856)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

პატარა კენგურუს, სახელად ბექას, ძალიან უყვარს მთების დალაშქვრა. თანაც, ერთხელ რომ მთას დალაშქრავს, მომავალში მასზე დაბალ მთას არ ირჩევს სამიზნედ. ბექას ძალიან დიდი ნახტომების გაკეთება შეუძლია, მთიდან მთაზე გადახტომაც არ არის მისთვის პრობლემა.
საახალწლოდ ბექას ბევრი დრო აქვს. მან ამოირჩია მთათა სისტემა, სადაც წრფივად N რაოდენობის მთაა განლაგებული. მისი ლაშქრობის სტრატეგია შემდეგია:
	● ახტება პირველი მთის მწვერვალზე
	● ამის შემდეგ ამოირჩევს პირველივე ისეთ მთას, რომელიც მოცემულ მთაზე მკაცრად მაღალია და მის მწვერვალზე გადახტება.
	● ასე აგრძელებს მანამ, სანამ ყველაზე მაღალი მთის მწვერვალზე არ მოხვდება.

დაადგინეთ, რამდენი ნახტომი დასჭირდება ბექას უმაღლესი მთის მწვერვალზე მოსახვედრად?

## Input

პირველ ხაზზე მოცემულია ერთი მთელი რიცხვი: N - მთების რაოდენობა
მომდევნო ხაზზე მოცემულია N რაოდენობის თითო ჰარით გამოყოფილი მთელი რიცხვი: h1, h2, ..., hN - სადაც hi არის i ნომერი მთის სიმაღლე

1 <= N <= 10³
1 <= hi <= 10⁹

## Output

გამობეჭდეთ ერთი რიცხვი: რამდენი ნახტომი დასჭირდება კენგურუ ბექას უმაღლეს მთაზე მოსახვედრად.
