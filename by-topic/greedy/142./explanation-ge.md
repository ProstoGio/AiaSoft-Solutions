# განმარტება — 142. ისევ ბორანი

## მიდგომა

საკმაოდ მარტივი ამოცანაა, მთავარია რიგის პრინციპი დაიცვა. ვინაიდან ტვირთების გადაადგილება არ შეგვიძლია, უბრალოდ ვუმატებთ თითოეულს ჯამურ წონას მანამ, სანამ ბორანის ლიმიტს არ გადავაჭარბებთ. როგორც კი შემდეგი ტვირთი 'ატვირთვას' აღარ ექვემდებარება, ვჩერდებით და გამოგვაქვს უკვე აკრეფილი ტვირთების რაოდენობა.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- აქ მასივები სულაც არაა საჭირო, პირდაპირ `cin`-ითაც აჯამებდი, მაგრამ `a[num]`-ითაც არაფერი დაშავებულა.
- მთავარი 'ტრიკი' ისაა, რომ წონას ვამოწმებთ ყოველი ახალი ტვირთის დამატებისას და მაშინვე ვწყვეტთ ციკლს, როგორც კი ზღვარს გადავცდებით.

## ხშირი შეცდომები

- ბოლო შემთხვევის გათვალისწინება, როცა ყველა ტვირთი ეტევა ბორანზე და ციკლი ბოლომდე გადის.
