# Problem Statement — 106. ლიდერები

> Copied from [AiaSoft.ge #106](https://aiasoft.ge/problem/106)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია მთელ რიცხვთა ერთგანზომილებიანი მასივი. მასივის რომელიმე x ელემენტს ეწოდება ლიდერი, თუ მის მარჯვნივ მდებარე (მეტი ინდექსის მქონე) ყველა რიცხვი ნაკლები ან ტოლია x - ზე. მასივის ბოლო ელემენტი ლიდერად ითვლება. 

დაწერეთ პროგრამა, რომელიც მოცემულ მასივში იპოვის ლიდერ რიცხვებს და გამოიტანს მათ იმ მიმდევრობით, რა მიმდევრობითაც გვხვდებიან ისინი მასივში.

## Input

სტანდარტული შეტანის პირველ სტრიქონში მოცემულია ერთი  მთელი დადებითი n რიცხვი (0 < n <= 10⁶) - მოცემულ მასივში ელემენტთა რაოდენობა. მეორე სტრიქონში ჩაწერილია თითო ჰარით გამოყოფილი მასივის ელემენტები, რომლებიც მთელი რიცხვებია დიაპაზონიდან [−10⁹, 10⁹]

## Output

სტანდარტული გამოტანის ერთადერთ სტრიქონში უნდა ჩაიწეროს თითო ჰარით გამოყოფილი ნაპოვნი ლიდერი რიცხვები იმ მიმდევრობით, რა მიმდევრობითაც ისინი გვხვდება მასივში.
