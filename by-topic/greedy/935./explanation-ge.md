# განმარტება — 935. ეშმაკური იდეა

## მიდგომა

თავდაპირველად ვითვლით 1-დან N-მდე რიცხვების ჯამს, რაც არის n*(n+1)/2. თუ ეს ჯამი ზუსტად S-ის ტოლია, მაშინ არაფრის შეცვლა არ გვჭირდება. თუ S მეტია ჯამზე, საკმარისია ერთი ფირფიტის შეცვლა, რადგან ნებისმიერი რიცხვის ჩანაცვლება შეიძლება. ყველაზე საინტერესო შემთხვევაა, როცა S ნაკლებია ჯამზე: აქ ვეძებთ მინიმალურ k-ს, რომლის ფარგლებშიც შესაძლებელი იქნება ჯამის შემცირება საჭირო ოდენობით.

## სირთულე

| | |
|-|--|
| დრო | O(log n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- ჯამის მაქსიმალურად შესამცირებლად უნდა შევცვალოთ ყველაზე დიდი რიცხვები.
- k ცალი უდიდესი რიცხვის შეცვლა იძლევა ყველაზე დიდ ეფექტს.
- Binary search-ით ვპოულობთ მინიმალურ k-ს, რომელიც აკმაყოფილებს პირობას.

## ხშირი შეცდომები

- მსხვილი რიცხვების ჯამის ფორმულაში შეცდომა.
- სხვადასხვა შემთხვევების (S=Sn, S>Sn) არასწორი დამუშავება.

## ნაბიჯ-ნაბიჯ ახსნა

კოდი ითვლის Sn-ს და ამოწმებს შედარებებს. თუ საჭიროა შემცირება, ვწერთ Binary search-ს 1-დან N-მდე დიაპაზონში. k ცვლადი გვიჩვენებს ფირფიტების რაოდენობას, ხოლო V-k არის ის მაქსიმალური სხვაობა, რომელსაც მივიღებთ ამ ფირფიტების 1-ზე ჩანაცვლებით. თუ ეს სხვაობა >= D-ს, ესე იგი k საკმარისია და ვცდილობთ უფრო პატარა კ-ს პოვნას.
