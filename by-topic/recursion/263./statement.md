# Problem Statement — 263. წვეროების რაოდენობა კომპონენტში

> Copied from [AiaSoft.ge #263](https://aiasoft.ge/problem/263)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

არაორიენტირებული გრაფი მოცემულია მეზობელთა წყვილების სახით. დაადგინეთ, სულ რამდენ წვეროს აერთიანებს კომპონენტი, რომლიც შეიცავს V წვეროს.

## Input

პირველ ხაზზე შემოდის N, M და V, შესაბამისად გრაფში წვეროების რაოდენობა, წიბოების რაოდენობა და ერთ-ერთი წვეროს ნომერი.
მომდევნო M ხაზზე მოცემულია ორ-ორი რიცხვი. კერძოდ i-ურ ხაზზე შემოვა vi და ui - მეზობელი წვეროების ნომრები. (გრაფის წვეროები დანომრილია 1-დან N-მდე)

1 <= N <= 20
0 <= M <= N*(N-1) / 2
1 <= V, vi, ui <= N

## Output

გამოიტანეთ ერთი მთელი რიცხვი, წვეროების რაოდენობა კომპონენტში, რომელიც შეიცავს V წვეროს.
