# განმარტება — 538. ჭადრაკის დაფა და დედოფლები

## მიდგომა

ეს ტიპური რეკურსიული backtracking ამოცანაა, სადაც თითოეულ რიგში ვცდილობ დედოფლის დასმას. რადგან დაფაზე დახურული უჯრებია, სანამ დედოფალს დავსვამ, ჯერ ვამოწმებ ცარიელია თუ არა და მერე check ფუნქციით ვარკვევ, ემუქრება თუ არა ვინმე. თუ ყველაფერი რიგზეა, ვსვამ '1'-ს, გადავდივარ შემდეგ რიგზე და რეკურსიის დასრულების მერე ისევ ვშლი, რომ სხვა ვარიანტებიც შევამოწმო.

## სირთულე

| | |
|-|--|
| დრო | O(8!) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- დიაგონალების შემოწმება ყველაზე მნიშვნელოვანია; ოთხივე მიმართულებით ვამოწმებ '1'-ს, რომ დედოფლები ერთმანეთს არ გადაეყარონ.
- რადგან ვიცით, რომ 8 დედოფალია საჭირო, რეკურსია პირდაპირ ითვლის 0-დან 8-მდე და ბაზისური ქეისიც ეგაა.

## ხშირი შეცდომები

- დაგავიწყდებათ '1'-ის '.-ზე გადაცვლა რეკურსიის მერე (backtracking-ის ძირითადი შეცდომა).

## ნაბიჯ-ნაბიჯ ახსნა

ავიღოთ პირველი რიგი: ვცდი ყოველ სვეტს. თუ დაფა თავისუფალია და check-იც 'true'-ს აბრუნებს, ვსვამ დედოფალს და მივდივარ მეორე რიგზე. თუ რომელიმე რიგში ვერსად ვსვამ, ფუნქცია ბრუნდება უკან და ცდილობს წინა ნაბიჯის შეცვლას.
