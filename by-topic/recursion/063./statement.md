# Problem Statement — 063. ციფრების ჯამი

> Copied from [AiaSoft.ge #63](https://aiasoft.ge/problem/63)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია მრავალნიშნა რიცხვი, რომლის ციფრების რაოდენობა არ აღემატება 500 სიმბოლოს.
დაწერეთ რეკურსიული ქვეპროგრამა, რომელიც გამოთვლის მოცემული რიცხვის ციფრების ჯამს

## Input

შემოდის ერთი არაუარყოფითი მთელი  რიცხვი.

## Output

გამოიტანეთ მოცემული რიცხვის ციფრების ჯამი.
