# განმარტება — 511. 2 - ის ხარისხი

## მიდგომა

მარტივი ლოგიკაა: თუ რიცხვი 2-ის ხარისხია, მას მანამ გავყოფთ ორზე, სანამ 1-ს არ მივიღებთ. კოდში ამისთვის `>>1` ბიტურ ოპერაციას ვიყენებ, რაც ძალიან სწრაფია. თუ დაყოფის პროცესში რიცხვი კენტი გახდა და ჯერ კიდევ 1-ზე მეტია, მაშინ ეგრევე ვხვდები, რომ 2-ის ხარისხი არ არის.

## სირთულე

| | |
|-|--|
| დრო | O(log n) |
| მეხსიერება | O(log n) |

## მთავარი დაკვირვებები

- 2-ის ხარისხი ბინარულში მხოლოდ ერთიანით იწყება, ამიტომ მარჯვნივ შიფტი ყოველ ჯერზე კლავს ერთ ბიტს.
- თუ ბოლოს 1-ზე გავჩერდი, ესე იგი თავიდანვე სწორ გზაზე ვიყავი.

## ხშირი შეცდომები

- 1-ის შემთხვევის გამოტოვება, როცა 2^0 = 1 და ბევრი ამას უბრალოდ 'კენტად' თვლის და 'NO'-ს ბეჭდავს.
