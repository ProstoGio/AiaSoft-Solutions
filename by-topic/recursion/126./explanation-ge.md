# განმარტება — 126. ფიბონაჩის მიმდევრობა 1

## მიდგომა

ფიბონაჩის ამ ამოცანისთვის სუფთა რეკურსია ძალიან ნელი იქნება, რადგან ერთი და იგივე მნიშვნელობებს ათასჯერ გადაითვლის. მარტივი გამოსავალია memoization — შევქმენი გლობალური მასივი, სადაც ვინახავ უკვე ნაპოვნ პასუხებს. როგორც კი ფუნქცია ახალ მნიშვნელობას დაითვლის, მაშინვე ვწერ მასივში, რომ შემდეგ ჯერზე პირდაპირ დავაბრუნო.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- აუცილებლად long long უნდა გამოვიყენოთ, რადგან 90-ე ფიბონაჩის რიცხვი სტანდარტულ int-ში ვერ ჩაეტევა.
- რეკურსიის დროს cache-ის გამოყენება ოპტიმალურია, რადგან დროის კომპლექსურობას პირდაპირ O(n)-ზე ჩამოვდივართ.

## ხშირი შეცდომები

- int-ის გამოყენება, რაც დიდ n-ებზე overflow-ს გამოიწვევს.
- მეხსიერების დაზოგვის მცდელობა memoization-ის გარეშე, რაც დროს გაწელავს.
