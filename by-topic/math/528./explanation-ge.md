# განმარტება — 528. ფერადი ბურთები

## მიდგომა

ეს პრობლემა იმაზე მარტივია, ვიდრე ერთი შეხედვით ჩანს. თუ ბურთების რაოდენობა ლუწია, შუაზე გაყოფ და ნაშთი 0 იქნება. კენტი რიცხვის შემთხვევაში, ვერანაირად ვერ გაყოფ ზუსტად შუაზე, ამიტომ მინიმალური სხვაობა ყოველთვის 1 დარჩება. რეალურად, მარტო n-ის ლუწ-კენტობაზეა დამოკიდებული ყველაფერი.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- ლუწი რაოდენობის შემთხვევაში ორ ყუთს შორის სხვაობა ყოველთვის 0-ია.
- კენტი რაოდენობის დროს 1-იანი სხვაობა გარდაუვალია, რაც ამ შემთხვევაში ოპტიმალურია.

## ხშირი შეცდომები

- ზედმეტი გართულება და მასივების ან ციკლების გამოყენება იქ, სადაც მარტივი if-else საკმარისია.
