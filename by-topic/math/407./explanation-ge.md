# განმარტება — 407. თოვლის ბაბუა და საჩუქრები

## მიდგომა

ამ ამოცანის არსი იმაშია, რომ დავადგინოთ, იყოფა თუ არა საჩუქრების რაოდენობა ბავშვების რაოდენობაზე უნაშთოდ. თუ N იყოფა M-ზე, ეს ნიშნავს, რომ ყველას თანაბრად შეხვდება საჩუქარი და არაფერი დარჩება. მარტივი modulo ოპერატორით (N % M) ზუსტად ამას ვამოწმებთ. არანაირი ციკლი არ სჭირდება, პირდაპირ ამოწმებს პირობას.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- N % M == 0 პირობა ზუსტად იმას ნიშნავს, რომ საჩუქრები თანაბრად გადანაწილდება.
- ამოცანა მათემატიკურად მარტივია და არ მოითხოვს დამატებით მეხსიერებას ან რთულ ალგორითმებს.

## ხშირი შეცდომები

- ციკლების გამოყენება საჩუქრების გამოსაკლებად, რაც ტყუილი წვალებაა.
- გამომავალი ტექსტის არასწორი ფორმატით დაბეჭდვა.

## ნაბიჯ-ნაბიჯ ახსნა

შემოგვაქვს N და M, შემდეგ ვამოწმებთ if(n % m == 0). თუ პირობა სრულდება, ვბეჭდავთ 'YES', სხვა შემთხვევაში - 'NO'.
