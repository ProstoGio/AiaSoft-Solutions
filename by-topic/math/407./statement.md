# Problem Statement — 407. თოვლის ბაბუა და საჩუქრები

> Copied from [AiaSoft.ge #407](https://aiasoft.ge/problem/407)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

თოვლის ბაბუა საჩუქრების დარიგებას იწყებს, მას აქვს N რაოდენობის საჩუქარი და უნდა დაურიგოს M რაოდენობის ბავშვს, თოვლის ბაბუას უნდა რომ ყველა ბავშვმა ერთნაირი რაოდენობის საჩუქარი მიიღოს და ამასთანავე ყველა საჩუქარი დაარიგოს. დაეხმარეთ თოვლის ბაბუას იმის გარკვევაში, შეძლებს თუ არა ის ყველა ბავშვისთვის ერთნაირი რაოდენობის საჩუქრების დარიგებას.

## Input

მოცემულია ორი დადებითი რიცხვი: N-საჩუქრების რაოდენობა, M-ბავშვების რაოდენობა.
2<=N,M<=10³

## Output

გამობეჭდეთ YES თუკი თოვლის ბაბუა შეძლებს ყველა ბავშვისთვის ერთნაირი რაოდენობის საჩუქრის დარიგებას ისე რომ, თოვლის ბაბუას არ დარჩეს საჩუქარი, წინააღმდეგ შემთხვევაში გამობეჭდეთ NO.
