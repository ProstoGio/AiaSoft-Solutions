# განმარტება — 701. მასივში ციფრთა ჯამი

## მიდგომა

ამოცანა საკმაოდ მარტივია: მასივის თითოეული რიცხვისთვის უნდა ვიპოვოთ ციფრთა ჯამი და დავბეჭდოთ. ამისთვის დავწერე დამხმარე ფუნქცია calc, რომელიც modulo 10-ის გამოყენებით ბოლო ციფრს იღებს და შემდეგ 10-ზე გაყოფით რიცხვს ამცირებს მანამ, სანამ ნული არ დარჩება. ბოლოს, უბრალოდ ვუვლით მასივს, ვატარებთ თითოეულ ელემენტს ფუნქციაში და შედეგებს ვბეჭდავთ.

## სირთულე

| | |
|-|--|
| დრო | O(N * log10(max_element)) |
| მეხსიერება | O(N) |

## მთავარი დაკვირვებები

- x % 10 ოპერაცია გვაძლევს ბოლო ციფრს, ხოლო x /= 10 აშორებს მას.
- რადგან რიცხვები 10^5-ს არ აღემატება, while ციკლი თითოეულ რიცხვზე მაქსიმუმ 5-6-ჯერ შესრულდება.

## ხშირი შეცდომები

- რიცხვის სტრიქონად ქცევა ციფრთა ჯამის საპოვნელად - მუშაობს, მაგრამ მათემატიკურად ბევრად სწრაფია.
- შედეგების დაბეჭდვისას დამატებითი სიმბოლოების არევით ფორმატის გაფუჭება.

## ნაბიჯ-ნაბიჯ ახსნა

ჯერ ვკითხულობთ N-ს. შემდეგ ვქმნით მასივს, სადაც შევინახავთ მონაცემებს. ციკლში თითოეულ ელემენტს ვუგზავნით calc ფუნქციას, რომელიც აბრუნებს ციფრთა ჯამს. საბოლოოდ, მიღებულ პასუხებს ვბეჭდავთ ჰარით გამოყოფილს.
