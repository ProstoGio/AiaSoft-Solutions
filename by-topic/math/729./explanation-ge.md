# განმარტება — 729. შეფ ვასოს კერძები

## მიდგომა

ეს ამოცანა სინამდვილეში ელემენტარული გეომეტრიაა. როცა პიცას ცენტრში ჭრი დიამეტრალურად, ყოველი ერთი გაჭრა ორად ყოფს არსებულ ნაჭრებს. ანუ, ყოველი გაჭრა ამატებს ორ ახალ ნაჭერს, შესაბამისად, ჯამური რაოდენობა არის 2 გამრავლებული n-ზე.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- ფორმულა არის 2 * n, რადგან ყოველი გაჭრა ორ ნაწილად ანაწილებს პიცას.
- პირობის თანახმად, მთლიანი პიცა ნაჭრად არ ითვლება, ამიტომ n=0-ისთვის პასუხი 0 უნდა იყოს.

## ხშირი შეცდომები

- შეცდომით 'Lazy Caterer's sequence'-ის (n*(n+1)/2 + 1) გამოყენება, რომელიც მაშინ გამოიყენება, როცა ჭრილები ერთმანეთს არ კვეთენ ცენტრში.
- n=0 შემთხვევის არასწორად დამუშავება.

## ნაბიჯ-ნაბიჯ ახსნა

კოდი უბრალოდ კითხულობს n-ს და ბეჭდავს 2*n-ს. თუ 0-ს შევიყვანთ, პროგრამა გამობეჭდავს 0-ს, რაც ზუსტად ისაა, რაც პირობაში გვიწერია - მთლიანი პიცა არ ითვლებაო.
