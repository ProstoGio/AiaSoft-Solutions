# Problem Statement — 729. შეფ ვასოს კერძები

> Copied from [AiaSoft.ge #729](https://aiasoft.ge/problem/729)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

შეფი ვასო სახელგანთქმულია თავისი უგემრიელესი კერძებით. ვასოს სამზარეულოს სტუმრებს განსაკუთრებით პიცა "მარგარიტა" მოსწონთ. დღეს ვასიკომ გადაწყვიტა ყველა სტუმრისთვის საჩუქარი გაეკეთებინა და უუუზარმაზარი მარგარიტა გამოაცხო, შემდეგ მომზადებული პიცა n-ჯერ გაჭრა დიამეტრალურად.
თუ ერთი ნაჭერი ერთ სტუმარს ერგება, რამდენი სტუმრის გამასპინძლებას შეძლებს შეფ ვასო თავისი მომზადებული პიცით?

* გაითვალისწინეთ, თუ მთლიანი პიცა ნაჭრად არ ითვლება.

## Input

ერთ ხაზზე მოცემულია ერთი არაუარყოფითი მთელი რიცხვი: n (0 <= n <= 57) - გაჭრების რაოდენობა

## Output

გამობეჭდეთ ერთი მთელი რიცხვი: რამდენ სტუმარს გაუმასპინძლდება ვასო პიცით?
