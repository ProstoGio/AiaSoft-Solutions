# Problem Statement — 208. სამკუთხა რიცხვები

> Copied from [AiaSoft.ge #208](https://aiasoft.ge/problem/208)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

წარმოიდგინეთ ერთმანეთზე შემდეგი სახით დალაგებული წრეები:

`~https://storage.googleapis.com/problems_api_photos/images/P9c7HS96Um3ijmXq7EMt.png`

თითოეული სამკუთხედის შესაქმნელად საჭირო წრეების რაოდენობებია, შესაბამისად: 1, 3, 6, 10, 15. მომდევნო სამკუთხედის შექმნას, ცხადია, დასჭირდება 21 წრე.

წრეების რაოდენობას, რომელიც საჭიროა სამკუთხედის შესაქმნელად, სამკუთხა რიცხვი ეწოდება. 1, 3, 6, 10, 15, 21, ... სამკუთხა რიცხვების მიმდევრობაა.

მოცემულია ნატურალური რიცხვი n, სამკუთხა რიცხვის ნომერი სამკუთხა რიცხვების მიმდევრობაში. იპოვეთ შესაბამისი სამკუთხა რიცხვი.

## Input

შემოდის ერთი ნატურალური რიცხვი n, 1<=n<5*10⁹
გამოთვალეთ შესაბამისი სამკუთხა რიცხვის მნიშვნელობა.

## Output

გამოიტანეთ შესაბამისი სამკუთხა რიცხვის მნიშვნელობა.
