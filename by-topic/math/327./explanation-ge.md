# განმარტება — 327. კენტი რიცხვები

## მიდგომა

აქ რთული არაფერია, პირდაპირ ვიწყებთ ათვლას 1-დან. იმის ნაცვლად, რომ ყველა რიცხვი შევამოწმოთ 'if'-ით, უბრალოდ ყოველ ჯერზე 2-ს ვუმატებთ. ეს მიდგომა ყველაზე ოპტიმალურია, რადგან ზედმეტ ოპერაციებს ვერიდებით და კოდიც უფრო მარტივი გამოდის.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- increment-ზე 2-ის დამატება პირდაპირ კენტი რიცხვების მიმდევრობას გვაძლევს.
- while ციკლის პირობა i <= n ზუსტად აკონტროლებს, რომ N თუ კენტია, ისიც დაიბეჭდოს.

## ხშირი შეცდომები

- ციკლის საწყის პირობაში არევა, რამაც შეიძლება ლუწი რიცხვებიც გამოგვატანინოს.
