# Problem Statement — 497. ოთხი ფერის ბურთულა

> Copied from [AiaSoft.ge #497](https://aiasoft.ge/problem/497)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

სანდროს ძალიან უყვარს ლოგიკური ამოცანების კეთება, ამჯერად მან ასეთი ამოცანის გაკეთება გადაწყვიტა: ყუთში დევს A - რაოდენობის თეთრი, B - რაოდენობის შავი, C - რაოდენობის ლურჯი და D - რაოდენობის მწვანე ბურთულა, მინიმუმ რამდენი ბურთულა უნდა ამოვიღოთ ყუთში ჩაუხედავად რომ ამოღებულ ბურთულებს შორის აუცილებლად იყოს ყველა ფერის 1 ბურთულა მაინც. სანდროსთვის მოცემული ამოცანა რთული აღმოჩნდა, დაეხმარეთ სანდროს ამოცანის ამოხსნაში.

## Input

ერთ ხაზზე მოცემულია ოთხი მთელი რიცხვი - A, B, C, D.
1<=A,B,C,D<=200

## Output

ერთ ხაზზე გამობეჭდეთ, მინიმუმ რამდენი ბურთულა უნდა ამოვიღოთ ყუთიდან(ყუთში ჩაუხედავათ), რომ ამოღებულ ბურთულებს შორის აუცილებლად იყოს ყველა ფერის ბურთულა.
