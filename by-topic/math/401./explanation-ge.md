# განმარტება — 401. საუკუნე

## მიდგომა

ამ ამოცანის ამოსახსნელად ყველაზე მარტივი გზაა if-else პირობების გამოყენება. უბრალოდ ვამოწმებ, ეკუთვნის თუ არა შემოსული რიცხვი 1900-1999 ან 2000-2099 შუალედებს. თუ არცერთში არ ჯდება, გამოაქვს -1. მეტი არაფერი სჭირდება.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- საუკუნეები მოცემულია ჩათვლით, ამიტომ >= და <= ოპერატორებია საჭირო.
- ინტერვალები არ გადაიკვეთება, ამიტომ თანმიმდევრობას დიდი მნიშვნელობა არ აქვს.

## ხშირი შეცდომები

- საზღვრების არასწორად დადგენა, მაგალითად < 2000-ის გამოყენება.
- გამოტანის ფორმატში შეცდომის დაშვება, რადგან b21 და b20-ის ნაცვლად რიცხვების დაწერა.

## ნაბიჯ-ნაბიჯ ახსნა

შემოგვაქვს N, ვამოწმებთ არის თუ არა [2000, 2099]-ში - გამოგვაქვს b21. თუ არა, ვამოწმებთ [1900, 1999]-ს - გამოგვაქვს b20. სხვა შემთხვევაში ვბეჭდავთ b-1.
