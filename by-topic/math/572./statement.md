# Problem Statement — 572. სათამაშოები

> Copied from [AiaSoft.ge #572](https://aiasoft.ge/problem/572)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

თაზოს შეუზღუდავი რაოდენობის სათამაშო აქვს. თითოეული სათამაშო მართკუთხედის ფორმისაა ზომით 1 x 2. დაადგინეთ, მაქსიმუმ რამდენი სათამაშოს დაწყობას შეძლებს თაზო 2 x N ზომის დაფაზე ისე, რომ სათამაშოები ერთმანეთს თავზე არ გადაადოს. სათამაშოების შეხება შესაძლებელია თუმცა გადაფარვა არა.

## Input

მოცემულია ერთი ნატურალური რიცხვი: N, (1 <= N <= 10¹⁸)

## Output

გამობეჭდეთ სათამაშოების მაქსიმალური რაოდენობა, რომელთა დაწყობასაც თაზო შეძლებდა დაფაზე.
