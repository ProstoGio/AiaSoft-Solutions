# განმარტება — 754. ფაქტორიალი

## მიდგომა

რადგან N მაქსიმუმ 15-ია, რეკურსია ყველაზე ელეგანტური გზაა ამის დასაწერად. ფაქტორიალის მათემატიკური განმარტება პირდაპირ გვეუბნება, რომ N! = N * (N-1)!, ამიტომ კოდიც ზუსტად ასე ჩავწერე. ბაზისურ შემთხვევად 0 ავირჩიე, რადგან 0! ერთის ტოლია და ლოგიკას იდეალურად ამთავრებს.

## მთავარი დაკვირვებები

- მიუხედავად იმისა, რომ ფაქტორიალი სწრაფად იზრდება, 15-ისთვის long long სავსებით საკმარისია და არაფერი აგვერევა.
- რეკურსიული ფუნქციის ბაზისური შემთხვევა (x=0) აუცილებელია, თორემ სთექი აივსება.

- ხშირად ავიწყდებათ 0-ის შემთხვევა და რეკურსია უსასრულოდ მიდის.
