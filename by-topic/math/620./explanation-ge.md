# განმარტება — 620. მშვიდი მიმდევრობა

## მიდგომა

ეს ტიპური ფიბონაჩის მიმდევრობაა, ოღონდ ცოტა გადაკეთებული. დაუკვირდი: თუ ბოლო ციფრი 0-ია, მანამდე ნებისმიერი მშვიდი მიმდევრობა შეიძლება იყოს (n-1-ისთვის). თუ ბოლო ციფრი 1-ია, მაშინ მის წინ აუცილებლად 0 უნდა იდგეს, ანუ ეს ნაწილი n-2-ს უბრუნდება. ასე ვიღებთ მარტივ რეკურსიულ ფორმულას.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- ჯამში ვიღებთ d[i] = d[i-1] + d[i-2], რაც ზუსტად ფიბონაჩის ლოგიკაა.
- მთავარია base case-ები სწორად დავსვათ: n=1-ისთვის 2 ვარიანტი გვაქვს, ხოლო n=2-ისთვის 3.

## ხშირი შეცდომები

- ინდექსების აღრევა DP-ში; ყურადღება მიაქციე, რომ d[1]=2-დან იწყებ და არა d[0]=1-დან.

## ნაბიჯ-ნაბიჯ ახსნა

მაგალითად n=3: 000, 001, 010, 100, 101. სულ 5. წინა ორის ჯამი (2+3=5) ზუსტად ემთხვევა.
