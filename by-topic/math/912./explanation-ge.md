# განმარტება — 912. საახალწლო ნათურები

## მიდგომა

ნათურების თანმიმდევრობა ციკლურია და მეორდება ოთხნიშნა ბლოკებად: 2, 0, 2, 5. იმის მაგივრად რომ დიდი მასივი ავაწყოთ, უბრალოდ K-ს გავყოფთ 4-ზე და ნაშთს გამოვიყენებთ. თუ ნაშთი 0-ია, ესე იგი ნათურა ბოლო პოზიციაზეა, ანუ 5-იანზე, დანარჩენ შემთხვევაში კი ნაშთი პირდაპირ გვეუბნება ინდექსს ჩვენს სტრიქონში.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- პერიოდულობა 4-ია, ამიტომ % ოპერატორი ზუსტად იმას აკეთებს, რაც გვინდა.
- k%4-ის დროს ფრთხილად უნდა ვიყოთ 0-თან, რადგან 1-ლი ინდექსი '2'-ია, მეოთხე კი '5'.

## ხშირი შეცდომები

- 1-based ინდექსაციის არასწორი დამუშავება სტრიქონის 0-based ინდექსებთან მიმართებაში.
