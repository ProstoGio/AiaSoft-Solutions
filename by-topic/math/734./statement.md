# Problem Statement — 734. შებრუნებული რიცხვი

> Copied from [AiaSoft.ge #734](https://aiasoft.ge/problem/734)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

პატარა კატოს რიცხვებით თამაში ძალიან უყვარს, მან ასეთი თამაში მოიგონა, მისი მეგობარი ვატო ასახელებს ორნიშნა რიცხვს, კატომ კი ამ რიცხვის შებრუნებული (ციფრების უკუთანმიმდევობით ჩაწერილი) რიცხვი უნდა გამოიცნოს. დაეხმარეთ კატოს ამოცანის ამოხსნაში.

## Input

ერთ სტრიქონზე შემოდის ორნიშნა დადებითი რიცხვი.

## Output

პირველ სტრქიონზე გამოიტანეთ მოცემული რიცხვის შებრუნებული რიცხვი
