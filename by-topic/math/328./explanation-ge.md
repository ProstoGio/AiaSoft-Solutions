# განმარტება — 328. ლუწი რიცხვები

## მიდგომა

აქ არაფერია რთული, უბრალოდ a-დან b-მდე უნდა გავიაროთ ყველა რიცხვი. ციკლს ვატრიალებ და თითოეულ რიცხვს ვამოწმებ, 2-ზე იყოფა თუ არა. პირობა i % 2 == 0 ზუსტად იმას აკეთებს რაც გვჭირდება, დანარჩენი უკვე ჩვეულებრივი C++-ის სინტაქსია.
