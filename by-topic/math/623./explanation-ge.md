# განმარტება — 623. სტრიქონები წიგნში

## მიდგომა

აქ მთავარია მივხვდეთ, როგორ გადავანაწილოთ N სტრიქონი K-ზომიან გვერდებზე. თუ N-ს გავყოფთ K-ზე, მივიღებთ გვერდების რაოდენობას, მაგრამ თუ ნაშთი გვაქვს, ესე იგი უკვე შემდეგ გვერდზე გადავედით. ყველაზე მეტად რაზეც უნდა იწვალოთ, არის შემთხვევა, როცა N ზუსტად იყოფა K-ზე — მაგ დროს ნაშთი 0 გამოდის, ჩვენ კი გვჭირდება რომ ხაზი იყოს K, და არა 0.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- ნაშთიანი გაყოფისას, თუ ნაშთი 0-ია, ეს ნიშნავს, რომ სტრიქონი გვერდის ბოლო ხაზია.
- გვერდის ნომერი მარტივად ითვლება: თუ ნაშთი გვაქვს, მაშინ N/K-ს ვუმატებთ ერთს.

## ხშირი შეცდომები

- N % K == 0 შემთხვევის უგულებელყოფა, რაც 0-იან ხაზს დაგიბეჭდავთ.
