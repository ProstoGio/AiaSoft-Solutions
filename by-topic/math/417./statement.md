# Problem Statement — 417. ორი ბაჭია

> Copied from [AiaSoft.ge #417](https://aiasoft.ge/problem/417)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ორი ბაჭია, ბაბი და ბუბი, ზის უჯრედებიანი რვეულის ერთსტრიქონიან ზოლზე, კიდურა უჯრებში. ბაბი ზის მარცხენა, ხოლო ბუბი მარჯვენა კიდეში.
ბაჭიებს უფლება აქვთ გააკეთონ ნახტომი, რომლის სიგრძე არ აღემატება k უჯრას, `bრომელი მიმართულებითაც უნდათ - წინ ან უკან`.
აკრძალული აქვთ გადაახტნენ იმ უჯრას, რომელზეც მეგობარი ბაჭია ზის. აგრეთვე აკრძალულია რვეულის ზოლის დატოვება.
თამაშს იწყებს ბაბი, ბაჭიები რიგრიგობით აკეთებენ სვლებს. წაგებული ის ბაჭია, რომელსაც სვლის გაკეთების საშუალება აღარ აქვს. 
ცნობილია, რომ საკუთარი სვლის დროს ბაჭია აუცილებლად აკეთებს ნახტომს.
დაწერეთ პროგრამა, რომელიც დაადგენს, თუ რომელი ბაჭია გაიმარჯვებს.

## Input

შემოდის ერთმანეთისაგან ჰარით დაშორებული ორი ნატურალური რიცხვი n და k.  შესაბამისად, უჯრედებიანი ზოლის უჯრების რაოდენობა და ბაჭიას ნახტომის მაქსიმალური სიგრძე. 2<=n,k<=1000.

## Output

გამოიტანეთ ერთი სიტყვა Babi ან Bubi - გამარჯვებული ბაჭიას სახელი.
