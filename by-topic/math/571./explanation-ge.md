# განმარტება — 571. ხაჭაპური

## მიდგომა

ეს ამოცანა რეალურად მარტივი მათემატიკაა, ზედმეტი სირთულეების გარეშე. ერთი ცენტრიდან გამავალი ხაზი ყოველთვის ორად ყოფს არსებულ ნაჭრებს, შესაბამისად, ყოველი დამატებითი ჭრა ნაჭრების რაოდენობას ორმაგად ზრდის. ერთადერთი მომენტია, როცა ნოლი გაქვს - მაშინ ხაჭაპური მთლიანია და პასუხიც 1 უნდა იყოს.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- 0 გაჭრის შემთხვევაში 1 ნაჭერი გვაქვს, ამაზე ბევრს ავიწყდება ხოლმე ყურადღების გამახვილება.
- თითოეული დიამეტრალური ჭრა ორჯერ ზრდის სულ ნაჭრების რაოდენობას.

## ხშირი შეცდომები

- N=0 შემთხვევის გამოტოვება და ავტომატურად 2*N-ის დაბეჭდვა, რაც ნულს მოგცემდათ.
