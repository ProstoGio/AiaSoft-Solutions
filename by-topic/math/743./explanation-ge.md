# განმარტება — 743. ციფრთა ჯამი - 1

## მიდგომა

აქ არაფერია საეჭვო, პირდაპირი მიდგომა საუკეთესოა. 10⁵ შეზღუდვა იმდენად პატარაა, რომ უბრალო ციკლით 'a'-დან 'b'-მდე სიარული მყისიერად მუშაობს. calc ფუნქციაში modulo-ს და გაყოფის კომბინაციით ვშლი რიცხვს ციფრებად და ვკრებ მათ. არაფერი რთული, უბრალოდ ციფრების ამოკრეფაა.

## სირთულე

| | |
|-|--|
| დრო | O(N log M) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- log M იმიტომ გვექნება, რომ თითოეულ რიცხვზე ციფრების რაოდენობის პროპორციული ოპერაცია სრულდება.
- 10⁵-ზე ოპტიმიზაციაზე ფიქრიც არ ღირს, პირდაპირი რეალიზაცია წამზე ბევრად სწრაფად იტრიალებს.

## ხშირი შეცდომები

- შეიძლება ვინმემ დაგეგმოს რთული მათემატიკური ფორმულა, როცა ჩვეულებრივი loop საკმარისია.
