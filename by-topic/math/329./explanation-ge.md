# განმარტება — 329. კენტი რიცხვები 2

## მიდგომა

ამოცანის არსი იმაშია, რომ N-დან 1-მდე კლებადობით გამოვიტანოთ მხოლოდ კენტი რიცხვები. ყველაზე პირდაპირი გზაა მარტივი `while` ციკლი: დავიწყოთ N-ით და ყოველ ჯერზე თითო ერთეულით დავიკლოთ. შიგნით უბრალოდ ვამოწმებთ, ნაშთი 2-ზე გაყოფისას ნულის ტოლია თუ არა; თუ არაა, ესე იგი რიცხვი კენტია და ეკრანზე გამოგვაქვს.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- modulo-ს გამოყენება `i % 2 != 0` ყველაზე საიმედოა კენტობის შესამოწმებლად.
- ციკლის პირობაში `i > 0`-ის გამოყენება გვარანტირებს, რომ 1-საც ჩავთვლით.

## ხშირი შეცდომები

- ციკლის პირობის არასწორად დაწერა, რის გამოც 1-იანი ხშირად იკარგება.
