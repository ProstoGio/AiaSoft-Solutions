# განმარტება — 409. გუგლიკო და შერჩევა

## მიდგომა

თუ პატარა N-ებზე გატესტავ, მიხვდები, რომ გადარჩენილი ყოველთვის არის N-ზე ნაკლები ან ტოლი უდიდესი 2-ის ხარისხი. რეალურად, ეს პრობლემა იოზეფუსის ამოცანის ერთ-ერთი სახეობაა, სადაც კენტების ყოველ ჯერზე მოცილებით მხოლოდ 2-ის ხარისხები რჩებიან. ნაცვლად იმისა, რომ მთელი პროცესი დავასიმულიროთ, უბრალოდ ვპოულობთ ყველაზე დიდ 2-ის ხარისხს, რომელიც N-ს არ აღემატება.

## სირთულე

| | |
|-|--|
| დრო | O(log n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- კენტი ნომრების მოშორება ნიშნავს, რომ ყოველი იტერაციის შემდეგ კანდიდატების რაოდენობა ნახევრდება და რჩება მხოლოდ 2-ის ხარისხი.
- 10^18 საკმაოდ დიდი რიცხვია, ამიტომ O(N) სიმულაცია ჩავარდება; O(log N) არის ერთადერთი გზა.

## ხშირი შეცდომები

- ვექტორის ან მასივის გამოყენება სიმულაციისთვის, რაც მემორიას და დროსაც წაიღებს.
