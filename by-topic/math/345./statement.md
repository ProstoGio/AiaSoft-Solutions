# Problem Statement — 345. ბაბის მულტფილმი 2

> Copied from [AiaSoft.ge #345](https://aiasoft.ge/problem/345)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ბაბის დაბადების დღეზე ტელეარხმა "ბაჭიების ოცნება" ბაბისთვის საჩუქრად მულტფილმის გაშვება გადაწყვიტა. ცნობილია, რომ მულტფილმი იწყება h1 საათსა და m1 წუთზე და გრძელდება h2 საათისა და m2 წუთის განმავლობაში.
ბაბის აინტერესებს, რომელი საათი იქნება მულტფილმის დამთავრების მომენტში.

## Input

კლავიატურიდან შემოდის ოთხი ნატურალური რიცხვი h1, m1, h2, m2 - მულტფილმის დაწყების დრო და მისი ხანგრძლივობა.
0<=h1, h2<=23,  0<=m1, m2<=59.

## Output

გამოიტანეთ ორი რიცხვი h, m, 0<=h<=23,  0<=m<=59 - ბაბის საათის ჩვენება მულტფილმის დამთავრებისას, პირველი რიცხვი უნდა გამოსახავდეს საათებს, ხოლო მეორე წუთებს
