# Problem Statement — 514. მიიღე მესამე

> Copied from [AiaSoft.ge #514](https://aiasoft.ge/problem/514)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია სამი არაუარყოფითი მთელი რიცხვი. დაადგინეთ, პირველი ორისგან მიიღება თუ არა მესამე, ქვემოთმოყვანილი არითმეტიკული ოპერაციებიდან რომელიმე ერთის პირველ ორს შორის ჩასმის შედეგად (გაითვალისწინეთ, რომ მოცემული რიცხვების გადანაცვლება არ შეიძლება).
დასაშვები არითმეტიკული ოპერაციებია:
`b+` (შეკრება), `b-` (გამოკლება), `b*` (გამრავლება), `b/` (მთელად გაყოფა), `b%` (ნაშთით გაყოფა).

## Input

შემოდის ერთმანეთისაგან ჰარით გამოყოფილი სამი არაუარყოფითი მთელი რიცხვი a,  b და c, (0<=a, b, c <= 100).

## Output

დადებითი პასუხის შემთხვევაში პირველ სტრიქონზე გამოიტანეთ სიტყვა "YES", ხოლო მეორე სტრიქონზე ჭეშმარიტი არითმეტიკული წინადადება (იხილეთ ნიმუში).
უარყოფითი პასუხის შემთხვევაში გამოიტანეთ ერთადერთი სიტყვა "NO".
გაქვთ იმისი გარანტია, რომ პასუხი ერთადერთი იქნება.
