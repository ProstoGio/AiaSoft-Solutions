# Problem Statement — 194. სიმბოლოს გადაჩოჩება

> Copied from [AiaSoft.ge #194](https://aiasoft.ge/problem/194)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

N ერთეულით სიმბოლოს გადაჩოჩება ვუწოდოთ ანბანში სიმბოლოს რიგითი ნომრის N-ით გაზრდას. იმ შემთხვევაში თუ გადაჩოჩებისას რიგითი ნომერი გაცდება ანბანის სიმბოლოების რაოდენობას, მაშინ ათვლას ვაგრძელებთ 1-დან.

მაგალითად:
   * a-ს 4-ით გადაჩოჩებისას მივიღებთ e-ს 
   * y-ს 3-ით გადაჩოჩებისას მივიღებთ b-ს

## Input

პირველ ხაზზე შემოდის ინგლისური ანბანის ერთი პატარა სიმბოლო, რომლის გადაჩოჩებაც გვსურს.

მეორე ხაზზე შეოდის ერთი მთელი რიცხვი - N, რამდენი ერთეულით უნდა გადაჩოჩდეს სიმბოლო. (0 <= N <= 10⁶)

## Output

გამოიტანეთ გადაჩოჩების შედეგად მიღებული სიმბოლო
