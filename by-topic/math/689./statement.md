# Problem Statement — 689. ფერადი კარტები

> Copied from [AiaSoft.ge #689](https://aiasoft.ge/problem/689)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

პატარა ნიკომ ახალი კარტი იყიდა, რომელშიც a რაოდენობის წითელი და b რაოდენობის ლურჯი კარტია. ნიკომ კარტებისთვის სპეციალური მაგიდის შეძენა გადაწყვიტა, თუმცა თითოეული მაგიდა გათვლილია კარტების გარკვეულ რაოდენობაზე. ნიკოს აინტერესებს სულ რამდენი კარტი უნდა ეტეოდეს მაგიდაზე, რომ მან შეძლოს ორივე ფერის ყველა კარტის განლაგება.

## Input

ერთ ხაზზე შემოდის ორი ნატრალური რიცხვი a და b (1<=a,b<=10⁵)

## Output

გამოიტანეთ ერთუ რუცხვუ: სულ რამდენ კარტზე გათვლილი მაგიდის შეძენა დაჭირდება ნიკოს.
