# განმარტება — 333. გამყოფები 2

## მიდგომა

ამ ამოცანის იდეა ძალიან მარტივია: უბრალოდ ვამოწმებთ ყველა რიცხვს 1-დან N-ის ჩათვლით. თუ რიცხვი იყოფა N-ზე უნაშთოდ, ესე იგი გამყოფია და პირდაპირ ვუმატებთ ჩვენს ჯამს. კოდი პირდაპირ კითხულობს N-ს და ციკლით გადის ყველა კანდიდატს, რაც ძალიან ინტუიციური მიდგომაა.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- გამყოფების პოვნა 1-დან N-მდე ციკლით ყველაზე მარტივია, თუმცა O(sqrt(n)) მიდგომა უფრო ეფექტური იქნებოდა.
- ჯამის შესანახად `long long` ტიპის გამოყენება ჯობია, თუ რიცხვები ძალიან დიდი იქნება, თორემ `int` შეიძლება გადაივსოს.

## ხშირი შეცდომები

- k ცვლადის საწყის მნიშვნელობად 0-ის არ მიცემა.

## ნაბიჯ-ნაბიჯ ახსნა

მაგალითად, N=6 რომ იყოს: p=1 (6 იყოფა 1-ზე, k=1), p=2 (6 იყოფა 2-ზე, k=1+2=3), p=3 (6 იყოფა 3-ზე, k=3+3=6), p=4 და p=5 არ გვაძლევს ნაშთს 0-ს, და ბოლოს p=6 (6 იყოფა 6-ზე, k=6+6=12).
