# განმარტება — 747. გამყოფები - 4

## მიდგომა

ამოცანა გვთხოვს [a, b] შუალედში თითოეული რიცხვის გამყოფების რაოდენობა ვიპოვოთ. რადგან a და b მაქსიმუმ 10^4-ია, მარტივი ციკლით თითოეულ რიცხვზე გავლა სრულიად საკმარისია. გამყოფებს sqrt(n)-მდე ვამოწმებ, რადგან თუ i გამყოფია, მაშინ n/i-ც ავტომატურად გამყოფი იქნება.

## სირთულე

| | |
|-|--|
| დრო | O((b-a) * sqrt(b)) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- გამყოფების წყვილებად დათვლა საგრძნობლად აჩქარებს მუშაობას.
- კვადრატული რიცხვების შემთხვევაში ფრთხილად უნდა ვიყოთ, რომ root არ დავითვალოთ ორჯერ.

## ხშირი შეცდომები

- if(i*i != t) შემოწმების გამოტოვება, რაც კვადრატული რიცხვების შემთხვევაში არასწორ პასუხს იძლევა.
- შიდა ციკლის 0-დან დაწყება, რაც ნულზე გაყოფის შეცდომას (Runtime Error) გამოიწვევს.

## ნაბიჯ-ნაბიჯ ახსნა

გარე ციკლით მივყვები a-დან b-მდე ყველა რიცხვს. თითოეულისთვის მაქვს ცვლადი cnt=0. შიდა ციკლი 1-დან sqrt(t)-მდე ტრიალებს. თუ t იყოფა i-ზე, ვზრდი cnt-ს და დამატებით ვამოწმებ, i*i != t ხომ არ არის. თუ არ არის, მეორე გამყოფიც (t/i) დამყავს. ბოლოს ვბეჭდავ შედეგებს.
