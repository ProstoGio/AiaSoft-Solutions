# Problem Statement — 940. მარტივი ციკლები

> Copied from [AiaSoft.ge #940](https://aiasoft.ge/problem/940)

**Time limit:** 0.25s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია N წვეროიანი და M წიბოიანი არაორიენტირებული გრაფი. ცნობილია, რომ გრაფი ბმულია, ანუ ნებისმიერი წვეროდან შეგვიძლია სხვა ნებისმიერ წვეროში მისვლა.

`bმარტივი ციკლი` ვუწოდოთ ისეთ ციკლს, რომლის შემადგენელი წვეროების არცერთი ქვესიმრავლე არ ქმნის განსხვავებულ ციკლს. თქვენი დავალებაა მოცემულ გრაფში დათვალოთ მარტივი ციკლების რაოდენობა.

## Input

პირველ ხაზზე მოცემულია რიცხვი Q - ტესტების რაოდენობა.
ყოველი ტესტის პირველ ხაზზე მოცემულია ორი რიცხვი, N და M.
მომდევნო M ხაზზე მოცემულია ორ-ორი რიცხვი. კერძოდ, i-ურ ხაზზე მოცემული იქნება i-ური წიბოს აღწერა: vi და ui, წიბოს ბოლოები.

1≤Q≤100
1≤vi,ui≤N≤2*10⁵
0≤M≤2*10⁵

## Output

თითოეული ტესტისთვის სხვადასხვა სტრიქონზე დაბეჭდეთ მარტივი ციკლების რაოდენობა.
