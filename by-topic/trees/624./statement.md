# Problem Statement — 624. წიბოს ხარისხი

> Copied from [AiaSoft.ge #624](https://aiasoft.ge/problem/624)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია არაორიენტირებული ბმული გრაფი, რომელიც არ შეიცავს ციკლებს.

გრაფში `bწიბოს ხარისხი` ვუწოდოთ განსხვავეული (v, u) წყვილების რაოდენობას, სადაც v-დან u-მდე გზაზე გვხვდება მოცემული წიბო. (ნახეთ ნიმუშის გარჩევა).

გამოთვალეთ მოცემული გრაფის ყოველი წიბოს ხარისხი.

* გაითვალისწინეთ, რომ (v, u) და (u, v) წყვილები განსხვავებულად არ ითვლება.

## Input

პირველ ხაზზე მოცემულია ერთი რიცხვი: N - გრაფში წვეროების რაოდენობა (წვეროები დანომრილია 1-დან N-ის ჩათვლით)
მომდევნო N-1 ხაზზე შემოდის ორ-ორი რიცხვი: vi, ui, რაც აღნიშნავს i ნომერი წიბოს დაბოლოებებს. (წიბოები დანომრილია 1-დან N-1-ის ჩათვლით)

2 <= N <= 10⁵

## Output

ცალ-ცალკე ხაზზე გამობეჭდეთ N-1 რაოდენობის რიცხვი, კერძოდ, i ხაზზე გამოიტანეთ i ნომერი წიბოს ხარისხი.
