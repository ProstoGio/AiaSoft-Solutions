# Problem Statement — 546. ყველაზე შორს

> Copied from [AiaSoft.ge #546](https://aiasoft.ge/problem/546)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია არაორიენტირებული ბმული გრაფი, რომელიც არ შეიცავს ციკლებს (ასეთ გრაფს ხე ეწოდება). ხის წვეროები გადანომრილის 1-დან N-მდე, ხოლო ხის ფესვს წარმოადგენს წვერო ნომრით - 1. თქვენი დავალებაა დაადგინოთ ხის ფესვიდან უდიდეს მანძილზე მყოფი ყველა წვერო. ორ წვეროს შორის მანძილი არის მათ შორის მდებარე წვეროების რაოდენობა, მათი ჩათვლით.

## Input

პირველ ხაზზე მოცემულია N წვეროების რაოდენობა ხეში.
მომდევნო N-1 ხაზზე მოცემულია ორ-ორი რიცხვი. კერძოდ i-ურ ხაზზე შემოვა vi და ui - მეზობელი წვეროების ნომრები. (გრაფის წვეროები დანომრილია 1-დან N-მდე)
2<=N<=10⁵

## Output

პირველ ხაზზე გამობეჭდეთ წვეროების რაოდენობა, რომლებიც მაქსიმალური მანძილით არიან დაშორებულები ხის ფესვიდან.
ხოლო მომდევნო ხაზზე გამობეჭდეთ ეს წვეროები ზრდადობით.
