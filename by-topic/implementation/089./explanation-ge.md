# განმარტება — 089. მორები

## მიდგომა

ეს ამოცანა სუფთა ლოგიკაა. როცა ერთ მორს შუაზე ვჭრით, ორი მორი გვრჩება, ანუ ერთი ჭრით მორის რაოდენობა ერთით იზრდება. გამოდის, რომ თუ თავიდან k მორი გვქონდა და m-ჯერ გადავხერხეთ, ბოლოს გვექნება k + m მორი. შესაბამისად, საწყისი მორების საპოვნელად n-ს ვაკლებ m-ს.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- თითოეული ჭრა ზრდის მორების რაოდენობას ზუსტად ერთით.
- თუ m-ზე მეტი ჭრა მოხდა, ვიდრე საბოლოო მორებია, ეგ ამოცანა შეუძლებელია.

## ხშირი შეცდომები

- უარყოფითი რიცხვის მიღების შემოწმების დავიწყება.
- იმ მომენტის გამოტოვება, როცა მორები საერთოდ არ იყო, მაგრამ m > 0.

## ნაბიჯ-ნაბიჯ ახსნა

კოდში ეგრევე ვითვლი k = n - m. თუ k უარყოფითია, ესე იგი მოცემული m და n ერთმანეთს ეწინააღმდეგება და 'Impossible'-ს ვბეჭდავ. ასევე, ვამოწმებ k=0 და m>0 შემთხვევას, რადგან თუ მორები არ გვაქვს, ვერაფერს გადავხერხავთ.
