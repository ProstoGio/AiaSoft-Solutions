# Problem Statement — 089. მორები

> Copied from [AiaSoft.ge #89](https://aiasoft.ge/problem/89)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მორების სახერხში, სადაც, რა თქმა უნდა, მორებს ხერხავენ, რამდენიმე მორი m-ჯერ გადახერხეს (m არის გადახერხვების საერთო რაოდენობა) , თითო გადახერხვისას ხერხზე მხოლოდ ერთი მორი იდო.

დღის ბოლოს სახერხში n რაოდენობის პატარა მორი აღმოჩნდა.

რამდენი მორი იყო თავიდან სახერხში?

## Input

შემოდის ორი მთელი რიცხვი,  m და n, 0 leq m leq n leq100

## Output

გამოიტანეთ ერთი მთელი რიცხვი - საწყისი მორების რაოდენობა, ანდა სიტყვა Impossible, თუ მოცემული სიტუაცია შეუძლებელია
