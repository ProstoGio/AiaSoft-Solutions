# განმარტება — 813. სიმეტრიულის გამყოფები

## მიდგომა

რიცხვის შებრუნება ყველაზე მარტივად ციფრების გამოყოფით გამოდის: ვყოფთ 10-ზე, ვიღებთ ნაშთს და ვაწყობთ ახალ რიცხვს. გამყოფების დასათვლელად პირდაპირ 1-დან `reversed_num`-მდე ციკლის გატარება ნელი იქნებოდა, ამიტომ `sqrt(N)`-მდე ვჩერდები. ეს კლასიკური ტრიუკია — როცა `i` გამყოფია, `reversed_num/i`-ც ავტომატურად გამყოფი გამოდის, რაც დროში გვზოგავს.

## სირთულე

| | |
|-|--|
| დრო | O(sqrt(N)) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- რიცხვის შებრუნება სტრინგების გარეშე, მარტივი მათემატიკით, ბევრად სუფთა გამოსავალია.
- კვადრატული ფესვის გამოყენება გამყოფების საპოვნელად O(N)-ს O(sqrt(N))-მდე ამცირებს, რაც გაცილებით ეფექტურია.

## ხშირი შეცდომები

- თუ რიცხვი სრული კვადრატია, ფესვი ორჯერ არ უნდა დავითვალოთ, რასაც `if (i != sqrt(reversed_num))` პირობით ვაკონტროლებთ.
