# განმარტება — 487. მკაცრად მონოტონურობა

## მიდგომა

ამოცანის არსი იმაშია, რომ თავიდანვე უნდა მივხვდეთ, მასივი იზრდება თუ კლებულობს. ამისთვის პირველ ორ ელემენტს ვადარებ ერთმანეთს. თუ ტოლია, მაშინვე NO, რადგან 'მკაცრად' მონოტონური უნდა იყოს. სხვა შემთხვევაში, ერთი ციკლით ვამოწმებ, მთელ მასივში ინახება თუ არა იგივე მიმართულება. თუ სადმე პირობა დაირღვა, პროგრამას მაშინვე ვწყვეტ.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- მკაცრად მონოტონურში ტოლი ელემენტები არ უნდა იყოს.
- მიმართულებას პირველი ორი ელემენტი განსაზღვრავს.
- n=1 შემთხვევა ცალკე უნდა დამუშავდეს, რომ მეხსიერების შეცდომა არ ამოაგდოს.

## ხშირი შეცდომები

- a[0] == a[1] შემთხვევის გამოტოვება.
- სიმკაცრის პირობის არევა (>= ან <= გამოყენება).
- ციკლის საზღვრების არასწორად გაწერა (a[i+1] წვდომა როცა n-1-ს სცდება).

## ნაბიჯ-ნაბიჯ ახსნა

თავიდანვე ვითხოვ მასივს. თუ ერთელემენტიანია, პასუხი ავტომატურად YES-ია. თუ ორზე მეტია, ვნახულობ, a[0] > a[1]-ია თუ a[0] < a[1]. პირველ შემთხვევაში მთელ მასივში კლებას ვამოწმებ, მეორეში - ზრდას. როგორც კი პირობა დაირღვევა, გამოვაქვს NO და ვასრულებ მუშაობას.
