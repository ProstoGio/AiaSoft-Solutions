# Problem Statement — 487. მკაცრად მონოტონურობა

> Copied from [AiaSoft.ge #487](https://aiasoft.ge/problem/487)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მასივს ეწოდება მკაცრად მონოტონური, თუ იგი ან მკაცრად ზრდადია, ან მკაცრად კლებადია.
მოცემულია მასივი, დაადგინეთ მკაცრად მონოტონურია თუ არა იგი.

## Input

პირველ სტრიქონზე შემოდის ერთი რიცხვი - მასივის ელემენტების რაოდენობა, რომელიც არ აღემატება 100-ს.
მეორე სტრიქონზე შემოდის თვითონ მასივი.

## Output

გამოიტანეთ ერთი სიტყვა  - YES, თუ მასივი მკაცრად მონოტონურია , ან  NO, თუ მასივი მკაცრად მონოტონური არ არის.
