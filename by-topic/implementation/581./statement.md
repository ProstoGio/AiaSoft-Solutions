# Problem Statement — 581. სიმპატიური მატრიცა

> Copied from [AiaSoft.ge #581](https://aiasoft.ge/problem/581)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია NxM ზომის მატრიცა, რომელიც შედგება მხოლოდ 0-ებისა  და 1-ებისგან. მატრიცას ეწოდება სიმპატიური თუკი მასში არარის არცერთი კვადრატი 2x2, ისეთი რომ ოთხივე უჯრა მხოლოდ 1-იანია ან მხოლოდ 0-იანი. თქვენი დავალებაა ყოველი ტესტისთვის დაადგინოთ არის თუ არა მატრიცა სიმპატიური.

## Input

პირველ ხაზზე მოცმეულია T - ტესტების რაოდენობა.
ყოველი ტესტის პირველ ხაზზე მოცემულია N და M - მატრიცაში სტრიქონების და სვეტების რაოდენობა შესაბამისად.
მომდევნო ხაზზე მოცემულია NxM ზომის მატრიცა, რომელიც შედგენა მხოლოდ 0 და 1-ისგან. 
1<=T<=10
1<=N,M<=100

## Output

ყოველი ტესტისთვის გამობეჭდეთ, არის თუ არა შესაბამისი მატრიცა სიმპატიური.
