# განმარტება — 581. სიმპატიური მატრიცა

## მიდგომა

მატრიცა რომ 'სიმპატიური' იყოს, არცერთი 2x2 კვადრატი არ უნდა შედგებოდეს ერთნაირი ციფრებისგან. ამიტომ, უბრალოდ გადავუყვები მატრიცას (N-1)-მდე და (M-1)-მდე, რათა ყოველი უჯრისტვის შევამოწმო მისი მეზობლები ქვემოთ, მარჯვნივ და დიაგონალზე. თუ ოთხივე უჯრა ერთი და იგივეა, ესე იგი ეგ 2x2 კვადრატი 'ცუდია' და მატრიცა ავტომატურად არასიმპატიურია.

## სირთულე

| | |
|-|--|
| დრო | O(N * M) |
| მეხსიერება | O(N * M) |

## მთავარი დაკვირვებები

- საკმარისია მხოლოდ ის შევამოწმოთ, ოთხივე უჯრა ტოლია თუ არა; არ აქვს მნიშვნელობა 0-ებია თუ 1-ები.
- ციკლები უნდა შეჩერდეს N-1 და M-1-ზე, თორემ ინდექსი გასცდება მასივს.

## ხშირი შეცდომები

- ინდექსების არევა და მასივის გარეთ გასვლა (Out of bounds).
- ზედმეტი პირობების წერა, როცა ერთი შედარება საკმარისია.

## ნაბიჯ-ნაბიჯ ახსნა

კოდი კითხულობს მატრიცას და `ind` ცვლადს ვუყენებ true-ზე. თუ სადმე ვიპოვე 2x2 უბანი, სადაც ყველა უჯრა ერთმანეთის ტოლია, `ind`-ს ვაქცევ false-ად, ვწყვეტ ციკლს და ვბეჭდავ 'NO'-ს.
