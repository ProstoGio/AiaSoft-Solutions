# განმარტება — 828. კატო და საყვარელი სიმბოლო

## მიდგომა

მოკლედ, ამოცანა მარტივია: სტრიქონში უნდა დავითვალო კონკრეტული სიმბოლოს რაოდენობა. მთავარი 'ტრიუკი' აქ ისაა, რომ ASCII ცხრილში დიდი და პატარა ასოები ერთმანეთისგან ზუსტად 32 ერთეულით განსხვავდება. ასე რომ, `if`-ში უბრალოდ ვამოწმებ თვითონ სიმბოლოს, ან მის ვარიაციებს `±32`-ით და ეგაა, ზედმეტი ბიბლიოთეკების გარეშეც მშვენივრად მუშაობს.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- ASCII-ში 'A'-ს კოდი 65-ია, 'a'-სი კი 97. 32-ზე მანიპულაციით მარტივად ვფარავთ ორივე შემთხვევას.
- სტრიქონის სიგრძე მაქსიმუმ 500-ია, ამიტომ O(n) სირთულე წამში უპრობლემოდ ჩაეტევა.

## ხშირი შეცდომები

- მხოლოდ ერთი რეგისტრის შემოწმება, როცა პირობა ზოგად შემთხვევას გულისხმობს.
