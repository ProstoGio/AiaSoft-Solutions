# განმარტება — 757. მიმდევრობა

## მიდგომა

აქ არაფერია საწვალებელი, პირდაპირი მიდგომაა. მთელი მასივის მეხსიერებაში შენახვაც კი არ გჭირდება, რადგან მხოლოდ დათვლა გვაინტერესებს. უბრალოდ ციკლში ვამოწმებ თითოეულ სიმაღლეს და თუ მიშოზე დაბალია, `cnt` ცვლადს ვზრდი ერთით.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- არ არის საჭირო `vector`-ის ან მასივის გამოცხადება, პირდაპირ `cin`-ის დროსვე შეგვიძლია შედარება.
- ყურადღება მიაქციე პირობას, 'ზუსტად მიშოს სიმაღლის ბავშვი არ ითვლება' - ანუ მხოლოდ მკაცრი უტოლობა გვჭირდება.

## ხშირი შეცდომები

- შედარებისას შეცდომით `<=` ოპერატორის გამოყენება.
