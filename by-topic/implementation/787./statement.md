# Problem Statement — 787. დაადგინე არის თუ არა მარტივი(მარტივი ვერსია)

> Copied from [AiaSoft.ge #787](https://aiasoft.ge/problem/787)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია N ცალი რიცხვისგან შემდგარი მასივი. თქვენი დავალებაა თითოეული წევრისთვის დაადგინოთ არის თუ არა მოცემული რიცხვი მარტივი.

## Input

პირველ სტრიქონზე შემოდის N რიცხვი.
მეორე სტრიქონზე შემოდის ერთმანეთისგან ჰარით გამოყოფილი N ცალი რიცხვი.
თითოეული რიცხვი მოცემული მასივიდან არის [1,10^3] შუალედში.
1<=N<=10⁶

## Output

ერთ სტრიქონზე გამოიტანეთ თითოეული რიცხვის პასუხი. თუ რიცხვი მარტივია დაბეჭდეთ 1, წინააღმდეგ შემთხვევაში 0.
