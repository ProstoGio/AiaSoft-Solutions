# განმარტება — 572. სათამაშოები

## მიდგომა

რადგან დაფა არის 2xN ზომის, ხოლო სათამაშოები 1x2-ის, სულ გვაქვს 2*N ფართობი. თითო სათამაშო იკავებს 2 ერთეულ ფართობს, ამიტომ მაქსიმალური რაოდენობა არის (2*N) / 2, რაც უბრალოდ N-ია. არ აქვს მნიშვნელობა ჰორიზონტალურად დააწყობ თუ ვერტიკალურად, დაფა სრულად შეივსება და ცარიელი ადგილი არ დარჩება.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- მთლიანი ფართობი იყოფა სათამაშოს ფართობზე და ვიღებთ N-ს.
- N შეიძლება იყოს 10^18-მდე, ამიტომ აუცილებლად long long უნდა გამოვიყენოთ.

## ხშირი შეცდომები

- int ტიპის გამოყენება, რაც დიდ რიცხვებზე overflow-ს გამოიწვევს.
- პრობლემის ზედმეტად გართულება და რაღაც რთული ალგორითმების ძებნა.

## ნაბიჯ-ნაბიჯ ახსნა

კოდი მარტივია: შემოგვაქვს N და პირდაპირ გამოგვაქვს იგივე რიცხვი ეკრანზე. მეტი არაფერია საჭირო.
