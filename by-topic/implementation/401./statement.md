# Problem Statement — 401. საუკუნე

> Copied from [AiaSoft.ge #401](https://aiasoft.ge/problem/401)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

მოგეხსენებათ, რომ 2000 წლიდან 2099 წლის ჩათვლით 21-ე საუკუნეში ვიმყოფებით, ხოლო 1900-დან 1999-წლის ჩათვლით მე-20 საუკუნე იყო.

მოცემულია წელიწადის ნომერი. დაადგინეთ რომელ საუკუნეს ეკუთვნის მოცემული წელი: მეოცეს, ოცდამეერთეს თუ არცერთს.

## Input

მოცემულია ერთი დადებითი მთელი რიცხვი: N - წელიწადის ნომერი.

1000 <= N <= 3000

## Output

გამობეჭდეთ `b21` თუ N ეკუთვნის 21-ე საუკუნეს, `b20` თუ N ეკუთვნის მე-20 საუკუნეს და `b-1` თუ არცერთ მათგანს არ მიეკუთვნება N.
