# Problem Statement — 573. სუნთქვა

> Copied from [AiaSoft.ge #573](https://aiasoft.ge/problem/573)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ზუკა შეუწყვეტლად ითვლის 1-დან N-მდე. თან აკეთებს სუნთქვის პროცედურას შემდეგნაირად: თვლაზე "1-2" ჩაისუნთავს, ხოლო თვლაზე "3-4" ამოისუნთქავს, შემდეგ "5-6"-ზე ჩაისუნთქავს, "7-8"-ზე ისევ ამოისუნთავს და ა. შ.

დაადგინეთ რომელი უფრო მეტჯერ გააკეთა ზუკამ N-მდე დათვლაში: სრულად ჩასუნთქვა თუ სრულად ამოსუნთქვა.

## Input

მოცემულია ერთი ნატურალური რიცხვი: N, (1 <= N <= 100)
ზუკა ითვლის 1-დან N-ის ჩათვლით.

## Output

გამობეჭდეთ "Breathe in" თუ ზუკამ უფრო მეტჯერ ჩაისუნთქა, "Breathe out" - თუ უფრო მეტჯერ ამოისუნთქა და "-1" თუ ორივე მოქმედება თანაბარი რაოდენობით შეასრულა.
