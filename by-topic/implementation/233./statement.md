# Problem Statement — 233. ჩოგბურთის მატჩის ოქმი

## Task

„დიდი მუზარადის“ ტურნირებზე ჩოგბურთის მატჩები გრძელდება 3 სეტის მოგებამდე. სეტი შეუძლებელია ფრედ დამთავრდეს - მას აუცილებლად იგებს ერთ-ერთი ჩოგბურთელი. ასევე შეუძლებელია ფრედ დამთავრდეს მთლიანი მატჩიც. ცხადია, რომ ერთ მატჩში ჩატარებული სეტების რაოდენობა 5-ს არ აღემატება, ამიტომ მატჩის ოქმში მითითებულია 5 გრაფა სეტების შედეგთა ჩასაწერად. თუ მატჩი სამ ან ოთხ სეტში დამთავრდა, ჩაუტარებელი სეტების გრაფებში იწერება 0-ები. დაწერეთ პროგრამა, რომელიც  ოქმის მიხედვით დაადგენს, თუ რომელმა მოთამაშემ მოიგო მატჩი - პირველმა თუ მეორემ და ამასთანავე გამოითვლის თითოეული მოთამაშის მიერ ჯამურად დაგროვებულ ქულებს.

## Input

შესატანი მონაცემები შედგება ხუთი სტრიქონისაგან. თითოეულ სტრიქონში მოცემულია თითო ჰარით გამოყოფილი ორ-ორი მთელი რიცხვი 0-დან 7-მდე დიაპაზონში - პირველად წერია შესაბამის სეტში პირველი მოთამაშის მიერ დაგროვებული ქულები, ხოლო მეორედ - მეორე მოთამაშის მიერ დაგროვებული ქულები იმავე სეტში.

## Output

პირველ სტრიქონში გამოიტანეთ სიტყვა “FIRST”- თუ მატჩი მოიგო პირველმა მოთამაშემ და სიტყვა “SECOND” - თუ მატჩი მოიგო მეორე მოთამაშემ. მეორე სტრიქონში გამოიტანეთ თითოეული მოთამაშის მიერ მთელი მატჩის განმავლობაში დაგროვებული ქულები.
