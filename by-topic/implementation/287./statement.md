# Problem Statement — 287. არჩევნები

> Copied from [AiaSoft.ge #287](https://aiasoft.ge/problem/287)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

ბოლივიაში მალე საპრეზიდენტო არჩევნები გაიმართება. პრეზიდენტად სულ რაღაც ორი კანდიდატი განიხილება, თაზო და ლერი. ორივე მათგანს სურს გაიგოს მინიმუმ რამდენი ხმა უნდა მოაგროვოს, რომ გაიმარჯვოს არჩევნებში. გაიმარჯვებს ის კანდიდატი ვისაც მეტი ხმა ექნება.

ამომრჩეველთა რაოდენობა ბოლივიაში არის N (ლერის და თაზოს ჩათვლით).
დაეხმარეთ პრეზიდენტობის კანდიდატებს პასუხის გაგებაში.

* `iგარანტირებულია, რომ არჩევნებში ყველა ამომრჩეველი მიიღებს მონაწილეობას და შემოხაზავს მხოლოდ ერთ-ერთ კანდიდატს`.

## Input

მოცემულია ერთი მთელი რიცხვი: N - ამომრჩეველთა რაოდენობა ბოლივიაში. (2<=N<=100)

## Output

გამოიტანეთ ერთი მთელი რიცხვი - გამარჯვების მოსაპოვებლად საჭირო მინიმალური ხმების რაოდენობა.
