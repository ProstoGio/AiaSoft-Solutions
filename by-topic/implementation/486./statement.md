# Problem Statement — 486. მკაცრად ზრდადობა

## Task

მოცემულია მასივი. დაადგინეთ არის თუ არა იგი მკაცრად ზრდადი.

* მიმდევრობას ეწოდება მკაცრად ზრდადი, თუ მისი ყოველი წევრი (მეორედან დაწყებული) `bმეტია` წინაზე.

## Input

პირველ სტრიქონზე შემოდის ერთი რიცხვი n - მასივის ელემენტების რაოდენობა, რომელიც არ აღემატება 100-ს.
მეორე სტრიქონზე შემოდის თვითონ მასივის ელემენტები.

## Output

გამოიტანეთ ერთი სიტყვა -  YES, თუ მასივი მკაცრად ზრდადია და NO, თუ არ არის მკაცრად ზრდადი,
