# განმარტება — 332. გამყოფები 1

## მიდგომა

ამოცანა მარტივია: უნდა დავითვალოთ N-ის ყველა გამყოფი. პირდაპირი მიდგომით, შეგვიძლია 1-დან N-მდე ყველა რიცხვზე გავიაროთ, მაგრამ ოპტიმიზაციისთვის საკმარისია N/2-მდე შემოწმება, რადგან N-ზე დიდი გამყოფი (თვითონ N-ის გარდა) არ არსებობს. მრიცხველი 1-ით დავაწყებინე, რადგან N ყოველთვის იყოფა საკუთარ თავზე, შემდეგ კი ციკლით ვპოულობთ დანარჩენებს.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- N-ის გამყოფი ვერ იქნება N/2-ზე მეტი, გარდა თავად N-ისა.
- კოდი `k=1`-ით იწყება, რითაც N-ის გამყოფობას ავტომატურად ვითვალისწინებთ.

## ხშირი შეცდომები

- თუ ციკლს N/2-მდე აჩერებ, არ დაგავიწყდეს, რომ საბოლოო პასუხს თავად N-იც უნდა დაუმატო.
