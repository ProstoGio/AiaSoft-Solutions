# განმარტება — 426. ტრისკაიდეკაფობია

## მიდგომა

ამ ამოცანის იდეა მარტივია: უნდა ვადევნოთ თვალი გივიკოს სართულს და გავითვალისწინოთ, რომ მე-13 სართული არ არსებობს. როცა მაღლა ავდივართ და 12-ს გადავცდებით, 13-ის ნაცვლად პირდაპირ 14-ზე ვხვდებით. იგივე პრინციპი მოქმედებს ქვევით ჩამოსვლისას, უბრალოდ პირიქით.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- თუ სართული 12-ზე მაღლაა, ის რეალურად 1-ით მეტია, ვიდრე ლოგიკური რიცხვთა მიმდევრობა გვაძლევს.
- გადასვლისას 13-ის გამოტოვება ხდება როგორც ასვლის, ისე ჩამოსვლის შემთხვევაში.

## ხშირი შეცდომები

- იმის დავიწყება, რომ ჩამოსვლის დროსაც 13-იანი უნდა გამოვტოვოთ.
- რთული ლუპების წერა, როცა ჩვეულებრივი `if-else` სრულიად საკმარისია.

## ნაბიჯ-ნაბიჯ ახსნა

პირველ რიგში, ვთვლი სართულს, სადაც გივიკო ავა. თუ სტარტი 13-ზე ქვემოთაა და მიზანი 13 ან მეტია, ვამატებ 1-ს, რომ 13-იან სართულს გადავახტე. შემდეგ ზუსტად იგივე ლოგიკას ვიყენებ ჩამოსვლისას, ოღონდ `b`-ს გამოკლებისას ვამოწმებ ხომ არ გადავიკვეთეთ 13-იან ნიშნული. საკმაოდ პრიმიტიული მიდგომაა, მაგრამ 0-დან 50-მდე შეზღუდვისთვის იდეალურად მუშაობს.
