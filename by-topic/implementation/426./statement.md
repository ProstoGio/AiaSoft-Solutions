# Problem Statement — 426. ტრისკაიდეკაფობია

> Copied from [AiaSoft.ge #426](https://aiasoft.ge/problem/426)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ტრისკაიდეკაფობია 13-ის შიშს ეწოდება.
ლიფტის სამონტაჟო კომპანიის პრეზიდენტი, ისევე როგორც მთელი ქვეყანა, შეპყრობილია ტრისკაიდეკაფობიით. შესაბამისად, ეს კომპანია ლიფტის ღილაკებზე 13-ს არ აწერს, მეცამეტე ნომერი სართული არც მრავალსართულიან სახლებშია. მე-12 ნომრის მერე მოდის მე-14 ნომერი.
გივიკო შედის 100 სართულიანი ცათამბჯენის ლიფტში, k ნომერ სართულზე, აპირებს მაღლა ასვლას  a სართულით, შემდეგ კი დაბლა უნდა დაეშვას b სართულით. რომელ ნომერ სართულზე აღმოჩნდება გივიკო ლიფტიდან გამოსვლის შემდეგ?

## Input

შემოდის სამი ნატურალური რიცხვი  k, a და b - შესაბამისად იმ სართულის ნომერი, სადაც გივიკო შევიდა ლიფტში, იმ სართულების რაოდენობა, რამდენითაც ავიდა მაღლა და იმ სართულების რაოდენობა, რამდენითაც ჩამოვიდა დაბლა.
0<=k, a, b<=50

## Output

გამოიტანეთ ერთი რიცხვი, იმ სართულის ნომერი, რომელზეც გამოვიდა ლიფტიდან გივიკო.
