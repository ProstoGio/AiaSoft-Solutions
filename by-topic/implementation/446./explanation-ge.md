# განმარტება — 446. დაკარგული რიცხვი

## მიდგომა

მოკლედ, საქმე იმაშია, რომ გვაქვს რიცხვები 1-დან n-მდე, ოღონდ ერთი აკლია. ყველაზე მარტივი გზა, რაც თავში მომივიდა, არის 'ფრექუენსი ერეის' გამოყენება. მოკლედ, ავიღე მასივი, სადაც თითოეულ რიცხვს მივუთითებ, შეგვხვდა თუ არა. ბოლოს უბრალოდ ვამოწმებ, რომელ ინდექსზეა ნული დარჩენილი და ეგაა ჩვენი დაკარგული კოჭი.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- ჯამების გამოთვლას სჯობია მარტივი 'ფლაგების' სისტემა ავაწყოთ, ასე არც ოვერფლოუს რისკი გვაქვს და არც ზედმეტი მათემატიკური ოპერაციები გვჭირდება.
- მასივის ზომა მაქსიმალურ n-ს უნდა გასწვდეს, ამიტომ კონსტანტად 1e5-ის დაწერა სრულიად საკმარისია.

## ხშირი შეცდომები

- ხშირად ავიწყდებათ, რომ მარყუჟი 1-დან n-მდე უნდა ტრიალებდეს, თორემ თუ n-ია დაკარგული, მასივი ვერ დაინახავს.
