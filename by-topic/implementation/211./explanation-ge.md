# განმარტება — 211. სიმბოლო

## მიდგომა

ამ ამოცანის ამოხსნა ყველაზე მარტივად `isdigit()` ფუნქციით შეიძლება, რომელიც პირდაპირ გვეუბნება არის თუ არა სიმბოლო რიცხვი. მე სტრინგად წავიკითხე შემოტანილი მნიშვნელობა, რომ არ ვიწვალო `cin`-ის ბუფერში დარჩენილ სიმბოლოებზე. კოდი იმდენად მოკლეა, რომ აქ ზედმეტი სირთულეები არაფერში გვჭირდება. უბრალოდ ვამოწმებთ პირველ ინდექსს და პასუხიც მზადაა.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- `isdigit()` ფუნქცია ყველაზე ოპტიმალური ვარიანტია, რადგან ის პირდაპირ ASCII კოდების მიხედვით ამოწმებს დიაპაზონს.
- მონაცემის string-ის სახით წაკითხვა უსაფრთხოა, რადგან თავიდან გვარიდებს cin-ის სპეციფიკურ ქცევას char-ის კითხვისას.

## ხშირი შეცდომები

- ხანდახან სტუდენტები ცდილობენ ხელით შეამოწმონ if-ით, მაგალითად (c >= '0' && c <= '9'), რაც სწორია, მაგრამ `isdigit()` მაინც უფრო პროფესიონალური და მოკლე მიდგომაა.
