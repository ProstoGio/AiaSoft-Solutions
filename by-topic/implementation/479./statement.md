# Problem Statement — 479. სამშენებლო წელიწადი

> Copied from [AiaSoft.ge #479](https://aiasoft.ge/problem/479)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

2021 წელი თანამედროვეობის ლიდერმა სამშენებლო კომპანია `iaiasofter`-მა მშენებლობის წლად გამოაცხადა. მათ გრძელვადიანი პროექტების განსახორციელებლად სატვირთო ავტომობილები სჭირდებათ. სამწუხაროდ ამ ქალაქში ყველა მანქანას თანაბარი (m ტონიანი) ტვირთამწეობა აქვს. თქვენ, როგორც `iaiasofter`-ის პროგრამისტს, დაგევალათ გამოთვალოთ მინიმუმ რამდენი მანქანის დაქირავება იქნება საჭირო, რომ `bერთი გადატანით` მოხერხდეს n ტონა ტვირთის წაღება A პუნქტიდან B პუნქტში.

## Input

ერთ ხაზზე შემოდის ორი მთელი რიცხვი n და m, შესაბამისად გადასატანი ტვირთის რაოდენობა (ტონებში) და თითოეული მანქანის ტვირთამწეობა (ტონებში).
1<=n,m<=100

## Output

ერთ ხაზზე გამობეჭდეთ ერთი რიცხვი. ის მინიმალური მანქანების რაოდენობა, რომელიც საჭიროა n ტონა ტვირთის გადასატანად.
