# Problem Statement — 701. მასივში ციფრთა ჯამი

> Copied from [AiaSoft.ge #701](https://aiasoft.ge/problem/701)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემული N ელემენტიანი მიმდევრობა, თქვენი დავალებაა ყოველი რიცხვისთვის დაბეჭდოთ ამ რიცხვის ციფრთა ჯამი.

## Input

პირველ ხაზზე მოცემულია N - მიმდევრობაში ელემენტების რაოდენობა.
მომდევნო ხაზზე მოცემულია N მთელი დადებითი რიცხვი, რომლებიც არ აღემატება 10⁵

## Output

ერთ ხაზზე გამობეჭდეთ N მთელი დადებითი რიცხვი, ყოველი გამობეჭდილი k-ური რიცხვი უნდა აღნიშნავდეს მიმდევრობაში k პოზიციაზე არსებულ რიცხვთა ციფრთა ჯამი
