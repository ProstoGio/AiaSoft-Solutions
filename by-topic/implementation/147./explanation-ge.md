# განმარტება — 147. განსხვავებული რიცხვების რაოდენობა 1

## მიდგომა

თავიდან 'სეთების' გამოყენებაზე ვიფიქრე, მაგრამ ამ ზომის მასივისთვის სორტირება უფრო გამარტივებული გზაა. როგორც კი რიცხვებს დაალაგებ ზრდადობით, ყველა ერთნაირი რიცხვი გვერდიგვერდ აღმოჩნდება. მერე უბრალოდ თითოეულ ელემენტს ვადარებდი წინა ნაპოვნს და თუ განსხვავებული იყო, მაშინვე ვითვლიდი. კოდი იმდენად მოკლე გამოდის, რომ მეტი არაფერია საჭირო.

## სირთულე

| | |
|-|--|
| დრო | O(n log n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- დალაგებული მასივის შემთხვევაში დუბლიკატები ყოველთვის გვერდით დგას, რაც საშუალებას გვაძლევს ერთი 'პასით' დავთვალოთ ყველაფერი.
- მეხსიერების დასაზოგად დამატებითი კონტეინერების გარეშე, მარტივი ცვლადით შედარება სავსებით საკმარისია.

## ხშირი შეცდომები

- თუ მრიცხველს 0-დან დაიწყებ, 1 განსხვავებული რიცხვის შემთხვევაში პასუხში შეცდომას მოგცემს.
