# განმარტება — 814. ლამაზი წევრები

## მიდგომა

მოკლედ, ამოცანა არის მარტივი იტერაცია მასივზე, ოღონდ უნდა გავითვალისწინოთ, რომ პირველ და ბოლო ელემენტს მეზობლები არ ეყოფათ, ამიტომ ციკლს ვუშვებთ პირველიდან ბოლოსწინა ინდექსამდე. ყოველი ასეთი ელემენტისთვის ვამოწმებთ ორ რამეს: ლუწია თუ არა და ტოლია თუ არა მეზობლების საშუალო არითმეტიკულის. თუ პირობა სრულდება, მთვლელს ვზრდით და ბოლოს თუ არაფერი ვიპოვეთ, -1-ს გამოვიტანთ.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- ინდექსების სწორად შერჩევა გადამწყვეტია – 0-ს და n-1-ს ავტომატურად ვტოვებთ, რადგან მათ ორივე მხარეს მეზობელი არ გააჩნიათ.
- საშუალო არითმეტიკულის შემოწმებისას (a[i-1] + a[i+1]) % 2 == 0 დამატება გვიცავს იმაში, რომ გაყოფა მთელი რიცხვი იყოს.

## ხშირი შეცდომები

- მასივის საზღვრების არევა და მეხსიერების არასწორი მისამართის წაკითხვა.
- პირობის არასრული შემოწმება, მაგალითად, მხოლოდ ლუწობის გათვალისწინება.
