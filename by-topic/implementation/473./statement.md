# Problem Statement — 473. არითმეტიკული ქვემიმდევრობა

> Copied from [AiaSoft.ge #473](https://aiasoft.ge/problem/473)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია n ელემენტიანი მიმდევრობა a1, a2, ..., an.
დაადგინეთ მინიმალური ინდექსი p, ისეთი, რომ ap, a(p+1), a(p+2), ..., an წარმოადგენდეს არითმეტიკულ ქვემიმდევრობას (ანუ არითმეტიკულ პროგრესიას).

## Input

შემოდის რამდენიმე ტესტი (გაურკვეველი რაოდენობით).
თითოეული ტესტი შეიცავს შემდეგ ინფორმაციას:
● პირველ ხაზზე მოცემულია ერთი მთელი რიცხვი: n - მიმდევრობაში ელემენტების რაოდენობა
● მეორე ხაზზე მოცემულია ჰარით გამოყოფილი n მთელი რიცხვი: a1, a2, ..., an - მიმდევრობის წევრები

* გარანტირებულია, რომ ტესტების რაოდენობა არ აღემატება 531-ს.

1 <= n <= 10³
0 <= a1, a2, ..., an <= 10⁹

## Output

ყოველი ტესტისთვის ცალკეულ ხაზზე გამობეჭდეთ ერთიმთელი რიცხვი: მინიმალური p
