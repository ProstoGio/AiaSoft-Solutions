# Problem Statement — 260. მარყუჟი

> Copied from [AiaSoft.ge #260](https://aiasoft.ge/problem/260)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

მოცემულია გრაფი მეზობლობის მატრიცით. დაადგინეთ შეიცავს თუ არა მოცემული გრაფი მარყუჟს.

`iმარყუჟი ეწოდება წიბოს, რომელიც წვეროს თავის თავთან აერთებს`

## Input

პირველ ხაზზე მოცემულია გრაფის წვეროების რაოდენობა: N (1 <= N <= 50)

შემდეგ მოცემულია N სტრიქონი, თითოეულზე N ჰარით გამოყოფილი რიცხვი, რომელიც ერთიანობაში წარმოადგენს NxN ზომის გრაფის წვეროების მეზობლობის მატრიცას. თუ a(i,j) = 1 მაშინ i და j მეზობელი წვეროებია, ხოლო თუ a(i,j) = 0 - მაშინ არა.

## Output

გამოიტანეთ ერთი სიტყვა: `bYES` თუ გრაფი შეიცავს ერთ მარყუჟს მაინც, წინააღმდეგ შემთხვევაში `bNO`.
