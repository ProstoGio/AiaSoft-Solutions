# Problem Statement — 854. სრული პალინდრომი

> Copied from [AiaSoft.ge #854](https://aiasoft.ge/problem/854)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ყველამ კარგად ვიცით, რომ პალინდრომი არის სტრიქონი, რომელიც მარცხნიდან და მარჯვნიდან ერთნაირად იკითხება. გაითვალისწინეთ, რომ სიმბოლოების რეგისტრს ყურადღება არ ექცევა. მაგალითად პაილდრომებია: Level, level, LevEl, Ana, ana, anA
შემოვიტანოთ ახალი ტერმინი, `bსრული პალინდრომი`, რომელიც შემდეგნაირად განიმარტება - სტრიქონი წარმოადგენს სრულ პალინდრომს, თუ იგი პალინდრომია და მისი ნემისმიერი ქვესტრიქონიც პალინდრომია.

მოცემულია სტრიქონი, დაადგინეთ წარმოადგენს თუ არა იგი სრულ პალინდრომს.

## Input

ერთ ხაზზე მოცემულია ლათინური ანბანის სიმბოლოებისგან შედგენილი სტრიქონი, რომლის სიგრძეც არ აღემატება 1000-ს.

## Output

გამობეჭდეთ `bTrue` თუ მოცემული სტრიქონი სრული პალინდრომია, წინააღმდეგ შემთხვევაში `bFalse`.
