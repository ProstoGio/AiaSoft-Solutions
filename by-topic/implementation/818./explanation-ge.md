# განმარტება — 818. ელემენტები დიაპაზონიდან

## მიდგომა

სიმართლე რომ გითხრა, აქ არაფერია რთული. ორი ცვლადი მჭირდებოდა, ერთი რაოდენობის დასათვლელად და მეორე ჯამის შესანახად. პირდაპირ `cin`-ის დროსვე ვამოწმებ, რიცხვი შუალედშია თუ არა, ასე რომ მეორე ციკლი აღარ მჭირდება. თუ ბოლოს აღმოვაჩენ, რომ მთვლელი 0-ია, უბრალოდ -1-ს ვბეჭდავ და მორჩა.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- ჯობია ეგრევე, input-ის წაკითხვის პროცესშივე დავთვალოთ ყველაფერი, ზედმეტი სირთულეების გარეშე.
- თუ counter არ გაიზრდება, ეგ პირდაპირ მიგვანიშნებს იმაზე, რომ პირობა არცერთმა რიცხვმა არ დააკმაყოფილა.

## ხშირი შეცდომები

- შეიძლება ვინმეს დაავიწყდეს ცვლადების ნულზე გაინიციალიზება, რის გამოც ნაგავი რიცხვები გამოვა.
