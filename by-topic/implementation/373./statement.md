# Problem Statement — 373. გამოტოვებული რიცხვი

> Copied from [AiaSoft.ge #373](https://aiasoft.ge/problem/373)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია ზრდადობით დალაგებული N არაუარყოფითი მთელი რიცხვისაგან შედგენილი მიმდევრობა. თქვენი მიზანია იპოვოთ `bმინიმალური არაუარყოფითი` რიცხვი, რომელიც არ მოიძებნება მოცემულ მიმდევრობაში.

* გარანტირებულია, რომ მიმდევრობის ყველა ელემენტი განსხვავებულია

## Input

პირველ ხაზზე მოცემულია ერთი მთელი რიცხვი: N - მიმდევრობის წევრების რაოდენობა.
მომდევნო ხაზზე მოცემულია N რაოდენობის არაუარყოფითი რიცხვი: a1, a2, ..., an - მიმდევრობის წევრები.

1 <= N <= 2 * 10⁵
0 <= ai <= 10⁶

## Output

გამოიტანეთ მინიმალური არაუარყოფითი მთელი რიცხვი, რომელიც არ მოიძებნება მოცემულ მიმდევრობაში.
