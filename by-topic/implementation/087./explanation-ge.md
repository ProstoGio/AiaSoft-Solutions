# განმარტება — 087. N რიცხვის ჯამი

## მიდგომა

აქ მთავარია `long long`-ის გამოყენება არ დაგავიწყდეს, თორემ მარტივი ჯამია. რიცხვები 10⁹-მდეა და 20 ცალია, შესაბამისად ჯამი `int`-ის ლიმიტს გადაცდება. მე `array`-ში შევინახე და მერე ცალკე ფუნქციაში დავუჯამეთ, თუმცა რეალურად ეგრევე `cin`-ის დროსაც შეიძლება დაჯამება.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- `long long` აუცილებელია, რადგან ჯამი მარტივად აცდება 32-bit `int`-ის მაქსიმალურ значени-ს.
- N-ის ლიმიტი 20-ია, ამიტომ `array[20]` სრულიად საკმარისია და მეხსიერების პრობლემა არ შეგექმნება.

## ხშირი შეცდომები

- სტანდარტული `int`-ის გამოყენება ჯამისთვის, რაც overflow-ს გამოიწვევს.
