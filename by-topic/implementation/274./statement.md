# Problem Statement — 274. ციკლები გრაფში - 2

> Copied from [AiaSoft.ge #274](https://aiasoft.ge/problem/274)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 128 MB

## Task

მოცემულია არაორიენტირებული ბმული გრაფი.
დაადგინეთ, `iრომელი` წიბოები უნდა წაიშალოს, რომ დარჩენილი გრაფი წარმოადგენდეს ხეს.

## Input

პირველ ხაზზე შემოდის N და M, შესაბამისად გრაფში წვეროებისა და წიბოების რაოდენობა.
მომდევნო M ხაზზე მოცემულია ორ-ორი რიცხვი. კერძოდ i-ურ ხაზზე შემოვა vi და ui - მეზობელი წვეროების ნომრები. (გრაფის წვეროები დანომრილია 1-დან N-მდე)

1 <= N <= 20
0 <= M <= N*(N-1) / 2
1 <= vi, ui <= N

## Output

პირველ ხაზზე გამობეჭდეთ ერთი მთელი რიცხვი: წასაშლელი წიბოების რაოდენობა.
მეორე ხაზზე გამობეჭდეთ წასაშლელი წიბოების ნომრები (წიბოები გადანომრილია 1-დან M-მდე შემოსვლის თანმიმდევრობის მიხედვით)

ამოშლის რამდენიმე ვარიანტის არსებობის შემთხვევაში, გამობეჭდეთ რომელიმე ერთი მათგანი.
