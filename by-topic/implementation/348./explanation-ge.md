# განმარტება — 348. ინდექსები

## მიდგომა

საკმაოდ მარტივი ამოცანაა, მთავარია თანმიმდევრობა არ აგვერიოს. თავიდან ვითხოვთ N-ს და X-ს, მერე ვავსებთ მასივს და პირდაპირ ერთ 'for' ციკლში ვამოწმებთ ელემენტებს. თუ a[i] უდრის X-ს, პირდაპირ ვბეჭდავთ ინდექსს. იმისთვის, რომ ამოცანის პირობა დავიცვათ და თუ არაფერია ნაპოვნი, -1 დავბეჭდოთ, ერთი 'cnt' ცვლადი დავამატე, რომელიც მინიშნებას გვაძლევს, საერთოდ ვიპოვეთ თუ არა რამე.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- ინდექსები ისედაც ზრდადობით მოდის, თუ მარცხნიდან მარჯვნივ ჩავუყვებით მასივს, ამიტომ სორტირებაზე ფიქრი არაა საჭირო.
- cnt ცვლადი გვეხმარება მარტივად გავიგოთ, შესრულდა თუ არა პირობა ციკლის დასრულების შემდეგ.

## ხშირი შეცდომები

- ციკლის შიგნით -1-ის დაბეჭდვა, რაც გამოიწვევს იმას, რომ ყოველ ჯერზე, როცა რიცხვი არ ემთხვევა, ეკრანზე მინუს ერთები გამოვა.
