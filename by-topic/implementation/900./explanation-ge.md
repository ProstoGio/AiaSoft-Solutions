# განმარტება — 900. ორი მარტოხელა

## მიდგომა

რადგან მასივში მხოლოდ ორი რიცხვია უნიკალური, ხოლო დანარჩენი წყვილებად გვხვდება, სორტირება საუკეთესო ვარიანტია. დალაგების შემდეგ მსგავსი რიცხვები გვერდიგვერდ დადგება და ის 'მარტოხელა' ელემენტები ადვილად გამოსაყოფი ხდება. უბრალოდ ვამოწმებ თითოეულ ინდექსზე: თუ მარცხნივაც და მარჯვნივაც სხვადასხვა რიცხვია, ესე იგი ის ელემენტი ჩვენი საძიებელია. ბოლო და პირველი ელემენტები ცალკე ამოვიღე პირობებში, რომ 'out of bounds' არ მიმეღო.

## სირთულე

| | |
|-|--|
| დრო | O(n log n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- სორტირება ყველაფერს აადვილებს, რადგან წყვილებს ერთმანეთის გვერდით აყენებს.
- თუ ელემენტი არც წინას უდრის და არც მომდევნოს, ის უეჭველი 'მარტოხელაა'.

## ხშირი შეცდომები

- მასივის დასაწყისისა და ბოლო ელემენტის შემოწმებისას ინდექსების არევა.
