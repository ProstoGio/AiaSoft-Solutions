# Problem Statement — 788. ვინ დგას შუაში?

> Copied from [AiaSoft.ge #788](https://aiasoft.ge/problem/788)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია მასივი, გამობეჭდეთ მასივის შუაში მდგარი ელემენტი, თუკი ელემენტების რაოდენობა კენტია.
თუ მასივის ელემენტების რაოდენობა ლუწია, მაშინ გამობეჭდეთ შუაში მდგომი ორი მეზობელი ელემენტი.

## Input

შემომავალი ფაილის პირველ სტრიქონში ჩაწერილია მასივის ელემენტების რაოდენობა n, 1 <= n <=20.
მეორე სტრიქონი შეიცავს მასივის ელმენტებს ak, -100 <= ak <=100, 0<=k<n.

## Output

გამობეჭდეთ შუაში მდგომი ელემენტი, თუ ელემენტების რაოდენობა კენტია.
წინააღმდეგ შემთხვევაში შუაში მდგომი ორი მეზობელი ელემენტი.
