# Problem Statement — 251. მასივის სეგმენტის შემობრუნება - 1

> Copied from [AiaSoft.ge #251](https://aiasoft.ge/problem/251)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 128 MB

## Task

მოცემულია N წევრიანი მასივი. შემოსულ მასივში შემოაბრუნეთ სეგმენტი [a, b]. იხილეთ ნიმუშები.

## Input

პირველ ხაზზე შემოდის ერთი მთელი რიცხვი: N - მასივის წევრების რაოდენობა (1 <= N <= 20)
მეორე ხაზზე შემოდის N ცალი მთელი რიცხვი: x1, x2, ..., xn - მასივის წევრები (1 <= xi <= 100)
მესამე ხაზზე შემოდის ორი მთელი რიცხვი: a და b - შემოსაბრუნებელი სეგმენტის საწყისი და ბოლო ინდექსი. (1 <= a <= b <= N)

## Output

შეაბრუნეთ მოთხოვნილი სეგმენტი და ერთ ხაზზე გამობეჭდეთ მიღებული მასივი.
