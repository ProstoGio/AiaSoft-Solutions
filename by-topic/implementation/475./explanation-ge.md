# განმარტება — 475. ორის ხარისხების ჯამი

## მიდგომა

პირველივე შეხედვით ხვდები, რომ 2^10^6-ს ვერცერთი standard ტიპი ვერ დაიტიევს, ამიტომ პირდაპირი ჯამი გამოირიცხა. სინამდვილეში ეს არის ჩვეულებრივი ორობითი სისტემის მიმატება: როცა ორი ცალი 2^i გვაქვს, ისინი ავტომატურად გადადიან 2^(i+1)-ში. მთელი ხრიკი იმაშია, რომ გავაკეთოთ ფრეკვენსი არეი (frequency array) და 'კარი' (carry) გავატაროთ დაბლიდან ზემოთ, სანამ ყველა პოზიციაზე 0 ან 1 არ დარჩება.

## სირთულე

| | |
|-|--|
| დრო | O(N + max(ai)) |
| მეხსიერება | O(max(ai)) |

## მთავარი დაკვირვებები

- freq[i]-ში ვინახავთ რამდენი 2^i გვხვდება, მერე კი მარტივი დივიზიონით ვანაწილებთ ზედა ბიტებზე.
- ბოლოში როცა ყველგან 0 ან 1 გვექნება, უბრალოდ მასივის ბოლოდან თავისკენ გამოგვაქვს ბიტები და პასუხიც მზადაა.

## ხშირი შეცდომები

- იმის დავიწყება, რომ კარი (carry) შესაძლოა მაქსიმალურ შეყვანილ რიცხვზე მეტ პოზიციაზე გადავიდეს.
- ოპტიმიზაციისთვის ზედმეტი სორტირების გამოყენება, რაც აქ საერთოდ არ გვჭირდება.

## ნაბიჯ-ნაბიჯ ახსნა

მაგალითად თუ გვაქვს [0, 0], freq[0] გახდება 2. 2/2 არის 1, ანუ 0-იანი პოზიცია გახდება 0, ხოლო 1-ლი პოზიცია გახდება 1. შედეგად მივიღებთ 2^1, ანუ 10.
