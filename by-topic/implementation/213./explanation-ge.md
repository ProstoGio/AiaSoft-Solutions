# განმარტება — 213. სიმბოლოს გარდაქმნა

## მიდგომა

ამ ამოცანის მთავარი ხრიკი ASCII ტაბულის ცოდნაა. ლათინური დიდი ასოები და პატარა ასოები ერთმანეთისგან ზუსტად 32 ერთეულით განსხვავდებიან. ამიტომ, მაგივრად იმისა, რომ სტანდარტული ბიბლიოთეკების ფუნქციები ვეძებო, პირდაპირ ვუმატებ 32-ს თითოეულ სიმბოლოს. ეს არის ყველაზე მარტივი და ეფექტური გზა ამის გასაკეთებლად.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- ASCII-ში დიდი 'A' არის 65, ხოლო პატარა 'a' არის 97, სხვაობა 32-ია.
- char-ის მნიშვნელობებზე არითმეტიკული მოქმედებების შესრულება C++-ში სრულიად მისაღები და სწრაფია.

## ხშირი შეცდომები

- პასუხის გამოტანისას დავიწყება იმისა, რომ სიმბოლოები ერთმანეთისგან ჰარითაა დაშორებული.
