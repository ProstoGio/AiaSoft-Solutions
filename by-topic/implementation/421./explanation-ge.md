# განმარტება — 421. ტურისტი

## მიდგომა

ამ ამოცანის ამოხსნა ყველაზე მარტივად სიმულაციით გამოდის. უბრალოდ ვინახავთ გავლილ მანძილს და ყოველდღიურ ნამატს. while ციკლით ვამოწმებთ, სანამ გავლილი მანძილი N-ზე ნაკლებია, მანამდე ვაგრძელებთ მოძრაობას. ყოველ ნაბიჯზე ვზრდით დღეების რაოდენობას და ვამატებთ ახალ მანძილს, რომელიც წინა დღის მანძილს + B-ს უდრის.

## სირთულე

| | |
|-|--|
| დრო | O(sqrt(N)) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- ყოველდღიური მანძილი არის არითმეტიკული პროგრესია.
- რადგან N არის 10^9-მდე, ციკლი საკმარისად სწრაფად მიაღწევს პასუხს.

## ხშირი შეცდომები

- N-ის ტიპად int-ის გამოყენება, თუ მანძილი ძალიან დიდი იქნება (ჯობია long long გამოვიყენოთ).
- დღეების მთვლელის არასწორი ინიციალიზაცია.

## ნაბიჯ-ნაბიჯ ახსნა

ვიწყებთ პირველი დღით, სადაც გავიარეთ A მანძილი. ციკლში ვამოწმებთ, გვაქვს თუ არა საჭირო რაოდენობა. თუ არა, ვამატებთ შემდეგი დღის დისტანციას (წინა დღე + B) და ვზრდით დღეების მთვლელს 1-ით.
