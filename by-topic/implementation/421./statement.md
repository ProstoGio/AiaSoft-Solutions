# Problem Statement — 421. ტურისტი

> Copied from [AiaSoft.ge #421](https://aiasoft.ge/problem/421)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ტურისტი ერთი ქალაქიდან მეორე ქალაქში წასასვლელად ემზადება. ქალაქებს შორის მანძილი N კილომეტრია. ტურისტი პირველ დღეს A კილომეტრს გადის, ხოლო ყოველ შემდეგ დღეს B კილომეტრით მეტს, ვიდრე წინა დღეს. დაეხმარეთ ტურისტს იმის დადგენაში თუ მერამდენე დღეს მიაღწევს დანიშნულების ადგილს.

## Input

პირველ სტრიქონზე შემოდის N - ქალაქებს შორის მანძილი.
1<=N<=10⁹
მეორე სტრიქონზე შემოდის ჰარით გამოყოფილი A და B - პირველ დღეს გავლილი მანძილი და ყოველი შემდეგი დღის ნამატი.
1<=A<=10⁴ 
0<=B<=10⁴

## Output

გამოიტანეთ ერთი რიცხვი - მერამდენე დღეს მიაღწევს ტურისტი დანიშნულების ადგილს.
