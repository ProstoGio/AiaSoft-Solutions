# განმარტება — 123. სტრიქონის შეტრიალება

## მიდგომა

ამოცანაში რეკურსიის გამოყენებაა მოთხოვნილი, მაგრამ სიმართლე რომ ვთქვა, მარტივი ციკლით გაკეთება უფრო ნაკლებ თავის ტკივილს იწვევს. უბრალოდ ვიღებ სტრიქონის ბოლო ინდექსს და ნაბიჯ-ნაბიჯ უკუღმა ვკითხულობ სიმბოლოებს. ასე ახალი სტრიქონის შექმნა და მასში მნიშვნელობების ჩაყრა ყველაზე სწრაფი და გასაგები გზაა.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- მთავარია ინდექსებზე ფრთხილად ვიყოთ, `s.size()-1`-დან დაწყებაა საჭირო, რომ მეხსიერების მისამართს არ გავცდეთ.
- თუ რეკურსია გინდათ, იგივე ლოგიკაა — უბრალოდ ფუნქციამ სტრიქონის ბოლო სიმბოლო უნდა დააბრუნოს და მერე თავის თავს შეუძახოს დარჩენილი ნაწილით.

## ხშირი შეცდომები

- ციკლის პირობაში `<=` ნიშნის გამოყენება, რაც სტრიქონის საზღვრებს არღვევს.
