# განმარტება — 865. კენტი რიცხვების ჯამი

## მიდგომა

ამ ამოცანის ამოხსნა ყველაზე მარტივი გზით მინდოდა. უბრალოდ ვიღებ ცვლადს ჯამისთვის, შევდივარ ლუპში და თითოეულ რიცხვს ვამოწმებ 'if(a[i]%2!=0)'-ით. თუ კენტია, პირდაპირ ვუმატებ ჯამს და მორჩა, ზედმეტი სირთულეების გარეშე.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- მასივის შენახვა რეალურად არ გჭირდება, პირდაპირ შეყვანისას შეგიძლია აჯამო კენტი რიცხვები.
- კენტი რიცხვის გამოსაცნობად ყველაზე სწრაფი მეთოდი მოდულოთი შემოწმებაა.

## ხშირი შეცდომები

- ჯამის ცვლადისთვის საწყისი მნიშვნელობის (0) მინიჭების დავიწყება.
