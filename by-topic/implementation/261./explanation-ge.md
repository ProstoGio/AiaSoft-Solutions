# განმარტება — 261. მეზობლების მატრიცა

## მიდგომა

სინამდვილეში ეს უბრალოდ წვეროებს შორის კავშირების ცხრილად გადაწერაა. რადგან გრაფი არაორიენტირებულია, როცა ვიღებ წიბოს v და u წვეროებს შორის, მატრიცაში ორივე უჯრა უნდა ავანთო: g[v][u] და g[u][v]. N ძალიან პატარაა, მაქსიმუმ 20, ამიტომ მარტივი NxN მატრიცა იდეალური გამოსავალია. თავიდან ყველაფერს ნულიანად ვტოვებ, შეტანისას კი მხოლოდ იმ უჯრებს ვცვლი, სადაც წიბო არსებობს.

## სირთულე

| | |
|-|--|
| დრო | O(N² + M) |
| მეხსიერება | O(N²) |

## მთავარი დაკვირვებები

- რადგან გრაფი არაორიენტირებულია, მატრიცა ყოველთვის სიმეტრიული გამოდის მთავარი დიაგონალის მიმართ.
- მასივის ზომა N+1-ზე რომ ავიღო, 1-დან დაწყებული ინდექსებით პირდაპირ ვმუშაობ და არ ვწვალობ i-1 და j-1-ებით.

## ხშირი შეცდომები

- დავიწყება იმისა, რომ არაორიენტირებული გრაფის შემთხვევაში ორივე მიმართულება (v-დან u-სკენ და u-დან v-სკენ) უნდა აღინიშნოს.
