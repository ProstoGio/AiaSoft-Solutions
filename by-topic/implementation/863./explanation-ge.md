# განმარტება — 863. დაითვალე კენტები

## მიდგომა

აქ არაფერია რთული, პირდაპირ ვიკითხულობთ რიცხვებს და ვამოწმებთ. მასივი სინამდვილეში არც გვჭირდება, იმიტომ რომ კენტი რიცხვების რაოდენობა მხოლოდ ერთი მთელი ცვლადით შეგვიძლია დავითვალოთ. უბრალოდ 'cnt' ცვლადს დავამატებთ ერთს, როცა შემოტანილი რიცხვი 2-ზე გაყოფისას ნაშთს ტოვებს.

- ნაშთის ოპერატორი (%) ყველაზე მარტივი გზაა კენტობის შესამოწმებლად.
