# Problem Statement — 249. მასივის ელემენტების წანაცვლება - 1

> Copied from [AiaSoft.ge #249](https://aiasoft.ge/problem/249)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

მოცემულია N წევრიანი მასივი. წაანაცვლეთ მასივის ელემენტები მარცხნივ ერთხელ.

* `bმარცხნივ წანაცვლება` - ვუწოდოთ შემდეგ მოქმედებას მასივზე: პირველი ელემენტი გადავიტანოთ ბოლოში, დანარჩენები კი გადმოვაჩოჩოთ მარცხნივ 1 უჯრით.

მაგალითად, წავანაცვლოთ მარცხნივ მასივი: 1 2 3 4,  მივიღებთ: 2 3 4 1.

## Input

პირველ ხაზზე შემოდის ერთი მთელი რიცხვი: N - მასივის სიგრძე (1 <= N <= 100)
მეორე ხაზზე შემოდის N ცალი მთელი რიცხვი: a1, a2, ..., an - მასივის წევრები. (1 <= ai <= 1000)

## Output

შეასრულეთ მარცხნივ წანაცვლების ოპერაცია და გამობეჭდეთ მიღებული მასივი ერთ ხაზზე.
