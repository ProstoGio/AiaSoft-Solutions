# განმარტება — 249. მასივის ელემენტების წანაცვლება - 1

## მიდგომა

ამოცანა მოითხოვს პირველი ელემენტის ბოლოში გადატანას და დანარჩენების მარცხნივ გადმოწევას. იმის ნაცვლად, რომ დამხმარე მასივი შევქმნა, უბრალოდ გვერდიგვერდ მდგომ ელემენტებს ვანაცვლებ მარცხნიდან მარჯვნივ. ასე პირველი ელემენტი 'მიცოცავს' ბოლო პოზიციამდე და ყველა დანარჩენი თავისით გადაინაცვლებს მარცხნივ.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- მიმდევრობითი swap-ები საშუალებას გვაძლევს წანაცვლება დამატებითი მეხსიერების გარეშე გავაკეთოთ.
- ციკლი n-1-მდეა საკმარისი, რადგან ყოველი ნაბიჯით პირველი ელემენტი ერთი პოზიციით მარჯვნივ ინაცვლებს.

## ხშირი შეცდომები

- ახალი მასივის ზედმეტად გამოყოფა.
- ციკლის არასწორი საზღვრები, რამაც შეიძლება 'out of bounds' გამოიწვიოს.

## ნაბიჯ-ნაბიჯ ახსნა

თავიდან ვკითხულობ N-ს და ვავსებ ვექტორს. ციკლი მიდის 0-დან n-2-მდე და ყოველ ნაბიჯზე ვუცვლი ადგილს a[i]-ს და a[i+1]-ს. თუ გვაქვს [1, 2, 3, 4], პირველი swap-ის შემდეგ იქნება [2, 1, 3, 4], მეორეს მერე [2, 3, 1, 4] და ბოლოს [2, 3, 4, 1].
