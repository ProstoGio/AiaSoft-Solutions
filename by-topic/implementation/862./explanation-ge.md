# განმარტება — 862. ნახევარმატრიცა

## მიდგომა

ამოცანა საკმაოდ მარტივია: კვადრატულ მატრიცაში მთავარი დიაგონალი და მის ზემოთ არსებული ელემენტები უნდა შევავსოთ x-ით, ხოლო დანარჩენი ნულები დავტოვოთ. ყველაზე მოხერხებული გზაა, ჯერ მთელი მატრიცა ნულებით გავავსოთ და მერე მხოლოდ საჭირო ნაწილს მივხედოთ. თუ შიდა ციკლს დავიწყებთ i ინდექსიდან, ავტომატურად მოვიცავთ მთავარ დიაგონალს და მის მარჯვნივ არსებულ ელემენტებს, რაც ზუსტად ისაა, რაც გვინდა.

## სირთულე

| | |
|-|--|
| დრო | O(n²) |
| მეხსიერება | O(n²) |

## მთავარი დაკვირვებები

- შიდა ციკლის j=i-დან დაწყება ყველაზე 'სუფთა' მეთოდია, რომ დამატებითი if-ების გარეშე შეავსო დიაგონალი და ზედა ნაწილი.
- თავიდანვე ნულებით შევსება ზოგავს დროს იმაზე ფიქრში, თუ სად ჩავწეროთ 0 და სად x.

## ხშირი შეცდომები

- ციკლის საზღვრების არევა, მაგალითად j = i+1-ით დაწყება, რაც დიაგონალს ცარიელს დატოვებს.
