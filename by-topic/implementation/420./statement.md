# Problem Statement — 420. საფეხურები

> Copied from [AiaSoft.ge #420](https://aiasoft.ge/problem/420)

**Time limit:** 0.5s &nbsp;|&nbsp; **Memory limit:** 128 MB

## Task

ჩვენთვის ცნობილია რომ ახალი კომპანიის აიაბილდინგ-ის მიერ აშენებულ კორპუსებში სართულებს შორის N რაოდენობის საფეხურია. ჩვენი მიზანია გავიგოთ, სულ რამდენ საფეხურს გავივლით თუ a სართულიდან ავალთ b სართულზე და შემდეგ ჩამოვალთ c სართულზე.

## Input

შემოდის სამი ნატურალური რიცხვი N,a,b,c სართულებს შორის კიბეების რაოდენობა, საწყისი სართული, შუალედური სართული და საბოლოო სართული.
1<=N<=5*10⁵
1<=a,b,c<=4000

## Output

გამოიტანეთ ერთი რიცხვი - საფუხერების ჯამური რაოდენობა.
