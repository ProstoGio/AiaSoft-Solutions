# განმარტება — 420. საფეხურები

## მიდგომა

ამოცანა საკმაოდ მარტივია: უბრალოდ უნდა დავთვალოთ, სულ რამდენი სართული გავიარეთ. ჯერ ავდივართ a-დან b-მდე, მერე ჩავდივართ b-დან c-მდე. ამიტომ, ჯამური სართულების რაოდენობა არის |b-a| + |b-c|. ვინაიდან თითო სართულს შორის N საფეხურია, ამ ჯამს ვამრავლებთ N-ზე და ეგაა.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- ჯამური მანძილი არის (abs(b-a) + abs(b-c)) * N
- არ დაგავიწყდეთ, რომ სართულების სხვაობა აუცილებლად დადებითი უნდა იყოს

## ხშირი შეცდომები

- სართულების სხვაობის უარყოფითად დატოვება
- მონაცემთა ტიპების არასწორი შერჩევა

## ნაბიჯ-ნაბიჯ ახსნა

შემოგვაქვს N, a, b, c. ვთვლით სართულების ჯამურ სხვაობას (b-a) + (b-c). ამას ვამრავლებთ N-ზე და გამოგვაქვს შედეგი.
