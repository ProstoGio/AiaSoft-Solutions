# განმარტება — 316. ნაყინები

## მიდგომა

ეს ამოცანა კლასიკური 'ფროbeniusის მონეტების' პრობლემაა, სადაც ვამოწმებთ, N შეიძლება თუ არა წარმოვადგინოთ 3-იანებისა და 5-იანების ჯამის სახით. თავიდან პირდაპირ ვამოწმებ 3-ზე, 5-ზე და 15-ზე გაყოფადობას, რადგან ეს ყველაზე სწრაფი გზაა. თუ ამით ვერ გადავწყვიტეთ, მაშინ ციკლით ვცდი რამდენჯერმე 5-ის გამოკლებას, სანამ დარჩენილი თანხა 3-ზე არ გაიყოფა. სინამდვილეში, მათემატიკურად ეს ამოცანა 8-ზე მეტი ყველა რიცხვისთვის 'YES'-ს გვაძლევს, მაგრამ loop-იც მშვენივრად მუშაობს ამ შეზღუდვებში.

## სირთულე

| | |
|-|--|
| დრო | O(N) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- 3-ისა და 5-ის კომბინაციით ნებისმიერი 8-ზე მეტი მთელი რიცხვის აწყობა შესაძლებელია.
- N=10^9-ისთვის ციკლი ცოტა ნელია, ამიტომ ჯობია პირდაპირ მათემატიკური შემოწმება (მაგალითად, if N > 7 მაშინ YES).

## ხშირი შეცდომები

- პატარა რიცხვების (1, 2, 4, 7) უგულებელყოფა, რაც არასწორ პასუხს გვაძლევს.
