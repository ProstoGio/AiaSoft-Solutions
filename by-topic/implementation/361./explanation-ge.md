# განმარტება — 361. ციფრთა ჯამი

## მიდგომა

ციფრთა ჯამის ამოსაღებად ჩვეულებრივი `while` ციკლის მაგივრად რეკურსია გამოვიყენე, რადგან კოდი უფრო მოკლე და ელეგანტური გამოდის. ლოგიკა მარტივია: რიცხვს ვყოფთ 10-ზე, სანამ მარტო ბოლო ციფრი არ დარჩება, თან პარალელურად `modulo`-ს საშუალებით ჯამს ვითვლით. `calc(x/10) + x%10` ზუსტად ამას აკეთებს – ჯერ რიცხვს ამცირებს, ბოლოს კი უკან დაბრუნებისას ციფრებს ერთმანეთს უმატებს.

## სირთულე

| | |
|-|--|
| დრო | O(log n) |
| მეხსიერება | O(log n) |

## მთავარი დაკვირვებები

- x % 10 ყოველთვის გვაძლევს ბოლო ციფრს, ხოლო x / 10 რიცხვს აპატარავებს ერთი თანრიგით.
- რეკურსიაში ბაზური შემთხვევა x < 10 აუცილებელია, თორემ stack overflow-ს მივიღებთ.

## ხშირი შეცდომები

- ბაზური პირობის დავიწყება, რაც გაუთავებელ რეკურსიას იწვევს.
