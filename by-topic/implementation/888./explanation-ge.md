# განმარტება — 888. πბონაჩის მიმდევრობა

## მიდგომა

ეს არის ჩვეულებრივი ფიბონაჩის მიმდევრობა, უბრალოდ 1-ის ნაცვლად 3.14-ით ვიწყებთ. რადგან n მაქსიმუმ 35-ია, საერთოდ არ არის საჭირო რამე ოპტიმიზაცია, პირდაპირ მასივში ვინახავთ მნიშვნელობებს და ვუმატებთ წინა ორ წევრს.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- ფორმულა იგივეა: d[i] = d[i-1] + d[i-2], უბრალოდ ბაზისური შემთხვევები გვაქვს 0 და 3.14.
- fixed და setprecision(2) აუცილებელია, რომ პასუხი ზუსტად ორ ნიშნამდე დაიტიპოს.

## ხშირი შეცდომები

- თუ setprecision-ს დაგავიწყდა fixed, მაშინ პასუხს შეიძლება არასწორი ფორმატით გიჩვენებდეს.
