# განმარტება — 118. ჯვარი

## მიდგომა

აქ მთავარი იდეა ისაა, რომ ცენტრიდან ოთხივე მიმართულებით გავიაროთ გზა მანამ, სანამ '1'-ებს ვხედავთ. რადგან გვაქვს უფლება, ერთი '0' შევცვალოთ '1'-ით, ეს ნიშნავს, რომ შეგვიძლია ან ცენტრი გავხადოთ '1', თუ ის '0'-ია, ან ერთ-ერთი მიმართულების პირველივე ნულოვანი უჯრა 'შევაკეთოთ'. კოდში ამას ოთხივე მხარეს ცალ-ცალკე ვამოწმებ და ვუყურებ, ხომ არ გამოვიყენე უკვე ეს უფლება.

## სირთულე

| | |
|-|--|
| დრო | O(T * (M + N)) |
| მეხსიერება | O(M * N) |

## მთავარი დაკვირვებები

- თუ ცენტრი '0'-ია, ავტომატურად ის უნდა შეიცვალოს, სხვა შემთხვევაში ჯვარი ვერ აიწყობა.
- ჯვრის ფორმის შენარჩუნება აუცილებელია, ამიტომ კუთხეებში მდგარი ჯვრები თავიდანვე გამოსარიცხია.

## ხშირი შეცდომები

- საზღვრების შემოწმების დავიწყება, რამაც შეიძლება 'Index Out of Bounds' გამოიწვიოს.
- ერთზე მეტი '0'-ის გაერთიანების მცდელობა ერთსა და იმავე ჯვარში.

## ნაბიჯ-ნაბიჯ ახსნა

თითოეული მოთხოვნისას ვამოწმებ ცენტრის გარშემო ოთხ მიმართულებას. თუ მაგალითად (x+1, y) არის '0', ვამოწმებ, შემიძლია თუ არა ამის ჩათვლა. მთავარია, რომ ჯვრის განმარტება პირობის მიხედვით უნდა სრულდებოდეს: ანუ ცენტრის ორივე მხარეს მაინც უნდა იყოს რაღაც მონაკვეთი.
