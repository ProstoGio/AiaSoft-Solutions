# განმარტება — 866. ლუწი რიცხვების ჯამი

## მიდგომა

ამ ამოცანაში სტრატეგია ძალიან მარტივია: არ გვაქვს საჭიროება, რომ მთლიანი მასივი შევინახოთ მეხსიერებაში. უბრალოდ ვკითხულობთ რიცხვებს სათითაოდ და თუ ლუწია, პირდაპირ ვუმატებთ ჯამს. კოდი იმდენად პატარაა, რომ ზედმეტი სირთულეების გარეშე, პირდაპირ loop-ში ვაგვარებ ყველაფერს.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- არ არის აუცილებელი 100-ელემენტიანი მასივის დეკლარაცია, რადგან ჯამის დათვლა პარალელურად შეგვიძლია.
- ლუწობის შესამოწმებლად x % 2 == 0 პირობა სტანდარტული და ყველაზე ეფექტური გზაა.

## ხშირი შეცდომები

- ცვლადის (cnt) დაუინიციალიზებლობა, რაც ხშირად იწვევს შემთხვევითი რიცხვების მიმატებას შედეგზე.
