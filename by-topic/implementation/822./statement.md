# Problem Statement — 822. ნულების ამოგდება

> Copied from [AiaSoft.ge #822](https://aiasoft.ge/problem/822)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია ნატურალური რიცხვი, რომლის ციფრების რაოდენობა არ აღემატება 9-ს. 
თქვენი დავალებაა მოცემულ რიცხვში ყველა ნულის ტოლი ციფრი წაშალოთ და შედეგი გამობეჭდოთ ეკრანზე. ნულების ამოგდების შემდეგ უნდა იყოს დაცული თავდაპირველ რიცხვში მოცემული სხვა (ნულისგან განსხვავებული) რიცხვების რიგითობა.

## Input

ერთადერთ ხაზზე მოცემულია ერთი ნატურალური რიცხვი N, რომელშიც ციფრთა რაოდენობა არ აღემატება 9-ს.
გარანტირებულია რომ N არ უდრის 0-ს.

## Output

ერთ ხაზზე გამობეჭდეთ მიღებული რიცხვი.
