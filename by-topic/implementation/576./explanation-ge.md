# განმარტება — 576. ლუწი ინდექსები

## მიდგომა

ამ ამოცანის იდეა არის მარტივი: მასივში 0-დან იწყება ინდექსაცია, ამიტომ ლუწი ინდექსები იქნება 0, 2, 4 და ა.შ. უბრალოდ ვკითხულობთ n-ს, მერე მთლიან მასივს, და ბოლოს `for` ციკლით ვამოწმებთ, `i % 2 == 0` არის თუ არა. თუ პირობა სრულდება, ეგრევე ვბეჭდავთ იმ ელემენტს.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- არ დაგავიწყდეთ, რომ C++-ში ნული ლუწ რიცხვად ითვლება, შესაბამისად მასივის პირველი ელემენტიც ლუწ ინდექსზეა.
- ინდექსების დათვლა 0-დან რომ ვიწყებთ, მთელ ლოგიკას ძალიან ამარტივებს.

## ხშირი შეცდომები

- ხანდახან ხალხს ერევა და 1-დან იწყებენ ათვლას, რის გამოც კენტი ინდექსები გამოდის ლუწი.
