# განმარტება — 140. მეორე ნომერი ცელქობა

## მიდგომა

ამ ამოცანაში მთავარი ისაა, რომ არ აგერიოთ თავბრუ ცელქი ბავშვების წესებით. მასწავლებელმა ხომ თავიდანვე სიმაღლის მიხედვით დააყენა ბავშვები, ანუ ჩვენი მიზანია უბრალოდ აღვადგინოთ ეს დალაგებული მდგომარეობა. არ აქვს მნიშვნელობა ვინ ვის გაუცვალა ადგილი, თუ ბოლოს მაინც ზრდადობით უნდა დალაგდეს, მაშინ `sort()`-ი არის ყველაზე მარტივი და სწორი გზა.

## სირთულე

| | |
|-|--|
| დრო | O(n log n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- ნუ ეცდებით ამ 'ცელქობის' სიმულაციას, ეს უბრალოდ სატყუარაა.
- თუ მასივი დალაგებული უნდა იყოს, `std::sort`-ი ყოველთვის საუკეთესო არჩევანია.

## ხშირი შეცდომები

- სიმულაციის წერა მაშინ, როცა პირდაპირი დალაგება საკმარისია.
