# განმარტება — 861. წაშალეთ სიმბოლო

## მიდგომა

აქ მთავარი იდეა ისაა, რომ პირდაპირ სტრიქონში ვეძებთ K სიმბოლოს და თუ ვიპოვეთ, ეგრევე ვშლით. erase ფუნქციას როცა იყენებ, სტრიქონი იკუმშება და შემდეგი ელემენტები მარცხნივ ინაცვლებს. ამიტომ, `i` ცვლადს რომ არ გავაოროთ, აუცილებლად `i--` უნდა მივაწეროთ წაშლის მერე, თორემ ალგორითმი ერთ პოზიციას გამოტოვებს.

## სირთულე

| | |
|-|--|
| დრო | O(n²) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- რადგან `erase` ფუნქცია O(n)-ია, ციკლში ჩასმული ეს მეთოდი O(n²)-ს გვაძლევს, რაც ამ ზომის დატასთვის იდეალურია.
- მნიშვნელოვანია, რომ წაშლის მერე ინდექსი უკან დავწიოთ, თორემ გამოგვრჩება მომდევნო სიმბოლო, რომელიც წაშლილის ადგილზე გადმოხტა.

## ხშირი შეცდომები

- წაშლის შემდეგ `i`-ს არ დაკლება, რის გამოც კოდი ზოგიერთ სიმბოლოს უბრალოდ გვერდს აუვლის.
