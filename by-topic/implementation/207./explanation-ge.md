# განმარტება — 207. ლებრონი და კალათბურთი

## მიდგომა

ამ ამოცანაში მთელი „მაგია“ იმაშია, რომ ქულები სწორად დავთვალოთ. კალათბურთში ერთქულიანი, ორქულიანი და სამქულიანი განსხვავებული წონისაა, ამიტომ უბრალოდ მათი ჯამი არ გვაწყობს. ცალ-ცალკე ვანგარიშობ ლებრონის გუნდის ქულებს `a + b*2 + c*3` ფორმულით და მოწინააღმდეგისას `x + y*2 + z*3`-ით. მერე ერთი მარტივი შედარებაა დარჩენილი და ეგაა, მეტი არაფერი უნდა.
