# განმარტება — 607. რიცხვების XOR - 1

## მიდგომა

N მაქსიმუმ 10⁵-ია, რაც იმას ნიშნავს, რომ მარტივი ციკლით ამოხსნა თავისუფლად გაქაჩავს. უბრალოდ ვინახავ ცვლადს, რომელიც 1-ით იწყება და 2-დან N-ის ჩათვლით ყველა რიცხვს 'ვაქსორებ' (XOR) ამ ცვლადზე. სულ ესაა, მეტი არაფერი სჭირდება, რადგან დროის ლიმიტი 1 წამია და ეს მიდგომა მილიწამებში მორჩება.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- 10⁵-ისთვის O(n) სირთულე იდეალურია, ტყუილად არ ღირს მათემატიკური ფორმულების ძებნა თუ ისედაც მუშაობს.
- XOR ოპერაცია არის ასოციაციური, ანუ თანმიმდევრობას მნიშვნელობა არ აქვს.

## ხშირი შეცდომები

- შესაძლოა ვინმემ პასუხი 0-ით დააინიციალიზოს, რაც მაშინვე გააფუჭებს შედეგს.
