# Problem Statement — 409. გუგლიკო და შერჩევა

> Copied from [AiaSoft.ge #409](https://aiasoft.ge/problem/409)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

გუგლიკო კომპანია გეგლსოფტის შერჩევაზე მივიდა. 
გეგლსოფტს ერთი ვაკანტური ადგილი აქვს.
გეგლსოფტი ზოგადად  შერჩევების უცნაური ფორმებით გამოირჩევა. 
ამჯერად შერჩევის წესი ასეთია: შერჩევაზე გამოცხადებულ n რაოდენობის კანდიდატს შესთავაზეს ერთი ნატურალური ნომრის არჩევა 1-დან n-ის ჩათვლით (ერთი ნომერი უნდა აირჩიოს მხოლოდ ერთმა კანდიდატმა). შემდეგ კანდიდატები ჩაამწკრივეს არჩეული ნომრის მიხედვით და გამოუცხადეს საოცარი გადაწყვეტილება, რომ კენტი ნომრის მქონე კანდიდატები შერჩევიდან მოხსნილნი არიან!!. 
ამის შემდე უფრო მეტი საოცრება ჩაიდინეს: კანდიდატებს ჩამოურიგეს ახალი ნომრები (არ არჩევინეს!) ისე, რომ მწკრივიდან გამოსვლის უფლება და ერთმანეთს შორის ადგილის გაცვლის უფლება არ მისცეს. ახალი ნომრები დაურიგეს ზრდადობით, 1,2,3,4, და ა.შ. ნომრის გამოუტოვებლად. ამის შემდეგ გაიმეორეს იგივე: კენტი ნომრის მქონენი შერჩევიდან დაითხოვეს. მერე ისევ ახალი ნომრები დაურიგეს იგივე წესებით და ასე გააგრძელეს, სანამ მხოლოდ ერთი კანდიდატი არ დარჩა. აი, ეს კანდიდატი აიყვანეს სამსახურში.
და ეს იყო ბედნიერი გუგლიკო, რადგან ჯადოსნური საწყისი ნომერი ჰქონდა. 
მიბაძეთ გუგლიკოს!! 
თქვენი პროგრამის მოვალეობაა გუგლიკოს საწყისი ნომრის გამოცნობა.

## Input

შემოდის ერთი ნატურალური რიცხვი n, 1<=n<=10¹⁸ - შერჩევაში მონაწილე კანდიდატების რაოდენობა.

## Output

გამოიტანეთ ერთი ნატურალური რიცხვი - გუგლიკოს საწყისი ნომერი.
