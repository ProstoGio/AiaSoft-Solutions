# განმარტება — 606. დათვალე ერთები

## მიდგომა

ამ ამოცანის მთავარი მუღამი არის ორობით სისტემაში ერთიანების სწრაფად დათვლა. ყველაზე მარტივი გზაა C++-ის ჩაშენებული `__builtin_popcount` ფუნქციის გამოყენება, რომელიც პირდაპირ აბრუნებს პასუხს. მე უბრალოდ მასივში შევინახე შედეგები და ბოლოს გამოვიტანე. არაფერი რთული, უბრალოდ სუფთა კოდი.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- `__builtin_popcount` არის ყველაზე ოპტიმალური ვარიანტი, რადგან ის პირდაპირ აპარატურულ დონეზე ასრულებს მოქმედებას.
- რადგან N-ის მაქსიმალური მნიშვნელობა 1000-ია, მეხსიერების პრობლემა არ შეგვექმნება.

## ხშირი შეცდომები

- ხელით წერა ბიტების სათვლელად (while ციკლით), რაც ტყუილი წვალებაა.
