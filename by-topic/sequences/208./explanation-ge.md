# განმარტება — 208. სამკუთხა რიცხვები

## მიდგომა

ეს არის კლასიკური მათემატიკური ამოცანა, სადაც n-ური სამკუთხა რიცხვი 1-დან n-მდე რიცხვების ჯამია. ამის გამოსათვლელად ფორმულა n*(n+1)/2-ია. რადგან n შეიძლება იყოს 5*10^9-მდე, n*(n+1) ჩვეულებრივ 64-ბიტიან ტიპში, ანუ unsigned long long-ში, დარეინჯდება. ამიტომ, შუალედური ნამრავლისთვის გამოვიყენე __int128, რათა overflow ავიცილო თავიდან.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- სამკუთხა რიცხვების მიმდევრობა არითმეტიკული პროგრესია, რომლის ჯამიც n(n+1)/2-ით ითვლება.
- 5*10^9-ის შემთხვევაში, n*(n+1) დაახლოებით 2.5*10^19-ია, რაც unsigned long long-ის მაქსიმუმს (დაახლოებით 1.8*10^19) ცდება.
- __int128 არის საუკეთესო გზა C++-ში ამ მასშტაბის რიცხვებთან სამუშაოდ.

## ხშირი შეცდომები

- უბრალო long long-ის გამოყენება, რაც დიდ შეყვანებზე არასწორ პასუხს მოგცემთ.
- ციკლის გამოყენება, რაც ამ ზომის n-ისთვის Time Limit-ს დაარღვევს.

## ნაბიჯ-ნაბიჯ ახსნა

შემომაქვს n, ვკასტავ მას __int128-ად, რომ ნამრავლი არ გადაივსოს. ვამრავლებ n-ს (n+1)-ზე, ვყოფ ორზე და შედეგი გამომაქვს ეკრანზე. მეტი არაფერია საჭირო.
