# განმარტება — 473. არითმეტიკული ქვემიმდევრობა

## მიდგომა

იდეა ძალიან მარტივია: ჩვენ ვეძებთ ბოლო მომენტს, სადაც პროგრესია ირღვევა. თავიდან ვუშვებთ, რომ პროგრესია დასაწყისიდანვე მოდის, მაგრამ როგორც კი დავინახავთ, რომ მეზობელი ელემენტების სხვაობა (d) შეიცვალა, ვხვდებით, რომ წინა ნაწილი უკვე აღარ გვაინტერესებს. მაგ მომენტში `ind` ცვლადს ვწევთ წინ, რადგან სწორედ მანდ იწყება ახალი 'პოტენციური' არითმეტიკული მიმდევრობა. ბოლო დარჩენილი `ind` იქნება ჩვენი პასუხი.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- ერთი ან ორი ელემენტი ყოველთვის არითმეტიკული პროგრესიაა, ამიტომ ლოგიკა კარგად მუშაობს მცირე ზომის მასივებზეც.
- როცა `a[i+1]-a[i] != d` პირობა სრულდება, ეს ნიშნავს, რომ ძველი პროგრესია დასრულდა და ახალი იწყება.

## ხშირი შეცდომები

- ინდექსების არევა და p-ს ნულოვანი ან არასწორი ინდექსით დაბრუნება (გახსოვდეს, რომ 1-ზე დაფუძნებული ინდექსია საჭირო).
