# Problem Statement — 757. მიმდევრობა

> Copied from [AiaSoft.ge #757](https://aiasoft.ge/problem/757)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მიშო ახლახანს ახალ კლასში გადავიდა და მას აინტერესებს რამდენი ბავშვია ახალ კლასში მასზე დაბალი. მოცემულია N (კლასში ბავშვების რაოდენობა, მიშოს ჩაუთვლელად) ელემენტიანი მიმდევრობა, რომელიც აღწერს კლასში ბავშვების სიმაღლეებს და K - მიშოს სიმაღლე. დათვალეთ რამდენი ბავშვია კლასში მიშოზე დაბალი (ზუსტად მიშოს სიმაღლის ბავშვი არ ითვლება მასზე დაბლად).

## Input

სტანდარტული შეტანის პირველ სტრიქონში ჩაწერილია ორი მთელი დადებითი რიცხვი: N და K - ბავშვების რაოდენობა კლასში და მიშოს სიმაღლე. მეორე სტრიქონი შეიცავს N რაოდენობის მთელ რიცხვს - თითოეული ბავშვის სიმაღლეს.

1<=N<=5*10⁵
1<=K<=10⁹

## Output

სტანდარტული გამოტანის ერთადერთ სტრიქონში უნდა ჩაიწეროს ერთი მთელი რიცხვი - კლასში მიშოზე დაბალი ბავშვების რაოდენობა.
