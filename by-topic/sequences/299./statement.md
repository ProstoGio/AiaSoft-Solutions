# Problem Statement — 299. კიბე და ბურთულა

> Copied from [AiaSoft.ge #299](https://aiasoft.ge/problem/299)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

ბურთულა მოთავსებულია N საფეხურიანი კიბის უმაღლეს საფეხურზე და ნელნელა იწყებს ქვემოთ ვარდნას. ბურთულამ ხტუნვისას შესაძლოა გადაინაცვლოს ერთი, ორი ან სამი საფეხურით დაბლა. ანუ მაგალითად, თუ ახლა ბურთულა იმყოფება მე-8 საფეხურზე, მომდევნო გადახტომით შესაძლოა აღმოჩნდეს მე-7, მე-6 ან მე-5 საფეხურზე.

გამოთვალეთ, სულ რამდენი განსხვავებული გზით შეიძლება ჩამოიაროს ბურთმა კიბეზე.

## Input

შემოდის ერთი მთელი რიცხვი: N - კიბის საფეხურების რაოდენობა. 1 <= N <= 30

## Output

გამოიტანეთ ერთი მთელი რიცხვი, კიბის ჩამოვლის ვარიანტების რაოდენობა.
