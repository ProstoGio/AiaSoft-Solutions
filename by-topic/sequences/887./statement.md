# Problem Statement — 887. πრველი კენტების ჯამი

> Copied from [AiaSoft.ge #887](https://aiasoft.ge/problem/887)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

`bკარლ ფრიდრიხ გაუსი` იყო ყველა დროის ერთ-ერთი უდიდედესი მათემატიკოსი. მან უდიდესი წვლილი შეიტანა მათემატიკის განვითარებაში. არსებობს ლეგენდა, რომ გაუსმა გააოცა სკოლის მასწავლებელი, როდესაც მან აღმოაჩინა პირველი n ნატურალური რიცხვის ჯამის გამოსათვლელი ფორმულა, საკლასო დავალების დროს. მან შეამჩნია კანონზომიერება და დაადგინა, რომ პირველი n ნატურალური რიცხვის ჯამი გამოითვლება ფორმულით (n*(n+1))/2

თქვენ გეძლევათ გაუსისაგან ოდნავ განსხვავებული ამოცანა, იპოვეთ პირველი n `bკენტი` ნატურალური რიცხვის ჯამი

## Input

პირველ ხაზზე შემოდის ნატურალური რიცხვი n
1<=n<=10⁹

## Output

გამობეჭდეთ პირველი n კენტი ნატურალური რიცხვის ჯამი
