# განმარტება — 138. მასივი 1

## მიდგომა

ეს დავალება საკმაოდ მარტივია, არაფერი ზედმეტი. თავიდან მასივს მთლიანად ვკითხულობთ და ვიმახსოვრებთ, რადგან პირველი და ბოლო ელემენტების ადგილების გასაცვლელად დაგვჭირდება ორივე მათგანზე წვდომა. ყველაზე მარტივი გზაა, ერთი ცვლადი ავიღოთ დროებითი მნიშვნელობისთვის, რომ არ დაგვეკარგოს ინფორმაცია გადაწერის დროს.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- პროგრამირებაში 'swap'-ის კლასიკური მეთოდი მუშაობს: დროებით ცვლადში შეინახე ერთი მნიშვნელობა და მერე გადაანაცვლე.
- არ დაგავიწყდეთ, რომ მასივში ინდექსირება 0-დან იწყება, ამიტომ ბოლო ელემენტი ყოველთვის n-1 ინდექსზე ზის.

## ხშირი შეცდომები

- n-1 ინდექსის ნაცვლად n-ის გამოყენება, რაც 'out of bounds' ერორს გამოიწვევს.
