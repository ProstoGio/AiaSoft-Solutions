# Problem Statement — 126. ფიბონაჩის მიმდევრობა 1

> Copied from [AiaSoft.ge #126](https://aiasoft.ge/problem/126)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

რიცხვით მიმდევრობას F1, F2, F3, ... რომლისთვისაც F1=1, F2=1 ხოლო ყოველი i>2  - თვის Fi=Fi-2 + Fi-1 ფიბონაჩის მიმდევრობა ეწოდება. ანუ ფიბონაჩის მიმდევრობას აქვს სახე 1, 1, 2, 3, 5, 8, 13, 21, 34, ...

მოცემულია ნატურალური რიცხვი n. იპოვეთ და გამოიტანეთ Fn

## Input

პირველ ხაზზე შემოდის ნატურალური რიცხვი n
1<=n<=90

## Output

ერთადერთ სტრიქონში გამოიტანეთ ფიბონაჩის მიმდევრობის n - ური წევრი ანუ Fn
