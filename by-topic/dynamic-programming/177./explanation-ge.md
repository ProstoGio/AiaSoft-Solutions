# განმარტება — 177. ჩატეხილი კიბე 1

## მიდგომა

ეს ტიპური ფიბონაჩის მიმდევრობის ვარიაციაა. სტანდარტულად, ყოველ საფეხურზე ასვლის ვარიანტები წინა ორი საფეხურის ჯამია, მაგრამ აქ ჩატეხილი საფეხური გვაქვს, რომელზეც ვერ დავადგამთ ფეხს. მარტივი გამოსავალია, უბრალოდ ნულად ჩავთვალოთ ჩატეხილ საფეხურზე მოხვედრის შანსი. როცა DP-ს ვაკეთებ, თუ 'x'-ს ნულს მივანიჭებ, ავტომატურად ყველა გზა, რომელიც ამ წერტილზე გადის, 'გაინულდება'.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- DP ფორმულა d[i] = d[i-1] + d[i-2] მუშაობს ყველგან, გარდა ჩატეხილი საფეხურისა.
- თუ ჩატეხილ საფეხურს ნულს მივანიჭებთ, ის ავტომატურად ბლოკავს ნებისმიერ გზას, რომელიც მასზე გადის.

## ხშირი შეცდომები

- პირველი და მეორე საფეხურის ინიციალიზაციისას შეცდომები, თუ ჩატეხილი საფეხური თავიდანვე ახლოსაა.
