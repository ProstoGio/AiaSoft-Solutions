# Problem Statement — 120. ყუთები

> Copied from [AiaSoft.ge #120](https://aiasoft.ge/problem/120)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია ერთ რიგში, მიმდევრობით დალაგებული n რაოდენობის ყუთი, რომელთაგან თითოეულს მართკუთხა პარალელეპიპედის ფორმა აქვს.  i - ური ყუთის განზომილებები შესაბამისად არის hi, ai და bi, სადაც hi ყუთის სიმაღლეა, ხოლო ai და bi კი მისი ფუძის გვერდების ზომებია. ჩვენ შეგვიძლია მოცემული რიგიდან  ამოვიღოთ ზოგიერთი ყუთი და დარჩენილებით კი ავაგოთ კოშკი მათი ერთმანეთზე დალაგების საშუალებით საწყის განლაგებაში რიგითობის დაურღვევლად. ანუ, დარჩენილებიდან პირველის ზედა ფუძეზე ვდებთ რიგში მომდევნო ყუთს, მერე მის ზედა ფუძეზე რიგში მის მომდევნო ყუთს და ასე შემდეგ. ერთი ყუთი მეორეზე შეიძლება დავდოთ მხოლოდ იმ შემთხვევაში, თუ ზედა ყუთის ქვედა ფუძე არ სცდება ქვედა ყუთის ზედა ფუძის საზღვრებს, რაც აუცილებლობის შემთხვევაში ფუძის მობრუნებითაც შეიძლება იქნას მიღწეული.

დაწერეთ პროგრამა, რომელიც დაადგენს, თუ რა მაქსიმალური სიმაღლის ასეთი კოშკის აგებაა შესაძლებელი მოცემული ყუთების საშუალებით ზემოთ აღწერილი წესების დაცვით.

## Input

სტანდარტული შეტანის პირველ სტრიქონში მოცემულია ერთი  მთელი დადებითი  n რიცხვი (1 ≤ n ≤ 10 000) - მოცემული ყუთების რაოდენობა. მომდევნო n რაოდენობის სტრიქონიდან თითოეულში ჩაწერილია სამი მთელი დადებითი რიცხვი - მორიგი ყუთის სიმაღლე და ფუძის გვერდების ზომები შესაბამისად. არცერთი ყუთის არცერთი განზომილება არ აღემატება 10 000 - ს.
სტრიქონებში მონაცემები ერთმანეთისაგან თითო ჰარითაა გამოყოფილი.

## Output

სტანდარტული გამოტანის ერთადერთ სტრიქონში უნდა გამოიტანოთ ერთი მთელი რიცხვი - აგებული კოშკის მაქსიმალური სიმაღლე.
