# Problem Statement — 266. უდიდესი ზრდადი ქვემიმდევრობა

> Copied from [AiaSoft.ge #266](https://aiasoft.ge/problem/266)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

მოცემულია მთელი რიცხვების N ელემენტიანი მიმდევრიბა. იპოვეთ მოცემული მიმდევრობის უდიდესი მკაცრად ზრდადი ქვემიმდევრობის სიგრძე.

## Input

პირველ ხაზზე მოცემულია მიმდევრობის ელემენტების რაოდენობა: N (1 <= N <=100)
მეორე ხაზზე მოცემულია N რაოდენობის მთელი რიცხვი: a1, a2, ..., an - მიმდევრობის წევრები (0 <= ai <= 100)

## Output

გამოიტანეთ ერთი მთელი რიცხვი: უდიდესი ზრდადი ქვემიმდევრობის სიგრძე.
