# განმარტება — 611. XOR ოპერაცია

## მიდგომა

XOR ოპერაციის მთავარი მუღამი ისაა, რომ თუ 1-ების რაოდენობა კენტია, შედეგი 1 იქნება, ხოლო ლუწი რაოდენობის შემთხვევაში — 0. შესაბამისად, ამოცანა დაიყვანება კომბინატორიკაზე: N-დან რამდენნაირად შეგვიძლია ავირჩიოთ 1, 3, 5... (ანუ კენტი) რაოდენობის პოზიცია 1-ებისთვის. პასკალის სამკუთხედით მარტივად ვიანგარიშებთ კომბინაციების რაოდენობას და ბოლოს უბრალოდ ვკრებთ იმ სვეტების მნიშვნელობებს, სადაც 1-ების რაოდენობა კენტია.

## სირთულე

| | |
|-|--|
| დრო | O(n²) |
| მეხსიერება | O(n²) |

## მთავარი დაკვირვებები

- XOR 1-ს გვაძლევს მხოლოდ მაშინ, როცა ერთიანების ჯამი კენტია.
- პასკალის სამკუთხედი ყველაზე მოსახერხებელი გზაა C(n, k)-ს გამოსათვლელად DP-ით.

## ხშირი შეცდომები

- ლუწი რაოდენობის 1-ების არასწორად ჩათვლა პასუხში.
