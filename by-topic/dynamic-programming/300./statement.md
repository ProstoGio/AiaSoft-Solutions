# Problem Statement — 300. ფასიანი კიბე

> Copied from [AiaSoft.ge #300](https://aiasoft.ge/problem/300)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

პატარა თაზოს სურს კიბის ბოლო საფეხურზე ასვლა. თითოეულ საფეხურზე ფეხის დაბიჯება გარკვეული თანხა ღირს. დავადგინოთ, რა მინიმალური თანხის გადახდა მოუწევს ბიჭუნას კიბეზე ასასვლელად, თუ მას ერთი სვლით შეუძლია ერთი ან ორი საფეხურით მაღლა ასვლა.

## Input

პირველ ხაზზე შემოდის კიბის საფეხურების რაოდენობა - N.
მეორე ხაზზე შემოდის N-ცალი რიცხვი, i-ური რიცხვი წარმოადგენს i საფეხურზე დაბიჯების საფასურს.

## Output

გამოიტანეთ ერთი მთელი რიცხვი, კიბის ავლის მინიმალური საფასური.
