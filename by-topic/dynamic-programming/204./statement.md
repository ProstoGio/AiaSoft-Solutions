# Problem Statement — 204. კალია გრასი

> Copied from [AiaSoft.ge #204](https://aiasoft.ge/problem/204)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ჭაობის ზედაპირზე, ერთი ნაპირიდან მეორემდე გამწკრივებულია დუმფარას  n რაოდენობის ფოთოლი. პირველ ფოთოლზე კალია გრასი ზის. გრასის ბოლო ფოთოლზე უნდა გადასვლა, თანაც, რაც შეიძლება, სწრაფად.

გრასი სუპერრეაქტიულია, შეუძლია როგორც ფოთლის მომდევნო ფოთოლზე გადახტომა, ასევე ახერხებს იმ ფოთოლზე გადახტომასაც, რომლის ნომერი გრასის ფოთლის ნომერზე ორჯერ ან სამჯერ მეტია. მაგალითად, თუ გრასი ზის მე-7 ნომერ ფოთოლზე, მაშინ შეძლებს, როგორც მე-8, ასევე მე-14 ან 21-ე ნომერ ფოთოლზე გადასვლას.  სხვა ფოთლებზე გრასი მე-7 ფოთლიდან ვერაფრით მოხვდება. გრასი უკან არასოდეს ბრუნდება.

დაეხმარეთ გრასის გაარკვიოს, მინიმუმ რამდენ ნახტომში აღმოჩნდება მე-n ნომერ ფოთოლზე?

## Input

შემოდის ერთი ნატურალური რიცხვი n, 1<=n<=10⁵, დუმფარას ფოთლების რაოდენობა.

## Output

გამოიტანეთ ერთი ნატურალური რიცხვი, გრასის ნახტომების რაოდენობა.
