# განმარტება — 204. კალია გრასი

## მიდგომა

კალია გრასის ამოცანა კლასიკური DP-ა, სადაც პირდაპირ წინ სვლას ჯობს უკუღმა დავიწყოთ ფიქრი. ლოგიკა მარტივია: იმის მაგივრად, ვიფიქროთ სად გადავხტეთ, ვკითხულობთ, საიდან შეიძლებოდა მოვსულიყავით მიზნამდე. სამი ვარიანტი გვაქვს: x-1, x/2 ან x/3. დანარჩენი უკვე რეკურსიაა, რომელსაც memoization-ს ვუმატებთ, რომ ერთი და იგივე ფოთლის ნომერი თავიდან არ ვითვალოთ.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- უკუმიმართულებით აზროვნება ამ ამოცანის მთავარი გასაღებია, თორემ წინსვლით ძალიან აირირეოდი.
- memoization-ის მასივი რომ არ გამომეყენებინა, დროში ვერ ჩავეტეოდით და TLE-ს მივიღებდი.

## ხშირი შეცდომები

- x/2 ან x/3-ს გაყოფისას მოდულური შემოწმების (if x%2==0) დავიწყება, რაც სერიოზულ ლოგიკურ შეცდომას მოგვცემდა.

## ნაბიჯ-ნაბიჯ ახსნა

ვთქვათ n=10. გრასი 10-ზეა და ამოწმებს: 9-დან მოვიდა, 5-დან თუ არაფრიდან. 5-ზე რომ მოხვდეს, 4-დან ან 2-დან უნდა მოსულიყო, 2-ზე კი 1-დან. ასე, ნაბიჯ-ნაბიჯ, რეკურსია ითვლის ყველაზე მოკლე გზას და ინახავს m[] მასივში.
