# განმარტება — 315. ირაკლი და მიმდევრობა

## მიდგომა

ეს ამოცანა სუფთა DP-ა, უბრალოდ უნდა დაიჭირო კანონზომიერება, რით შეიძლება ბოლოვდებოდეს ვალიდური მიმდევრობა. რადგან '111' აკრძალულია, მიმდევრობა შეიძლება მთავრდებოდეს ან '0'-ით, ან '01'-ით, ან '011'-ით. თუ ბოლო სამი სიმბოლო რომელიმე ამათგან განსხვავებულია, ესე იგი უკვე დაარღვიე წესი. შესაბამისად, N-ისთვის პასუხი არის N-1, N-2 და N-3-ისთვის მიღებული პასუხების ჯამი.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- თუ მიმდევრობა მთავრდება 0-ით, წინა N-1 სიმბოლო ნებისმიერი ვალიდურია.
- თუ მთავრდება 01-ით, წინა N-2 სიმბოლო უნდა იყოს ვალიდური.
- თუ მთავრდება 011-ით, წინა N-3 სიმბოლო უნდა იყოს ვალიდური.

## ხშირი შეცდომები

- N=1, 2, 3-ისთვის ბაზისური შემთხვევების არასწორად გაწერა.

## ნაბიჯ-ნაბიჯ ახსნა

მაგალითად, N=4-ისთვის ავიღოთ N=3-ის 7 ვარიანტი (ბოლოში 0-ით), N=2-ის 4 ვარიანტი (ბოლოში 01-ით) და N=1-ის 2 ვარიანტი (ბოლოში 011-ით), ჯამში 13.
