# Problem Statement — 055. გაფუჭებული ვაშლები

> Copied from [AiaSoft.ge #55](https://aiasoft.ge/problem/55)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 128 MB

## Task

ლანამ გუშინ სოფელში ვაშლი მოკრიფა, საგულდაგულოდ გადაარჩია და მართკუთხედის ფორმის მაგიდაზე დაალაგა. ვაშლები სულ N ზოლად აწყვია და თითოეულ მწკრივში M ცალია. სამწუხაროდ ლანამ შემთხვევით ერთი გაფუჭებული ვაშლი დადო მაგიდაზე, კეროძოდ x ზოლზე მე-y ადგილზე.

ცნობილია, რომ დაავადებული ვაშლი ერთ კვირაში მის ახლოს მყოფ სხვა ვაშლებსაც აფუჭებს. დაადგინეთ რომელი ვაშლები იქნება გაფუჭებული ერთი კვირის შემდეგ.

`~https://storage.googleapis.com/problems_api_photos/images/problem55.png`

სურათზე ნაჩვენებია ვაშლების 5 მწკრივი, თითოეულში 6 ვაშლი (N=5, M=6). მესამე ზოლის მეოთხე ვაშლი ფაგუჭებულია (ანუ x=3, y=4). ერთ კვირაში ისრით ნაჩვენები ვაშლებიც გაფუჭდება, სხვები კი ჯანსაღი დარჩება.

## Input

ერთ ხაზზე შემოდის 4 რიცხვი: N, M, x, y

1 ≤ N, M ≤ 100
1 ≤ x ≤ N
1 ≤ y ≤ M

## Output

გამოიტანეთ N x M ზომის მატრიცა, მატრიცის (i,j)-წევრი არის 1 თუ ის გაფუჭებულია, წინააღმდეგ შემთხვევაში 0.
