# Problem Statement — 548. ოთახების დათვლა

> Copied from [AiaSoft.ge #548](https://aiasoft.ge/problem/548)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

თქვენ მოცემული გაქვთ AIATOWER-ის პირველი სართულის გეგმა. გეგმა შედგება n სტრიქონისა და m სვეტისგან. თუ რომელიმე უჯრაში წერია # ეს აღნიშნავს რომ AIATOWER-ის პირველი სართულის ამ ადგილას კედელია, ხოლო თუ წერია წერტილი ეს ნიშნავს, რომ ეს ადგილი თავისუფალია და ჩვენ შეგვიძლია ამ უჯრაში გადაადგილება. თქვენი დავალებაა დათვალოთ AIATOWER-ის პირველი სართულის ოთახების რაოდენობა. გადაადგილება შეგიძლიათ თქვენ უშუალო მეზობლებში, ანუ თუ დგახართ i სტრიქონის j უჯრაში გადაადგილება შეგიძლიათ მხოლოდ: (i+1,j), (i-1,j), (i,j+1), (i,j-1). გეგმაში აღნიშნული არ არის რომ AIATOWER-ის პირველი სართული შემოსაზღვრულია კედლებით, ამიტომ ეს ფაქტი უნდა გაითვალისწინოთ. ოთახი ეწოდება თავისუფალი უჯრების ერთობლიობას, სადაც ნებისმიერი უჯრიდან შეგვიძლია ნებისმიერ უჯრაში მისვლა.

## Input

პირველ ხაზზე შემოდის ორი რიცხვი n, m - სტრიქონების და სვეტების რაოდენობა კარტაზე.
მომდევნო n ხაზზე მოცემულია კარტის აღწერა, თუ i სტრიქონის j უჯრაში წერია #, მაშინ ეს ადგილი დაკავებულია და იქ გადაადგილება არ შეგვიძლია, წინააღმდეგ შემთხვევაში იქ მოძრაობა დასაშვებია.
1<=n,m<=10³

## Output

ერთ ხაზზე გამობეჭდეთ AIATOWER-ში ოთახების რაოდენობა.
