# განმარტება — 907. ფიფქების კვადრატი

## მიდგომა

სინამდვილეში, აქ მთავარია ფიგურის პერიმეტრის დაჭერა. 30-ზე მეტი არ არის N, ჰოდა 2D მასივი რომ გამოვიყენე, კოდი საკმაოდ გასაგები გამოვიდა. თავიდან ყველგან ნულები ჩავყარე, მერე კი მარტივად: ზედა და ქვედა სტრიქონები, პლიუს პირველი და ბოლო სვეტები გადავწერე ვარსკვლავებით. ბოლოს, დაბეჭდვის დროს ნულები უბრალოდ ცარიელი სივრცით ჩავანაცვლე.

## სირთულე

| | |
|-|--|
| დრო | O(n²) |
| მეხსიერება | O(n²) |

## მთავარი დაკვირვებები

- კვადრატის საზღვრები ემთხვევა მასივის 0-იან და n-1-იან ინდექსებს.
- არ არის აუცილებელი მასივი, პირდაპირ print-ითაც შეიძლებოდა, მაგრამ ეს მიდგომა უფრო მოწესრიგებულია.

## ხშირი შეცდომები

- კუთხეების გაორმაგება ან გამოტოვება ლუპების დროს.
