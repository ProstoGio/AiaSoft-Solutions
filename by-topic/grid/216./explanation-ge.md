# განმარტება — 216. მარცხენა მთავარი დიაგონალი

## მიდგომა

ეს დავალება საკმაოდ მარტივია, პირდაპირ მატრიცის სტრუქტურაზეა. მთავარი დიაგონალის ელემენტებს ის თავისებურება აქვთ, რომ ინდექსები ერთმანეთის ტოლია, ანუ i უდრის j-ს. უბრალოდ ვკითხულობთ მატრიცას და მეორე ციკლით წვალების ნაცვლად, პირდაპირ a[k][k]-ებს ვუმატებთ ჯამს.

## სირთულე

| | |
|-|--|
| დრო | O(n²) |
| მეხსიერება | O(n²) |

## მთავარი დაკვირვებები

- მთავარი დიაგონალი ყოველთვის არის ის ელემენტები, სადაც მატრიცის ინდექსი i და j ერთმანეთს ემთხვევა.
- მიუხედავად იმისა, რომ შეიძლება პირდაპირ წაკითხვისთანავე დავითვალოთ ჯამი, 20x20 მატრიცისთვის მეხსიერების გაფლანგვა არაფერს გვიშავებს.

## ხშირი შეცდომები

- მთავარი დიაგონალის აგრევა მეორად დიაგონალთან, სადაც i+j == n-1 პირობა უნდა დაგვეწერა.
