# Problem Statement — 301. ლაბირინთი

> Copied from [AiaSoft.ge #301](https://aiasoft.ge/problem/301)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

თეკლას ძალიან უყვარს ლაბირინთში თამაში. მან გადაწყვიტა, რომ ლაბირინთიდან გასასვლელამდე რაც შეიძლება ნაკლებ სვლაში მიაღწიოს.

ლაბირინთი მოცემულია N x M ზომის მატრიცით. თამაშის საწყისი წერტილი აღნიშნულია S-ით, ხოლო საბოლოო - F-ით. ლაბირინთის კედლები აღნიშნულია X-ით, ხოლო თავისუფალი ტერიტორია წერტილით.

ერთ სვლაზე თეკლას შეუძლია გადაინაცვლოს ერთი უჯრით ზემოთ, ქვემოთ, მარცხნივ ან მარჯვნივ. დაეხმარეთ მას, გამოთვალოს მინიმუმ რამდენი სვლა დასჭირდება ლაბირინთიდან გასასვლელად.

`bგარანტირებულია`, რომ S-დან F-მდე მიღწევა შესაძლებელია და ლაბირინთი შემოღობილია კედლებით.

## Input

პირველ ხაზზე შემოდის ორი მთელი რიცხვი: N და M.    (4 <= N, M <= 20)
მომდევნო N ხაზი შეიცავს M სიმბოლოს.

## Output

გამოიტანეთ ერთი მთელი რიცხვი, საწყისი წერტილად გასასვლელამდე მისასვლელად საჭირო მინიმალური სვლების რაოდენობა.
