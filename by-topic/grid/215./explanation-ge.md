# განმარტება — 215. მატრიცის შევსება

## მიდგომა

ამ ამოცანის იდეა არის მარტივი მატრიცის შევსება - უბრალოდ გვაქვს n ზომის კვადრატული მატრიცა, სადაც ყველგან x უნდა ჩავწეროთ. ორი ჩაშენებული `for` ციკლი საკმარისია იმისთვის, რომ ყველა უჯრა x-ით აივსოს, ხოლო მეორე წყვილი ციკლი უბრალოდ ეკრანზე გამოსატანად მჭირდება.

## სირთულე

| | |
|-|--|
| დრო | O(n²) |
| მეხსიერება | O(n²) |

## მთავარი დაკვირვებები

- რეალურად მატრიცის მეხსიერებაში შენახვა საჭიროც არაა, პირდაპირ `cout`-ითაც შეგიძლია დაბეჭდო x, თუ გინდა რომ O(1) space complexity მიიღო.
- VLA-ს გამოყენება `int matrix[n][n]` სახით ყველაზე მარტივია, თუმცა თუ n ძალიან დიდი იქნება, stack overflow-ს საფრთხე გაჩნდება.

## ხშირი შეცდომები

- დაბეჭდვის დროს ინტერვალების (spaces) დავიწყება, რაც output-ს აფუჭებს.
