# Problem Statement — 934. დიდი რიცხვები

> Copied from [AiaSoft.ge #934](https://aiasoft.ge/problem/934)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია ორი რიცხვი. პირველი მათგანი $N-$ნიშნაა, ხოლო მეორე - $M-$ნიშნა და ორივე მათგანის ჩანაწერში მხოლოდ ერთი $D$ ციფრია გამოყენებული.დაწერეთ პროგრამა, რომელიც იპოვის მოცემული რიცხვების უდიდეს საერთო გამყოფს.

## Input

სტანდარტული შეტანის პირველ სტრიქონში მოცემულია ერთი ჰარით გამოყოფილი ორი მთელი $N$ და $M$ რიცხვი - ციფრების რაოდენობა პირველ და მეორე რიცხვებში შესაბამისად. მეორე სტრიქონში ჩაწერილია ერთი მთელი $D$ რიცხვი  - ციფრი, რომლისგანაც შედგებიან მოცემული რიცხვები.

$1 ≤ N,M ≤ 10^18$
$1 ≤ D ≤ 9$

## Output

თუ ნაპოვნი უდიდესი საერთო გამყოფი შედგება არაუმეტეს $10^6$ რაოდენობის ციფრისაგან, მაშინ სტანდარტული გამოტანის ერთადერთ სტრიქონში უნდა ჩაიწეროს თვით ეს რიცხვი, წინააღმდეგ შემთხვევაში კი - მისი ბოლო $10^6$ რაოდენობის ციფრი. პასუხი არ უნდა შეიცავდეს არანიშნად ნულებს.
