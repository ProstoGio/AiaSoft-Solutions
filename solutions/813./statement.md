# Problem Statement — 813. სიმეტრიულის გამყოფები

> Copied from [AiaSoft.ge #813](https://aiasoft.ge/problem/813)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია მთელი ნატურალური რიცხვი, რომლის ციფრების რაოდნეობა არ აღემატება 9-ს. დაწერეთ პროგრამა, რომელიც შეატრიალებს ამ რიცხვს სიმეტრიულად და შედეგს გამოიტანს ეკრანზე, ასევე იპოვის სიმეტრიულად შეტრიალებული რიცხვის გამყოფების რაოდენობას და შედეგს გამოიტანს ეკრანზე.

## Input

ერთადერთ ხაზზე შემოდის ერთი ნატურალური რიცხვი N - რომლის ციფრთა რაოდენობა არ აღემატება 9-ს.

## Output

ერთადერთ ხაზზე გამობეჭდეთ ორი მთელი დადებითი რიცხვი, N-ის სიმეტრულად შეტრიალების შედეგად მიღებული რიცხვი და შეტრიალებული რიცხვის გამყოფების რაოდენობა.
