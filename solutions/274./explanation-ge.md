# განმარტება — 274. ციკლები გრაფში - 2

## მიდგომა

მოკლედ რომ ვთქვათ, ეს არის კლასიკური DSU-ს (Disjoint Set Union) ამოცანა. ხე რომ მივიღოთ, უბრალოდ წიბოები ისე უნდა შევარჩიოთ, რომ ციკლები არ შევქმნათ. თითოეულ წიბოს ვუყურებთ და ვამოწმებთ, ორივე წვერო უკვე ერთ კომპონენტში ხომ არაა. თუ ერთ კომპონენტში არიან, ესე იგი წიბოს დამატებით ციკლს ვკრავთ და შესაბამისად, წასაშლელი წიბოების სიაში ვამატებთ.

## სირთულე

| | |
|-|--|
| დრო | O(M * α(N)) |
| მეხსიერება | O(N) |

## მთავარი დაკვირვებები

- ხეს N წვეროთი ზუსტად N-1 წიბო უნდა ჰქონდეს და ციკლები არ უნდა გააჩნდეს.
- path compression-ის გამოყენებით find_set-ს ძალიან სწრაფს ვხდით, რაც პრაქტიკულად O(1)-ია თითო ოპერაციაზე.

## ხშირი შეცდომები

- თუ წიბო ციკლს არ ქმნის, გვავიწყდება union_set ფუნქციის გამოძახება, რაც მთლიან ლოგიკას ანგრევს.
- 1-დან N-მდე დანომრილი წვეროების მასივის ზომა თუ არასწორად გვექნება განსაზღვრული.
