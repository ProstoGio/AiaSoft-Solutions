# განმარტება — 068. ბიოლოგიური რიტმები

## მიდგომა

ამ ამოცანის მთავარი მუღამი ისაა, რომ ბიორიტმების პერიოდები პატარა რიცხვებია (23, 28, 33). თითოეულ მაქსიმუმს ვთარგმნით წლის დასაწყისიდან გასულ დღეების რაოდენობაში და მერე უბრალოდ ვამოწმებთ, რომელი დღე დააკმაყოფილებს პირობას. რადგან ამ სამი ციკლის საერთო პერიოდი 21,252-ია (მათი LCM), უბრალო ლუპი, რომელიც დღეებს ამოწმებს, წამზე ნაკლებ დროში იპოვის პასუხს. ზედმეტი მათემატიკური ფორმულების გარეშეც მშვენივრად მუშაობს.

## სირთულე

| | |
|-|--|
| დრო | O(21252) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- სამივე ციკლის დამთხვევის პერიოდი არის 23 * 28 * 33 = 21,252. ამიტომ brute-force-იც კი არ გვიქმნის პრობლემას.
- თარიღების დღეებად გადაქცევისას არ დაგავიწყდეთ თვეების დღეების ჯამი სწორად დაასტრიქჩეროთ მასივში.

## ხშირი შეცდომები

- პასუხში იმ დღის ჩათვლა, რომელიც მოცემულია შეყვანაში (უნდა დავიწყოთ ათვლა მომდევნო დღიდან).
