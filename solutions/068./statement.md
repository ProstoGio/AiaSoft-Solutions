# Problem Statement — 068. ბიოლოგიური რიტმები

## Task

არსებობს თეორია, რომ ადამიანის ცხოვრებაში არსებობს სამი ციკლი, რომლებიც მისი დაბადებისას იწყება და მის ფიზიკურ, ემოციურ და ინტელექტუალურ მდგომარეობას განსაზღვრავს. მათი პერიოდებია, შესაბამისად, 23, 28 და 33 დღე და მათ ბიორიტმებს უწოდებენ. თითოეულ ციკლს გააჩნია მაქსიმუმები. მაგალითად, ფიზიკური ციკლის მაქსიმუმის მიღწევის დღიდან 23 დღის შემდეგ ისევ მიიღწევა ფიზიკური ციკლის მაქსიმუმი, 23 დღის შემდეგ ისევ მაქსიმუმი და ა.შ. ანალოგიურად ემოციური და ინტელექტუალური ციკლებისთვისაც, თუმცა პერიოდებით 28 და 33.
რომელიმე მათგანის მაქსიმუმის დროს ადამიანი შესაბამის სფეროში (ფიზიკური, ემოციური, ინტელექტუალური) თავისი შესაძლებლობების მაქსიმალურ რეალიზაციას ახერხებს. რადგანაც განსხვავებული პერიოდები გააჩნიათ, ამიტომ მათი მაქსიმუმები განსხვავებულ დღეებში მიიღწევა. თუმცა, ხანდახან სამივე მაქსიმუმის მიღწევის თარიღები ერთმანეთს ემთხვევა. რა თქმა უნდა, ბევრ ადამიანს აინტერესებს, როდის მიიღწევა მათი ბიორიტმების სამმაგი მაქსიმუმი. დაწერეთ პროგრამა, რომელიც დაადგენს დღეების რაოდენობას მოცემული თარიღიდან მომდევნო სამმაგი მაქსიმუმის მიღწევის თარიღამდე, თუ ცნობილია თითოეული ციკლის მაქსიმუმის მიღწევის თარიღი (გარანტირებულია რომ ყოველი მოცემული მაქსიმუმის თარიღი წინარ უსწრებსმოცემულ თარიღს).
სამმაგი მაქსიმუმის მიღწევისათვის საჭირო დღეების რაოდენობაში არ ითვლება თარიღი, რომლიდანაც ვიწყებთ ათვლას (მოცემული თარიღი). მაგალითად, თუ მოცემული თარიღია 20/02 და მომდევნო სამმაგი მაქსიმუმის მიიღწევა 22/02 - ში, მაშინ პასუხია 2 და არა 3. თუ მოცემულ თარიღში უკვე მიიღწევა სამმაგი მაქსიმუმი, მაშინ უნდა იპოვნოთ მომდევნო სამმაგი მაქსიმუმის მიღწევამდე საჭირო დღეების რაოდენობა.

## Input

სტანდარტული შეტანის ოთხ ხაზზე შემოდის ორი, ერთმანეთისგან ჰარით გამოყოფილი, რიცხვი: დღე და თვე. პირველი თარიღი წარმოადგენს ფიზიკური ციკლის მაქსიმუმის მიღწევის თარიღს, მეორე - ემოციური ციკლის მაქსიმუმის მიღწევის თარიღს, ხოლო მესამე კი - ინტელექტუალური ციკლის მაქსიმუმის მიღწევის თარიღს. მეოთხე თარიღი წარმოადგენს მოცემულ თარიღს, საიდანაც უნდა დავთვალოთ დღეების რაოდენობა მომდევნო სამმაგი მაქსიმუმის მიღწევისთარიღამდე.

## Output

სტანდარტული გამოტანის ერთადერთ სტრიქონში უნდა გამოიტანოთ ერთი მთელი დადებითი რიცხვი - დღეების რაოდენობა მოცემული თარიღიდან მომდევნო სამმაგი მაქსიმუმის მიღწევის თარიღამდე.
