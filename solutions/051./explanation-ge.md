# განმარტება — 051. შესაკრებთა წყვილების პოვნა

## მიდგომა

მასივის დალაგება ყველაზე სწრაფი გზაა საქმის გასამარტივებლად. დახარისხებული მასივის შემთხვევაში, ორი პოინტერით მუშაობა ძალიან მოსახერხებელია: ერთი თავიდან იწყებს, მეორე ბოლოდან. თუ მათი ჯამი X-ზე ნაკლებია, მარცხენა პოინტერს ვწევთ წინ, რომ ჯამი გავზარდოთ; თუ მეტია - მარჯვენას ვწევთ უკან. ასე ვმოძრაობთ მანამ, სანამ პოინტერები ერთმანეთს არ შეხვდებიან.

## სირთულე

| | |
|-|--|
| დრო | O(n log n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- დალაგება საშუალებას გვაძლევს ამოცანა O(n²)-დან O(n log n)-მდე ჩამოვიყვანოთ.
- ორი პოინტერის მეთოდი იდეალურია, რადგან ყოველ ნაბიჯზე ვხვდებით, რომელი მხარე უნდა შევცვალოთ.

## ხშირი შეცდომები

- Nested loop-ის გამოყენება, რომელიც 10^5 შეზღუდვისას უეჭველი TLE-ს მოგცემთ.

## ნაბიჯ-ნაბიჯ ახსნა

ვთქვათ გვაქვს [1, 2, 3, 4, 5] და X=6. პოინტერები დგას 1-ზე და 5-ზე (ჯამი 6), ვპოულობთ წყვილს და ვცვლით ორივეს. შემდეგ დგანან 2-ზე და 4-ზე (ჯამი 6), კიდევ ერთი წყვილი. ბოლოს, როცა 3-ზე შეხვდებიან, ციკლი წყდება.
