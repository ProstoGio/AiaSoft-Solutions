# Problem Statement — 684. კიტრი და პომიდორი

> Copied from [AiaSoft.ge #684](https://aiasoft.ge/problem/684)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ბაზარში ერთი კილოგრამი კიტრი x თეთრი ღირს, ხოლო პომიდორი - y თეთრი. დაადგინეთ რამდენი ლარი და რამდენი თეთრი დაგეხარჯებათ თუ n კილოგრამ კიტრს და m კილოგრამ პომიდორს შეიძენთ.

გაითვალისწინეთ, თუ პროდუქტის საერთო ღირებულება თეთრებში არ არის ხუთის ჯერადი, მაშინ ღირებულება გაიზრდება უახლოეს ხუთის ჯერადამდე. მაგალითად, თუ ჯამურად გიწევთ 5 ლარის და 23 თეთრის გადახდა, სინამდვილეში გადაიხდით 5 ლარს და 25 თეთრს. ეს იმიტიმ ხდება, რომ ქართულ ფულის ერთეულებში მინიმუმ 5 თეთრიანი მონეტა გვაქვს.

## Input

ერთ ხაზზე შემოდის ოთხი მთელი რიცხვი: x, y, n text( და ) m.
1<= x, y <= 100
0<=n,m<=100

* პროდუქტების ფასები მოცემულია თეთრებში.

## Output

გამობეჭდეთ ორი მთელი რიცხვი: a და b, რაც ნიშნავს, რომ თქვენ მოგიწევთ a ლარის და b თეთრის გადახდა.

* შეგახსენებთ, რომ ერთი ლარი ასი თეთრია.
