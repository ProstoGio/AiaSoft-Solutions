# Problem Statement — 704. ინტერვალის ჯამი

> Copied from [AiaSoft.ge #704](https://aiasoft.ge/problem/704)

**Time limit:** 2.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია $N$ ელემენტიანი მიმდევრობა, რომლის ელემენტებიც გადანომრილია 1-დან $N$-ის ჩათვლით (შემოსვლის თანმიმდევრობის მიხედვით).
ასევე მოცემულია $Q$ რაოდენობის შეკითხვა. თითოეულ შეკითხვა შეიცავს ორ რიცხვს: $l, r$ - რაც ნიშნავს, რომ საპოვნია მასივის $[l, r]$ ინტერვალში (ანუ $l$ ინდექსიდან $r$ ინდექსის ჩათვლით) მოთავსებული ელემენტების ჯამი.

## Input

პირველ ხაზზე მოცემულია ორი მთელი რიცხვი: $N, Q$.
მეორე ხაზზე მოცემულია თითო ჰარით გამოყოფილი $N$ მთელი რიცხვი - $a_1, a_2, ..., a_N$ - მიმდევრობის წევრები.
მომდევნო $Q$ რაოდენობის ხაზზე მოცემულია ორ-ორი რიცხვი: $l, r$

$1 <= N, Q <= 2*10^5$
$1 <= a_i <= 10^9$
$1 <= l <= r <= N$

## Output

ყოველი შეკითხვისთვის ცალკე ხაზზე გამობეჭდეთ პასუხი.
