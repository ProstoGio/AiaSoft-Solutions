# Problem Statement — 139. პირველი ნომერი ცელქობა

> Copied from [AiaSoft.ge #139](https://aiasoft.ge/problem/139)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ფიზკულტურის მასწავლებელმა მოსწავლეები სიმაღლის ზრდადობის მიხედვით დააყენა და ბურთის შემოსატანად გავიდა.

ცელქმა ბავშვებმა მაშინვე გამოიყენეს `bპირველი ნომერი ცელქობა`, ამ ცელქობის წესებში წერია, რომ `bყოველმა` ლუწი ნომრის მქონე მოსწავლემ ადგილი უნდა გაუცვალოს მის მომდევნო მოსწავლეს. თუ რომელიმე ლუწი ნომრის მქონე ბავშვს არ ჰყავს მომდევნო მეგობარი, მაშინ, რა თქმა უნდა, ეს ბავშვი ადგილზე რჩება. მოსწავლეთა  დანომრვა იწყება ნულით.

მოცემული გაქვთ მოსწავლეთა სიმაღლების `bპირველი ნომერი ცელქობის` წესებით არეული მასივი, დაწერეთ პროგრამა, რომელიც მოცემულ მასივს დაუბრუნებს პირვანდელ ფორმას.

## Input

შემომავალი ფაილის პირველ სტრიქონში ჩაწერილია მასივის ელემენტების რაოდენობა $n$, $1 <= n <=20$.
მეორე სტრიქონი შეიცავს არეული მასივის $a_k$ ელემენტებს - მოსწავლეთა სიმაღლეებს სანტიმეტრებში, $120 <= a_k <=160$, $1<=k<=n$.

## Output

გამოიტანეთ სიმაღლეთა მასივის საწყისი ვარიანტი.
