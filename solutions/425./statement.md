# Problem Statement — 425. სიმრავლეები

> Copied from [AiaSoft.ge #425](https://aiasoft.ge/problem/425)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია სიმრავლე, რომლის ელემენტებიც სიმრავლეებია, რომლებიც შესაძლოა თავის მხრივ სიმრავლეებს შეიცავდეს და ა. შ. მაგალითად:
	● {} - ცარიელი სიმრავლეა
	● {{}} - ერთ ელემენტიანი სიმრავლეა
	● {{}{{}}} - ორ ელემენტიანი სიმრავლეა. {} და {{}} ამ სიმრავლის ელემენტებია
	● {{{}{}}} - ასევე ერთ ელემენტიანი სიმრავლეა და ა. შ.

მოცემული სიმრავლე შესაძლოა შეიცავდეს რამდენიმე ერთნაირ ელემენტს. თქვენი მიზანია იპოვოთ მაქსიმალური სიხშირის მქონე ელემენტი.

ასევე გაითვალისწინეთ, რომ სიმრავლეებში ელემენტთა თანმიმდევრობას მნიშვნელობა არ აქვს. მაგალითად {a, b} და {b, a} ტოლი სიმრავლეებია.

## Input

ერთ ხაზზე მოცემულია სიმრავლე $S$, რომელიც შედგება მხოლოდ '{' და '}' სიმბოლოებისგან. გარანტირებულია, რომ $S$ სიმრავლე ერთ ელემენტს მაინც შეიცავს.

$4 <= |S| <= 1000$

## Output

ერთ ხაზზე გამობეჭდეთ ერთი რიცხვი, შემოსული სიმრავლის ელემენტებიდან მაქსიმალური სიხშირის მქონე ელემენტის სიხშირე.
