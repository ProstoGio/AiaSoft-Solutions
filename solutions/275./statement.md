# Problem Statement — 275. ცხოველები - 1

> Copied from [AiaSoft.ge #275](https://aiasoft.ge/problem/275)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

ზოოპარკში ახალი მიყვანილი ცხოველებისთვის 2 დიდი გალია აქვთ გამოყოფილი, რომლებშიც შეუზღუდავი რაოდენობით ცხოველის დატევაა შესაძლებელი. ახლახანს ზოოპარკში $N$ ახალი ცხოველი მოიყვანეს და ერთ-ერთ თანამშრომელს, თემოს, დაევალა მათი განაწილება ორ გალიაში ისე, რომ ერთ გალიაში ერთმანეთის მიმართ აგრესიულად განწყობილი ცხოველები არ მოხვდნენ.

ცხოველები გადანომრილია $1$-დან $N$-ის ჩათვლით და ცნობილია $M$ კავშირი, თუ რომელი ორი ცხოველია ერთმანეთის მიმართ აგრესიული.

ეს ერთი შეხედვით ადვილი საქმე საკმაოდ რთული გამოდგა. ამიტომ თემომ დასახმარებლად თქვენ მოგმართათ. 

დაწერეთ პროგრამა, რომელიც დაადგენს შესაძლებელია თუ არა მოყვანილი $N$ ცხოველის გადანაწილება ორ სხვადასხვა გალიაში, ისე, რომ ერთმანეთის მიმართ აგრესიული ცხოველები ერთად არ მოხვდნენ.

## Input

პირველ ხაზზე მოცამულია ორი რიცხვი: $N$ და $M$ - მოყვანილი ცხოველების რაოდენობა და კავშირების რაოდენობა.
მომდევნო $M$ ხაზზე შემოდის ორ-ორი რიცხვი. კერძოდ $i$-ურ ხაზზე მოცემული იქნება $v_i$ და $u_i$ - ერთმანეთის მიმართ აგრესიული ცხოველების ნომრები.

$1 <= N <= 20$
$0 <= M <= N*(N-1) / 2$
$1 <= v_i, u_i <= N$

## Output

გამოიტანეთ ერთი სიტყვა: `bYES` თუ პირობის დაკმაყოფილება შესაძლებელია და `bNO` - წინააღმდეგ შემთხვევაში.
