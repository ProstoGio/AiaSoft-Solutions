# განმარტება — 275. ცხოველები - 1

## მიდგომა

სინამდვილეში ეს არის კლასიკური Bipartite Graph-ის ამოცანა. წარმოიდგინე, რომ უნდა დავყოთ წვეროები ორ ფერად ისე, რომ არცერთი მეზობელი არ იყოს ერთნაირი ფერის. მე ავირჩიე DFS, რადგან ყველაზე მარტივად აკეთებს ამ საქმეს: ყოველი ახალი ცხოველი უნდა გავაფერადო იმ ფერით, რომელიც მისი წინაპრისგან განსხვავდება. თუ აღმოვაჩენ, რომ მეზობელი უკვე გაფერადებულია და ჩემს ფერს ემთხვევა, ეგრევე ვამბობ 'NO'-ს.

## სირთულე

| | |
|-|--|
| დრო | O(N + M) |
| მეხსიერება | O(N + M) |

## მთავარი დაკვირვებები

- პრობლემა 1-ში 1-თან გადადის გრაფის ორფერადოვნების (bipartite check) შემოწმებაში.
- თუ გრაფი დაშლილია, მთავარი ციკლი აუცილებლად უნდა გადიოდეს ყველა კვანძზე, თორემ რომელიმე კომპონენტი გაგეპარება.

## ხშირი შეცდომები

- მხოლოდ ერთი DFS გამოძახება, როცა გრაფი შესაძლოა შედგებოდეს რამდენიმე დამოუკიდებელი ნაწილისგან.

## ნაბიჯ-ნაბიჯ ახსნა

მაგალითად, თუ გვაქვს 1-2 და 2-3 კავშირები: ვიწყებთ 1-ით (ფერი 1), შემდეგ 2 ხდება 0, ბოლოს 3 ხდება ისევ 1. რადგან არსად შეგვხვდა მეზობელი იგივე ფერით, კოდი მორჩება მუშაობას და დაბეჭდავს 'YES'-ს.
