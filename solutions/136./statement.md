# Problem Statement — 136. ქეთი და კოტე

> Copied from [AiaSoft.ge #136](https://aiasoft.ge/problem/136)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

ქეთი ძალიან არის შეყვარებული კოტეზე თუმცა მან არ იცის კოტეს გრძნობების შესახებ. ამიტომ მან გადაწყვიტა გავიდეს მინდორში და მოძებნოს მაგიური გვირილა, რომელიც დაეხმარება ქეთის რომ გაერკვეს კოტეს გრძნობებში.

მაგიური გვირილა შემდეგნაირად მუშაობს: ქეთიმ ნელ-ნელა უნდა დააცალოს გვირილას ფოთლები და თან ყოველი ფოთლის მოწყვეტისას გულში უნდა თქვას შემდეგი ორი ვარიანტიდან ერთ-ერთი:
1. ვუყვარვარ
2. არ ვუყვარვარ

დაადგინეთ, უყვარს თუ არა კოტეს ქეთი, თუ გვირილას N ფოთოლი აქვს და ქეთიმ პირველი ფოთლის მოწყვეტისას თქვა - ვუყვარვარ.

## Input

შემოდის ერთი მთელი რიცხვი: N - ქეთის ნაპოვნ გვირილაზე ფოთლების რაოდენობა.  ( 1<= N <= 10³ )

## Output

გამოიტანეთ YES თუ კოტეს უყვარს ქეთი, წინააღმდეგ შემთხვევაში - NO.
