# Problem Statement — 298. ცხენის სვლა

> Copied from [AiaSoft.ge #298](https://aiasoft.ge/problem/298)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

მოცემულია N x M ზომის ჭადრაკის დაფა. დაფის (x1, y1) კოორდინატზე მოთავსებულია ცხენი, რომლის მიზანიცაა (x2, y2) კოორდინატზე მიღწევა.

დაადგინეთ, მინიმუმ რამდენი სვლა დასჭირდება ცხენს საწყისი წერტილიდან დანიშნულების ადგილამდე მისასვლელად.

## Input

პირველ ხაზზე მოცემულია ორი მთელი რიცხვი: N და M.
მეორე ხაზზე შემოდის ოთხი მთელი რიცხვი: x1, y1, x2, y2.

1 <= N, M <= 20
1 <= x1, x2 <= N
1 <= y1, y2 <= M

## Output

თუ ცხენს შეუძლია მიაღწიოს (x2, y2) კოორდინატს, მაშინ გამობეჭდეთ მინიმალური სვლების რაოდენობა. წინააღმდეგ შემთხვევაში გამოიტანეთ `b-1`.
