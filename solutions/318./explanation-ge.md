# განმარტება — 318. კეთილი ფერია

## მიდგომა

სიმართლე რომ გითხრა, არ მინდოდა რაღაც რთული ლოგიკის გამოგონება. რადგან მაქსიმალური ხუთნიშნა რიცხვი გვინდა, ყველაზე მარტივი გზაა 99999-დან დავიწყოთ კლება და პირველივე რიცხვი, რომელიც პირობას დააკმაყოფილებს, ეგ იქნება ჩვენი პასუხი. ციკლი უბრალოდ თითოეულ ციფრს ყოფს და კრებს, როგორც კი ჯამი N-ს გაუტოლდება, პროგრამას ვწყვეტთ.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- რადგან დიაპაზონი ძალიან პატარაა, brute-force მიდგომა იდეალურად მუშაობს და დროშიც ვეტევით.
- 99999-დან დაწყება იმის გარანტიაა, რომ პირველი ნაპოვნი რიცხვი ავტომატურად იქნება მაქსიმალური.

## ხშირი შეცდომები

- ზედმეტად გართულებული ალგორითმების წერა, როცა 100 000-მდე იტერაცია წამებში სრულდება.
