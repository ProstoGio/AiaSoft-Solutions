# Problem Statement — 318. კეთილი ფერია

> Copied from [AiaSoft.ge #318](https://aiasoft.ge/problem/318)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

ნინიკოს ძალიან უყვარს ზღაპრები. დღეს დედამ ნინიკოს ზღაპარი უამბო, რომელშიც მოთხრობილია ბოროტი ჯადოქრისა და კეთილი ფერიას დაპირისპირების ამბავი. ბოროტმა ჯადოქარმა ფერია დაწყევლა.  წყევლის მოხსნა მხოლოდ მაგიური რიცხვის წარმოთქმით შეიძლება, მაგიური რიცხვი ისეთი მაქსიმალური ხუთნიშნა რიცხვია, რომლის ციფრთა ჯამი $N$-ის ტოლია.

დაეხმარეთ ფერიას მაგიური რიცხვის პოვნაში, რათა გათავისუფლდეს ჯადოსგან.

## Input

მოცემულია ერთი მთელი რიცხვი: $N$ - მაგიური რიცხვის ციფრთა ჯამი. $(1 <=N<=45)$

## Output

იპოვეთ და გამობეჭდეთ მაგიური რიცხვი.
