# Problem Statement — 465. უდიდესი კომპონენტის წვეროები

> Copied from [AiaSoft.ge #465](https://aiasoft.ge/problem/465)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია არაორიენტირებული გრაფი. დაადგინეთ რომელ წვეროებს შეიცავს უდიდესი კომპონენტი და გამობეჭდეთ ეს წვეროები ზრდადობით. 

კომპონენტების სიდიდე მათში მოთავსებული წვეროების რაოდენობით განისაზღვრება.

* რამდენიმე პასუხის არსებობის შემთხვევაში უნდა გამობეჭდოთ ლექსიკოგრაფიულად ყველაზე პატარა ვარიანტი.

## Input

პირველ ხაზზე შემოდის N და M, შესაბამისად გრაფში წვეროებისა და წიბოების რაოდენობა.
მომდევნო M ხაზზე მოცემულია ორ-ორი რიცხვი. კერძოდ i-ურ ხაზზე შემოვა vi და ui - მეზობელი წვეროების ნომრები. (გრაფის წვეროები დანომრილია 1-დან N-მდე)

1 <= N <= 20
0 <= M <= N*(N-1) / 2
1 <= vi, ui <= N

## Output

ერთ ხაზზე გამობეჭდეთ თითო ჰარით გამოყოფილი უდიდეს კომპონენტში შემავალი წვეროების ნომრები ზრდადობით.
