# Problem Statement — 766. ჯამი

> Copied from [AiaSoft.ge #766](https://aiasoft.ge/problem/766)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

პანდემიის დროს რომ არ მოეწყინა, ანასტასიამ ასეთი გასართობი მოიფიქრა: იგი ფურცელზე წერს მიმდევრობით აღებულ ნატურალურ რიცხვებს 1-დან რაიმე დიდი რიცხვამდე. შემდეგ ის თითოეულ ასეთ რიცხვს ცვლის მისი ციფრების ჯამით მანამ, სანამ ჩამოწერილი რიცხვებიდან ყველა მათგანი ერთნიშნა არ გახდება. მაგალითად, ასეთი მიმდევრობის 978-ე წევრი თავდაპირველად 978- ის ტოლია. შემდეგ იგი შეიცვლება მისი ციფრთა ჯამით: 9+7+8=24. ამის შემდეგ კი მიღებული რიცხვი, ანუ 24, კვლავ შეიცვლება მისი ციფრთა ჯამით: 2+4=6. ანასტასიამ გადაწყვიტა გამოცდა მოუწყოს მარიამს მათემატიკის ცოდნაში და აძლევს მას $Q$ რაოდენობის ერთნაირი ტიპის შეკითხვებს: რას უდრის რიცხვთა ახალ მიღებულ მიმდევრობაში $S$ და $F$ პოზიციებს შორის მდგომი წევრების ჯამი? (თვით $S$ და $F$ პოზიციების ჩათვლით).
 დაეხმარეთ მარიამს და დაწერეთ პროგრამა, რომელიც პასუხს გასცემს ანასტასიას ყველა ასეთ შეკითხვას.

## Input

სტანდარტული შეტანის პირველ სტრიქონში ჩაწერილია ერთი მთელი დადებითი $Q$ რიცხვი - ანასტასიას მიერ დასმული შეკითხვების რაოდენობა. მომდევნო $Q$ რაოდენობის სტრიქონიდან თითოეულში მოცემულია თითო ჰარით გამოყოფილი ორი ნატურალური $S_i$ და $F_i$ რიცხვი - $i-$ურ შეკითხვაში დასახელებული პოზიციების ნომრები.
$1<=Q<=100$
$1<=S_i<=F_i<=2^60$

## Output

სტანდარტულ გამოტანაში ცალ-ცალკე სტრიქონებში უნდა გამოიტანოთ ანასატასიამ მიერ დასმულ შეკითხვებზე პასუხები იმ მიმდევრობით, რა მიმდევრობითაც იგი ამ შეკითხვებს სვამს.
