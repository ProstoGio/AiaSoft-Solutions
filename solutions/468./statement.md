# Problem Statement — 468. ირაკლის კვადრატი

> Copied from [AiaSoft.ge #468](https://aiasoft.ge/problem/468)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

ირაკლი პატარა მათემატიკოსია. მას განსაკუთრებით უყვარს გეომეტრია და გეომეტრიული ფიგურები. ამ ეტაპზე მან გადაწყვიტა მართკუთხედის ფორმის ქაღალდისგან კვადრატი გამოჭრას. ირაკლის ფურცლის ნებისმიერი მიმართულებით გაჭრა შეუძლია.

დაადგინეთ მაქსიმუმ რა ფართობის მქონე კვადრატს გამოჭრის ირაკლი.

## Input

ერთ ხაზზე მოცემულია ორი მთელი რიცხვი: $N$ და $M$ - ქაღალდის გვერდების სიგრძეები. $1 <= N, M <= 1000$

## Output

გამობეჭდეთ ერთი მთელი რიცხვი: კვადრატის მაქსიმალური ფართობი, რომლის გამოჭრაც შესაძლებელია ფურცლისგან.
