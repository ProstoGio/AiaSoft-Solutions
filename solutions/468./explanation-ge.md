# განმარტება — 468. ირაკლის კვადრატი

## მიდგომა

აქ არაფერია რთული, უბრალოდ ლოგიკას უნდა მიჰყვე. მართკუთხედიდან კვადრატს რომ ჭრი, მისი გვერდი ვერასდროს იქნება მართკუთხედის მოკლე გვერდზე დიდი, თორემ ფურცელს გასცდება. ამიტომ, პოულობ მინიმუმს N-სა და M-ს შორის და უბრალოდ კვადრატში სვამ.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- კვადრატის გვერდის მაქსიმალური სიგრძე ყოველთვის ტოლია ფურცლის ყველაზე პატარა გვერდის.
- ფართობი არის გვერდი გამრავლებული თავის თავზე, ანუ min(N, M)^2.

## ხშირი შეცდომები

- ზედმეტი პირობების (if-else) წერა იქ, სადაც `min()` ფუნქცია იდეალურად აკეთებს საქმეს.
