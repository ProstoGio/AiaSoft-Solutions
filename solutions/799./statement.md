# Problem Statement — 799. კარგი სამეული

> Copied from [AiaSoft.ge #799](https://aiasoft.ge/problem/799)

## Task

$(a, b, c)$ სამეულს ვუწოდოთ ,,`bკარგი`'', თუ მათ შორის რომელიმე ორი ერთმანეთის ტოლია, ხოლო მესამე მათი არატოლი. მაგალითად, სამეული $(1, 1, 2)$ ,,`bკარგი`'' სამეულია. ხოლო სამეულები $(5, 5, 5)$ და $(4, 5, 1)$ არა.

მოცემულია სამი ნატურალური რიცხვი, დაადგინეთ არის თუ არა ის ,,`bკარგი`'' სამეული

## Input

პირველ ხაზზე შემოდის ჰარით გამოყოფილი სამი ნატურალური რიცხვი $a$, $b$ და $c$

$1<=a,b,c<=100$

## Output

გამობეჭდეთ Yes თუ მოცემულია ,,`bკარგი`''  სამეული, წინააღმდეგ შემთხვევაში - No
