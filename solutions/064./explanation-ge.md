# განმარტება — 064. ფრჩხილებიანი მეორეკლასელები

## მიდგომა

ამ ამოცანის ამოხსნისას პირველი რაც თავში მოდის არის სტეკის გამოყენება, მაგრამ ვინაიდან მხოლოდ ერთი ტიპის ფრჩხილი გვაქვს, უბრალო ინტეგერ ცვლადითაც მშვენივრად გავართმევთ თავს. მთელი იდეა იმაშია, რომ თითოეულ გახსნილ ფრჩხილზე counter-ს ვზრდით, ხოლო დახურულზე — ვაკლებთ. თუ პროცესში counter-ი ნულზე დაბლა ჩამოვიდა, ესე იგი დახურული ფრჩხილი მანამდე დადგა, ვიდრე გახსნილი, რაც პირდაპირ 'NO'-ს ნიშნავს. ბოლოს მთავარია counter ზუსტად ნული იყოს, თორემ ესე იგი რაღაც ფრჩხილი დარჩა გაუხსნელი ან პირიქით.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- სტეკის ნაცვლად ერთი მთელი რიცხვის გამოყენება მეხსიერებას ზოგავს და კოდსაც უფრო ამარტივებს.
- თუ გამოსახულების რომელიმე წერტილში დახურული ფრჩხილების რაოდენობა გახსნილს აჭარბებს, გამოსახულება ავტომატურად მცდარია.

## ხშირი შეცდომები

- ბოლოში იმის შემოწმების გამორჩენა, უდრის თუ არა counter ნულს.
