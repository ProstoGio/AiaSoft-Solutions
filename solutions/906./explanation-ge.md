# განმარტება — 906. ფიფქების სამკუთხედი -1

## მიდგომა

ეს ამოცანა ჩვეულებრივი nested loop-ების თემაა. იდეა მარტივია: გვინდა ფიფქების სამკუთხედი, რომელიც ზემოთ ფართოა და ნელ-ნელა ვიწროვდება. ამიტომ, გარე ციკლი i-თი მიყვება სტრიქონებს, შიდა ციკლი კი n-დან i-მდე ტრიალებს და ბეჭდავს ვარსკვლავებს. როგორც კი ხაზს დავამთავრებთ, გადავდივართ ახალ ხაზზე და ვიმეორებთ პროცესს.

## სირთულე

| | |
|-|--|
| დრო | O(n²) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- შიდა ციკლის საზღვრები უნდა იყოს n და i, რათა თითოეულ ნაბიჯზე ერთი ვარსკვლავით ნაკლები დაიბეჭდოს.
- მარტივი მათემატიკაა: პირველ ხაზზე გვინდა n ცალი, მეორეზე n-1 და ასე შემდეგ, სანამ 1-მდე არ ჩამოვალთ.

## ხშირი შეცდომები

- შიდა ციკლის პირობის არასწორად დაწერა, რის გამოც შეიძლება ან ზედმეტი ვარსკვლავები დაიბეჭდოს, ან პირიქით.
