# Problem Statement — 860. მასივის კენტები

> Copied from [AiaSoft.ge #860](https://aiasoft.ge/problem/860)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია ნატურალური რიცხვების ერთგანზომილებიანი მასივი. დავადგინოთ მასივის კენტი ელემენტების `bრაოდენობა` და `bჯამი`.
თუ მასივი კენტ ელემენტებს არ შეიცავს, გამოიტანეთ -1.

## Input

პირველი სტრიქონზე ჩაწერილია მასივის  სიგრძე $N, 0<N<100$.
მომდევნო სტრიქონზე მოცემულია ერთმანეთისგან ჰარით გამოყოფილი მასივის ელემენტები.

## Output

ერთ სტრიქონზე გამოიტანეთ ერთმანეთისაგან ჰარით გამოყოფილი ორი ნატურალური რიცხვი - კენტი ელემენტების რაოდენობა და ჯამი.
თუ კენტი ელემენტები არ არსებობს, მაშინ გამოიტანეთ -1.
