# განმარტება — 860. მასივის კენტები

## მიდგომა

აქ არაფერია რთული, უბრალოდ მასივში ვინახავ რიცხვებს და მერე ერთ ციკლს ვატარებ მათზე. თითოეულ რიცხვს ვამოწმებ მოდულით, თუ კენტია, პირდაპირ ვუმატებ ჯამს და რაოდენობის მთვლელს ვზრდი. ბოლოს, თუ 't' ცვლადი ისევ ნულია, ესე იგი მასივში კენტი რიცხვი არ გვქონია და პირდაპირ -1-ს ვბეჭდავ.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- კენტი რიცხვის გამოსავლენად `a[i] % 2 != 0` ყველაზე ოპტიმალური გზაა.
- ერთი დამხმარე ცვლადის შემოღება, რომელიც ითვლის რამდენჯერ ვიპოვეთ კენტი რიცხვი, გვიმარტივებს პირობის შემოწმებას ბოლოს.

## ხშირი შეცდომები

- მთავარია -1-ის შემთხვევა არ გამოგვრჩეს, თუ მასივში არცერთი კენტი რიცხვი არ არის.
