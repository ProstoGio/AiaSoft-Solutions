# Problem Statement — 667. ბაბიმ თვლა ისწავლა

> Copied from [AiaSoft.ge #667](https://aiasoft.ge/problem/667)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ბაჭია ბაბიმ უკვე იცის სამამდე თვლა, რა თქმა უნდა, ამ ციფრების წერაც ისწავლა. დაიწყო სხვადასხვა სიგრძის სტრიქონების ჩაწერა ციფრებით 1, 2 და 3. ოღონდ, ფიქრობს, რომ ჩანაწერი იმ შემთხვევაში იქნება `bლამაზი`, თუ ციფრი 2 მოთავსებული იქნება აუცილებლად 1-სა და 3-ს შორის ან პირიქით, 3 - სა და 1-ს შორის, ან საერთოდ არ იქნება. ამასთანავე, ორი ერთნაირი ციფრი ერთმანეთის გვერდით არ უნდა იდგეს.
მაგალითად, თუ სტრიქონის სიგრძე არის 3, მაშინ შესაძლებელია შემდეგი სახის `bლამაზი` სტრიქონების ჩაწერა: 123, 321, 131, 313.
ბაბის აინტერესებს, სხვადასხვა სიგრძის რამდენი `bლამაზი` სტრიქონის წაწერაა შესაძლებელი.

## Input

შემოდის სრიქონის სიგრძე n, 1<=n<=30.

## Output

გამოიტანეთ ერთი რიცხვი, n სიგრძის ლამაზი სტრიქონების რაოდენობა.
