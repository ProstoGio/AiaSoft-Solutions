# განმარტება — 689. ფერადი კარტები

## მიდგომა

ეს პრობლემა იმაზე მარტივია, ვიდრე ერთი შეხედვით ჩანს. ნიკოს აქვს a რაოდენობის წითელი და b რაოდენობის ლურჯი კარტი, ხოდა მაგიდაზე ორივე ფერის ერთად დასაწყობად უბრალოდ ამ ორი რიცხვის ჯამი გვჭირდება. მეტი არაფერი, უბრალო მიმატებაა მთელი ამბავი.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- მაგიდის ტევადობა არის უბრალოდ a და b-ს ჯამი.
- რიცხვები იმდენად პატარაა, რომ int ტიპი სრულიად საკმარისია და overflow-ს არ უნდა ველოდოთ.

## ხშირი შეცდომები

- ზედმეტი ფიქრი და რაღაც რთული ლოგიკის ძებნა, როცა პასუხი უბრალოდ a+b-ა.
