# ამოხსნის ახსნა

მოცემულია ორნიშნა რიცხვი.

საჭიროა ვიპოვოთ მისი უდიდესი ციფრი.

---

# მთავარი იდეა

ორნიშნა რიცხვი შედგება ორი ციფრისგან:

- ათეულების ციფრი;
- ერთეულების ციფრი.

ჯერ გამოვყოფთ ამ ორ ციფრს, შემდეგ კი ერთმანეთს შევადარებთ.

---

# კოდის განხილვა

## რიცხვის წაკითხვა

```cpp
int a, b, c;
cin >> a;
```

`a` ინახავს შემოსულ ორნიშნა რიცხვს.

---

## ციფრების გამოყოფა

```cpp
b = a / 10;
c = a % 10;
```

- `a / 10` გვაძლევს ათეულების ციფრს.
- `a % 10` გვაძლევს ერთეულების ციფრს.

მაგალითად, თუ:

```text
a = 84
```

მაშინ:

```text
b = 8
c = 4
```

---

## ციფრების შედარება

```cpp
if(b > c)
    cout << b;
else
    cout << c;
```

თუ ათეულების ციფრი უფრო დიდია, იბეჭდება `b`.

წინააღმდეგ შემთხვევაში იბეჭდება `c`.

---

# რატომ მუშაობს?

რადგან ორნიშნა რიცხვს მხოლოდ ორი ციფრი აქვს, მათი გამოყოფის შემდეგ საკმარისია უბრალოდ შევადაროთ ერთმანეთს და გამოვიტანოთ უფრო დიდი.
