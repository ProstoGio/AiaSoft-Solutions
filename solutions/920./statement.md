# Problem Statement — 920. წყალდიდობა

> Copied from [AiaSoft.ge #920](https://aiasoft.ge/problem/920)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ჭიანჭველების მიწისყვეშა ქალაქში წყალდიდობის შედეგად რამდენიმე გვირაბი ჩამოიშალა, რის გამოც კომუნიკაცია შეწყდა მრავალ დარბაზს შორის.
დაწერეთ პროგრამა რომელიც გაარკვევს მინიმუმ რამდენი გვირაბი უნდა აღდგეს, რათა შესაძლებელი გახდეს ნებისმიერი დარბაზიდან ნებისმიერ დარბაზში გადასვლა.

## Input

პირველ ხაზზე შემოდის დარბაზებისა და გვირაბების რაოდენობა, $N<=50, M<=N*(N-1)/2$
ყოველ მომდევნო სტრიქონზე შემოდის ორი მთელი რიცხვი - იმ დარბაზების ნომრები, რომელთაც გვირაბი აერთებს.

## Output

გამოიტანეთ ერთი მთელი რიცხვი - იმ გვირაბების რაოდენობა, რომელთ აღდგენაცაა საჭირო.
