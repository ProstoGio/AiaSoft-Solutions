# Problem Statement — 265. უდიდესი კომპონენტი

> Copied from [AiaSoft.ge #265](https://aiasoft.ge/problem/265)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

მოცემულია არაორიენტირებული გრაფი. დაადგინეთ უდიდეს კომპონენტში შემავალი წვეროების რაოდენობა.

## Input

პირველ ხაზზე შემოდის N და M, შესაბამისად გრაფში წვეროებისა და წიბოების რაოდენობა.
მომდევნო M ხაზზე მოცემულია ორ-ორი რიცხვი. კერძოდ i-ურ ხაზზე შემოვა vi და ui - მეზობელი წვეროების ნომრები. (გრაფის წვეროები დანომრილია 1-დან N-მდე)

1 <= N <= 20
0 <= M <= N*(N-1) / 2
1 <= vi, ui <= N

## Output

გამოიტანეთ ერთი მთელი რიცხვი: უდიდეს კომპონენტში შემავალი წვეროების რაოდენობა.
