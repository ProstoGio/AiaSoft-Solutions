# Problem Statement — 337. რეგისტრის ცვლილება

> Copied from [AiaSoft.ge #337](https://aiasoft.ge/problem/337)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია ლათინური ანბანის რამდენიმე სიმბოლო. საჭიროა მათი რეგისტრის შეცვლა. ანუ, თუ სიმბოლო მთავრულია, უნდა დაპატარავდეს, და პირიქით, თუ პატარა ასოა, უნდა გამთავრულდეს.

## Input

პირველ სტრიქონზე შემოდის ერთი ნატურალური რიცხვი - სიმბოლოების რაოდენობა N, 1<=N<=20.
მომდევნო N რაოდენობის სტრიქონიდან თითოეულზე შემოდის ლათინური ანბანის ერთი სიმბოლო.

## Output

გამოიტანეთ N რაოდენობის სტრიქონი. თითოეულ მათგანზე უნდა გამოიბეჭდოს ლათინური ანბანის ერთი სიმბოლო - შესაბამისი შემომავალი სიმბოლო შეცვლილი რეგისტრით (პატარა ასო უნდა გამთავრულდეს და პირიქით).
