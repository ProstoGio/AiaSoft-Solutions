# განმარტება — 287. არჩევნები

## მიდგომა

არჩევნებში მოსაგებად უბრალოდ ნახევარზე მეტი ხმა გჭირდება. თუ N ადამიანი აძლევს ხმას, შენი გამარჯვება გარანტირებულია, როგორც კი N/2-ზე ერთით მეტ ხმას აიღებ. C++-ში მთელი რიცხვების გაყოფა ისედაც ამრგვალებს ქვემოთ, ასე რომ n/2 + 1 პირდაპირ გვაძლევს პასუხს, რაც ძალიან ამარტივებს კოდს.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- C++-ის integer division n/2-ს ავტომატურად ამრგვალებს ქვემოთ, რაც ჩვენს სასარგებლოდ მუშაობს.
- ფორმულა n/2 + 1 უნივერსალურია როგორც ლუწი, ისე კენტი N-ისთვის.

## ხშირი შეცდომები

- ცალ-ცალკე if-ების წერა ლუწი და კენტი შემთხვევებისთვის, როცა ერთი ხაზით გვარდება ყველაფერი.
