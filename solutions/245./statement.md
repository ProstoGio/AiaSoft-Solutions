# Problem Statement — 245. ძაღლების თავშესაფარი

> Copied from [AiaSoft.ge #245](https://aiasoft.ge/problem/245)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

ირაკლის ძალიან უყვარს ძაღლები. მას უპატრონო ძაღლების თავშესაფარი აქვს და ამ ეტაპზე სულ $n$ ძაღლი ჰყავს. ცნობილია, რომ თითოეული ძაღლი კვირაში $k$ კგ პედიგრს მიირთმევს.

ირაკლის მარაგში $m$ კგ პედიგრი აქვს.
დაეხმარეთ მას გამოთვალოს, მერამდენე კვირას დასჭირდება მარაგის შევსება.

* `iმარაგის შევსება საჭიროა მხოლოდ მაშინ, როცა მოცემული კვირის სამყოფი პედიგრი აღარ დარჩება`

## Input

ერთ ხაზზე შემოდის სამი მთელი რიცხვი: $n, k$ და $m$.
$1 <= n, k, m <= 100$

## Output

გამოიტანეთ ერთი მთელი რიცხვი, იმ კვირის რიგირთი ნომერი, როცა მარაგის შევსება აუცილებელი იქნება.
