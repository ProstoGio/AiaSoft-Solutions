# ამოხსნის ახსნა

მოცემულია ერთი რიცხვი `N`.

საჭიროა გამოვთვალოთ:

```text
1 ⊕ 2 ⊕ 3 ⊕ ... ⊕ N
```

სადაც `⊕` აღნიშნავს ბიტურ **XOR** ოპერაციას.

---

# მაგალითი

თუ:

```text
N = 4
```

მაშინ უნდა გამოვთვალოთ:

```text
1 ⊕ 2 ⊕ 3 ⊕ 4
```

ეტაპობრივად:

```text
1 ⊕ 2 = 3
3 ⊕ 3 = 0
0 ⊕ 4 = 4
```

პასუხია:

```text
4
```

---

# მთავარი იდეა

პასუხი თავიდან უდრის:

```cpp
1
```

შემდეგ სათითაოდ ვამატებთ ყველა შემდეგ რიცხვს XOR ოპერაციით.

---

# კოდის განხილვა

## რიცხვის წაკითხვა

```cpp
int n;
cin >> n;
```

---

## საწყისი მნიშვნელობა

```cpp
int answer = 1;
```

რადგან გამოსახულება იწყება რიცხვით `1`, საწყისი პასუხიც `1` იქნება.

---

## XOR-ის გამოთვლა

```cpp
for(int i=2; i<=n; i++){
    answer ^= i;
}
```

ციკლი გადის `2`-დან `N`-ის ჩათვლით.

ყოველ ნაბიჯზე სრულდება:

```cpp
answer = answer ^ i;
```

ან მოკლედ:

```cpp
answer ^= i;
```

ასე ეტაპობრივად ვითვლით:

```text
1 ⊕ 2
```

შემდეგ:

```text
1 ⊕ 2 ⊕ 3
```

შემდეგ:

```text
1 ⊕ 2 ⊕ 3 ⊕ 4
```

და ასე გაგრძელდება `N`-მდე.

---

## პასუხის გამოტანა

```cpp
cout << answer;
```

ბოლოს იბეჭდება მიღებული XOR-ის მნიშვნელობა.

---

# რატომ მუშაობს?

XOR ოპერაცია შეგვიძლია თანმიმდევრულად გამოვთვალოთ.

თუ უკვე ვიცით:

```text
1 ⊕ 2 ⊕ ... ⊕ k
```

მომდევნო რიცხვის დამატება ძალიან მარტივია:

```text
(1 ⊕ 2 ⊕ ... ⊕ k) ⊕ (k+1)
```

ზუსტად ამას აკეთებს ციკლი.

როდესაც ყველა რიცხვი დამუშავდება, `answer` შეიცავს საბოლოო პასუხს.

---

# სირთულე

### დროითი სირთულე

```text
O(N)
```

რადგან ციკლი ერთხელ გადის ყველა რიცხვზე.

---

### მეხსიერების სირთულე

```text
O(1)
```

გამოიყენება მხოლოდ ერთი დამატებითი ცვლადი.

---

# შეჯამება

პროგრამა იწყებს რიცხვით `1` და შემდეგ `2`-დან `N`-ის ჩათვლით ყველა რიცხვს ეტაპობრივად ამატებს XOR ოპერაციით.

ციკლის დასრულების შემდეგ მიღებული მნიშვნელობა სწორედ:

```text
1 ⊕ 2 ⊕ ... ⊕ N
```

გამოსახულების პასუხია.

---

# შენიშვნა

ამ ამოცანის ამოხსნა შესაძლებელია `O(1)` დროშიც, რადგან `1`-დან `N`-მდე რიცხვების XOR ოთხი მნიშვნელობის ციკლით მეორდება (`N % 4`-ის მიხედვით).

თუმცა მოცემულ შეზღუდვებში (`N ≤ 10⁵`) ხაზოვანი (`O(N)`) ამოხსნაც სრულიად საკმარისია და მარტივად გასაგებია. ( იხ. ამოცანა `608. რიცხვების XOR - 2` ).
