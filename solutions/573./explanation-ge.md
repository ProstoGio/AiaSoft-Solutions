# განმარტება — 573. სუნთქვა

## მიდგომა

თავიდან მინდოდა ციკლი დამეწერა, მაგრამ მერე მივხვდი, რომ 1-დან 100-მდე ეგრე წვალება ზედმეტია. სინამდვილეში, ზუკა 4-ნაბიჯიან ციკლებს აკეთებს: 1-2 ჩასუნთქვა, 3-4 ამოსუნთქვა. თუ N-ს 4-ზე გავყოფთ, ნაშთი პირდაპირ გვეუბნება, სად გავჩერდით. 2-ზე რომ გავჩერდეთ, ე.ი. ჩასუნთქვა დავამთავრეთ, 3-ზეც იგივეა, რადგან ჩასუნთქვა ჯერ კიდევ მეტჯერაა შესრულებული.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- პატერნი მეორდება 4-ზე, ამიტომ N % 4 საკმარისია იმის გასაგებად, თუ რომელი ფაზის ბოლოს ვართ.
- თუ ნაშთი 2 ან 3-ია, ეს ნიშნავს რომ ჩასუნთქვები ამოსუნთქვებზე მეტია.

## ხშირი შეცდომები

- ზედმეტი if-else განშტოებების წერა მაშინ, როცა მათემატიკურად პირდაპირ ჩანს პასუხი.
