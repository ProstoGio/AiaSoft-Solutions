# Problem Statement — 798. დათვალე ბურთები

> Copied from [AiaSoft.ge #798](https://aiasoft.ge/problem/798)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

თარაშს უამრავი წითელი და ლურჯი ბურთი აჩუქეს და გადაწყვიტა ისინი მწკრივში განალაგოს. თავდაპირველად მწკრივი ცარიელია და რადგან თარაში ძალიან მომთმენია, იგი აპირებს მწრივში განალაგოს მინიმუმ 10¹⁰⁰ ცალი ბურთი შემდეგი პრინციპით:

● იღებს a ცალ ლურჯ ბურთს და ალაგებს მწკრივში უკვე დალაგებული ბურთების მარჯვნივ
● შემდეგ b ცალ წითელ ბურთს და ალაგებს ისევ მწკრივის მარჯვნივ
● იმეორებს პირველ ორ ნაბიჯს სანამ მინიმუმ 10¹⁰⁰ ბურთი არ დალაგდება

თქვენი დავალებაა დაადგინოთ რამდენი ცალი ლურჯი ბურთი იქნება განლაგებული პირველ n ცალ ბურთს შორის

## Input

პირველ ხაზზე შემოდის ჰარით გამოყოფილი სამი მთელი რიცხვი n, a და b

1≤n≤10¹⁸
a, b>=0
0<a+b<=10¹⁸

## Output

გამობეჭდეთ ერთი ნატურალური რიცხვი, მწკრივში პირველ n ბურთში ლურჯი ბურთების რაოდენობა.
