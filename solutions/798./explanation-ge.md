# განმარტება — 798. დათვალე ბურთები

## მიდგომა

ეს ამოცანა სუფთა მათემატიკაა, უბრალოდ ციფრებია დიდი. რადგან 'a' ლურჯი და 'b' წითელი ბურთები რიგრიგობით ლაგდება, ჩვენი პატერნი მეორდება ყოველ 'a + b' ბურთზე. ჯერ ვთვლით რამდენი სრული ციკლი ეტევა n-ში და ამას ვამრავლებთ a-ზე. ბოლოს, თუ ნაშთი დაგვრჩა, ვუყურებთ, ეს ნაშთი 'a'-ზე მეტია თუ არა; თუ მეტია, მაშინ მთლიანი 'a' ლურჯი ბურთი შევა, თუ ნაკლები - მაშინ რამდენი ნაშთიც გვაქვს, იმდენი.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- სრული ციკლების რაოდენობა ვიპოვოთ n // (a + b)-ით და გავამრავლოთ a-ზე.
- ნაშთის ნაწილი დავამატოთ min(r, a) ლოგიკით, რათა დავიცვათ ბურთების ზუსტი რაოდენობა.

## ხშირი შეცდომები

- n-ის ძალიან დიდობის გამო 'long long'-ის გამოყენების დავიწყება.
