# Problem Statement — 472. ოთახის ფართობი

> Copied from [AiaSoft.ge #472](https://aiasoft.ge/problem/472)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია $N$x$N$ ზომის მატრიცა, რომელიც მხოლოდ '#' და '.' სიმბოლოებს შეიცავს.
მოსაზღვრე `bწერტილების` ერთობლიობა მატრიცაში ქმნის ოთახებს. ორი წერტილი მხოლოდ იმ შემთხვევაში ითვლება მოსაზღვრედ, თუ მათ საერთო გვერდი აქვთ.

თქვენი დავალებაა იპოვოთ უდიდესი ფართობის მქონე ოთახი მატრიცაში.
ოთახის ფართობი მასში არსებული წერტილების რაოდენობით განისაზღვრება.

## Input

პირველ ხაზზე მოცემულია ერთი მთელი რიცხვი: $N$ - მატრიცის ზომა $(1 <= N <= 150)$.
მომდევნო $N$ სტრიქონზე მოცემულია $N$ სიმბოლო - მატრიცის აწერა.

## Output

გამოიტანეთ ერთი მთელი რიცხვი - უდიდესი ოთახის ფართობი მატრიცაში.
