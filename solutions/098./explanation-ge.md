# განმარტება — 098. მაქსიმალური სხვაობა

## მიდგომა

ამოცანის არსი იმაშია, რომ დავალაგოთ რიცხვები ზრდადობით და შევადაროთ მეზობელი ელემენტები. როგორც კი მასივს დაასორტირებ, თითოეული რიცხვისთვის მომდევნო 'უმცირესი მეტი' რიცხვი ავტომატურად გვერდითა წევრი ხდება. შესაბამისად, უბრალოდ ვამოწმებთ სხვაობას a[i+1] და a[i]-ს შორის მთელ მასივზე და ვინახავთ მაქსიმალურს. სულ ეს არის, მეტი არაფერი სჭირდება.

## სირთულე

| | |
|-|--|
| დრო | O(n log n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- სორტირება მთელ სირთულეს გვიხსნის, რადგან აღარ გვჭირდება თითოეულ რიცხვზე ხელახლა ძებნა.
- თუ მასივში დუბლიკატი რიცხვებია, სხვაობა იქნება 0, რაც მაქსიმუმის ძებნაში ხელს არ გვიშლის.

## ხშირი შეცდომები

- შეიძლება ვინმეს მოუვიდეს აზრად O(n²) მიდგომა, სადაც თითოეული რიცხვისთვის ეძებს მასზე დიდს მთელ მასივში, რაც დროში ვერ ჩაეტია.
- დავიწყებოდათ `sort`-ის გაკეთება და არეულ მასივზე სხვაობების ძებნა.
