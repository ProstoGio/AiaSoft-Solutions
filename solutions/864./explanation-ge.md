# განმარტება — 864. დაითვალე ლუწები

## მიდგომა

ეს ამოცანა არის კლასიკური 'warm-up' ტიპის ამოცანა, სადაც ზედმეტი სირთულეების გარეშე უნდა დავითვალოთ ლუწი რიცხვები. საერთოდ არ არის საჭირო მთელი მასივის მეხსიერებაში შენახვა, რადგან რიცხვებს ერთმანეთის მიყოლებით ვკითხულობთ და მაშინვე შეგვიძლია შევამოწმოთ პირობა. ერთი ცვლადი, `cnt` (count), სავსებით საკმარისია პასუხის გასათვლელად.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- მასივის გამოცხადება `int a[n]` დამატებით მეხსიერებას იკავებს, რაც ამ ამოცანაში საჭირო არაა; ჯობია პირდაპირ `cin`-ით წავიკითხოთ რიცხვი და მაშინვე შევამოწმოთ.
- თუ რიცხვის `2`-ზე გაყოფისას ნაშთი 0-ია, ის ლუწია - ეს არის მთელი ლოგიკა.

## ხშირი შეცდომები

- იმის მცდელობა, რომ ჯერ მთელი მასივი შევავსოთ და მერე ცალკე ციკლით დავთვალოთ ლუწები, როცა ეს ყველაფერი ერთ ციკლში კეთდება.
