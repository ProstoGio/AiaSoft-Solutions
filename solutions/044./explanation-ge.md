# განმარტება — 044. კუბიკების კიბე

## მიდგომა

ამ ამოცანის არსი არის რიცხვის ისეთი დაშლა, სადაც თითოეული ნაწილი წინაზე ნაკლებია. DP-ში `d[i][j]` აღნიშნავს რამდენი გზით შეგვიძლია ავაწყოთ კიბე `i` კუბიკით ისე, რომ ზედა საფეხური ზუსტად `j` კუბიკისგან შედგებოდეს. შემდეგ საფეხურზე გადასვლისას უბრალოდ ვუმატებთ `j`-ს ისეთ კომბინაციებს, სადაც ზედა საფეხური `j`-ზე პატარა იყო — ამით ავტომატურად ვინარჩუნებთ კლებადობის პირობას.

## სირთულე

| | |
|-|--|
| დრო | O(n³) |
| მეხსიერება | O(n²) |

## მთავარი დაკვირვებები

- DP-ის მდგომარეობა უნდა ინახავდეს არა მარტო დარჩენილი კუბიკების რაოდენობას, არამედ ზედა საფეხურის ზომასაც, რომ კლებადობა არ დაირღვეს.
- ბოლო პასუხისთვის უბრალოდ ვკრებთ `d[n][i]` მნიშვნელობებს ყველა შესაძლო ზედა საფეხურის ზომისთვის.

## ხშირი შეცდომები

- პირველი საფეხურის იგნორირება, სადაც შესაძლოა მთლიანი N კუბიკი ერთ საფეხურად გამოიყენო.

## ნაბიჯ-ნაბიჯ ახსნა

მაგალითად, n=5-ისთვის: თუ ზედა საფეხურზე 2 კუბიკი გვაქვს, დაგვრჩა 3 კუბიკი. უნდა ვიპოვოთ ყველა ისეთი კიბე, სადაც ჯამი 3-ია და ზედა საფეხური 2-ზე პატარაა (ანუ 1-ის ტოლი). ამიტომ d[5][2] ივსება d[3][1]-დან.
