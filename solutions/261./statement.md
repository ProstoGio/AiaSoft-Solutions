# Problem Statement — 261. მეზობლების მატრიცა

> Copied from [AiaSoft.ge #261](https://aiasoft.ge/problem/261)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

არაორიენტირებული გრაფი მოცემულია მეზობელთა წყვილების სახით. გარდაქმენით იგი წვეროთა მეზობლობის მატრიცად.

`i*გარანტირებულია, რომ გრაფი არ შეიცავს ორმაგ წიბოებს.`

## Input

პირველ ხაზზე შემოდის N და M, შესაბამისად გრაფში წვეროების და წიბოების რაოდენობა.
მომდევნო M ხაზზე მოცემულია ორ-ორი რიცხვი. კერძოდ i-ურ ხაზზე შემოვა vi და ui - მეზობელი წვეროების ნომრები. (გრაფის წვეროები დანომრილია 1-დან N-მდე)

1 <= N <= 20
0 <= M <= N*(N-1) / 2
1 <= vi, ui <= N

## Output

გამობეჭდეთ NxN მატრიცა, გრაფის წვეროების მეზობლობის მატრიცული აღწერა.
რიცხვები გამოყავით ჰარით.

მატრიცის i,j უჯრაში უნდა ეწეროს 1 თუ გრაფის i და j ნომერი წვერო მეზობელია, წინააღმდეგ შემთხვევაში 0.
