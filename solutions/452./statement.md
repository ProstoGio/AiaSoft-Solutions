# Problem Statement — 452. კვადრების ქვეყანა

> Copied from [AiaSoft.ge #452](https://aiasoft.ge/problem/452)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ერთ კვადრატულ ქვეყანაში ცხოვრობდნენ კვადრატული ადამიანები და ყველაფერი დანარჩენიც იყო კვადრატული. ამრიგად, კვადრატულმა პარლამენტმა მიიღო კვადრატული მიწის კანონი. ამ კანონის თანახმად, ადამიანს შეეძლო მხოლოდ კვადრატული მიწის შეძენა. კვადრატული მიწის სიგრძეც და სიგანეც უნდა ყოფილიყო მთელი დადებითი რიცხვები. თუ კვადრატული ადამიანი შეიძენდა მიწას, რომლის სიგრძეც იყო a მეტრი, მაშინ ის გადაიხდიდა a² კვადრს.


სანდრომ, რომელიც კვადრატული ქვეყნის მოქალაქეა, გადაწყვიტა მთელი მისი N კვადრით ეყიდა მიწა. მას რა თქმა უნდა შეეძლო 1x1 ზომის მიწის ყიდვა, მაგრამ მას სურს, რომ მის მიერ ნაყიდი მიწების რაოდენობა რაც შეიძლება ცოტა გამოვიდეს.

სანდრომ დახმარებისთვის თქვენ მოგმართათ, დაეხმარეთ მას და უთხარით თუ მინიმუმ რამდენი მიწის ყიდვა მოუწევს a კვადრით, ცხადია ფული ბოლომდე უნდა დაიხარჯოს.

## Input

ერთ ხაზზე შემოდის N - კვადრების რაოდენობა, რომლითაც სანდრომ გადაწყვიტა მიწის ყიდვა კვადრების ქვეყანაში.
1<=N<=60 000

## Output

ერთ ხაზზე გამობეჭდეთ იმ მინიმალური მიწების რაოდენობა, რომლებიც შეგვიძლია ვიყიდოთ N კვადრად.
