# Problem Statement — 221. ქალაქი ჩე

> Copied from [AiaSoft.ge #221](https://aiasoft.ge/problem/221)

**Time limit:** 2.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ქალაქ ჩეს ცენტრში საფეხმავლო ქუჩაა - მოსეირნეთა ერთ-ერთი ყველაზე პოპულარული ადგილი. აქ სეირნობა ძალიან სასიამოვნოა, რადგან ამ ქუჩაზე  $n$  ცალი საინტერესო ძეგლია.

გოგონას თავისი ორი სკოლელი ვაჟი მოსწონს, მაგრამ მათ შორის არჩევანს ვერ აკეთებს. ბოლოს გადაწყვიტა, ორივესთვის ერთსა და იმავე დროს დაენიშნა შეხვედრა, გოგონას უნდა ორი ძეგლის არჩევა, სადაც ბიჭები დაელოდებიან. ამასთან ეს ძეგლების განლაგება ისეთი უნდა იყოს, რომ ბიჭებმა ერთმანეთი ვერ შენიშნონ. ნისლის გამო ბიჭები ერთმანეთს დაინახავენ მხოლოდ იმ შემთხვევაში, თუ მათ შორის მანძილი არ აღემატება $r$ მეტრს.
                                                                                             
გოგონას აინტერესებს, ორი ძეგლის არჩევის რამდენი ვარიანტი აქვს მას.

## Input

პირველ სტრიქონსი ორი მთელი რიცხვია - $n$ და $r$ ($2 \le n \le 300000$, $1 \le r \le 10^9$) - ძეგლების რაოდენობა და მაქსიმალური მანძილი, რომლის იქეთაც ბიჭები ერთმანეთს ვერ დაინახავენ.

მეორე სტრიქონში მოცემულია  $n$  დადებითი რიცხვი  $d_1, \ldots, d_n$, სადაც $d_i$ - არის $i$ ნომერი ძეგლის დაშორება ქუჩის დასაწყისიდან. ყველა ძეგლი გასხვავებულ ადგილზე დგას. ძეგლების მანძილები მოცემულია მნიშვნელობების ზრდის მიხედვით ($1 \le d_1 < d_2 < \ldots < d_n \le 10^9$).

## Output

გამოიტანეთ ერთი რიცხვი - ორი ძეგლის არჩევის ვარიანტების რაოდენობა.
