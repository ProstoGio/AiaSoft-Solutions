# Problem Statement — 212. სიმბოლოთა ჯაჭვი

> Copied from [AiaSoft.ge #212](https://aiasoft.ge/problem/212)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია ლათინური ანბანის ორი მთავრული ასო. საჭიროა დაიბეჭდოს ყველა სიმბოლო პირველი ასოდან მეორეს ჩათვლით, ანბანის მიხედვით, თანმიმდევრობით.

## Input

ერთ სტრიქონზე შემოდის ერთმანეთისაგან ჰარით გამოყოფილი ორი ლათინური მთავრული ასო.

## Output

გამობეჭდეთ ყველა სიმბოლო პირველი შემოსული სიმბოლოდან ბოლო სიმბოლოს ჩათვლით, თანმიმდევრობით. სიმბოლოები ერთმანეთისაგან ერთი ჰარით გამოყავით.
