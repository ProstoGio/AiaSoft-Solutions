# Problem Statement — 571. ხაჭაპური

> Copied from [AiaSoft.ge #571](https://aiasoft.ge/problem/571)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ხაჭაპური, რომელიც წრიული ფორმისაა, $N$-ჯერ გაჭრეს დიამეტრალურად (სწორი ხაზით, ცენტრის გავლით), და თანაბრად დაურიგეს სტუმრებს. ჭრის ხაზები ერთმანეთს არ ემთხვევა. დაადგინეთ, მაქსიმუმ რამდენი სტუმრის გამასპინძლება შეიძლება ხაჭაპურის $N$-ჯერ გაჭრის შემდეგ.

## Input

მოცემულია ერთი არაუარყოფითი რიცხვი: $N$ - გაჭრების რაოდენობა, $(0 <= N <= 100)$

## Output

გამობეჭდეთ ერთი მთელი რიცხვი: $N$-ჯერ გაჭრის შემდეგ მიღებული ნაჭრების რაოდენობა.
