# Problem Statement — 871. კარგი სამეული - 3

> Copied from [AiaSoft.ge #871](https://aiasoft.ge/problem/871)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

სამეულს ეწოდება კარგი, თუ მათგან ორი ერთმანეთის ტოლია, მესამე კი განსხვავებული.
მოცემულია სამი რიცხვი $a, b, c$. თქვენ იცით, რომ მოცემული სამი რიცხვი ქმნის კარგ სამეულს. თქვენი მიზანია, დაადგინოთ, რომელ ცვლადშია მოთავსებული რიცხვი, რომელიც ამ სამეულში მხოლოდ ერთხელ გვხვდება.

## Input

ერთ სტრიქონზე შემოდის სამი მთელი დადებითი რიცხვი, $a, b, c$.
$1<=a,b,c<=100$

## Output

დაბეჭდეთ იმ ცვლადის სახელი, რომელშიც მოცემული რიცხვი კარგ სამეულში მხოლოდ ერთხელ გვხვდება.
