# განმარტება — 913. მანდარინები და ფორთოხლები

## მიდგომა

მოკლედ რომ ვთქვათ, ეს ამოცანა სუფთა მათემატიკაა და GCD-ზე გადის. რადგან ორივე ხილის კალათა თანაბარი რაოდენობის დონეებად უნდა დავყოთ და თითო დონეზე თანაბარი ხილი უნდა იყოს, გამოდის, რომ დონეების რაოდენობა უნდა იყოს N-ის და M-ის საერთო გამყოფი. რომ მაქსიმალური სიმაღლე მივიღოთ, უბრალოდ ყველაზე დიდი საერთო გამყოფი, ანუ GCD უნდა ვიპოვოთ. მეტი არაფერი უნდა, `__gcd()` ფუნქცია ყველაფერს აკეთებს.

## სირთულე

| | |
|-|--|
| დრო | O(log(min(N, M))) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- დონეების რაოდენობა უნდა იყოს როგორც N-ის, ისე M-ის გამყოფი.
- მაქსიმალური სიმაღლე პირდაპირ გულისხმობს უდიდეს საერთო გამყოფს.

## ხშირი შეცდომები

- წრფივი ციკლის გამოყენება 1-დან მინიმალურ რიცხვამდე, რაც 10^9-ზე ნამდვილად აგდებს TLE-ს.
