# Problem Statement — 913. მანდარინები და ფორთოხლები

> Copied from [AiaSoft.ge #913](https://aiasoft.ge/problem/913)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ახალი წლის ღამეა და ალექსი მასპინძლობს გრანდიოზულ წვეულებას! იმისთვის, რომ ზეიმი განსაკუთრებული ყოფილიყო, ალექსმა მოამზადა ორი სახის ხილის კალათა: ერთი მანდარინით და მეორე ფორთოხლით.

თუმცა, ალექსს უცნაური ტრადიცია აქვს. მას სურს შექმნას სპეციალური ორი ხილის კოშკი მანდარინებისგან და ფორთოხლებისგან ისე, რომ დაკმაყოფილდეს შემდეგი პირობები:
	● თითოეულ კოშკში მხოლოდ ერთი ტიპის ხილი უნდა იყოს გამოყენებული
	● კოშკის ყველა დონე ერთი და იგივე რაოდენობის ხილს შეიცავს
	● ორივე კოშკი თანაბარი რაოდენობის დონეებს უნდა შეიცავდეს
	● აუცილებელია, რომ ხილი არ მორჩეს

ალექსს სურს მაქსიმალურად გაზარდოს ხილის კოშკების სიმაღლე (დონეების რაოდენობა).

შეგიძლიათ დაეხმაროთ ალექსს ხილის კოშკების მაქსიმალური სიმაღლის პოვნაში?

## Input

ერთ ხაზზე შემოდის ორი ნატურალური რიცხვი: $N$ და $M$ $(1 <= N, M <= 10^9)$ - შესაბამისად მანდარინებისა და ფორთოხლების რაოდენობა

## Output

გამობეჭდეთ ერთი მთელი რიცხვი - ხილის კოშკის მაქსიმალური სიმაღლე
