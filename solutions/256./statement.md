# Problem Statement — 256. რიცხვის ძიება მასივში - 1

> Copied from [AiaSoft.ge #256](https://aiasoft.ge/problem/256)

**Time limit:** 2.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია მთელი რიცხვებისგან შედგენილი მასივი. დაადგინეთ, შეიცავს თუ არა შემოსული მასივი $x$-ს.

* `iმასივი დალაგებულია ზრდადობით`

## Input

პირველ ხაზზე შემოდის ორი მთელი რიცხვი: $N$ და $x$ - შესაბამისად მასივის წევრების რაოდენობა და საძიებელი რიცხვი.
მეორე ხაზზე შემოდის $N$ ცალი მთელი რიცხვი: $a_1, a_2, ..., a_n$

$1 <= N <= 10^6$
$0 <= x, a_i <= 10^9$

## Output

გამოიტანეთ ერთი სიტყვა: `bYES` თუ x რიცხვი ერთხელ მაინც გვხვდება  შემოსულ მასივში, ხოლო `bNO` - წინააღმდეგ შემთხვევაში.
