# Problem Statement — 415. რა დღეა?

> Copied from [AiaSoft.ge #415](https://aiasoft.ge/problem/415)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ვთქვათ, კვირის დღეები დანომრილია 1-დან 7-ის ჩათვლით ნატურალური რიცხვებით.
ორშაბათი - 1, სამშაბათი - 2, ოთხშაბათი - 3, და ა.შ. 

მოცემულია კვირის დღის ნომერი და საათი. დაადგინეთ, რომელი დღე იქნება n საათის შემდეგ.

## Input

შემოდის ერთმანეთისაგან ჰარით გამოყოფილი სამი რიცხვი: k - კვირის დღის ნომერი, h - საათის ჩვენება, n - გასული საათების რაოდენობა.

1<=k<=7
0<=h<=23
0<=n<=1000

## Output

გამოიტანეთ ერთი რიცხვი - კვირის დღის ნომერი n საათის გასვლის შემდეგ.
