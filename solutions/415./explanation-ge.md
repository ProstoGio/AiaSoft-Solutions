# განმარტება — 415. რა დღეა?

## მიდგომა

აქ მთავარია მივხვდეთ, რომ დღეები იცვლება მხოლოდ 24-საათიანი ინტერვალებით. 'h'-ს და 'n'-ის ჯამს რომ გავყოფთ 24-ზე, ზუსტად გავიგებთ რამდენი სრული დღე გავიდა, რაც უნდა დავუმატოთ საწყის დღეს. მერე მოდულო 7-ით მარტივად ვპოულობთ ახალ ინდექსს.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- 24-ზე მთელი რიცხვით გაყოფა გვაძლევს სრული დღეების რაოდენობას, რაც 'k'-ს უნდა დაემატოს.
- მოდულო 7-ის შემდეგ 0 რომ მივიღოთ, ეგ პირდაპირ ნიშნავს, რომ კვირა დღეზე ვართ, ანუ 7-ს უნდა გავუტოლოთ.

## ხშირი შეცდომები

- ყველაზე ხშირად ის ავიწყდებათ, რომ 7-ზე გაყოფისას ნაშთი 0-ია და თუ დღეები 1-დან 7-მდეა დანომრილი, ეგ ცალკე უნდა გაითვალისწინო.
