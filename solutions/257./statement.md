# Problem Statement — 257. რიცხვის ძიება მასივში - 2

> Copied from [AiaSoft.ge #257](https://aiasoft.ge/problem/257)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია მთელი რიცხვებისგან შედგენილი ორი მასივი $a$ და $b$. დაადგინეთ, $b$ მასივის რომელ წევრებს შეიცავს $a$ მასივი.

* `i$a$ მასივი დალაგებულია ზრდადობით`

## Input

პირველ ხაზზე შემოდის ერთი მთელი რიცხვი: $N$ - $a$ მასივის წევრების რაოდენობა. 
მეორე ხაზზე შემოდის $N$ რაოდენობის მთელი რიცხვი: $a_1, a_2, ..., a_n$
მესამე ხაზზე შემოდის ერთი მთელი რიცხვი: $M$ - $b$ მასივის წევრების რაოდენობა. 
მეოთხე ხაზზე შემოდის $M$ რაოდენობის მთელი რიცხვი: $b_1, b_2, ..., b_m$

$1 <= N <= 10^6$
$1 <= M <= 10^3$
$0 <= a_i, b_i <= 10^9$

## Output

$b$ მასივის  ყველა წევრისთვის (თანმიმდევრულად) ცალ-ცალკე ხაზზე დაბეჭდეთ `bYES` - თუ მოცემული წევრი მოიძებნება $a$ მასივში, ხოლო `bNO` - თუ არ მოიძებნება.
