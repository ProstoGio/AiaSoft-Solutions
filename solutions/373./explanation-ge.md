# განმარტება — 373. გამოტოვებული რიცხვი

## მიდგომა

რადგან რიცხვები უკვე დალაგებულია, ყველაზე მარტივი გზაა უბრალოდ აღვნიშნოთ რომელი რიცხვები შეგვხვდა. მე გამოვიყენე მასივი 'b', სადაც ინდექსი არის თავად რიცხვი, ხოლო მნიშვნელობა - მისი არსებობა. ბოლოს, უბრალოდ გავდივარ 0-დან n-მდე ციკლით და პირველივე '0'-ს რომელიც შემხვდება, ეგ იქნება პასუხი. ეს მიდგომა ბევრად უფრო სწრაფია, ვიდრე რაიმე რთული ძებნის ალგორითმი.

## სირთულე

| | |
|-|--|
| დრო | O(N) |
| მეხსიერება | O(M) |

## მთავარი დაკვირვებები

- თუ პირველი N რიცხვი რიგზეა, მაშინ გამოტოვებული რიცხვი აუცილებლად N იქნება, ამიტომ ციკლის უსასრულოდ გაგრძელება არ გვჭირდება.
- სიხშირეების მასივის გამოყენება ამარტივებს ძებნას, რადგან O(1)-ში ვამოწმებთ რიცხვის არსებობას.

## ხშირი შეცდომები

- მასივის ზომის არასწორად განსაზღვრა, როცა შეყვანის მონაცემები 10^6-მდეა.
