# Problem Statement — 025. მასივი და მისი ორი ელემენტი

> Copied from [AiaSoft.ge #25](https://aiasoft.ge/problem/25)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია n ელემენტიანი მასივი a1, a2, ... , an. აგრეთვე მოცემულია ნატურალური რიცხვი s. დაადგინეთ არსებობს თუ არა მასივში ორი ელემენტი ისეთი, რომ მათი ჯამი იყოს s.

## Input

პირველ ხაზზე შემოდის ჰარით გამოყოფილი ორი ნატურალური რიცხვი n და s. მეორე ხაზზე შემოდის ჰარით გამოყოფილი მასივის ელემენტები a1, a2, ... , an

1 ≤ n ≤ 10⁵
1 ≤ ai ≤ 10⁹
1 ≤ s ≤ 2*10⁹

## Output

გამოიტანეთ YES თუ არსებობს მასივში ორი ელემენტი რომელთა ჯამი არის s. წინააღმდეგ შემთხვევაში გამოიტანეთ NO
