# განმარტება — 887. πრველი კენტების ჯამი

## მიდგომა

აქ არაფერია რთული, უბრალოდ მათემატიკური კანონზომიერების დაჭერაა საჭირო. თუ დავწერთ პირველ კენტ რიცხვებს: 1, 3, 5, 7, ვნახავთ რომ ჯამები არის კვადრატები: 1=1², 1+3=4=2², 1+3+5=9=3². შესაბამისად, n-ური კენტი რიცხვების ჯამი უბრალოდ n²-ია, არანაირი ციკლი არ გვჭირდება.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- პირველი n კენტი რიცხვის ჯამი ყოველთვის n-ის კვადრატია.
- n არის 10⁹-მდე, ამიტომ n*n-ის გამოსათვლელად აუცილებლად long long-ი უნდა გამოვიყენოთ, თორემ int-ში დაგვივარდება და არასწორ პასუხს მოგვცემს.

## ხშირი შეცდომები

- long long-ის დავიწყება, რაც დიდი რიცხვების დროს overflow-ს იწვევს.
