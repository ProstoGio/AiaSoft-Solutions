# 542. ჯადოქარი ბიტუნა

## ამოცანის არსი
პრობლემა მოითხოვს რიცხვის დამუშავებას, სადაც შედეგი დამოკიდებულია ბოლო ციფრების მოდულ 4-ზე გაყოფის შედეგზე. რიცხვი დიდია (სტრიქონის სახით), ამიტომ მათემატიკური ოპერაციები მთლიან რიცხვზე არ გამოვა — აქ სტრიქონების მანიპულაციაა საჭირო.

## ლოგიკა და მიდგომა
ჩემი მიდგომა მარტივია: რიცხვის ბოლო ნაწილი განსაზღვრავს 4-ზე გაყოფის ნაშთს.

* **op1-ის შექმნა**: მე მჭირდება რიცხვი $n+1$. ამისთვის ბოლო ციფრს ვზრდი 1-ით. გავითვალისწინე 9-იანის შემთხვევა, სადაც ხდება `carry`-ს გადატანა წინა ციფრზე.
* **ბოლო 2 ციფრის ამოღება**: რადგან $100$ იყოფა 4-ზე, ნაშთისთვის მხოლოდ ბოლო ორი ციფრია მნიშვნელოვანი. კოდში ვიღებ მათ, ვატრიალებ (რადგან ბოლოდან ვიწყებ კითხვას) და ვაკონვერტირებ `int`-ში `-48` (ASCII) მანიპულაციით.
* **გადაწყვეტილება**: 
    * `m % 4 == 0` $\rightarrow$ პასუხია თავად $n$.
    * `m % 4 == 1` $\rightarrow$ პასუხია 1.
    * `m % 4 == 2` $\rightarrow$ პასუხია $n+1$ (op1).
    * `m % 4 == 3` $\rightarrow$ პასუხია 0.

## კოდი
```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    string n;
    cin >> n;
    string op1 = n;
    int sz = n.size() - 1;
    
    // ბოლო ციფრის გაზრდა (ინკრემენტი)
    if(op1[sz] == '9'){
        op1[sz] = '0';
        op1[sz-1] += 1;
    } else {
        op1[sz] += 1;
    }
    
    // ბოლო ორი ციფრის ამოღება და შებრუნება
    string num = "";
    int cnt = 0;
    for(int i = n.size() - 1; i >= 0 && cnt < 2; i--, cnt++){
        num += n[i];
    }
    reverse(num.begin(), num.end()); // შენი swap ლოგიკის ექვივალენტი
    
    // ციფრებიდან რიცხვის აწყობა
    int m = (num.size() == 2) ? (num[0]-48)*10 + (num[1]-48) : (num[0]-48);
    
    // პასუხის გამოტანა მოდულის მიხედვით
    if(m % 4 == 0) cout << n;
    else if(m % 4 == 1) cout << 1;
    else if(m % 4 == 2) cout << op1;
    else if(m % 4 == 3) cout << 0;
    
    return 0;
}
