# ამოხსნის ახსნა

ამოცანაში მოცემულია სიმრავლე, რომლის ელემენტებიც თავადაც სიმრავლეები შეიძლება იყვნენ. გარდა ამისა, ელემენტების თანმიმდევრობას მნიშვნელობა არ აქვს.

მაგალითად:

```text
{{}{{}}}
```

და

```text
{{{}}{}}
```

ერთსა და იმავე სიმრავლეს აღწერს, რადგან ორივე შეიცავს ელემენტებს:

```text
{}
{{}}
```

უბრალოდ სხვადასხვა მიმდევრობით.

ჩვენი მიზანია ვიპოვოთ შემოსული სიმრავლის ისეთი ელემენტი, რომელიც ყველაზე მეტჯერ გვხვდება და გამოვიტანოთ მისი სიხშირე.

---

## მთავარი იდეა

ორი სიმრავლის შედარება პირდაპირ რთულია, რადგან ელემენტების მიმდევრობა შეიძლება განსხვავდებოდეს.

ამიტომ ყველა სიმრავლეს გადავიყვანთ **კანონიკურ (სტანდარტულ) ფორმაში**.

იდეა ასეთია:

1. თითოეული სიმრავლე დავყოთ მის უშუალო ელემენტებად.
2. თითოეული ელემენტი რეკურსიულად ასევე გადავიყვანოთ კანონიკურ ფორმაში.
3. მიღებული ელემენტები დავალაგოთ.
4. თავიდან ავაწყოთ სიმრავლე.

ამის შემდეგ ტოლი სიმრავლეები ყოველთვის ზუსტად ერთნაირ სტრიქონად ჩაიწერება და მათი დათვლა მარტივი გახდება.

---

## 1. სიმრავლის უშუალო ელემენტებად დაყოფა

ფუნქცია:

```cpp
vector<string> sol(string s)
```

იღებს ერთ სიმრავლეს და აბრუნებს მის უშუალო ელემენტებს.

მაგალითად:

```text
{{}{{}}}
```

გადაიქცევა:

```text
{}
{{}}
```

ამისათვის ვიყენებთ სტეკს.

```cpp
if(s[i]=='{'){
    st.push('{');
    temp += '{';
}

if(s[i]=='}'){
    st.pop();
    temp += '}';
}
```

ყოველ გახსნილ ფრჩხილზე სტეკში ვამატებთ ელემენტს, ხოლო ყოველ დახურულზე ვშლით.

---

როდესაც სტეკი ცარიელი ხდება:

```cpp
if(st.empty()){
    l.pb(temp);
    temp="";
}
```

ეს ნიშნავს, რომ ერთი სრული ელემენტის წაკითხვა დავასრულეთ და შეგვიძლია შევინახოთ.

---

## რატომ მუშაობს ეს?

მაგალითად:

```text
{{}{{}}}
```

შიგნით გვაქვს:

```text
{}{{}}
```

პირველი `{}`-ის დამთავრებისას სტეკი ცარიელდება.

შემდეგ იწყება ახალი ელემენტი:

```text
{{}}
```

და მისი დასრულების შემდეგ სტეკი ისევ ცარიელია.

ამგვარად ზუსტად ვპოულობთ უშუალო ელემენტების საზღვრებს.

---

## 2. სიმრავლის კანონიკურ ფორმაში გადაყვანა

ფუნქცია:

```cpp
void srt(string &str)
```

ამ ფუნქციის მიზანია ნებისმიერი სიმრავლის ერთნაირი წარმოდგენის მიღება.

ჯერ ვყოფთ ელემენტებად:

```cpp
vector<string> v = sol(str);
```

---

## 3. ელემენტების დალაგება

რადგან ელემენტების მიმდევრობას მნიშვნელობა არ აქვს, საჭიროა მათი დალაგება.

```cpp
stable_sort(v.begin(), v.end(),
    [](const string &a, const string &b){
        return a.length() < b.length();
    });
```

აქ ელემენტები იზრდება სიგრძის მიხედვით.

მაგალითად:

```text
{{{}}{}}
```

გადაიქცევა:

```text
{{}{{}}}
```

რადგან მოკლე ელემენტი პირველი მოვა.

---

## 4. რეკურსიული დამუშავება

თუ ელემენტი თავადაც შეიცავს ელემენტებს, მასზეც იგივე პროცედურა უნდა შესრულდეს.

```cpp
for(int i=0; i<v.size(); i++){
    if(v[i].size() <= 2)
        continue;
    else
        srt(v[i]);
}
```

---

რატომ არის პირობა:

```cpp
v[i].size() <= 2
```

?

რადგან

```text

```

უკვე ყველაზე მარტივი სიმრავლეა და მასში აღარაფერია დასალაგებელი.

---

## 5. სიმრავლის თავიდან აწყობა

ყველა ელემენტის დალაგებისა და დამუშავების შემდეგ თავიდან ვაგებთ სტრიქონს.

```cpp
string tmp = "{";
```

ვამატებთ ყველა ელემენტს:

```cpp
for(int i=0; i<v.size(); i++){
    for(int j=0; j<v[i].size(); j++){
        tmp += v[i][j];
    }
}
```

და ვხურავთ სიმრავლეს:

```cpp
tmp += '}';
```

საბოლოოდ:

```cpp
str = tmp;
```

---

ამ ეტაპზე ნებისმიერი ორი ტოლი სიმრავლე ზუსტად ერთნაირ სტრიქონად იქნება წარმოდგენილი.

მაგალითად:

```text
{{}{{}}}
```

და

```text
{{{}}{}}
```

ორივე გადაიქცევა:

```text
{{}{{}}}
```

## 6. ზედა დონის ელემენტების გამოყოფა

მთავარ ფუნქციაში თავიდან ვყოფთ შემოსულ სიმრავლეს მის უშუალო ელემენტებად.

```cpp
for(int i=1; i<s.size()-1; i++){
    ...
}
```

შედეგად ვიღებთ:

```cpp
vector<string> lst
```

რომელშიც ინახება ყველა ელემენტი.

## 7. ყველა ელემენტის ნორმალიზაცია

შემდეგ თითოეულ ელემენტს ვატარებთ `srt()` ფუნქციაში.

```cpp
for(int i=0; i<lst.size(); i++){
    srt(lst[i]);
}
```

ამის შემდეგ ყველა ტოლი სიმრავლე ერთნაირი სტრიქონი ხდება.

---

## 8. სიხშირეების დათვლა

ამისათვის ვიყენებთ `map`-ს.

```cpp
for(int i=0; i<lst.size(); i++){
    m[lst[i]]++;
}
```

მაგალითად:

```text
{}
{{}}
{}
{}
```

გახდება:

```text
{} -> 3
{{}} -> 1
```

---

## 9. მაქსიმალური სიხშირის პოვნა

```cpp
int answer = 0;

for(int i=0; i<lst.size(); i++){
    answer = max(answer, m[lst[i]]);
}
```

აქ ვპოულობთ ყველაზე დიდ მნიშვნელობას.

## სირთულე

### დროითი სირთულე

```text
O(n² log n)
```

სადაც `n` არის სტრიქონის სიგრძე.

რეკურსიული დაყოფისა და დალაგების გამო ზუსტი შეფასება რთულია, მაგრამ მოცემული შეზღუდვისთვის (`|S| ≤ 1000`) ეს სრულად საკმარისია.

---

### მეხსიერების სირთულე

```text
O(n)
```

რადგან ვინახავთ სიმრავლის ელემენტებს, სტეკს და სიხშირეების ცხრილს.

---

## შეჯამება

ამოხსნის მთავარი იდეაა ყველა სიმრავლის გადაყვანა ერთსა და იმავე კანონიკურ ფორმაში.

ამის შემდეგ:

1. ვყოფთ ზედა დონის ელემენტებად;
2. რეკურსიულად ვალაგებთ მათ შიდა სტრუქტურებს;
3. ტოლი სიმრავლეები ერთნაირ სტრიქონებად იქცევა;
4. `map`-ის საშუალებით ვითვლით სიხშირეებს;
5. ვპოულობთ მაქსიმალურ სიხშირეს.

ასე ვიღებთ პასუხს მაშინაც კი, როდესაც ტოლი სიმრავლეები სხვადასხვა მიმდევრობით არიან ჩაწერილი.
