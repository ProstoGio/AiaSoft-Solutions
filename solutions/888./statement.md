# Problem Statement — 888. πბონაჩის მიმდევრობა

> Copied from [AiaSoft.ge #888](https://aiasoft.ge/problem/888)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

`bπბონაჩის მიმდევრობა` განვსაზღვროთ შემდეგი წესით: პირველი ორი წევრი არის 0 და π, ხოლო ყოველი შემდეგი წევრი არის წინა ორი რიცხვის ჯამი.

მოცემულია ნატურალური რიცხვი n, თქვენი დავალებაა იპოვოთ და გამობეჭდოთ πბონაჩის მიმდევრობის n - ური წევრი.
ჩავთვალოთ, რომ pi=3.14

## Input

ერთადერთ ხაზზე შემოდის ნატურალური რიცხვი n

1<=n<=35

## Output

გამოიტანეთ ერთადერთი რიცხვი, πბონაჩის მიმდევრობის n - ური წევრი

პასუხი გამოიტანეთ 10−² სიზუსტით და წერტილის შემდეგ უნდა გამობეჭდოთ `bზუსტად ორი ციფრი`.
