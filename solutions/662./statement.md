# Problem Statement — 662. ტყე

> Copied from [AiaSoft.ge #662](https://aiasoft.ge/problem/662)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ბმულ არაორიენტირებულ გრაფს, რომელშიც არაა ციკლი, ხე ეწოდება. ხოლო ხეების ერთობლიობას კი ტყე ჰქვია.

მოცემული გაქვთ $N$ წვეროიანი ხე. დაადგინეთ, მაქსიმუმ რამდენი წიბოს ამოგდებაა შესაძლებელი ისე, რომ მივიღოთ ტყე, რომლიც ყველა კომპონენტშიც ლუწი რაოდენობის წვერო იქნება.

## Input

პირველ ხაზზე მოცემულია ორი მთელი რიცხვი: $N$ და $M$ - გრაფში წვეროების და წიბოების რაოდენობა.
მომდევნო $M$ ხაზზე შემოდის ორი-ორი რიცხვი: $v$ და $u$ - წიბოების დაბოლოებები

 * გარანტირებულია, რომ $N$ ყოველთვის ლუწია.

$2 <= N <= 10^5$

## Output

გამობეჭდეთ ერთი მთელი რიცხვი: მაქსიმალური წასაშლელი წიბოების რაოდენობა.
