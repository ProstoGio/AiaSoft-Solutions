# განმარტება — 479. სამშენებლო წელიწადი

## მიდგომა

აქ მთავარი იდეა მარტივია: n ტვირთი უნდა გავანაწილოთ m-ტონიან მანქანებზე. თუ ნაშთი არ გვაქვს, უბრალოდ n/m-ია პასუხი, მაგრამ თუ n იყოფა m-ზე ნაშთით, გამოდის რომ კიდევ ერთი დამატებითი მანქანა დაგვჭირდება იმ დარჩენილი ტვირთისთვის. კოდში ამას მარტივი if-ით ვაკეთებთ, რადგან C++-ში გაყოფა ავტომატურად ამრგვალებს ქვემოთ (floor-ს აკეთებს).

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- თუ n%m არ არის ნული, ეს ნიშნავს რომ ერთი არასრული დატვირთვა მაინც გვრჩება, რისთვისაც ცალკე მანქანაა საჭირო.
- არ გვჭირდება არანაირი რთული მათემატიკური ფუნქცია, ჩვეულებრივი `if` საკმარისია.

## ხშირი შეცდომები

- ხშირად ავიწყდებათ იმის შემოწმება, როცა n ზუსტად იყოფა m-ზე, რის გამოც პასუხს ზედმეტ ერთეულს უმატებენ.
