# Problem Statement — 140. მეორე ნომერი ცელქობა

> Copied from [AiaSoft.ge #140](https://aiasoft.ge/problem/140)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ფიზკულტურის მასწავლებელმა მოსწავლეები სიმაღლის ზრდადობის მიხედვით დააყენა და ბურთის შემოსატანად გავიდა.

ცელქმა ბავშვებმა მაშინვე გამოიყენეს `bმეორე ნომერი ცელქობა`, ამ ცელქობის წესებში წერია, რომ `bზოგიერთ` კენტი ნომრის მქონე მოსწავლეს შეუძლია ერთხელ გაუცვალოს ადგილი მის მომდევნო მოსწავლეს, თუკი ამის სურვილი ორივეს აქვს. თუ რომელიმე კენტი ნომრის მქონე ბავშვს არ ჰყავს მომდევნო მეგობარი, მაშინ, რა თქმა უნდა, ეს ბავშვი ადგილზე რჩება. მოსწავლეთა  დანომრვა იწყება ერთით.

მოცემული გაქვთ მოსწავლეთა სიმაღლეების `bმეორე ნომერი ცელქობის` წესებით არეული მასივი, დაწერეთ პროგრამა, რომელიც მოცემულ მასივს დაუბრუნებს პირვანდელ ფორმას.

## Input

შემომავალი ფაილის პირველ სტრიქონში ჩაწერილია მასივის ელემენტების რაოდენობა $n$, $1 <= n <=20$.
მეორე სტრიქონი შეიცავს არეული მასივის $a_k$ ელემენტებს - მოსწავლეთა სიმაღლეებს სანტიმეტრებში, $120 <= a_k <=160$, $1<=k<=n$.

## Output

გამოიტანეთ მასივის საწყისი ფორმა.
