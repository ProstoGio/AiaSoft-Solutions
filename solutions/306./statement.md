# Problem Statement — 306. მარიამის დაბადების დღე

> Copied from [AiaSoft.ge #306](https://aiasoft.ge/problem/306)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

დღეს მარიამის დაბადების დღეა და გადაწყვიტა მეგობრებთან ერთად გაერთოს. მან N მეგობარი დაპატიჟა წვეულებაზე და ისინი ორ მაგიდაზე უნდა გაანაწილოს. მარიამის მეგობრებიდან ზოგიერთს ერთმანეთთან უთანხმოება აქვს და ამიტომ იუბილარი დიდი სირთულის წინაშე დგას, მეგობრები ისე უნდა გაანაწილოს მაგიდებთან, რომ უთანხმოების მქონე მეგობრები ერთ მაგიდასთან არ მოხვდეს და დღემ მშვიდობიანად ჩაიაროს.

დაეხმარეთ მარიამს ამ პრობლემის მოგვარებაში.

## Input

პირველ ხაზზე მოცემულია ორი რიცხვი: N  და M - შესაბამისად მარიამის მეგობრების რაოდენობა და უთანხმოების მქონე წყვილების რაოდენობა
მომდევნო M ხაზზე მოცემულია ორ-ორი რიცხვი, მეგობრების ნომრები, რომელთაც სუფრასთან ერთად ყოფნა არ სურთ. (მეგობრები გადანომრილია 1-დან N-მდე რიცხვებით)

1 <= N <= 20
0 <= M <= (N * (N-1)) / 2
1 <= vi, ui <= N,  (vi != ui)

## Output

გამოიტანეთ `bYES` თუ მარიამის მეგობრების მაგიდებთან განაწილება შესაძლებელია, წინააღმდეგ შემთხვევაში გამოიტანეთ - `bNO`.
