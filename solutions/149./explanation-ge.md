# განმარტება — 149. განსხვავებული სიმბოლოების რაოდენობა

## მიდგომა

ამ ამოცანის მთავარი ხიბლი იმაშია, რომ ზედმეტი თავის ტკივილის გარეშე, სტანდარტული set-ით გადავჭრათ. C++-ის set-ი ბუნებრივად არ ინახავს დუბლიკატებს, ამიტომ უბრალოდ სტრიქონის ყველა ასოს ჩავყრით შიგნით და ის თავისით გაფილტრავს განმეორებად სიმბოლოებს. ბოლოს უბრალოდ ვამოწმებთ set-ის ზომას და ეგაა, პასუხიც მზადაა.

## სირთულე

| | |
|-|--|
| დრო | O(n log k) |
| მეხსიერება | O(k) |

## მთავარი დაკვირვებები

- std::set-ის გამოყენება ყველაზე ადეკვატური გზაა დუბლიკატების მოსაშორებლად.
- რადგან მხოლოდ 26 პატარა ლათინური ასო გვაქვს, k მუდმივია და ოპერაციები ძალიან სწრაფად სრულდება.

## ხშირი შეცდომები

- Nested loop-ების წერა, რაც ამ შემთხვევაში სრულიად არასაჭიროა და ზედმეტად ტვირთავს პროგრამას.
