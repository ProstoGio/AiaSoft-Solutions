# Problem Statement — 149. განსხვავებული სიმბოლოების რაოდენობა

## Task

მოცემულია ლათინური ანბანის პატარა სიმბოლოებისგან შედგენილი სტრიქონი. დაადგინეთ და გამოიტანეთ რამდენ განსხვავებულ სიმბოლოს შეიცავს იგი.

## Input

პირველ ხაზზე შემოდის ერთი სტრიქონი. გარანტირებულია რომ იგი შეიცავს მხოლოდ ლათინური ანბანის პატარა სიმბოლოებს და მისი სიგრძე არ აღემატება $100000$ - ს

## Output

გამოიტანეთ ერთი რიცხვი, სტრიქონში განსხვავებული სიმბოლოების რაოდენობა.
