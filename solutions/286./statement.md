# Problem Statement — 286. სინონიმები

> Copied from [AiaSoft.ge #286](https://aiasoft.ge/problem/286)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ქართული ენის სიტყვათა მარაგში უამრავი სინონიმი მოიძებნება. აქვე შეგახსენებთ, რომ სინონიმები ეწოდება იდენტური მნიშვნელობის მქონე სხვადასხვა სიტყვებს. ცხადია, რომ თუ A სიტყვა B-ს სინონიმია, ხოლო B კი C-სი, მაშინ A და C-ც სინონიმებია.

იდენტური მნიშვნელობის სიტყვები, ანუ სინონიმები, ერთიანდებიან ცალკეულ ჯგუფებში. მაგალითად, თუ A სინონიმია B-სი და B სინონიმია C-სი, მაშინ შეგვიძლია ვთქვათ რომ A, B და C ერთ ჯგუფში ერთიანდება.

მოცემულია $N$ ქართული სიტყვა, რომლებიც გადანომრილია $1$-დან $N$-ის ჩათვლით. ასევე ვიცით $M$ წყვილის შესახებ სინონიმური კავშირები.

დაადგინეთ, სულ რამდენი განსხვავებული ჯგუფი ჩამოყალიბდება მოცემული სიტყვებისგან და რამდენი სიტყვა იქნება გაერთიანებული უდიდეს ჯგუფში.

## Input

პირველ ხაზზე მოცემულია ორი რიცხვი: $N$ და $M$ - შესაბამისად სიტყვებისა და სინონიმური კავშირების რაოდენობა.
მომდევნო $M$ ხაზზე შემოდის ორ-ორი რიცხვი. კერძოდ $i$-ურ ხაზზე მოცემული იქნება $v_i$ და $u_i$ - რაც ნიშნავს, რომ $v_i$ და $u_i$ ნომერი სიტყვები სინონიმებია.

$1 <= N <= 500$
$0 <= M <= N*(N-1) / 2$
$1 <= v_i, u_i <= N$

## Output

ერთ ხაზზე გამოიტანეთ ჰარით გამოყოფილი ორი მთელი რიცხვი: განსხვავებული ჯგუფების რაოდენობა და უდიდეს ჯგუფში გაერთიანებული სიტყვების რაოდენობა.
