# Problem Statement — 058. ფხაჭუნა კატები

> Copied from [AiaSoft.ge #58](https://aiasoft.ge/problem/58)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ერთ-ერთ ქალაქში, ხმაურით შეწუხებული მოსახლეობის მრავალჯერადი მოთხოვნის შემდეგ, გადაწყდა მიეღოთ კანონი, რომლის თანახმადაც ქალაქის ტერიტორიაზე აკრძალული იქნებოდა ხმამაღალი ბგერების გამოცემა საღამოს 11 საათის შემდეგ.

ბიუროკრატიული ტრადიციების მიხედვით, კანონი აუცილებლად უნდა შეიცავდეს განმარტებას, თუ რას ნიშნავს ცნება "ხმამაღალი ბგერა". კარგახნის მსჯელობის შემდეგ აიკრძალა, მაგალითად, მუსიკალურ ინსტრუმენტებზე დაკვრა, ავეჯის გადაადგილება, ლურსმნების ჩაჭედება.

როცა უკვე უნდა მიეღოთ კანონი, ერთ დეპუტატს გაახსენდა, რომ მაცივარი ავეჯი არ არის და მისი გადაადგილება კანონის მოქმედების ქვეშ ვერანაირად ვერ ხვდება, სხვა დეპუტატებმაც დაიწყეს ისეთი დამატებითი აკრძალვების მოფიქრება, რომლებიც თავიდან კანონის მოქმედების სფეროში ვერ მოხვდა. საბოლოოდ აიკრძალა ღამით წრიპინი, ძაღლების ყეფა და კატების ფეხების ფხაჭუნი.

კანონის დარღვევა ისჯებოდა ფულის a ერთეულით.

ერთ-ერთ საცხოვრებელ სახლში n სართულია,  თითოეულ სართულზე m ბინაა. ბინები მთელ სახლში დანომრილია 1 -დან n*m-მდე.
თუ რომელიმე (არა ბოლო) სართულზე მდებარეობს ბინა x, მაშინ უშუალოდ მის ზემოთ არის ბინა, რომლის ნომერია x + m. 

ცნობილია, რომ  i ნომერ ბინაში bi რაოდენობის კატა ცხოვრობს. ცნობილია აგრეთვე, რომ რომელიმე ბინის მცხოვრები ზედა მეზობელს იმ შემთხვევაში უჩივლებს, თუ ზედას კატები უფრო ხმამაღლა აფხაჭუნებენ, ვიდრე მისი საკუთარი. 

ექსპერიმენტებით დადგენილია, რომ p  რაოდენობის კატა  იმ შემთხვევაში  აფხაჭუნებს უფრო ხმამაღლა, ვიდრე q რაოდენობისა, როცა p > 2q.

გამოარკვიეთ, სულ რა რაოდენობის თანხას გადაიხდის მთელი ბინა ჯარიმის სახით, თუ ცნობილია, რომ ის მცხოვრები, რომლის ზედა მეზობლის კატები უფრო ხმამაღლა აფხაჭუნებენ, ვიდრე მისი საკუთარი, ზედას აუცილებლად უჩივლებს და მას (ზედას) აუცილებლად დააჯარიმებენ.

## Input

შემომავალი ფაილის პირველი სტრიქონი შეიცავს სამ მთელ რიცხვს: n, m, a სართულების რაოდენობა, სართულზე ბინების რაოდენობა და ჯარიმის ზომა (1 ≤ n ≤ 20, 1 ≤ m ≤ 10, 1 ≤ a ≤ 1000). 
მომდევნო სტრიქონში მოცემულია n*m მთელი რიცხვი: b1, b2, \ldots , bnm, სადაც bi წარმოადგენს კატების რაოდენობას i ნომერ ბინაში (1 ≤ bi ≤ 30).

## Output

გამოიტანეთ მთელი ბინის საერთო ჯარიმა.
