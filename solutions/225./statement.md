# Problem Statement — 225. ხალისოს დერეფანი

## Task

ხალისო უცნაური ქალაქია. ხალისოს აეროპორტში უცნაური დერეფანია. უცნაურ დერეფანში უჯრედებიანი დაფაა დახატული. დაფის თითოეულ უჯრაში ჩაწერილია ნატურალური რიცხვი. 

ხალისოში ჩამოსულმა ნებისმიერმა სტუმარმა უნდა გაიაროს უცნაური დერეფანი და ცნობილი დაფა. ოღონდ დაფის გავლისას გარკვეული წესები უნდა დაიცვას. თავიდან სტუმარს შეუძლია პირველი სტრიქონის ნებისმიერ უჯრაზე დადგეს. ამის შემდეგ შეუძლია მხოლოდ მომდევო სტრიქონზე გადასვლა, ამასთან მხოლოდ თავისი უჯრის მეზობელ უჯრებზე (მეზობელი ჰქვია იმ უჯრას, რომელსაც ერთი საერთო წერტილი მაინც აქვს მოცემულ უჯრასთან).

სტუმარი უკან ვერ ბრუნდება, ნახტომს ვერ აკეთებს, დაფის სტრიქონში გვერდით უჯრაზე ვერ გადადის.  

დაფის გავლის შემდეგ აეროპორტის სპეციალური სამსახური შეკრებს იმ რიცხვებს, რომელთაც სტუმარმა დააბიჯა ფეხი. მიღებული ჯამი აღნიშნავს ხალისოური ხინკლის იმ რაოდენობას, რამდენ ხინკალსაც მიართმევენ მას უფასოდ ხალისოს მთავარ რესტორანში. 

რადგანაც ხალისოური ხინკალი განთქმულია, ამიტომაც ყველა სტუმარი ცდილობს, შეარჩიოს დაფაზე ისეთი მარშრუტი, რომ, რაც შეიძლება, მეტი ხინკალი შეხვდეს.

დაეხმარეთ სტუმარს იმის გარკვევაში, მაქსიმუმ რამდენი ხინკლის აღება შეუძლია ხალისოში სტუმრობისას, თუ თქვენთვის ცნობილია დაფის ყველა რიცხვი.

## Input

პირველ სტრიქონში შემოდის ორი რიცხვი $n$ და $m$,  $1<=n,m<=20$, დაფის სტრიქონებისა და სვეტების რაოდენობა.
მომდევნო $n$ რაოდენობის სტრიქონიდან თითოეულში ჩაწერილია $m$ რაოდენობის ნატურალური რიცხვი. რიცხვების მნიშვნელობა არ აღემატება 100-ს.

## Output

გამოიტანეთ ერთი რიცხვი - ხინკლის მაქსიმალური რაოდენობა.
