# Problem Statement — 604. შეკითხვები ტყეზე

> Copied from [AiaSoft.ge #604](https://aiasoft.ge/problem/604)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია "აია-ქალაქი"-ს გამწვანების გეგმა, რომელიც წარმოადგენს $N\timesN$ მატრიცას. გეგმაზე ფიფქით (`b'*'`) აღნიშნულია ყველა ის ადგილი სადაც უნდა დაირგოს ხე, ხოლო სხვა ყველა დანარჩენი წერტილით (`b'.'`). თქვენ დაგევალათ თითოეულ მართკუთხედში გამოთვალოთ დასარგავი ხეების რაოდენობა.

## Input

პირველ სტრიქონზე შემოდის ორი მთელი რიცხვი: $N, Q$ - მატრიცის სიგრძე-სიგანე და მართკუთხედების რაოდენობა.
შემდეგ მოცემულია $N\timesN$ ზომის მატრიცა.
შემდეგ $Q$ სტრიქონიდან თითოეულზე შემოდის ოთხი მთელი რიცხვი: $x_1,y_1,x_2,y_2$ - შესაბამისად მართკუთხედის ზედა მარცხენა და ქვედა მარჯვენა უჯრების კოორდინატები.

$1<=N<=1000$
$1<=Q<=2*10^5$
$1<=x_1,y_1,x_2,y_2<=N$

## Output

გამობეჭდეთ $Q$ რაოდენობის სტრიქონი, $i$-ური სტრიქონი უნდა შეესაბამებოდეს $i$-ურ მართკუთხედში დასარგავი ხეების რაოდენობას.
