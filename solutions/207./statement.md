# Problem Statement — 207. ლებრონი და კალათბურთი

> Copied from [AiaSoft.ge #207](https://aiasoft.ge/problem/207)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

პატარა ლებრონი სულ ახლახანს ჩაირიცხა კალათბურთის სახელმწიფო ნაკრებში. დღეს მის გუნდს შეჯიბრი ჰქონდა და მოწინააღმდეგეების კალათაში $a$ ერთქულიანი, $b$ ორქულიანი და $c$ სამქულიანი ბურთი ჩააგდეს. ხოლო მოწინააღმდეგე გუნდმა ლებრონის გუნდის კალათაში $x$ ერთქულიანი, $y$ ორქულიანი და $z$ სამქულიანი ბურთი ჩააგდო.

დაადგინეთ, გაიმარჯვა თუ ვერა ლებრონის გუნდმა.

## Input

პირველ ხაზზე შემოდის სამი მთელი რიცხვი: $a, b, c$ - ლებრონის გუნდის შედეგი
მეორე ხაზზე შემოდის ასევე სამი მღელი რიცხვი: $x, y, z$ - მოწინააღმდეგე გუნდის შედეგი

$0 <= a, b, c, x, y, z <= 100$

## Output

გამოიტანეთ `bYES` თუ ლებრონის გუნდმა გაიმარჯვა, წინააღმდეგ შემთხვევაში `bNO`.
