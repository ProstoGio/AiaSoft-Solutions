# Problem Statement — 912. საახალწლო ნათურები

> Copied from [AiaSoft.ge #912](https://aiasoft.ge/problem/912)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ბაქარმა საახალწლოდ ნაძვის ხე ნათურებით მორთო. ნათურებს მიმდევრობით ახალი  წლის რიცხვში შემავალი ციფრები აწერია (აღვნიშნოთ, რომ ამოცანის დაწერის მომენტში 2025 წელი დგება).

უფრო გასაგებად რომ ვთქვათ, თუ ნაძვის ხე მორთულია $N$ ნათურით, და ჩავთვლით რომ ნათურები გადანომრიალია 1-დან $N$-ის ჩათვლით, მაშინ პირველ ნათურაზე ეწერება ციფრი 2, მეორეზე - 0, მესამეზე - 2, მეოთხეზე - 5, მეხუთეზე ისევ - 2, მეექვსეზე 0 და ა. შ. - ანუ მას ექნება შემდეგი სახე: 202520252025...

ბაქარმა ნაძვის ხის მორთვის პროცესში პატარა თამაში წამოიწყო - მას აინტერესებს რა ციფრი ეწერება ნათურაზე რომლის ნომერიცაა - $K$

## Input

ერთ ხაზზე შემოდის ორი ნატურალური რიცხვი: $N$ და $K$, სადაც $4 <= N <= 100$ და $K <= N$

## Output

დაადგინეთ და გამობეჭდეთ მე-$K$ ნათურაზე დაწერილი ციფრი
