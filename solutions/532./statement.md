# Problem Statement — 532. მოხეტიალე ცხენი

> Copied from [AiaSoft.ge #532](https://aiasoft.ge/problem/532)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ჭადრაკის ცხენი NxM ზომის დაფის (1, 1) უჯრაზე დგას (ეს უჯრა დაფის უკიდურესი ზედა მარცხენა წერტილია). დაფაზე მოძრაობის შეზღუდული წესები მოქმედებს:
	დაფის ყოველ კოორდინატზე სპეციალური სიმბოლოებიდან (U, D, L, R) ერთ-ერთი წერია და თითოეული სიმბოლო გარკვეულ შეზღუდვას აღნიშნავს. კერძოდ, თუ (i, j) კოორდინატზე წერია U-მაშინ ცხენს შეუძლია შესაძლო სვლებიდან მხოლოდ ის სვლა განახორციელოს, რომელიც მას მოცემული სტრიქონიდან ზედა რომელიმე სტრიქონზე გადასვამს. შესაბამისად, თუ მოცემულ კოორდინატზე D წერია, მაშინ ცხენი ქვედა სტრიქონზე უნდა გადაადგილდეს. L-ის შემთხვევაში მარცხნივ და R-ის შემთხვევაში მარჯვნივ.

ფოტოზე ნაჩვენებია R და D შემთხვევები.

თქვენი დავალებაა გამოთვალოთ, მინიმუმ რამდენ სვლაში შეძლებს ცხენი დაფაზე წესების დაცვით სიარულისას (N, M) კოორდინატზე მისვლას.

## Input

პირველ ხაზზე მოცემულია ორი ნატურალური რიცხვი: N და M - დაფის ზომები
მომდევნო N ხაზზიდან თითოეულზე მოცემულია M სიმბოლო - დაფის აღწერა. დაფა შეიცავს მხოლოდ U, D, L, R სიმბოლოებს.

1 <= N, M <= 50

## Output

თუ (1, 1) კოორდინატიდან (N, M) კოორდინატზე მისვლა შესაძლებელია, გამობეჭდეთ მინიმალური სვლების რაოდენობა, წინააღმდეგ შემთხვევაში `b-1`.
