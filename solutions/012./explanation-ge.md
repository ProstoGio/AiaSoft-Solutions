# განმარტება — 012. მართკუთხედი

## მიდგომა

მართკუთხედის პერიმეტრი არის P = 2 * (a + b). რადგან გვერდები მთელი რიცხვებია, ჯამიც მთელი რიცხვია, ანუ პერიმეტრი აუცილებლად ლუწი უნდა იყოს, რომ 2-ზე გაიყოს. ასევე, პატარა მართკუთხედს რომ შევხედოთ, ყველაზე პატარა მხარეები 1-ია, შესაბამისად პერიმეტრი მინიმუმ 4 უნდა იყოს. ამიტომ, უბრალოდ ვამოწმებ P ლუწია თუ არა და მეტია თუ არა 2-ზე.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- პერიმეტრი P აუცილებლად უნდა იყოს 2-ის ჯერადი, ანუ P % 2 == 0.
- P=2 შემთხვევაში გვერდები იქნებოდა 0.5 და 0.5 ან მსგავსი, რაც პირობას არ აკმაყოფილებს, ამიტომ P > 2 უნდა იყოს.

## ხშირი შეცდომები

- მხოლოდ ლუწობის შემოწმება და P=2-ის გამოტოვება.
