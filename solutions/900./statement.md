# Problem Statement — 900. ორი მარტოხელა

> Copied from [AiaSoft.ge #900](https://aiasoft.ge/problem/900)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია ერთგანზომილებიანი მასივი $a_0, a_1, a_2, a_3,..., a_i,...,  1<=a_i <= 10^9$.
ცნობილია, რომ მასივის თითოეული ელემენტის რაოდენობა, გარდა ორისა,  არის ლუწი. მხოლოდ ორი ელემენტია წარმოდგენილი თითოჯერ.
დაადგინეთ იმ ელემენტების მნიშვნელობები, რომლებიც მხოლოდ ერთხელაა მასივში.

## Input

შემავალი მონაცემების პირველი ხაზი შეიცავს ერთ ნატურალურ რიცხვს $t, 1<=t<=1000$  ტესტების რაოდენობას.
თითოეული ტესტი შეიცავს ორ სტრიქონს, რომელთაგან პირველში მოცემულია მასივის სიგრძე $n. 1<=n<=10^5$, ხოლო მეორეში ჩაწერილია ერთმანეთისაგან ჰარით გამოყოფილი $n$ ცალი მთელი რიცხვი - მასივის ელემენტები.

## Output

თითოეული ტესტისათვის ცალკე სტრიქონზე გამოიტანეთ ერთმანეთისაგან ჰარით გამოყოფილი ორი მთელი რიცხვი (დაალაგეთ ზრდადობით) - მასივის იმ ელემენტების მნიშვნელობები, რომლებიც მხოლოდ ერთხელაა მასივში.
