# ამოხსნის ახსნა

თავდაპირველად მოსწავლეები იდგნენ სიმაღლის ზრდადობის მიხედვით.

შემდეგ მათ გააკეთეს მეორე ნომერი ცელქობა.

მისი წესები ასეთია:

- მხოლოდ კენტი ნომრის მქონე მოსწავლეს შეუძლია ადგილი გაუცვალოს მომდევნო მოსწავლეს;
- გაცვლა სავალდებულო არ არის, ზოგი ცვლის, ზოგი არა;
- თითოეულ წყვილს მაქსიმუმ ერთხელ შეუძლია ადგილის გაცვლა.

მოცემულია უკვე არეული მასივი და საჭიროა აღვადგინოთ საწყისი განლაგება.

---

# მთავარი დაკვირვება

თავდაპირველად მოსწავლეები უკვე დალაგებულები იყვნენ.

ცელქობის დროს მხოლოდ მეზობელი ელემენტები ცვლიან ადგილებს.

მაგალითად:

```text
120 125 130 135 140
```

შეიძლება გადაიქცეს:

```text
125 120 130 140 135
```

ან:

```text
120 125 135 130 140
```

ან საერთოდ არც შეიცვალოს.

---

მნიშვნელოვანია, რომ:

```text
არცერთი ელემენტი შორს არ გადადის.
```

ის მხოლოდ მეზობელთან შეიძლება შეიცვალოს.

---

# ყველაზე მნიშვნელოვანი იდეა

როგორიც არ უნდა იყოს გაცვლების კომბინაცია, ბოლოს მიღებული მასივი მაინც იმავე სიმაღლეებს შეიცავს.

ანუ:

```text
არცერთი სიმაღლე არ იკარგება
არც ახალი ემატება
```

---

ხოლო ვიცით, რომ თავიდან ყველა მოსწავლე სიმაღლის ზრდადობით იდგა.

ეს ნიშნავს, რომ საწყისი მასივი უბრალოდ:

```text
დალაგებული მასივია.
```

---

ამიტომ სრულიად საკმარისია მოცემული მასივი თავიდან დავალაგოთ.

---

# მაგალითი

მოცემულია:

```text
125 120 135 130 145 140
```

---

დალაგების შემდეგ მივიღებთ:

```text
120 125 130 135 140 145
```

რაც ზუსტად საწყისი განლაგებაა.

---

# კოდის განხილვა

## მონაცემების წაკითხვა

```cpp
int n;
cin >> n;

vector<int> a(n+1);

for(int i=1; i<=n; i++)
    cin >> a[i];
```

მასივში ვინახავთ მოსწავლეების სიმაღლეებს.

---

## დალაგება

```cpp
sort(a.begin(), a.end());
```

ეს ფუნქცია ელემენტებს ზრდადობით ალაგებს.

---

რადგან საწყისი მასივიც ზრდადობით იყო დალაგებული, სწორედ ეს მოქმედება აღადგენს პირვანდელ მდგომარეობას.

---

## პასუხის გამოტანა

```cpp
for(int i=1; i<=n; i++)
    cout << a[i] << " ";
```

ვბეჭდავთ უკვე აღდგენილ მასივს.

# რატომ მუშაობს?

საწყისი მიმდევრობა უკვე ზრდადობით იყო დალაგებული.

---

ცელქობის დროს მოსწავლეები მხოლოდ მეზობლებთან ცვლიან ადგილებს.

---

ამ ცვლილებების შემდეგ სიმაღლეების მნიშვნელობები არ იცვლება, იცვლება მხოლოდ მათი თანმიმდევრობა.

---

ამიტომ, თუ მიღებულ მასივს თავიდან დავალაგებთ:

```text
ყველა მოსწავლე ისევ იმავე ადგილას აღმოჩნდება,
სადაც თავიდან იდგა.
```

---

# შეჯამება

მიუხედავად იმისა, რომ ცელქობის დროს ზოგიერთი მეზობელი წყვილი ადგილს ცვლის, თავიდან მოსწავლეები უკვე სიმაღლის ზრდადობით იყვნენ განლაგებულები.

ამიტომ საწყისი მდგომარეობის აღსადგენად საკმარისია მიღებული მასივი თავიდან დავალაგოთ ზრდადობით.

ამ ამოცანის მთავარი იდეა სწორედ იმის შემჩნევაა, რომ საბოლოო პასუხი ყოველთვის დალაგებული მიმდევრობა იქნება, მიუხედავად იმისა, რომ გაცვლები მხოლოდ მეზობლებს შორის ხდებოდა.
