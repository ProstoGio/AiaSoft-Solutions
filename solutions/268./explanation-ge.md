# განმარტება — 268. ციკლი გრაფში

## მიდგომა

ამ ამოცანისთვის DSU-ს (Disjoint Set Union) გამოყენება ყველაზე ოპტიმალურია. გრაფი ითვლება ციკლურად, თუ მორიგი წიბოს დამატებისას ორივე წვერო უკვე ერთ კომპონენტში ზის. `find_set`-ში პათ კომპრესიას ვაკეთებ, რომ ხეები ძალიან არ დაგრძელდეს და მუშაობა სწრაფი დარჩეს. თუ წიბოს თავები ერთსა და იმავე 'მამას' უყურებენ, ესე იგი უკვე გვაქვს გზა მათ შორის და ახალი წიბო ავტომატურად ციკლს ქმნის.

## სირთულე

| | |
|-|--|
| დრო | O(m * α(n)) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- თუ ორი წვერო უკვე ერთ 'სეტშია', მათ შორის არსებული გზა და ახალი წიბო ციკლს კრავს.
- DSU-ს გამოყენება გრაფიკულად მარტივია და კოდიც უფრო მოკლე გამოდის ვიდრე DFS-ით.

## ხშირი შეცდომები

- დავიწყება `parent[i]=i` ინიციალიზაციის, რის გამოც `find_set` უსასრულო ციკლში ვარდება.
- გრაფის მიმართულებების არასწორი აღქმა, თუმცა აქ ამოცანა არაორიენტირებულზეა.
