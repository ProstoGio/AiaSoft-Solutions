# Problem Statement — 603. მონეტების ჯამი

> Copied from [AiaSoft.ge #603](https://aiasoft.ge/problem/603)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

თქვენ გაქვთ $n$ ცალი მონეტა გარკვეული მნიშვნელობებით. თქვენი ამოცანაა იპოვოთ ყველა ის თანხა, რომელიც შეგიძლიათ შექმნათ ამ მონეტების კომბინაციით.

## Input

პირველ ხაზზე მოცემულია $n$ - მონეტების რაოდენობა.
მეორე ხაზზე მოცეულია $n$ მთელი რიცხვი $x_1,x_2,x_3, ..., x_n$.
$1<=n<=100$
$1<=x_i<=1000$

## Output

პირველ ხაზზე გამობეჭდეთ $k$ ყველა შესაძლო განსხვავებული თანხა, რომელიც შეგვიძლია მონეტების კომბინაციით მივიღოთ.
მეორე ხაზზე გამობეჭდეთ $k$ მთელი დადებითი რიცხვი ზრდადობით - მიღებული თანხები.
