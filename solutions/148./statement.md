# Problem Statement — 148. განსხვავებული რიცხვების რაოდენობა 2

> Copied from [AiaSoft.ge #148](https://aiasoft.ge/problem/148)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია n ცალი მთელი რიცხვი a1,a2,...,an. დაადგინეთ და გამოიტანეთ რამდენი განსხვავებული რიცხვია მათ შორის.

## Input

პირველ ხაზზე შემოდის n (1<=n<=10⁵), რიცხვების რაოდენობა. მეორე ხაზზე შემოდის n ცალი მთელი რიცხვი, რომლებიც მოთავსებულია [-10⁹, 10⁹] შუალედში.

## Output

გამოიტანეთ ერთი რიცხვი, განსხვავებული რიცხვების რაოდენობა.
