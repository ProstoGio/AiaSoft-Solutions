# განმარტება — 043. მასივის ქვესეგმენტის შეტრიალება

## მიდგომა

ამ ამოცანის ამოხსნის ყველაზე მარტივი გზა ორი პოინტერის გამოყენებაა. l-ს დავსვამთ დასაწყისში, r-ს ბოლოში და მათ შორის არსებულ ელემენტებს ვუცვლით ადგილებს. როგორც კი პოინტერები შეხვდებიან ერთმანეთს ან გადაკვეთენ, შეტრიალებაც მორჩენილია. ზედმეტი მახსოვრობის დახარჯვა აქ საერთოდ არ გვჭირდება, რადგან ადგილზევე (in-place) ვანაცვლებთ ელემენტებს.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- while ციკლი ზუსტად შუამდე მიდის, რაც ოპტიმალურია, რადგან მეტს არაფერს აკეთებს.
- ტემპორარულ ცვლადს (t) ვიყენებთ მხოლოდ მნიშვნელობების გადასასმელად, სხვა დამხმარე მასივი არ გვჭირდება.

## ხშირი შეცდომები

- ზოგჯერ l და r ინდექსების არევა ხდება, ან პირობაში `l < r`-ის ნაცვლად არასწორი ლოგიკის ჩაწერა.
