# Problem Statement — 812. უდიდესი კენტი

> Copied from [AiaSoft.ge #812](https://aiasoft.ge/problem/812)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

იპოვეთ მოცემული მასივის უდიდესი კენტი ელემენტი. თუ ასეთი ელემენტები არ არსებობს, გამოიტანეთ -1.

## Input

პირველ სტრიქონზე მოცემულია მასივის სიგრძე $n, 1<=n<=100$.
მეორე სტრიქონზე მოცემულია ერთმანეთისაგან ჰარით გამოყოფილი მასივის ელემენტები $a_0,a_1,a_2…a_(n-1)$. მასივის ელემენტები ნატურალური რიცხვებია და არ აღემატება 1000-ს.

## Output

გამოიტანეთ ერთი მთელი რიცხვი - მასივის კენტ ელემენტებს შორის უდიდესი. თუ ასეთი ელემენტები არ არსებობს, გამოიტანეთ -1.
