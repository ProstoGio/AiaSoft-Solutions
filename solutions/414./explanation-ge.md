# განმარტება — 414. უდიდესი რიცხვი

## მიდგომა

ამოცანა მარტივია: გვაქვს ორი ციფრი და უნდა მივიღოთ მაქსიმალური შესაძლო რიცხვი. ცხადია, რომ ათეულების პოზიციაზე დიდი ციფრი უნდა დავსვათ, რომ რიცხვი მაქსიმალურად გაიზარდოს. შევადარებთ ორ რიცხვს, ავირჩევთ უფრო დიდს, გავამრავლებთ ათზე, მივუმატებთ მეორეს და ბოლოს ერთსაც მივუმატებთ.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- მაქსიმალური რიცხვი რომ მივიღოთ, უფრო დიდი ციფრი ათეულებში უნდა დავწეროთ.
- სტრინგებთან წვალება არ არის საჭირო, მათემატიკური ფორმულა `max(a, b) * 10 + min(a, b) + 1` სრულყოფილად მუშაობს.

## ხშირი შეცდომები

- თუ უბრალოდ პირველად შემოტანილ ციფრს გავამრავლებთ ათზე, შეიძლება შეცდომა დავუშვათ, რადგან შესაძლოა მეორე ციფრი უფრო დიდი იყოს.
