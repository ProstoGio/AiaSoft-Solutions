# Problem Statement — 829. კატო და საყვარელი სიმბოლოები

> Copied from [AiaSoft.ge #829](https://aiasoft.ge/problem/829)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

კატოს ძალიან უყვარს ავტობუსით მგზავრობა და მგზავრობისას მაღაზიების აბრებზე დაკვირვება. კატო სკოლაში წასვლამდე ირჩევს საყვარელ სიმბოლოს და აბრებზე დავირვებისას, მათში ითვლის არჩეული სიმბოლოს რაოდენობას.
მალე სკოლაში არდადეგები იწყება დაკატოს იშვიათად მოუწევს ხოლმე ავტობუსით მგზავრობა, რის გამოც ის დამწუხრებულია. თქვენი დავალებაა კატოს გამხიარულება შემდეგი თამაშით: კატო გეუბნებათ აბრაზე გამოკრულ სტრიქონს და ასევე მის საყვარელ სიმბოლოებს. თქვენ უნდა დათვალოთ სტრიქონში კატოს საყვარელი სიმბოლოების ჯამური რაოდენობა.

## Input

ერთ ხაზზე მოცემულია $S$ და $K$ - შესაბამისად აბრაზე გამოკრული სტრიქონი და კატოს საყვარელი სიმბოლოებისგან შედგენილი სტრიქონი. 
გარანტირებულია, რომ $S$ და $K$ შეიცავს მხოლოდ ლათინური ანბანის პატარა და დიდ სიმბოლოებს და $1<=|S|, |K|<=500$

## Output

გამობეჭდეთ აბრაზე კატოს საყვარელი სიმბოლოების ჯამური რაოდენობა.
