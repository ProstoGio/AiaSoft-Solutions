# განმარტება — 033. მასივის მაქსიმალური ელემენტი

## მიდგომა

აქ არაფერია რთული, უბრალოდ მასივის პირველ ელემენტს ვიღებთ საწყის 'მაქსიმუმად' და მერე მთლიან მასივს გავუყვებით თავიდან ბოლომდე. ყოველ ნაბიჯზე ვამოწმებ, თუ ახლანდელი ელემენტი ჩემს შენახულ მაქსიმუმზე დიდია, მაშინ მას ვანახლებ. ციკლის ბოლოს ის მნიშვნელობა, რაც ხელში მრჩება, ავტომატურად იქნება მთელი სიის უდიდესი რიცხვი.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- თუ გინდა მეხსიერება დაზოგო, მთლიანი მასივის შექმნა საჭიროც არაა; შეგიძლია პირდაპირ ციკლშივე წაიკითხო რიცხვები და ეგრევე შეადარო.
- მაქსიმუმისთვის საწყისი მნიშვნელობის მინიჭებისას ფრთხილად იყავი, არ დაწერო 0, რადგან თუ სიაში მარტო უარყოფითი რიცხვებია, ეგ ლოგიკა ჩაგივარდება.

## ხშირი შეცდომები

- მაქსიმუმის საწყის მნიშვნელობად 0-ის არჩევა, რაც უარყოფითი რიცხვების შემთხვევაში არასწორ პასუხს გვაძლევს.
