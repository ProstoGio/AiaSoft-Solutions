# Problem Statement — 062. პირველკლასელი (ოღონდ მეორე)

> Copied from [AiaSoft.ge #62](https://aiasoft.ge/problem/62)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

პირველკლასელს (მეორეს) აქვს სტრიქონი, რომელშიც მრავალნიშნა (მაქსიმუმ 8-ნიშნა) რიცხვების ჯამი, სხვაობა ან ნამრავლია ჩაწერილი. მაგ.: 12+23,  25-17, 321*42. 

დაეხმარეთ პირველკლასელს და დაუწერეთ პროგრამა, რომელიც იანგარიშებს მოცემული სტრიქონის რიცხვით მნიშვნელობას.

## Input

შემოდის ერთი სტრიქონი, რომელშიც შემდეგი არითმეტიკული ოპერაციების ნიშნებიდან +, - , * მხოლოდ ერთია მოთავსებული. ხოლო დანარჩენი სიმბოლოები წარმოადგენენ ციფრებს.

## Output

გამოიტანეთ არითმეტიკული გამოსახულების მნიშვნელობა.
