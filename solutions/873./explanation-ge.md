# განმარტება — 873. E = mc²

## მიდგომა

ფიზიკის ფორმულა მარტივია: E = mc². რადგან c = 300,000-ია, კვადრატში ის 90,000,000,000-ს გვაძლევს. m-ის მაქსიმალური მნიშვნელობა 10¹⁸-ია, რაც ნიშნავს, რომ პირდაპირი გამრავლებით `unsigned long long`-იც კი აუცილებლად overflow-ს მოგცემს. ამიტომ, ჯერ m გავამრავლე 9-ზე და ბოლოს უბრალოდ ათი ნული მივაწერე სტრიქონად, რაც ამ პრობლემას მარტივად ხსნის.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- c² ზუსტად 9-ია ათი ნულით, ამიტომ რიცხვითი ოპერაციების ნაცვლად სტრიქონის კონკატენაცია ყველაზე უსაფრთხო გზაა.
- 10¹⁸-ის 9-ზე გამრავლება `unsigned long long`-ში ეტევა, ამიტომ სრული რიცხვის შენახვა საჭირო არ არის.

## ხშირი შეცდომები

- გრძელი რიცხვის პირდაპირ გამრავლება, რაც 2⁶⁴-ს ცდება და არასწორ პასუხს გვაძლევს.
