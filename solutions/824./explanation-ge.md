# განმარტება — 824. ნინი და ხინკალი

## მიდგომა

ხინკალი რომ 'გემრიელი' იყოს, აუცილებელია სტრიქონში ინგლისური ანბანის ყველა 26-ვე ასო გვქონდეს. რადგან გვაქვს უფლება მაქსიმუმ ერთი სიმბოლო დავამატოთ ან შევცვალოთ, ეს ნიშნავს, რომ თუ უკვე 25 უნიკალური სიმბოლო მაინც გვაქვს, დავალებას მარტივად ვასრულებთ. ლოგიკა მარტივია: დავსორტოთ სტრიქონი, ამოვაგდოთ დუბლიკატები და ვნახოთ, დარჩენილი უნიკალური სიმბოლოების რაოდენობა 25-ს აღემატება თუ არა.

## სირთულე

| | |
|-|--|
| დრო | O(n log n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- მთავარია უნიკალური სიმბოლოების დათვლა, რისთვისაც `sort()` და `unique`-ის ლოგიკა საუკეთესოა.
- თუ უნიკალური ასოების რაოდენობა 25-ზე ნაკლებია, ერთი ოპერაციით 26-მდე ვერ ავალთ, ამიტომ პასუხი ავტომატურად NO იქნება.

## ხშირი შეცდომები

- სიმბოლოების რაოდენობის არასწორად დათვლა ან პირობის არასწორად გააზრება (მაგალითად, 26-ის ნაცვლად 25-ზე შემოწმება).
