# განმარტება — 811. მასივის ორმოების რაოდენობა

## მიდგომა

აქ მთავარია მივხვდეთ, რომ მასივის პირველი და ბოლო ელემენტი ორმო ვერასდროს იქნება, რადგან ორივე მხარეს მეზობელი არ ჰყავთ. ამიტომ, `for` ციკლს პირდაპირ მეორე ინდექსიდან ვიწყებ და ბოლომდე არ მივყვები - `n-1`-ზე ვჩერდები. ყოველ ჯერზე ვამოწმებ `a[i] < a[i-1]` და `a[i] < a[i+1]` პირობას. თუ დაემთხვა, კაუნტერს ვზრდი და ბოლოს თუ კაუნტერი 0-ია, უბრალოდ -1-ს ვბეჭდავ.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- ციკლის საზღვრები `1`-დან `n-1`-მდე ზუსტად გვაზღვევს, რომ მეხსიერებიდან არ გავვარდეთ და ინდექსაცია არ აგვერიოს.
- პირობა მკაცრად ნაკლებზეა, ამიტომ ტოლობა არ გაითვალისწინოთ.

## ხშირი შეცდომები

- ციკლის არასწორად დაწერა, რამაც შეიძლება გამოიწვიოს `out of bounds` ერორი, თუ `arr[i+1]`-ს წვდომას შევეცდებით ბოლო ელემენტის დროს.
