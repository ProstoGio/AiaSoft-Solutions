# განმარტება — 313. 0 და 1

## მიდგომა

სინამდვილეში ეს ტიპური ფიბონაჩია, ოღონდ ოდნავ სხვა კონტექსტში. დააკვირდი მიმდევრობას: თუ ბოლო ციფრი ნულია, წინა ნებისმიერი ვალიდური ვარიანტი მოსულა. თუ ბოლო ციფრი ერთიანია, მის წინ აუცილებლად ნული უნდა იდგეს, ანუ წინა-წინა მდგომარეობიდან მოდიხარ. ამიტომ, n-სიგრძის მიმდევრობების რაოდენობა არის (n-1)-ის და (n-2)-ის ჯამი.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- ლოგიკა მარტივია: ნებისმიერ ვალიდურ მიმდევრობას ან '0'-ს ვუმატებთ, ან '01'-ს, რათა ზედიზედ ორი ერთიანი არ მივიღოთ.
- პირველი ორი შემთხვევა, ანუ N=1 და N=2, ბაზისად უნდა დააფიქსირო, თორემ მერე ფორმულა არ აეწყობა.

## ხშირი შეცდომები

- ინდექსების აღრევა ან N=1-ისთვის 1-ის ჩაწერა 2-ის ნაცვლად.

## ნაბიჯ-ნაბიჯ ახსნა

ვთქვათ N=3: შესაძლო ვარიანტებია 000, 001, 010, 100, 101 — სულ 5. ჩვენი ფორმულით dp[3] = dp[2] + dp[1], ანუ 3+2=5. ყველაფერი ემთხვევა.
