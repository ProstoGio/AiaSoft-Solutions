# განმარტება — 547. ფოთლები

## მიდგომა

მოკლედ რომ ვთქვათ, ხეში ფოთოლი არის წვერო, რომელსაც მარტო ერთი მეზობელი ჰყავს. რადგან გრაფი არაორიენტირებულია, ნებისმიერი შვილი თავის მამასთანაა დაკავშირებული. მთავარია, ფესვი (პირველი წვერო) ცალკე გამოვყოთ, რადგან პირობა მკაცრად ამბობს, რომ ფესვი ფოთოლი არაა. უბრალოდ ვამოწმებთ ყველა წვეროს 1-იანის გარდა და ვითვლით, ვის აქვს ზუსტად ერთი მეზობელი.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- არაორიენტირებულ გრაფში ფოთლის ხარისხი ყოველთვის 1-ის ტოლია.
- ფესვი (node 1) არ უნდა ჩავთვალოთ ფოთლად, მაშინაც კი თუ მხოლოდ ერთი შვილი ჰყავს და ხარისხი 1 აქვს.

## ხშირი შეცდომები

- პირველი წვეროს (root-ის) ფოთლებში ჩათვლა, თუ მას მხოლოდ ერთი მეზობელი ჰყავს.
