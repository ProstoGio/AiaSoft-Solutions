# Problem Statement — 141. ბორანი

> Copied from [AiaSoft.ge #141](https://aiasoft.ge/problem/141)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მდინარეზე ბორანით გადააქვთ სხვადასხვა ტვირთი. ბორანის ტვირთამწეობა უდრის M ტონას (M-ზე მეტი მასის მქონე ტვირთს ვერ უძლებს). 
მდინარესთან, მეორე ნაპირზე გადასატანად მოიტანეს N რაოდენობის სხვადასხვა ტვირთი, რომელთაგან თითოეულის მასა ცნობილია.

დაადგინეთ, მოხერხდება თუ არა ბორანის ერთი გადასვლით ყველა ტვირთის გადატანა მდინარის მეორე ნაპირზე.

## Input

შემომავალი ფაილის პირველ სტრიქონზე მოცემულია ერთმანეთისაგან "ჰარით" გამოყოფილი ორი ნატურალური რიცხვი M და N. 1<=M<=1000, 1<=N<=100 - ბორანის ტვირთამწეობა და თვირთების რაოდენობა. 

მეორე სტრიქონზე მოცემულია N ცალი ნატურალური რიცხვი - თითოეული ტვირთის მასა ტონებში. 
ერთი ტვირთის მასა არ აღემატება 100 ტონას.

## Output

გამოიტანეთ სიტყვა YES, თუკი ბორანი შეძლებს ყველა ტვირთის გადატანას, წინააღმდეგ შემთხვევაში გამოიტანეთ სიტყვა NO.
