# Problem Statement — 620. მშვიდი მიმდევრობა

> Copied from [AiaSoft.ge #620](https://aiasoft.ge/problem/620)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ნულებისა და ერთებისაგან შედგენილ მიმდევრობას ეწოდება მშვიდი, თუ იგი არ შეიცავს ერთმანეთის გვერდით მდგარ ერთიანებს.
მაგალითად: 01001 მიმდევრობა მშვიდია, ხოლო 011001 მიმდევრობა არაა მშვიდი.
მოცემულია $n$. თქვენი დავალებაა იპოვოთ რამდენი განსხვავებული მშვიდი მიმდევრობა არსებობს ისეთი, რომლებიც შედგება ზუსტად $n$ ელემენტისგან,

## Input

ერთადერთ ხაზზე მოცემულია $n, 1<=n<=45$.

## Output

გამოიტანეთ ერთი მთელი რიცხვი - $n$ ელემენტიანი მშვიდი მიმდევრობების რაოდენობა.
