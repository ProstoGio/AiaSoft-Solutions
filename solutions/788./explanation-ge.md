# განმარტება — 788. ვინ დგას შუაში?

## მიდგომა

საკმაოდ მარტივი ამოცანაა, მთავარია სწორად დაიჭირო ინდექსები. თუ მასივის ზომა კენტია, შუა ელემენტი პირდაპირ n/2 ინდექსზე ზის. ლუწი ზომის შემთხვევაში კი, ცენტრში ორი ელემენტი გვხვდება: მარცხენა n/2-1 ინდექსზეა, ხოლო მარჯვენა — n/2-ზე.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- C++-ში მთელი რიცხვების გაყოფა (integer division) ავტომატურად ამრგვალებს ქვემოთ, რაც ზუსტად გვაძლევს შუა ინდექსს.
- მასივის სრულად გადარბენა სინამდვილეში არც გვჭირდება, თუ ინდექსს პირდაპირ გამოვთვლით, მაგრამ მარტივი ამოცანაა და 'for' ლუპითაც მშვენივრად მუშაობს.

## ხშირი შეცდომები

- ინდექსების არევა, განსაკუთრებით ლუწი შემთხვევისას, თუ დაგავიწყდა, რომ მასივი 0-დან იწყება.
