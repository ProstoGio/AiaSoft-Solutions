# განმარტება — 816. ტყუპი წევრები

## მიდგომა

აქ არაფერია რთული, უბრალოდ მასივში უნდა გაიარო და შეამოწმო მეზობელი ელემენტები. პირობის თანახმად, ორივე რიცხვი კენტი უნდა იყოს და სხვაობა ზუსტად 2. მე ამოვარჩიე ისეთი წყვილი, სადაც პირველი წევრი კენტია და სხვაობა ორის ტოლია — ეს ავტომატურად ნიშნავს, რომ მეორეც კენტი იქნება, რადგან კენტს რომ 2 მიუმატო ან გამოაკლო, ისევ კენტს მიიღებ.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- პირობა 'კენტი რიცხვები' მნიშვნელოვანია, თორემ მარტო სხვაობის შემოწმებით ლუწ წყვილებსაც დაიჭერდი.
- ციკლი `n-1`-მდე უნდა გაუშვა, თორემ `a[i+1]` ბოლო ელემენტის დროს გასცდება მასივის საზღვრებს.

## ხშირი შეცდომები

- მხოლოდ სხვაობის შემოწმება და კენტი რიცხვის პირობის დავიწყება.
