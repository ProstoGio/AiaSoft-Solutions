# Problem Statement — 013. მოქმედებები ვექტორზე

> Copied from [AiaSoft.ge #13](https://aiasoft.ge/problem/13)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 128 MB

## Task

ნიკას ძალიან უყვარს ვექტორებით თამაში. დღეს მან გადაწყვიტა შეადგინოს ვექტორზე მოქმედებების სია, რათა კარგად გაითავისოს ვექტორზე ოპერაციები.
ნიკამ Q ბრძანება ჩამოწერა ფურცელზე. სულ 5 სახის მოთხოვნისგან შედგება ნიკას ჩამოწერილი სია, ესენია: push, pop, insert, erase, print.

ნიკას სურს, რომ თავდაპირველად ვექტორი ცარიელი იყოს და შემდეგ შეასრულოს ჩამოწერილი მოქმედებები. თქვენი მიზანია დაწეროთ პროგრამა, რომელიც შეასრულებს ნიკას ჩამოწერილ ბრძანებებს.

## Input

პირველ ხაზზე შემოდის ნიკას ჩამოწერილი მოთხოვნების რაოდენობა:
შედეგ Q ხაზზე შემოდის ნიკას ჩამოწერილი მოქმედებები.

ნიკას ბრძანებები შემდეგნაირად აქვს ჩამოწერილი: ჯერ წერს ზემოთ ჩამოთვლილთაგან ერთ ერთ ბრძანებას (ანუ push, pop, insert, erase, print ამათგან ერთ-ერთს) შემდეგ კი საჭირო დეტალებს მიუწერს გვერდზე, კერძოდ:

- თუ ბრძანება არის push, მაშინ კიდევ ერთი x რიცხვი შემოვა ამავე ხაზზე, ანუ x რიცხვი უნდა ჩაემატოს ვექტორს ბოლოში.
- თუ ბრძანება არის pop, მაშინ პროგრამამ ვექტორიდან უნდა ამოიშალოს ბოლო წევრი.
- თუ ბრძანება არის insert, მაშინ კიდევ ორი რიცხვი შემოვა, p და x, ანუ x რიცხვი უნდა ჩაისვას ვექტორის p პოზიციაზე.
- თუ ბრძანება არის erase, მაშინ კიდევ ორი რიცხვი შემოვა, p1 და p2, ვექტორიდან უნდა ამოიშალოს [p1, p2] სეგმენტი (ანუ p1 პოზიციიდან p2 პოზიციის ჩათვლით ყველა წევრი)
- თუ ბრძანება არის print, მაშინ პროგრამამ უნდა გამობეჭდოს მოცემულ მომენტში ვექტორში არსებული ყველა წევრი.
$1 \leq Q \leq 100$

## Output

ყოველი print მოთხოვნისთვის ცალკე ხაზზე გამობეჭდეთ ერთმანეთისგან ჰარით გამოყოფილი ვექტორის ელემენტები.
