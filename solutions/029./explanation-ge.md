# განმარტება — 029. ნულების ჯგუფები

## მიდგომა

აქ მთავარია, ციკლში შესვლისას არ დაგვავიწყდეს, რომ ნულების ჯგუფი შეიძლება მასივის ბოლოშიც იყოს. `quanity`-ით ვითვლი მიმდინარე სერიას, და როგორც კი ნულისგან განსხვავებულ რიცხვს შევხვდები, ვამოწმებ, არის თუ არა ეს სერია რეკორდული და ვანულებ მრიცხველს. მარტივია, მთავარია ბოლო შემოწმება არ გამომრჩეს ციკლის გარეთ.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- ყველაზე ხშირი შეცდომაა იმის დავიწყება, რომ თუ მასივი ნულებით მთავრდება, ციკლის დასრულების შემდეგ შედეგი კიდევ ერთხელ უნდა შევამოწმოთ.
- არ გვჭირდება ზედმეტი მეხსიერება და მასივში შენახვა; პირდაპირ ნაკადიდან რომ ვკითხულობთ რიცხვებს, ეგრევე ვამუშავებთ.

## ხშირი შეცდომები

- ციკლის დასრულების შემდეგ `mainquanity`-ის განახლების გამოტოვება.
