# Problem Statement — 262. წიბოების რაოდენობა

> Copied from [AiaSoft.ge #262](https://aiasoft.ge/problem/262)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

მოცემულია არაორიენტირებული გრაფი მეზობლობის მატრიცით. დათვალეთ და გამობეჭდეთ წიბოების რაოდენობა.

* გრაფი შესაძლოა შეიცავდეს მარყუჟებს.

## Input

პირველ ხაზზე მოცემულია გრაფის წვეროების რაოდენობა: $N$ $(1 <= N <= 20)$

შემდეგ მოცემულია $N$ სტრიქონი, თითოეულზე $N$ რიცხვი, რომელიც ერთიანობაში წარმოადგენს $N$x$N$ ზომის გრაფის წვეროების მეზობლობის მატრიცას. თუ $a_(i,j) = 1$ მაშინ $i$ და $j$ მეზობელი წვეროებია, ხოლო თუ $a_(i,j) = 0$ - მაშინ არა.

## Output

გამობეჭდეთ ერთი მთელი რიცხვი. მოცემულ გრაფში წიბოების რაოდენობა.
