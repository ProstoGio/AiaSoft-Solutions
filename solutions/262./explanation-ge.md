# განმარტება — 262. წიბოების რაოდენობა

## მიდგომა

არაორიენტირებულ გრაფში მეზობლობის მატრიცა სიმეტრიულია. ეს იმას ნიშნავს, რომ თუ წვერო i უკავშირდება j-ს, მატრიცაში ეს ორჯერაა აღნიშნული: ერთხელ (i, j)-ში და მეორედ (j, i)-ში. უბრალოდ ყველაფრის დაჯამება და ორზე გაყოფა არ იმუშავებს, რადგან მარყუჟები (self-loops) მატრიცის დიაგონალზეა და ისინი მხოლოდ ერთხელაა ჩაწერილი. ამიტომ, დიაგონალზე მყოფ ერთიანებს ცალკე ვითვლი, დანარჩენებს ცალკე და ბოლოს ფორმულაც მარტივი გამოდის.

## სირთულე

| | |
|-|--|
| დრო | O(n²) |
| მეხსიერება | O(n²) |

## მთავარი დაკვირვებები

- დიაგონალს გარეთ არსებული წიბოები მატრიცაში ორჯერ გვხვდება, ამიტომ მათი ჯამი უნდა გავყოთ ორზე.
- თუ მატრიცის დიაგონალზე `g[i][i] == 1` გვაქვს, ეს მარყუჟია და ის პირდაპირ უნდა მივუმატოთ შედეგს.

## ხშირი შეცდომები

- ყველა ერთიანის დაჯამება და მხოლოდ ორზე გაყოფა, რაც მარყუჟების შემთხვევაში არასწორ პასუხს მოგცემს.
