# Problem Statement — 246. სამი კუ

> Copied from [AiaSoft.ge #246](https://aiasoft.ge/problem/246)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

სამი კუ a, b და c ერთმანეთს ცოცვაში ეჯიბრება, დაწერეთ პროგრამა, რომელიც დაადგენს, რომელი კუ მივა ფინიშთან ყველაზე ბოლოს, თუკი ჩვენთვის ცნობილია მათი სიჩქარეები.

## Input

კლავიატურიდან შემოდის სამი, ერთმანეთისაგან წყვილ-წყვილად განსხვავებული ნატურალური რიცხვი, შესაბამისად a, b, და c კუს სიჩქარეები მეტრ წუთებში.

## Output

თუ ფინიშთან ყველაზე ბოლოს მივა პირველი კუ, გამოიტანეთ სიმბოლო a, თუ ყველაზე ბოლოს მივა მეორე კუ, მაშინ გამოიტანეთ სიმბოლო b, წინააღმდეგ შემთხვევაში  - სიმბოლო c.
