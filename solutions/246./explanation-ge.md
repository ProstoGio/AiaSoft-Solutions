# განმარტება — 246. სამი კუ

## მიდგომა

აქ არაფერია რთული, ფიზიკიდან ხომ ვიცით რომ რაც უფრო ნაკლებია სიჩქარე, მით მეტი დრო სჭირდება კუს ფინიშამდე მისვლას. შესაბამისად, ვისაც ყველაზე პატარა რიცხვი აქვს მინიჭებული, ის მივა ბოლოს. უბრალოდ შევადარე სამივე რიცხვი `min` ფუნქციით და ამოვბეჭდე შესაბამისი სიმბოლო.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- ყველაზე პატარა რიცხვი ავტომატურად ნიშნავს ყველაზე დაბალ სიჩქარეს და შესაბამისად, ბოლო ადგილს.
- `min({a, b, c})` სინტაქსი C++-ში გაცილებით მოსახერხებელია, ვიდრე `if`-ების ბუდეები.

## ხშირი შეცდომები

- შეიძლება ვინმეს აერიოს და მაქსიმუმი ეძებოს მინიმუმის მაგივრად.
