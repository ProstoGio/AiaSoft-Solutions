# Problem Statement — 217. მთავარი დიაგონალები

> Copied from [AiaSoft.ge #217](https://aiasoft.ge/problem/217)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია მატრიცა, იპოვეთ მის მთავარ (მარცხენა და მარჯვენა) დიაგონალებზე მყოფი ელემენტების ჯამი.

## Input

პირველ სტრიქონზე შემოდის n, მასივის სტრიქონების რაოდენობა, 1<=n<=20.
მომდევნო n რაოდენობის სტრიქონზე მოცემულია კვადრატული მატრიცა, ელემენტებს შორის თითო ჰარია. თითოეული ელემენტის სიდიდე მოდულით არ აღემატება 100-ს.

## Output

გამოიტანეთ ერთი რიცხვი - მატრიცის დიაგონალების ელემენტების ჯამი.
გაქვთ გარანტია, რომ შედეგი არ აღემატება 2⁹-ს.
