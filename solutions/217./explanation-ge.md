# განმარტება — 217. მთავარი დიაგონალები

## მიდგომა

ამ ამოცანაში ყველაზე მარტივი გზაა ერთი ციკლით ორივე დიაგონალი დავთვალოთ. მთავარი დიაგონალი პირდაპირ a[i][i]-ით იკრიბება, ხოლო მეორესთვის მეორე ინდექსს, ვთქვათ k-ს, ბოლოდან ვიწყებთ და თითოეულ ნაბიჯზე ვაკლებთ. მთავარი მომენტი აქ ისაა, რომ თუ მატრიცა კენტია, შუა ელემენტი ორივე დიაგონალში ზის, ამიტომ ბოლოს ერთხელ უნდა გამოვაკლოთ, რომ ჯამი სწორი დარჩეს.

## სირთულე

| | |
|-|--|
| დრო | O(n²) |
| მეხსიერება | O(n²) |

## მთავარი დაკვირვებები

- ანტი-დიაგონალის ელემენტები ყოველთვის n-1-i ინდექსზე ზის.
- კენტი n-ის შემთხვევაში ცენტრალური ელემენტი იკრიბება ორჯერ, რაც შეცდომას იწვევს, თუ არ გავითვალისწინებთ.

## ხშირი შეცდომები

- შუა ელემენტის ორმაგი დათვლა კენტი მატრიცების დროს.
