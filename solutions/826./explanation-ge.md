# განმარტება — 826. სამნიშნას უდიდესი

## მიდგომა

სამნიშნა რიცხვია და ზედმეტი თავის ტკივილი არ გვჭირდება, მათემატიკურად დავშლი ციფრებად. ასეულებს ვიღებ 100-ზე გაყოფით, ათეულებს - 10-ზე გაყოფის და მერე მოდულოს გამოყენებით, ბოლოს კი ერთეულებს უბრალოდ 10-ზე მოდულოთი ვიგდებ ხელში. დანარჩენი უკვე მარტივია, სამივე ციფრს std::max-ში ჩავყრი და უდიდესს პირდაპირ დამიბრუნებს.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- ინტეჯერების გაყოფა C++-ში ავტომატურად აკლებს წილად ნაწილს, რაც ციფრების გამოსაცალკევებლად ზედგამოჭრილია.
- std::max-ის გამოყენება initializer list-ით ბევრად უფრო კომპაქტურია, ვიდრე if-else-ების რთული ჯაჭვების წერა.

## ხშირი შეცდომები

- რიცხვის სტრინგად გარდაქმნა მაშინ, როცა მათემატიკური ოპერაციებით უფრო სწრაფად და სუფთად კეთდება.
