# Problem Statement — 224. გადათხრილი გზები

> Copied from [AiaSoft.ge #224](https://aiasoft.ge/problem/224)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

საქართველოში გრანდიოზული სამუშაოები დაიწყო გზების შეკეთების მიზნით, რის გამოც ზოგიერთი საქალაქთაშორისო გზის გადათხრა გახდა საჭირო რეაბილიტაციის მიზნით. 
ქვეყანაში სულ $N$ ქალაქი (რომლებიც გადანომრილია $1$-დან $N$-ის ჩათვლით) და $M$ საქალაქთაშორისო გზაა. თითო გზა რომელიმე ორ ქალაქს აკავშირებს. ჩვენთვის ცნობილია რომელ გზაზე მიმდინარეობს სამუშაოები და რომელზე არა.
`~https://storage.googleapis.com/problems_api_photos/images/ELbPu4BiAh0SDcl02l4n.png`
ახლა გაიცანით ამოცანის გმირი, ბაქარა, რომელიც $A$ ქალაქში სამუშაოდ იყო წასული. მას სურს, რომ ქალაქ $B$-ში ჩავიდეს, სადაც მას მოუთმენლად ელის ოჯახი. ბაქარას აინტერესებს შეძლებს თუ არა გადათხრილი გზებისგან თავის არიდებით მიაღწიოს $B$ ქალაქამდე.

დაეხმარეთ ბაქარას და აჩვენეთ მას გზა, რომლითაც უნდა ჩავიდეს $A$-დან $B$-ში თუ ასეთი არსებობს.

## Input

პირველ ხაზზე შემოდის ოთხი მთელი რიცხვი $N, M, A$ და $B$.

მომდევნო $M$ ხაზზე შემოდის სამი მთელი რიცხვი, გზის აღწერა, კერძოდ, $i$-ურ ხაზზე შემოდის $v_i, u_i$ - მოცემული გზით დაკავშირებული ქალაქების ნომრები და $d_i$ - თუ $d_i=1$ მაშინ $i$ გზა დაზიანებულია, ხოლო თუ $d_i=0$ - მაშინ გზაზე გავლა შესაძლებელია.

$2 <= N <= 10^3$
$1 <= A, B, v_i, u_i <= N$
$0 <= M <= 10^5$
$0 <= d_i <= 1$

## Output

თუ $A$-დან $B$-ში ჩასვლა გადათხრილი გზებისგან თავის არიდებით შესაძლებელია, მაშინ პირველ ხაზზე გამოიტანეთ `bYES`, მეორე ხაზზე კი გამოიტანეთ `bგზა` $A$-დან $B$-მდე, ანუ ქალაქების თანმიმდევრობა, რომლებიც უნდა გაიაროს ბაქარამ. რამდენიმე განსხვავებული გზის არსებობის შემთხვევაში გამოიტანეთ ნებისმიერი ერთი.

წინააღმდეგ შემთხვევაში გამოიტანეთ `bNO`
