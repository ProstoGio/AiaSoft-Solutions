# Problem Statement — 349. იპოვე მინიმალური

> Copied from [AiaSoft.ge #349](https://aiasoft.ge/problem/349)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია $N$ მთელი რიცხვისგან შემდგარი მასივი. იპოვეთ მოცემული მასივის მინიმალური ელემენტი და გამობეჭდეთ.

* მასივი შეიძლება შეიცავდეს უარყოფით რიცხვებსაც.

## Input

პირველ ხაზზე მოცემულია ერთი მთელი რიცხვი: $N$ - მასივის ელემენტების რაოდენობა
მორე ხაზზე მოცემულია $N$ რაოდენობის მთელი რიცხვი: $a_1, a_2, ..., a_n$

$1 <= N <= 100$
$0 <= |a_i| <= 100$

## Output

გამობეჭდეთ ერთი მთელი რიცხვი: შემოსული მასივის მინიმალური ელემენტი.
