# განმარტება — 349. იპოვე მინიმალური

## მიდგომა

ეს არის კლასიკური ამოცანა, სადაც უბრალოდ მასივს უნდა გადავუაროთ. თავიდან პირველივე ელემენტს ვიღებ როგორც 'მინიმალურს' და მერე თითოეულ რიცხვს ვამოწმებ – თუ რამე ჩემს შენახულზე პატარა შემხვდა, იმას ვანახლებ. მეტი არაფერია საჭირო, ერთი ციკლით ვწყვეტთ მთლიანად.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- მინიმალურის საწყის მნიშვნელობად ყოველთვის მასივის პირველი წევრი დააყენე და არა 0, თორემ თუ ყველა რიცხვი დადებითი იქნება, ლოგიკა აგერევა.
- სივრცის მხრივ O(n)-ია, რადგან მასივს ვინახავთ, თუმცა რეალურად რიცხვების დამახსოვრება არც გვჭირდება, ერთ ცვლადშიც შეიძლება აკუმულაცია.

## ხშირი შეცდომები

- 0-ის მინიჭება მინიმუმისთვის – თუ მასივში მარტო დადებითი რიცხვებია, პასუხი ყოველთვის 0 გამოვა, რაც არასწორია.
