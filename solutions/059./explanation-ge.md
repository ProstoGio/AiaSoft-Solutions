# განმარტება — 059. კბილის ექიმი

## მიდგომა

სინამდვილეში, ეს არის ჩვეულებრივი რიგის სიმულაცია. რადგან ექიმი ყველაზე ადრე მოსულს ემსახურება, პირველ რიგში საჭიროა პაციენტების დალაგება მისვლის დროის მიხედვით. მერე უბრალოდ ვინახავთ `exitT`-ს, ანუ დროს, როცა ექიმი მორჩება კონკრეტულ პაციენტს. თუ შემდეგი პაციენტი მანამდე მოდის, სანამ ექიმი დაკავებულია, ის რიგში დგება; თუ ექიმი თავისუფალია, მაშინვე იწყებს მკურნალობას.

## სირთულე

| | |
|-|--|
| დრო | O(n²) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- რადგან N მაქსიმუმ 721-ია, Bubble Sort-იც კი საკმარისია და არ არის საჭირო რამე უფრო რთული.
- მთავარია სწორად გავითვალისწინოთ ექიმის 'დაკავებულობის' დრო: თუ ექიმი უკვე თავისუფალია, პაციენტის დაწყების დრო არის მისი მოსვლის დრო და არა წინა პაციენტის დამთავრების დრო.

## ხშირი შეცდომები

- იმის დავიწყება, რომ ექიმი არ აგვიანებს, თუ პაციენტი მისვლისას უკვე თავისუფალია.
