# Problem Statement — 476. უდიდესი ზრდადი ქვემიმდევრობა - 2

> Copied from [AiaSoft.ge #476](https://aiasoft.ge/problem/476)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია მთელი რიცხვების N ელემენტიანი მიმდევრიბა. იპოვეთ მოცემული მიმდევრობის უდიდესი მკაცრად ზრდადი ქვემიმდევრობა.

## Input

პირველ ხაზზე მოცემულია მიმდევრობის ელემენტების რაოდენობა: $N$ $(1 <= N <=100)$
მეორე ხაზზე მოცემულია $N$ რაოდენობის მთელი რიცხვი: $a_1, a_2, ..., a_n$ - მიმდევრობის წევრები $(0 <= a_i <= 100)$

## Output

პირველ ხაზზე გამობეჭდეთ მიღებული უდიდესი ზრდადი ქვემიმდევრობის სიგრძე.
მეორე ხაზზე გამობეჭდეთ მიღებული უდიდესი ზრდადი ქვემიმდევრობის ელემენტები.

* რამდენიმე პასუხის შემთხვევაში გამობეჭდეთ ნებისმიერი.
