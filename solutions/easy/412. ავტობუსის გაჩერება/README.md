# ამოხსნის ახსნა

თავდაპირველად ავტობუსში იმყოფება `N` მგზავრი.

გაჩერებაზე ელოდება კიდევ `M` მგზავრი.

ავტობუსში არის:

- `X` დასაჯდომი ადგილი;
- `Y` ფეხზე დასადგომი ადგილი.

ავტობუსიდან არავინ ჩადის, ამიტომ უნდა დავადგინოთ:

- რამდენი მგზავრი დაჯდება;
- რამდენი დარჩება ფეხზე;
- რამდენი საერთოდ ვერ ჩაეტევა ავტობუსში.

---

# მაგალითი

თუ მოცემულია:

```text
N = 8
M = 5

X = 10
Y = 2
```

მაშინ სულ გვყავს:

```text
8 + 5 = 13
```

მგზავრი.

---

ჯერ ივსება დასაჯდომი ადგილები:

```text
10
```

შემდეგ ფეხზე დასადგომი:

```text
2
```

ერთი მგზავრი კი ვეღარ ეტევა.

პასუხი იქნება:

```text
10 2 1
```

---

# მთავარი იდეა

ჯერ დავითვალოთ ავტობუსში მყოფი მგზავრების საერთო რაოდენობა.

```cpp
int total = n + m;
```

---

ამის შემდეგ ეტაპობრივად ვანაწილებთ მგზავრებს.

ჯერ ივსება სკამები.

შემდეგ, ვინც ვერ დაჯდა, ფეხზე დგება.

ბოლოს დარჩენილი მგზავრები ავტობუსში ვეღარ ეტევიან.

---

# სკამებზე მსხდომები

სკამებზე შეიძლება დაჯდეს მაქსიმუმ `X` ადამიანი.

თუ მგზავრი ნაკლებია, ყველა დაჯდება.

ამისთვის ვიყენებთ:

```cpp
min(total, x)
```

```cpp
int sitting = min(total, x);
```

---

მაგალითად:

```text
total = 7
X = 10
```

პასუხია:

```text
7
```

---

თუ:

```text
total = 15
X = 10
```

მაშინ დაჯდება:

```text
10
```

---

# ფეხზე მდგომები

ჯერ გამოვაკლებთ უკვე დამჯდარ მგზავრებს.

```cpp
total - sitting
```

შემდეგ ვნახავთ, რამდენი ეტევა ფეხზე დასადგომ ადგილებზე.

```cpp
int standing = min(total - sitting, y);
```

---

მაგალითად:

```text
დარჩა 6 მგზავრი
Y = 4
```

ფეხზე დადგება:

```text
4
```

---

თუ დარჩა:

```text
2
```

მაშინ ორივე დაეტევა.

---

# ვინც ვერ ჩაეტია

დარჩენილი მგზავრების რაოდენობა იქნება:

```cpp
int couldnt_fit = total - sitting - standing;
```

---

ესენი არიან ადამიანები, რომლებიც ვეღარ ავიდნენ ავტობუსში.

---

# კოდის განხილვა

## საერთო რაოდენობის დათვლა

```cpp
int total = n + m;
```

---

## დასაჯდომი ადგილების შევსება

```cpp
int sitting = min(total, x);
```

---

## ფეხზე მდგომების დათვლა

```cpp
int standing = min(total - sitting, y);
```

---

## ვინც ვერ ჩაეტია

```cpp
int couldnt_fit = total - sitting - standing;
```

---

## პასუხის გამოტანა

```cpp
cout << sitting << " "
     << standing << " "
     << couldnt_fit;
```

---

# რატომ მუშაობს?

მგზავრები ავტობუსში შემდეგი წესით ნაწილდებიან:

1. ჯერ ივსება ყველა თავისუფალი სკამი.
2. შემდეგ ივსება ფეხზე დასადგომი ადგილები.
3. თუ ამის შემდეგაც დარჩნენ მგზავრები, ისინი ავტობუსში ვეღარ ადიან.

პროგრამა ზუსტად ამავე ნაბიჯებს ასრულებს, ამიტომ ყოველთვის სწორ პასუხს აბრუნებს.

# შეჯამება

პირველ რიგში ვითვლით მგზავრების საერთო რაოდენობას.

შემდეგ ეტაპობრივად ვანაწილებთ მათ:

- ჯერ სკამებზე;
- შემდეგ ფეხზე დასადგომ ადგილებზე;
- ბოლოს კი ვითვლით იმ მგზავრებს, რომლებმაც ავტობუსში ასვლა ვერ მოახერხეს.

ასე ძალიან მარტივად ვიღებთ მოთხოვნილ სამივე პასუხს.
