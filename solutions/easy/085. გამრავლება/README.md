# ამოხსნის ახსნა

მოცემულია ორი მთელი რიცხვი:

```text
a და b
```

საჭიროა გამოვთვალოთ მათი ნამრავლი.

---

# მთავარი იდეა

ამოცანა ძალიან მარტივია.

ვკითხულობთ ორ რიცხვს, ვამრავლებთ ერთმანეთს და მიღებულ შედეგს ვბეჭდავთ.

---

# კოდის განხილვა

## ცვლადების გამოცხადება

```cpp
int sum, a, b;
```

- `a` და `b` ინახავენ შემოსულ რიცხვებს.
- `sum` ინახავს მათ ნამრავლს.

---

## მონაცემების წაკითხვა

```cpp
cin >> a >> b;
```

ვკითხულობთ ორ მთელ რიცხვს.

---

## ნამრავლის გამოთვლა

```cpp
sum = a * b;
```

`*` ოპერატორი ითვლის ორი რიცხვის ნამრავლს.

---

## პასუხის გამოტანა

```cpp
cout << sum;
```

ბეჭდავს მიღებულ შედეგს.

---

# რატომ მუშაობს?

პროგრამა იღებს ორ მთელ რიცხვს, ამრავლებს მათ ერთმანეთზე და მიღებულ ნამრავლს ბეჭდავს.
