# ამოხსნის ახსნა

მოცემულია ორი ლათინური მთავრული ასო.

საჭიროა დაიბეჭდოს ყველა ასო:

- პირველი ასოდან;
- მეორე ასომდე (ჩათვლით).

---

# მაგალითი

თუ შემოვიდა:

```text
A D
```

უნდა დაიბეჭდოს:

```text
A B C D
```

---

თუ შემოვიდა:

```text
M Q
```

პასუხია:

```text
M N O P Q
```

---

# მთავარი იდეა

კომპიუტერში თითოეულ სიმბოლოს თავისი რიცხვითი კოდი აქვს (ASCII კოდი).

მაგალითად:

```text
'A' = 65
'B' = 66
'C' = 67
...
'Z' = 90
```

რადგან ეს კოდები ერთმანეთის მიყოლებითაა განლაგებული, შეგვიძლია ერთი ასოდან მეორემდე უბრალოდ ციკლით გავიაროთ.

---

# სიმბოლოების მასივი

კოდში ჯერ იქმნება მასივი:

```cpp
char c[90];
```

შემდეგ:

```cpp
for(int i=41; i<=90; i++)
    c[i]=i;
```

---

აქ თითოეულ ინდექსზე ვინახავთ შესაბამის ASCII სიმბოლოს.

მაგალითად:

```text
c[65] = 'A'
c[66] = 'B'
...
c[90] = 'Z'
```

---

# მონაცემების წაკითხვა

```cpp
char a, b;
cin >> a >> b;
```

აქ:

- `a` არის პირველი ასო;
- `b` არის მეორე ასო.

---

# პირველი შემთხვევა

თუ:

```cpp
a <= b
```

მაშინ მივდივართ ანბანის ზრდის მიმართულებით.

```cpp
for(int i=a; i<=b; i++){
    cout << c[i] << " ";
}
```

---

მაგალითად:

```text
A D
```

ციკლი გაივლის:

```text
A
B
C
D
```

---

# მეორე შემთხვევა

თუ:

```cpp
a > b
```

მაშინ პირიქით მივდივართ.

```cpp
for(int i=a; i>=b; i--){
    cout << c[i] << " ";
}
```

---

მაგალითად:

```text
D A
```

დაიბეჭდება:

```text
D C B A
```

---

# რატომ მუშაობს?

ASCII ცხრილში ასოები მიმდევრობით არის განლაგებული.

მაგალითად:

```text
65 66 67 68
 A  B  C  D
```

ამიტომ თუ რიცხვებს თანმიმდევრობით გავივლით, შესაბამისი ასოებიც სწორი მიმდევრობით დაიბეჭდება.

---

# კოდის განხილვა

## სიმბოლოების მასივის შევსება

```cpp
for(int i=41; i<=90; i++)
    c[i]=i;
```

---

## ასოების წაკითხვა

```cpp
char a, b;
cin >> a >> b;
```

---

## ზრდადობით ბეჭდვა

```cpp
if(a <= b)
```

---

## კლებადობით ბეჭდვა

```cpp
else if(a > b)
```

---

# სირთულე

თუ ასოებს შორის მანძილია `K`:

### დროითი სირთულე

```text
O(K)
```

რადგან თითოეული სიმბოლო ერთხელ იბეჭდება.

---

### მეხსიერების სირთულე

```text
O(1)
```

---

# შეჯამება

პროგრამა იყენებს იმას, რომ ლათინური ასოების ASCII კოდები თანმიმდევრობით არის განლაგებული.

თუ პირველი ასო მეორე ასოზე ადრე მოდის, ციკლი წინ მიდის.

თუ პირველი ასო მეორე ასოზე გვიან მოდის, ციკლი უკუღმა მოძრაობს.

ასე იბეჭდება ყველა სიმბოლო პირველიდან ბოლომდე.

---

# შენიშვნა

ამ ამოცანის ამოხსნა შეიძლება კიდევ უფრო მარტივადაც.

რადგან `char` ტიპზე პირდაპირ შეგვიძლია ციკლის გამოყენება, `c` მასივის შექმნა საერთოდ საჭირო არ არის.

მაგალითად:

```cpp
for(char ch = a; ch <= b; ch++)
    cout << ch << " ";
```

ანალოგიურად შეიძლება კლებადობის შემთხვევაშიც.

ჩემი კოდი სწორად მუშაობს, თუმცა დამატებითი მასივი აუცილებელი არ არის.
