# ამოხსნის ახსნა

მოცემულია ორი მთელი რიცხვი:

```text
a და b
```

საჭიროა ვიპოვოთ მათი ჯამი.

---

# მთავარი იდეა

ამოცანა ძალიან მარტივია.

ვკითხულობთ ორ რიცხვს და ვბეჭდავთ მათ ჯამს.

---

# კოდის განხილვა

## მონაცემების წაკითხვა

```cpp
int num1, num2;
cin >> num1 >> num2;
```

აქ ვინახავთ ორ შემოსულ მთელ რიცხვს.

---

## ჯამის გამოთვლა

```cpp
cout << num1 + num2;
```

`+` ოპერატორი კრიბავს ორ რიცხვს, ხოლო `cout` მიღებულ შედეგს ბეჭდავს.

---

# რატომ მუშაობს?

პროგრამა იღებს ორ მთელ რიცხვს, ითვლის მათ ჯამს და პირდაპირ გამოაქვს პასუხი.

---

# სირთულე

### დროითი სირთულე

```text
O(1)
```

სრულდება მხოლოდ რამდენიმე მარტივი ოპერაცია.

---

### მეხსიერების სირთულე

```text
O(1)
```

გამოიყენება მხოლოდ ორი ცვლადი.

---

# შეჯამება

პროგრამა კითხულობს ორ მთელ რიცხვს, კრიბავს მათ და მიღებულ ჯამს ბეჭდავს.
