# განმარტება — 055. გაფუჭებული ვაშლები

## მიდგომა

აქ მთავარია მივხვდეთ, რომ დაავადებული ვაშლის გარშემო 8 მეზობელი ფუჭდება. არ გვჭირდება რთული ალგორითმები, უბრალოდ მატრიცა უნდა შევავსოთ ნულებით და სადაც გაფუჭებული ვაშლია, იქ და მისი რვა მეზობელი უჯრა გავხადოთ 1-იანები. პატარა დაფიქრება სჭირდება მხოლოდ საზღვრებს, ანუ თუ ვაშლი პირველ რიგშია, ზედა მეზობლების შემოწმებისას პრობლემა რომ არ შეგვექმნას.

## სირთულე

| | |
|-|--|
| დრო | O(N * M) |
| მეხსიერება | O(N * M) |

## მთავარი დაკვირვებები

- ყველაზე მარტივი გზაა მთელი მატრიცა თავიდანვე 0-ებით გავავსოთ და მერე 'ხელით' მივუთითოთ გაფუჭებული კოორდინატები.
- თუ ინდექსები 1-დან იწყება, დარწმუნდი რომ საზღვრებს გარეთ არ გახვალ, თუმცა პატარა მატრიცისთვის ეს ადვილად მოსაგვარებელია.

## ხშირი შეცდომები

- საზღვრის პირობების დავიწყება, რამაც შეიძლება 'segmentation fault' გამოიწვიოს თუ ვაშლი კუთხეშია.
- მატრიცის ინდექსების არასწორად აღქმა (0-დან იწყება თუ 1-დან).
