# Problem Statement — 917. ჭიანჭველები

> Copied from [AiaSoft.ge #917](https://aiasoft.ge/problem/917)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ჭიანჭველების მიწისქვეშა ქალაქში გამუდმებული სიბნელეა, ამიტომაც ქალაქის მთავარმა ჭიანჭველამ გადაწყვიტა თითოეულ დარბაზში საახალწლოდ განათება მოეწყო. გადაწყდა ორი სხვადასხვა ფერის, წითელის და მწვანის  გამოყენება.
მრავალფეროვნების მიზნით საჭირო იყო შემდეგი წესის დაცვა: ორ სხვადასხვა დარბაზში, რომელთაც აერთებდა პირდაპირი გვირაბი, არ შეიძლებოდა ერთიდაიგივე ფერის განათების მოწყობა. 
მთავარმა ჭიანჭველამ შეკრიბა ჭიანჭველების ქალაქის მთავარი საბჭო, რათა გადაეწყვიტათ, როგორ დაეძლიათ ქალაქის სიბნელე. მთავარი ჭიანჭველა ეჭვობს, საერთოდ, შესაძლებელია თუ არა ასეთი რამე.
ამ პროექტის სასწრაფო ხორცშეცხმისათვის, რა თქმა უნდა, პროგრამისტების ჩარევაა საჭირო.

## Input

პირველ ხაზზე შემოდის დარბაზებისა და გვირაბების რაოდენობა, N<=50, M<=N*(N-1)/2
ყოველ მომდევნო სტრიქონზე შემოდის ორი მთელი რიცხვი - იმ დარბაზების ნომრები, რომელთაც გვირაბი აერთებს.

## Output

გამოიტანეთ სიტყვა "YES", თუ ჭიანჭველების პრობლემა გადაჭრადია.
პრობლემის მოუგვარებლობის შემთხვევაში გამოიტანეთ სიტყვა "NO".
