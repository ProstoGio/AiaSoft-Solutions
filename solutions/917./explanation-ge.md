# განმარტება — 917. ჭიანჭველები

## მიდგომა

ეს არის კლასიკური ბიპარტიტული გრაფის (bipartite graph) ამოცანა. იდეა მარტივია: ვირჩევთ დაწყების წერტილს, ვაძლევთ პირველ ფერს და მერე რეკურსიულად მივყვებით მეზობლებს. ყოველი მომდევნო კვანძი უნდა იყოს საპირისპირო ფერის. თუ უცებ შევხვდით უკვე გაფერადებულ კვანძს, რომელსაც ჩვენი მიმდინარე ფერი აქვს, ესე იგი გრაფი ვერ დაიყოფა ორ ფერად და პირდაპირ ვბეჭდავთ NO-ს.

## სირთულე

| | |
|-|--|
| დრო | O(N + M) |
| მეხსიერება | O(N + M) |

## მთავარი დაკვირვებები

- გრაფი არის ბიპარტიტული მხოლოდ მაშინ, თუ მასში არ არის კენტი სიგრძის ციკლები.
- არ დაგავიწყდეთ `for` ციკლით მთავარ ფუნქციაში ყველა კომპონენტის შემოწმება, რადგან გრაფი შეიძლება იყოს დაშლილი.

## ხშირი შეცდომები

- მხოლოდ პირველი წვეროდან დაწყებული DFS-ის გაშვება, როცა გრაფი მთლიანად არაა დაკავშირებული.

## ნაბიჯ-ნაბიჯ ახსნა

წარმოიდგინე: ვარ 1-ელ დარბაზში, ვაფერადებ '1'-ით. მის ყველა მეზობელს ვაძლევ '0'-ს. თუ მეზობლის მეზობელიც '0'-იანია, მაშინ უკვე ვიცით, რომ კენტი ციკლია და ეგრევე ვწყვეტთ მუშაობას.
