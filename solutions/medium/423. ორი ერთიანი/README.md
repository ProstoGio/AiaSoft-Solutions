# ამოხსნის ახსნა

ამოცანაში მოცემულია რიცხვი `N`.

უნდა დავითვალოთ რამდენი ისეთი ნატურალური რიცხვი არსებობს, რომელიც:

- არ აღემატება `N`-ს;
- ორობით ჩანაწერში ზუსტად ორი `1` აქვს.

---

# მაგალითი

რიცხვი:

```text
5
```

ორობით სისტემაში იწერება:

```text
101
```

მასში ორი ერთიანია, ამიტომ ითვლება.

---

რიცხვი:

```text
6
```

ორობით სისტემაში:

```text

```

აქაც ზუსტად ორი ერთიანია.

---

```text

```

აქ უკვე სამი ერთიანია და არ ითვლება.

# მთავარი იდეა

რიცხვს ზუსტად ორი ერთიანი რომ ჰქონდეს, ის აუცილებლად ასეთი ფორმის იქნება:

```text
2^i + 2^j
```

სადაც:

```text
i ≠ j
```

---

მაგალითად:

```text
2^3 + 2^1
=
8 + 2
=
10
```

ორობით ჩანაწერი:

```text
1010
```

ზუსტად ორი ერთიანი.

---

კიდევ ერთი მაგალითი:

```text
2^4 + 2^2
=
16 + 4
=
20
```

ორობით ჩანაწერი:

```text
10100
```

კვლავ ორი ერთიანი.

---

ამიტომ ამოცანა რეალურად ითხოვს დავითვალოთ რამდენი წყვილი:

```text
(i, j)
```

არსებობს ისეთი, რომ

```text
2^i + 2^j ≤ N
```

---

# რატომ გვყოფნის 32 ბიტი?

შეზღუდვაა:

```text
N < 10^9
```

ხოლო:

```text
2^30 ≈ 10^9
```

ამიტომ 32 ბიტზე მეტი შემოწმება საჭირო არ არის.

---

# რიცხვის აგება

თითოეული წყვილისთვის ვაგებთ შესაბამის რიცხვს:

```cpp
(1 << i) + (1 << j)
```

---

მაგალითად:

```cpp
i = 4
j = 1
```

მივიღებთ:

```text
16 + 2 = 18
```

ორობითში:

```text
10010
```

სადაც ზუსტად ორი ერთიანია.

---

# შემოწმება

თუ მიღებული რიცხვი არ აღემატება `N`-ს:

```cpp
if((1<<i)+(1<<j)<=x){
    ans++;
}
```

ვზრდით პასუხს.

---

# რატომ მუშაობს `else` ბლოკი?

```cpp
else{
    cout<<ans;
    return 0;
}
```

გაითვალისწინეთ, რომ:

```text
2^i + 2^j
```

იზრდება როგორც `i` და `j` იზრდება.

---

როდესაც პირველად მივიღებთ რიცხვს, რომელიც უკვე მეტია `N`-ზე, შემდეგი კომბინაციები კიდევ უფრო დიდი იქნება.

ამიტომ შეგვიძლია პროგრამა მაშინვე დავასრულოთ.

# რატომ მუშაობს ეს ამოხსნა?

ნებისმიერ რიცხვს, რომელსაც ორობით ჩანაწერში ზუსტად ორი ერთიანი აქვს, შეგვიძლია ჩავწეროთ სახით:

```text
2^i + 2^j
```

და პირიქით,

ნებისმიერი:

```text
2^i + 2^j
```

ზუსტად ორი ერთიანის მქონე რიცხვს ქმნის.

---

ამიტომ ყველა შესაძლო წყვილის გადამოწმება ნიშნავს ყველა შესაძლო პასუხის გადამოწმებას.

არც ერთი ვარიანტი არ გვრჩება გამოტოვებული და არც ერთი არ ითვლება ორჯერ.

# შეჯამება

რიცხვი, რომლის ორობით ჩანაწერში ზუსტად ორი ერთიანია, ყოველთვის შეიძლება ჩაიწეროს ფორმით:

```text
2^i + 2^j
```

ამიტომ:

1. ვარჩევთ ორი ერთიანის პოზიციებს;
2. ვაგებთ შესაბამის რიცხვს;
3. ვამოწმებთ, აღემატება თუ არა `N`-ს;
4. თუ არა, ვითვლით პასუხში.

ასე სწრაფად და მარტივად ვპოულობთ ყველა ისეთ რიცხვს, რომლის ორობით ჩანაწერში ზუსტად ორი ერთიანია და რომლის მნიშვნელობაც არ აღემატება მოცემულ `N`-ს.
