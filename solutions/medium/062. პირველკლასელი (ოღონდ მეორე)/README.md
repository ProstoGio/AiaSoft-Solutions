# ამოხსნის ახსნა

ამოცანაში მოცემულია სტრიქონი, რომელიც შეიცავს ორ რიცხვს და მათ შორის ერთ არითმეტიკულ ოპერაციას:

- `+`
- `-`
- `*`

მაგალითად:

```text
12+23
25-17
321*42
```

საჭიროა ამოვიცნოთ:

1. პირველი რიცხვი;
2. ოპერაცია;
3. მეორე რიცხვი;

და შემდეგ გამოვთვალოთ შედეგი.

---

## მთავარი იდეა

რადგან სტრიქონში მხოლოდ ერთი ოპერაციაა მოთავსებული, შეგვიძლია:

1. ვიპოვოთ ოპერაციის ნიშანი;
2. მის მარცხნივ მდებარე სიმბოლოები შევინახოთ პირველ რიცხვად;
3. მის მარჯვნივ მდებარე სიმბოლოები შევინახოთ მეორე რიცხვად;
4. ორივე სტრიქონი გადავიყვანოთ რიცხვებად;
5. შევასრულოთ შესაბამისი ოპერაცია.

---

## 1. ათის ხარისხების მომზადება

პროგრამის დასაწყისში ვქმნით ვექტორს:

```cpp
vector<int> v(10);
```

რომელშიც ვინახავთ:

```text
1
10
100
1000
10000
...
```

ამისთვის ვიყენებთ ციკლს:

```cpp
int temp = 1;

for(int i=0; i<=9; i++){
    v[i] = temp;
    temp *= 10;
}
```

შედეგად:

```text
v[0] = 1
v[1] = 10
v[2] = 100
v[3] = 1000
...
```

ეს მოგვიანებით დაგვეხმარება სტრიქონის რიცხვად გადაყვანაში.

---

## 2. გამოსახულების წაკითხვა

```cpp
string s;
cin >> s;
```

მაგალითად:

```text
321*42
```

---

## 3. პირველი რიცხვის და ოპერაციის პოვნა

ვქმნით ცვლადებს:

```cpp
string a = "", b = "";
char c;
int idx;
```

სადაც:

- `a` → პირველი რიცხვი
- `b` → მეორე რიცხვი
- `c` → ოპერაციის ნიშანი
- `idx` → ოპერაციის პოზიცია

---

შემდეგ გადავდივართ მთელ სტრიქონზე:

```cpp
for(int i=0; i<s.size(); i++){
```

თუ სიმბოლო ოპერაცია არ არის:

```cpp
if(s[i]!='-' && s[i]!='+' && s[i]!='*'){
    a += s[i];
}
```

ვამატებთ პირველ რიცხვს.

---

მაგალითად:

```text
321*42
```

პირველი ნაბიჯების შემდეგ:

```text
a = "321"
```

---

როგორც კი ოპერაციას ვიპოვით:

```cpp
else{
    c = s[i];
    idx = i;
    break;
}
```

ვიმახსოვრებთ:

```text
c = '*'
idx = 3
```

და ვწყვეტთ ციკლს.

---

## 4. მეორე რიცხვის მიღება

ოპერაციის შემდეგ დარჩენილი ყველა სიმბოლო ეკუთვნის მეორე რიცხვს.

```cpp
for(int i=idx+1; i<s.size(); i++){
    b += s[i];
}
```

მაგალითად:

```text
321*42
```

მივიღებთ:

```text
b = "42"
```

---

## 5. სტრიქონების რიცხვებად გადაყვანა

ვქმნით ორ ცვლადს:

```cpp
long long x = 0, y = 0;
```

სადაც:

```text
x = პირველი რიცხვი
y = მეორე რიცხვი
```

---

### პირველი რიცხვის გარდაქმნა

```cpp
int sz1 = a.size()-1;
```

შემდეგ:

```cpp
for(int i=0; i<a.size(); i++){
    x += 1LL * (a[i]-48) * v[sz1];
    sz1--;
}
```

---

მაგალითად:

```text
a = "321"
```

ვიღებთ:

```text
3 × 100
2 × 10
1 × 1
```

ანუ:

```text
321
```

---

### რატომ `(a[i]-48)`?

ASCII ცხრილში:

```text
'0' = 48
'1' = 49
'2' = 50
...
'9' = 57
```

ამიტომ:

```cpp
a[i] - 48
```

გვაძლევს შესაბამის ციფრს.

მაგალითად:

```text
'7' - 48 = 7
```

---

### მეორე რიცხვის გარდაქმნა

ზუსტად იგივე პრინციპით:

```cpp
int sz2 = b.size()-1;

for(int i=0; i<b.size(); i++){
    y += 1LL * (b[i]-48) * v[sz2];
    sz2--;
}
```

მაგალითად:

```text
b = "42"
```

გახდება:

```text
4 × 10 + 2 × 1 = 42
```

---

## 6. ოპერაციის შესრულება

ახლა უკვე გვაქვს:

```text
x = პირველი რიცხვი
y = მეორე რიცხვი
c = ოპერაცია
```

რჩება შესაბამისი მოქმედების შესრულება.

---

### შეკრება

```cpp
if(c == '+'){
    cout << x + y;
}
```

მაგალითი:

```text
12+23
```

პასუხი:

```text
35
```

---

### გამოკლება

```cpp
else if(c == '-'){
    cout << x - y;
}
```

მაგალითი:

```text
25-17
```

პასუხი:

```text
8
```

---

### გამრავლება

```cpp
else if(c == '*'){
    cout << x * y;
}
```

მაგალითი:

```text
321*42
```

პასუხი:

```text
13482
```

## სირთულე

### დროითი სირთულე

```text
O(n)
```

სადაც `n` არის სტრიქონის სიგრძე.

სტრიქონს მხოლოდ რამდენჯერმე გადავუვლით.

---

### მეხსიერების სირთულე

```text
O(n)
```

რადგან ვინახავთ ორ ქვე-სტრიქონს (`a` და `b`).

---

## შეჯამება

ამოხსნის იდეა ძალიან მარტივია:

1. ვპოულობთ ოპერაციის სიმბოლოს;
2. სტრიქონს ვყოფთ ორ ნაწილად;
3. ორივე ნაწილს ხელით ვაქცევთ რიცხვად;
4. ვასრულებთ შესაბამის არითმეტიკულ ოპერაციას;
5. ვბეჭდავთ პასუხს.

ამგვარად პროგრამა სწორად მუშაობს როგორც შეკრებისთვის, ასევე გამოკლებისა და გამრავლებისთვის.
