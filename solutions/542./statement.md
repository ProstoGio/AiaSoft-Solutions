# Problem Statement — 542. ჯადოქარი ბიტუნა

> Copied from [AiaSoft.ge #542](https://aiasoft.ge/problem/542)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ბატუნა და ბიტუნა 1-დან $N$-ის ჩათვლით რიცხვებით თამაშობენ. ბიტუნამ რაღაც ჯადოქრობას მიმართა და ამ რიცხვების XOR ძალიან სწრაფად, ერთ წამზე ნაკლებ დროში გამოთვალა. შემდეგ იგივე დავალების შესრულებაში ბატუნა გამოიწვია. ბატუნა ბიტურ ოპერაციებში ვერ ერკვევა და თქვენთან მოვიდა დახმარების სათხოვნელად.

დაუწერეთ ბატუნას პროგრამა, რომელიც გამოთვლის 1-დან $N$-ის ჩათვლით რიცხვების XOR-ს.

## Input

მოცემულია ერთი მთელი რიცხვი: $N$, $(0 <= N <= 10^35)$

## Output

გამობეჭდეთ ერთი რიცხვი: 1-დან $N$-ის ჩათვლით რიცხვების XOR.
