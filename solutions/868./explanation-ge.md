# განმარტება — 868. დაბეჭდე ლუწები

## მიდგომა

ეს დავალება საკმაოდ მარტივია, არაფერი რთული აქ არ ხდება. უბრალოდ ვკითხულობთ N-ს, მერე მასივს ვავსებთ და თითოეულ ელემენტზე ვამოწმებთ, იყოფა თუ არა 2-ზე უნაშთოდ. თუ პირობა სრულდება, მაშინვე ვბეჭდავთ რიცხვს და გვერდით ვუწერთ ჰარს. სულ ესაა, მეტი არაფერი გინდა.

## სირთულე

|-|--|

| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- ჩვეულებრივი `a[i] % 2 == 0` შემოწმება საკმარისია ლუწების გამოსაყოფად.
- მასივის შენახვაც კი არ არის საჭირო, შეგიძლია პირდაპირ `cin`-ის დროსვე შეამოწმო და დაბეჭდო, თუ მეხსიერების ეკონომია გინდა.

## ხშირი შეცდომები

- დაგავიწყდეს რიცხვებს შორის ჰარის დატოვება, რამაც შეიძლება შეცდომა მოგცეს.
