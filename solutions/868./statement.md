# Problem Statement — 868. დაბეჭდე ლუწები

> Copied from [AiaSoft.ge #868](https://aiasoft.ge/problem/868)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია N ელემენტიანი მასივი. თქვენი მიზანია დაბეჭდოთ მოცემულ მასივში ლუწი რიცხვები იგივე თანმიმდევრობით, როგორი თანმიმდევრობითაც ისინი მოცემულია საწყის მასივში.

## Input

პირველ სტრიქონზე შემოდის ერთი მთელი დადებითი რიცხვი N, შემდეგ სტრიქონზე შემოდის N ცალი მთელი დადებითი რიცხვი.
$1<=N<=100$
$1<=a_i<=100$

## Output

ერთ სტრიქონზე გამოიტანეთ ყველა ლუწი რიცხვი, თითოეული მათგანი დაშორებული უნდა იყოს ჰარებით.
