# Problem Statement — 803. კრუიზი ავტობუსებით

> Copied from [AiaSoft.ge #803](https://aiasoft.ge/problem/803)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ბოლივიაში ყველა ქალაქი ოკეანის სხვადასხვა კუნძულზე მდებარეობს. ამიტომ ერთი ქალაქიდან მეორეში გადასასვლელად ხშირად გემებს იყენებენ ხოლმე.

ბოლივიის $A$ ქალაქის ერთ-ერთი სკოლიდან აპირებენ ბავშვები ექსკურსიაზე წაიყვანონ. პორტი სკოლიდან საკმაოდ შორსაა, ამიტომ ტრანსპორტირება შემდეგნაირად გადაწყდა, ბავშვები ჯერ ავტობუსებში უნდა გადანაწილდნენ, შემდეგ ამ ავტობუსებითვე ავლენ გემებზე, გაცურავენ $B$ ქალაქამდე და შემდეგ ისევ ავტობუსებით გააგრძელებენ გზას.

`~https://storage.googleapis.com/problems_api_photos/images/nZsvimm0LSahm14O527w.jpg`

ცნობილია, რომ სკოლაში სულ $n$ მოსწავლეა. ავტობუსში $m$ რაოდენობის ბავშვი ეტევა, ხოლო გემს შეუძლია ერთდროულად $k$ რაოდენობის ავტობუსის გადაყვანა.

დაადგინეთ, მინიმუმ რამდენი ავტობუსის და გემის დაქირავება დასჭირდება სკოლას ექსკურსიისთვის.

## Input

ერთ ხაზზე მოცემულია სამი მთელი რიცხვი: $n, m$ და $k$ $(1 <= n,m,k <= 10^3)$

## Output

ერთ ხაზზე გამობეჭდეთ ორი მთელი რიცხვი: შესაბამისად ავტობუსების და გემების მინიმალური რაოდენობა.
