# Problem Statement — 253. დიპლომები

> Copied from [AiaSoft.ge #253](https://aiasoft.ge/problem/253)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 128 MB

## Task

ირაკლი სკოლაში სიარულის პერიოდში ხშირად იღებდა მონაწილებას მათემატიკის, ინფორმატიკისა და ფიზიკის ოლიმპიადებში. გამომდინარე იქიდან, რომ ირაკლი ძალიან ბეჯითი მოსწავლე იყო, მან მრავალი დიპლომი აიღო ამ ოლიმპიადებზე. სკოლის დამთავრების შემდეგ მას სულ $N$ დიპლომი დაუგროვდა და თან შენიშნა, რომ ყველა მათგანი ერთი სიდიდის იყო: სიგრძით - $h$ და სიგანით - $w$.

ამ ეტაპზე ირაკლი საქართველოს ერთ-ერთ წამყვან უნივერსიტეტში ჩაირიცხა და საცხოვრებლად სასწავლებელთან ახლოს იქირავა სახლი. შემდეგ გადაწყვიტა თავისი ოთახის ერთ-ერთი კედელი თავისი სკოლის დროინდელი დიპლომებით მოერთო, თუმცა  რაკი დიპლომების პირდაპირ კედელზე მიკვრა ძალიან მოუხერხებელი იყო იფიქრა, რომ სპეციალურ ჩარჩოს იყიდდა დიპლომების ჩასაკრავად და შემდეგ ამ ჩარჩოს დაკიდებდა კედელზე. ირაკლის სურს, რომ ჩარჩო კვადრატული ფორმის იყოს და რაც შეიძლება პატარა ადგილი დაიკავოს კედელზე. 

ასევე:
	● ჩარჩოში დიპლომების მობრუნებით ჩაწებება არ შეიძლება (ყველა უნდა განლაგდეს ისე, რომ $w$ x $h$ ზომის ადგილს იკავებდეს)
	● დიპლომები ერთმანეთს არ უნდა კვეთდეს (შეხება შესაძლებელია)

დაწერეთ პროგრამა, რომელიც გამოთვლის კვადრატული ჩარჩოს გვერდის მინიმალურ სიგრძეს, სადაც ყველა დიპლომი დაეტევა.

## Input

ერთ ხაზზე შემოდის სამი მთელი რიცხვი: $N$, $w$ და $h$ - შესაბამისად დიპლომების რაოდენობა და მისი ზომები.

$1 <= N <= 10^6$
$1 <= w, h <= 10^9$

## Output

გამოიტანეთ ერთი მთელი რიცხვი, კვადრატის ჩარჩოს გვერდის მინიმალური სიგრძე, რომელშიც ჩაეტევა ყველა დიპლომი.
