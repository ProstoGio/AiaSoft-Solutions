# განმარტება — 088. გადაჭერი ლენტი

## მიდგომა

ამ ამოცანაში მთელი აზრი იმაშია, რომ ლენტი ორ თანაბარ მთელ რიცხვად დავყოთ. თუ რიცხვი კენტია, ნახევარი აუცილებლად წილადი გამოვა, რაც პირობას არ აკმაყოფილებს. ამიტომ, უბრალოდ ვამოწმებთ, იყოფა თუ არა N უნაშთოდ 2-ზე.

## მთავარი დაკვირვებები

- თუ N ლუწია, მაშინ N/2 არის მთელი რიცხვი და ორივე ნაწილი თანაბარი გამოდის.
- კენტი რიცხვის შემთხვევაში ყოველთვის გვექნება ნაშთი, ანუ 'yes'-ის პასუხი ფიზიკურად შეუძლებელია.

## ხშირი შეცდომები

- არასაჭირო ციკლების ან რთული პირობების წერა, როცა მარტივი მოდულო ოპერაცია საკმარისია.
