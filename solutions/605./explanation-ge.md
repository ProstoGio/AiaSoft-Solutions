# განმარტება — 605. გადაიყვანე ორობითში

## მიდგომა

ათობითიდან ორობითში გადასაყვანად ყველაზე კლასიკური გზა 2-ზე განაყოფების აღებაა. სანამ რიცხვი ნულზე მეტია, ვუყურებთ ნაშთს, რითაც ვიგებთ ბოლო ბიტს და შემდეგ რიცხვს ვყოფთ ორზე. შედეგად ბიტებს ვიღებთ უკუღმა, ამიტომ ბოლოს უბრალოდ ვარევერსებთ სტრიქონს. მთავარია ნულის დამუშავება არ დაგავიწყდეთ, თორემ ციკლი საერთოდ არ გაეშვება.

## სირთულე

| | |
|-|--|
| დრო | O(log N) |
| მეხსიერება | O(log N) |

## მთავარი დაკვირვებები

- ნაშთი ყოველთვის გვაძლევს ორობითი ჩანაწერის ბოლო ციფრს, განაყოფი კი მომდევნო ეტაპზე გადაგვყავს.
- char-ად გადაქცევა '0'-ის მიმატებით ძალიან კომფორტულია, რადგან ASCII-ში ციფრები მიყოლებითაა.

## ხშირი შეცდომები

- N=0 შემთხვევაში ცარიელი სტრიქონის დაბეჭდვა, რადგან while ციკლი ამ დროს არ შესრულდება.
