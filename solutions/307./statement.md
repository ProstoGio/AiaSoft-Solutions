# Problem Statement — 307. ჯამი მატრიცაში

> Copied from [AiaSoft.ge #307](https://aiasoft.ge/problem/307)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია NxM ზომის მატრიცა. ყოველი მოთხოვნილი (x, y) წყვილისთვის გამოთვალეთ იმ მართკუთხედში არსებული რიცხვების ჯამი, რომლის ზედა მარცხენა წვერო მდებარეობს (1, 1) კოორდინატზე, ხოლო ქვედა მარჯვენა (x, y)-ზე.

## Input

● პირველ ხაზზე მოცემულია ორი რიცხვი: N და M - მატრიცის ზომა
● შემდეგ მოცემულია თავად NxM ზომის მატრიცის წევრები
● მომდევნო ხაზზე მოეცმული იქნება Q - მოთოვნების რაოდენობა
● შემდეგი Q ხაზი შეიცავს მოთხოვნის შესახებ ინფორმაციას. კერძოდ i-ურ ხაზზე მოცემული იქნება xi და yi - მართკუთხედის ქვედა მარჯვენა კუთხის კოორდინატები მატრიცაში. 

1 <= N, M <= 10³
0 <= A(i, j) <= 10³
1 <= Q <= 10⁵
1 <= xi <= N
1 <= yi <= M

## Output

ყოველი მოთხოვნისთვის ცალ-ცალკე ხაზზე გამობეჭდეთ ჯამი.
