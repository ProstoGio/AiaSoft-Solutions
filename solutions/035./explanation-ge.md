# განმარტება — 035. ორი რიცხვის უდიდესი საერთო გამყოფი

## მიდგომა

ევკლიდეს ალგორითმი არის ამ ამოცანის "ოქროს სტანდარტი". ნაცვლად იმისა, რომ გამყოფები ვეძებოთ, რაც 10^18-ის შემთხვევაში უბრალოდ გაჭედავს კომპიუტერს, ვიყენებთ რეკურსიას: gcd(a, b) ტოლია gcd(b, a % b)-ის. რიცხვები ძალიან სწრაფად მცირდება და ბაზისური შემთხვევა მაშინ გვაქვს, როცა მეორე რიცხვი ნული ხდება.

## სირთულე

| | |
|-|--|
| დრო | O(log(min(a, b))) |
| მეხსიერება | O(log(min(a, b))) |

## მთავარი დაკვირვებები

- მთავარი ეშმაკობა ისაა, რომ a % b ოპერაცია ყოველი ნაბიჯისას რიცხვს საგრძნობლად ამცირებს.
- რეკურსია აქ იდეალურია, რადგან კოდი ძალიან სუფთად გამოიყურება და მეხსიერებასაც თითქმის არ ჭამს.

## ხშირი შეცდომები

- ინტების (int) გამოყენება long long-ის ნაცვლად, რაც ასეთი დიდი რიცხვების დროს overflow-ს გამოიწვევს.
