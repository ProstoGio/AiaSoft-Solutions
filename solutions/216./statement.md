# Problem Statement — 216. მარცხენა მთავარი დიაგონალი

> Copied from [AiaSoft.ge #216](https://aiasoft.ge/problem/216)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია მატრიცა, იპოვეთ მის მარცხენა მთავარ დიაგონალზე მყოფი ელემენტების ჯამი.

## Input

პირველ სტრიქონზე შემოდის $n$, მასივის სტრიქონების რაოდენობა. $1<=n<=20$.
მომდევნო $n$ რაოდენობის სტრიქონზე მოცემულია კვადრატული მატრიცა, ელემენტებს შორის თითო ჰარია. თითოეული ელემენტის სიდიდე მოდულით არ აღემატება 100-ს.

## Output

გამოიტანეთ ერთი რიცხვი - მატრიცის მთავარი დიაგონალის ელემენტების ჯამი.
გაქვთ გარანტია, რომ შედეგი არ აღემატება $2^9$-ს.
