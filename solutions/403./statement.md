# Problem Statement — 403. ბატები და ბატკნები

> Copied from [AiaSoft.ge #403](https://aiasoft.ge/problem/403)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ეზოში ბატები და ბატკნები თამაშობენ. გუგლიკომ დაადგინა, რომ სულ თავების რაოდენობა იყო H, ხოლო ფეხების რაოდენობა F. რამდენი ბატი და რამდენი ბატკანია გუგლიკოს ეზოში?

## Input

შემოდის ორი რიცხვი H და F, შესაბამისად, თავებისა და ფეხების რაოდენობა.
1<=H,F<=100.

## Output

გამოიტანეთ ერთმანეთისაგან ჰარით დაშორებული ორი რიცხვი, პირველი უნდა აღნიშნავდეს ბატების რაოდენობას, ხოლო მეორე ბატკნების რაოდენობას.
თუ აღმოჩნდა, რომ პასუხი არ არსებობს, გამოიტანეთ რიცხვი  -1.
