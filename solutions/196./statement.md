# Problem Statement — 196. სიმბოლოთა ამომცნობი

> Copied from [AiaSoft.ge #196](https://aiasoft.ge/problem/196)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

მოცემულია ერთი სტრიქონი, რომელიც შეიცავს ინგლისური ანბანის დიდ/პატარა სიმბოლოებს და ასევე ციფრებს.

თქვენი დავალებაა დაადგინოთ რამდენი ციფრისგან, რამდენი დიდი და რამდენი პატარა სიმბოლოსგან შედგება მოცემული სტრიქონი.

## Input

ერთ ხაზზე შემოდის სტრიქონი: s ($1 <= |s| <= 20$)

## Output

ერთ ხაზზე გამოიტანეთ სამი მთელი რიცხვი, რომელთაგან პირველი იქნება სტრიქონში ციფრების რაოდენობა, მეორე - სტრიქონში დიდი სიმბოლოების რაოდენობა, ხოლო მესამე - პატარა სიმბოლოების რაოდენობა.
