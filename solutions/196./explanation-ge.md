# განმარტება — 196. სიმბოლოთა ამომცნობი

## მიდგომა

ამ ამოცანაში ველოსიპედის გამოგონება არ გვჭირდება. უბრალოდ ვიღებთ სტრიქონს და თითოეულ სიმბოლოს ვამოწმებთ, რა ტიპისაა. სამი ცვლადი დავნიშნე: `digit`, `big`, და `small`. ციკლში ჩავდივარ სტრიქონის ბოლომდე და `isdigit`, `isupper` და `islower` ფუნქციების გამოყენებით ზუსტად ვარკვევ, რომელი კატეგორია უნდა გავზარდო. სულ ეს არის.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- C++-ის <cctype> ბიბლიოთეკის ფუნქციები საუკეთესო ვარიანტია სიმბოლოების ტიპის დასადგენად.
- O(n) სირთულე სრულიად საკმარისია ასეთი პატარა შეზღუდვისთვის.

## ხშირი შეცდომები

- საწყისი მნიშვნელობების (0-ების) მინიჭების დავიწყება.
