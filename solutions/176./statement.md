# Problem Statement — 176. კიბე

> Copied from [AiaSoft.ge #176](https://aiasoft.ge/problem/176)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

პატარა თინიკო ძალიან ხალისობს ხოლმე კიბეზე თამაშისას. დღეს მან გადაწყვიტა შემდეგნაირად ითამაშოს: უნდა დადგეს კიბის დასაწყისში (ქვემოთ) და უნდა მიაღწიოს კიბის ბოლოს (ყველაზე ზედა საფეხურს), ოღონდ ისე, რომ ერთ სვლაზე ერთი ან ორი საფეხურით მაღლა გადაინაცვლოს ხოლმე.

გამოთვალეთ, სვლების რამდენი განსხვავებული ვარიანტით შეიძლება მოხვდეს თინიკო კიბის ყველაზე ზედა საფეხურზე.

## Input

შემოდის ერთი მთელი რიცხვი $N$ - კიბის საფეხურების რაოდენობა. ($1 <= N <= 40$)

## Output

გამოიტანეთ ერთი მთელი რიცხვი: განსხვავებული ვარიანტების რაოდენობა.
