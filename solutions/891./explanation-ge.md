# განმარტება — 891. დაითვალე ლიდერები

## მიდგომა

მარცხნიდან მარჯვნივ ჩარბენა არაფერში გჭირდება, ეგ მხოლოდ წვალებაა. ბოლოდან წამოსვლა ბევრად მარტივია, რადგან ბოლო ელემენტი ხომ ავტომატურად ლიდერია. უბრალოდ გაიყოლე 'მაქსიმუმის' ცვლადი და უკნიდან გადმოდი; როგორც კი შეხვდები რიცხვს, რომელიც შენს მაქსიმუმზე დიდია ან ტოლი, ეგრევე ლიდერად ჩათვალე და მაქსიმუმი განაახლე.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- ლიდერების პირობა დამოკიდებულია მარჯვენა მხარეზე, ამიტომ უკნიდან სკანირება ყველაზე ოპტიმალურია.
- საკმარისია მხოლოდ მიმდინარე მაქსიმუმის ცოდნა, რომ თითოეულ ელემენტზე მარტივი შედარებით გავიგოთ ლიდერია თუ არა.

## ხშირი შეცდომები

- O(n²) მიდგომა, სადაც თითოეული ელემენტისთვის ცალკე ამოწმებენ მის მარჯვნივ არსებულ ყველა რიცხვს, რაც n=10⁶-ისთვის დროში გაგივარდება.
