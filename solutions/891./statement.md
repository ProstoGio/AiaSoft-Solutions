# Problem Statement — 891. დაითვალე ლიდერები

> Copied from [AiaSoft.ge #891](https://aiasoft.ge/problem/891)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია მთელ რიცხვთა ერთგანზომილებიანი მასივი. მასივის რომელიმე $x$ ელემენტს ეწოდება ლიდერი, თუ მის მარჯვნივ მდებარე (მეტი ინდექსის მქონე) ყველა რიცხვი ნაკლები ან ტოლია $x$ - ზე. მასივის ბოლო ელემენტი ლიდერად ითვლება. 

დაწერეთ პროგრამა, რომელიც მოცემულ მასივში დათვლის ლიდერების რაოდენობას.

## Input

სტანდარტული შეტანის პირველ სტრიქონში მოცემულია ერთი  მთელი დადებითი n რიცხვი $0<n≤10^6$ - მოცემულ მასივში ელემენტთა რაოდენობა. მეორე სტრიქონში ჩაწერილია თითო ჰარით გამოყოფილი მასივის ელემენტები, რომლებიც მთელი რიცხვებია დიაპაზონიდან $[−10^9,10^9]$

## Output

სტანდარტული გამოტანის ერთადერთ სტრიქონში უნდა გამოიტანოთ მოცემულ მასივში ლიდერების რაოდენობა.
