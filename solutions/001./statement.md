# Problem Statement — 001. ორი რიცხვის ჯამი

> Copied from [AiaSoft.ge #1](https://aiasoft.ge/problem/1)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია ორი მთელი რიცხვი $a$ და $b$. იპოვეთ მათი ჯამი.

## Input

ერთ ხაზზე შემოდის ჰარით გამოყოფილი ორი მთელი რიცხვი $a$ და $b$ სადაც $-10^9 <= a,b <= 10^9$

## Output

გამოიტანეთ $a+b$
