# განმარტება — 001. ორი რიცხვის ჯამი

## მიდგომა

ეს არის კლასიკური 'Warm-up' ამოცანა. არაფერი რთული, უბრალოდ ვიღებთ ორ რიცხვს და ვაჯამებთ. 10⁹-მდე რიცხვები ჩვეულებრივ int-ში მშვენივრად ეტევა, ასე რომ ზედმეტი თავის ტკივილი არ გვჭირდება.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- არითმეტიკული ჯამი კომპიუტერისთვის ელემენტარული ოპერაციაა, ამიტომ O(1) დროში ვეტევით.
- ყოველთვის თვალი გადაავლე კონსტრეინებს, რომ შემთხვევით integer overflow არ გამოგეპაროს.

## ხშირი შეცდომები

- ზოგჯერ ავიწყდებათ, რომ მიუხედავად იმისა, რომ a და b int-ში ეტევა, მათი ჯამი შეიძლება 2*10⁹-ს გადასცდეს, თუმცა აქ ეგ საფრთხე არ გვაქვს.
