# Problem Statement — 783. პირველობა რეიტინგში

> Copied from [AiaSoft.ge #783](https://aiasoft.ge/problem/783)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

დიმა და ალექსი ერთმანეთს პროგრამირების ამოცანების ამოხსნაში ეჯიბრებიან. ცნობილია, რომ ამ ეტაპზე დიმას N, ხოლო ალექსს - M რაოდენობის ამოცანა აქვს ამოხსნილი. დაადგინეთ ვის უფრო მეტი ამოცანა ექნება ამოხსნილი D დღის შემდეგ, თუ დიმა დღეში A და ალექსი B რაოდენობის ამოცანას ხსნის.

## Input

ერთ ხაზზე მოცემულია ხუთი მთელი რიცხვი: N, M, D, A და B
* გარანტირებულია, რომ ყველა რიცხვი მოთავსებულია [1, 10³] შუალედში.

## Output

გამობეჭდეთ ერთი სიტყვა: `bAlex` - თუ გამარჯვებული იქნება ალექსი, `bDima` - თუ გაიმარჯვებს დიმა, ხოლო წინააღმდეგ შემთხვევაში - `bDraw`.
