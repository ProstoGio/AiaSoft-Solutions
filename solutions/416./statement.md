# Problem Statement — 416. გივიკო, ნინიკო და კანფეტები

> Copied from [AiaSoft.ge #416](https://aiasoft.ge/problem/416)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

გივიკოს და ნინიკოს კანფეტები აქვთ. ჯერ გივიკომ მისცა ნინიკოს თავისი კანფეტების ნახევარი. ამის შემდეგ ნინიკომ მისცა გივიკოს იმ კანფეტების ნახევარი, რაც იმ მომენტში ჰქონდა. ამის შემდეგ გივიკოს აღმოაჩნდა a რაოდენობის კანფეტი, ხოლო ნინიკოს - b რაოდენობისა.
დაადგინეთ ბავშვების კანფეტების თავდაპირველი რაოდენობა.

## Input

შემოდის ერთმანეთისაგან ჰარით გამოყოფილი ორი რიცხვი a და b - გივიკოს და ნინიკოს კანფეტების საბოლოო რაოდენობა.
1<=a,b<=100.

## Output

გამოიტანეთ ერთმანეთისაგან ჰარით გამოყოფილი ორი რიცხვი - გივიკოს და ნინიკოს კანფეტების თავდაპირველი რაოდენობა, ხოლო თუ ამოცანის საწყისი მონაცემები არასწორია, მაშინ გამოიტანეთ -1 (მინუს ერთი).
