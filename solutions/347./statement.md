# Problem Statement — 347. ბაბი კინოთეატრში

> Copied from [AiaSoft.ge #347](https://aiasoft.ge/problem/347)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

კინოთეატრში სკამების N რაოდენობის მწკრივია. თითოეულ მწკრივში M სკამია.
ცარიელ კინოთეატრში ბაბიმ მეგობარი ბაჭიები შეიყვანა, ბაჭიების რაოდენობა ბაბის ჩათვლით არის B.
ბაჭიებმა კინოთეატრის შევსება დაიწყეს პირველი სკამიდან. შეავსეს პირველი რიგი და მერე გადავიდნენ მომდევნო რიგზე სკამების გამოუტოვებლად, სანამ ყველა ბაჭია არ მოთავსდა. 
ბაბი მეგობრების სიაში ყველაზე ბოლოა.
დაადგინეთ, რომელ რიგში და რიგის რომელ ნომერ სკამზე მოუწევს დაჯდომა ბაბის, თუ ცნობილია, რომ ბაჭიები კინოთეატრში აუცილებლად დაეტევიან და რიგების და სკამების დანომრვა იწყება ერთით.

## Input

შემოდის სამი ნატურალური რიცხვი N, M, B

1<=N,M<=1000
1<=B<=N*M

## Output

გამოიტანეთ ორი რიცხვი,  ბაბის რიგის ნომერი და მისი სკამის ნომერი რიგში.
რიცხვებს შორის მოათავსეთ ერთი ჰარი.
