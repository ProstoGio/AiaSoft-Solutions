# განმარტება — 365. გაასწორეთ წინადადება

## მიდგომა

სამუშაო მარტივია: სტრიქონი უნდა გავწმინდოთ ზედმეტი ჰარებისგან. `getline`-ით ვკითხულობ მთლიან წინადადებას, თორემ `cin` პირველივე ჰარესთან გაჩერდებოდა. მთავარი იდეა ისაა, რომ ყოველ ჯერზე ვამოწმებ, წინა სიმბოლო ხომ არ იყო ჰარე. თუ ახლა ჰარეს წავაწყდი და მანამდეც ჰარე იყო, ესე იგი ზედმეტია და უბრალოდ გამოვტოვებ.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- წინა სიმბოლოზე (s[i-1]) დაკვირვება იდეალურია იმისთვის, რომ დამატებითი flag-ები არ ვწერო.
- მნიშვნელოვანია, რომ ლოგიკა პირველი სიმბოლოს შემთხვევაშიც სწორად მუშაობდეს.

## ხშირი შეცდომები

- ხშირად ავიწყდებათ `getline`-ის გამოყენება და მერე უკვირთ, რატომ წყდება წინადადება პირველივე სიტყვაზე.
