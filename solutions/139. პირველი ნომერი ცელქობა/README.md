# ამოხსნის ახსნა

თავდაპირველად მოსწავლეები იდგნენ სიმაღლის ზრდადობის მიხედვით.

შემდეგ მათ გააკეთეს პირველი ნომერი ცელქობა:

- ყოველი ლუწი ინდექსის მქონე მოსწავლე ადგილს უცვლის მის მომდევნო მოსწავლეს;
- თუ ლუწ ინდექსზე მდგომ მოსწავლეს მომდევნო აღარ ჰყავს, მაშინ იგი ადგილზე რჩება.

მოცემულია უკვე არეული მასივი და საჭიროა მისი საწყისი მდგომარეობის აღდგენა.

---

# მთავარი იდეა

ამ ამოცანის საინტერესო ნაწილი ის არის, რომ იგივე მოქმედება ორჯერ შესრულების შემთხვევაში მასივი ისევ საწყის მდგომარეობას უბრუნდება.

მაგალითად:

```text
1 2 3 4
```

პირველი გაცვლის შემდეგ:

```text
2 1 4 3
```

იგივე გაცვლა კიდევ ერთხელ:

```text
1 2 3 4
```

ანუ თავდაპირველი მასივი აღდგა.

---

ამიტომ საკმარისია იგივე გაცვლები კიდევ ერთხელ შევასრულოთ.

# კოდის განხილვა

## მონაცემების წაკითხვა

```cpp
int n;
cin >> n;

int a[n];

for(int i=0; i<n; i++)
    cin >> a[i];
```

if(i % 2 == 0 && i + 1 < n)

# რატომ მუშაობს?

ყოველი გაცვლა მხოლოდ ორ მეზობელ ელემენტს ეხება.

თუ იმავე ორ ელემენტს კიდევ ერთხელ გავუცვლით ადგილებს:

```text
A B
```

↓

```text
B A
```

↓

```text
A B
```

ისინი ისევ საწყის პოზიციებზე დაბრუნდებიან.

რადგან ყველა წყვილზე ზუსტად იგივე მოქმედებას ვასრულებთ, საბოლოოდ მთელი მასივი პირვანდელ სახეს იღებს.

# შეჯამება

ცელქობის დროს ყოველი ლუწი ინდექსის მქონე ელემენტი ადგილს უცვლის შემდეგ ელემენტს.

იგივე გაცვლის კიდევ ერთხელ შესრულება თითოეულ წყვილს საწყის მდგომარეობაში აბრუნებს.

ამიტომ პროგრამა უბრალოდ კიდევ ერთხელ ასრულებს იმავე გაცვლებს და აღადგენს მოსწავლეების თავდაპირველ განლაგებას.
