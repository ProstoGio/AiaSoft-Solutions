# Problem Statement — 502. პიცა

> Copied from [AiaSoft.ge #502](https://aiasoft.ge/problem/502)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

თაკოს ოჯახში დღეს პიცის დღეა და გადაწყვიტა მეგობრები დაეპატიჟებინა. მეგობრებიდან ზოგიერთმა შესაძლოა მოსვლა ვერ მოახერხოს, თუმცა თაკომ ზუსტად იცის, რომ $N$ ან $M$ რაოდენობის სტუმარი ეყოლება.

მან დიდი პიცა გაკეთა და სანამ სტუმრები მოვლენ სურს ბარემ დაჭრაც მოასწროს. მინიმუმ რამდენ ნაჭრად უნდა დაჭრას თაკომ პიცა, რომ გარანტირებულად `bყველა სტუმარს (და თვითონ თაკოსაც)` თანაბარი რაოდენობის ნაჭერი შეხვდეს?

## Input

ერთ ხაზზე მოცემულია ორი მთელი რიცხვი: $N$ და $M$ $(1 <= N, M <= 10^5)$

## Output

გამობეჭდეთ ერთი მთელი რიცხვი: ნაჭრების მინიმალური რაოდენობა.
