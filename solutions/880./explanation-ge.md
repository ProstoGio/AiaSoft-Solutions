# განმარტება — 880. გრძელი მოედანი

## მიდგომა

ეს ამოცანა მარტივი გეომეტრიაა - უბრალოდ მართკუთხედისა და წრის ფართობები უნდა შევკრიბოთ. ორი ნახევარწრე გვერდებზე რომ შევაერთოთ, ერთი მთლიანი წრე გამოდის, რისი რადიუსიც r-ია. მართკუთხედის სიგრძე მოცემულია, ხოლო სიგანე წრის დიამეტრია, ანუ 2*r. ყველაფერი ერთ ფორმულაში ჩავწერე და ოთხი ნიშანი მძიმის შემდეგ fixed-ით დავაფიქსირე.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- ორივე ნახევარწრის რადიუსი ერთი და იგივეა, ამიტომ მათი ფართობი წრის ფორმულით (πr²) მარტივად გამოითვლება.
- მართკუთხედის სიგანის სწორად აღქმაა მთავარი - ის წრის დიამეტრის ტოლია.

## ხშირი შეცდომები

- სიზუსტის დარღვევა, როცა setprecision-ს ავიწყდებათ.
- რადიუსისა და დიამეტრის აღრევა მართკუთხედის ფართობის დათვლისას.
