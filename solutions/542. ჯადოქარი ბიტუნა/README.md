# ამოხსნის ახსნა

ამოცანაში საჭიროა ვიპოვოთ `1`-დან `N`-ის ჩათვლით ყველა რიცხვის XOR.

თუ რამდენიმე პატარა მაგალითს დავწერთ, საინტერესო კანონზომიერებას დავინახავთ:

| N | XOR(1...N) |
|---|---|
| 0 | 0 |
| 1 | 1 |
| 2 | 3 |
| 3 | 0 |
| 4 | 4 |
| 5 | 1 |
| 6 | 7 |
| 7 | 0 |
| 8 | 8 |

შედეგი ყოველ 4 რიცხვში მეორდება:

- თუ `N % 4 == 0` → პასუხია `N`
- თუ `N % 4 == 1` → პასუხია `1`
- თუ `N % 4 == 2` → პასუხია `N + 1`
- თუ `N % 4 == 3` → პასუხია `0`

პრობლემა ის არის, რომ `N` შეიძლება იყოს ძალიან დიდი (`10^35`-მდე), ამიტომ მისი შენახვა სტანდარტულ მთელ ტიპებში შეუძლებელია. სწორედ ამიტომ რიცხვს სტრიქონის (`string`) სახით ვინახავთ.

---

## 1. რიცხვის წაკითხვა

```cpp
string n;
cin >> n;
```

რადგან `N` ძალიან დიდია, `long long`-შიც კი ვერ დაეტევა. ამიტომ ვიყენებთ `string`-ს.

---

## 2. `N + 1`-ის წინასწარ გამოთვლა

როდესაც `N % 4 == 2`, პასუხი არის `N + 1`, ამიტომ თავიდანვე ვამზადებთ მის მნიშვნელობას.

```cpp
string op1 = n;
int sz = n.size() - 1;

if(op1[sz] == '9'){
    op1[sz] = '0';
    op1[sz - 1] += 1;
}
else{
    op1[sz] += 1;
}
```

ლოგიკა მარტივია:

- თუ ბოლო ციფრი 9 არ არის, უბრალოდ ვამატებთ 1-ს.
- თუ ბოლო ციფრი 9-ია, ის ხდება 0 და მარცხენა ციფრი იზრდება ერთით.

მაგალითად:

```text
12 -> 13
18 -> 19
29 -> 30
```

---

## 3. ბოლო ორი ციფრის აღება

`N % 4`-ის საპოვნელად მთელი რიცხვი არ გვჭირდება.

საკმარისია მხოლოდ ბოლო ორი ციფრი, რადგან 4-ზე გაყოფის ნაშთი მათზეა დამოკიდებული.

```cpp
int cnt = 0;
string num = "";

for(int i = n.size() - 1; i >= 0; i--){
    if(cnt < 2){
        num += n[i];
        cnt++;
    }
    else{
        break;
    }
}
```

ამ ციკლის შემდეგ `num` უკუღმა იქნება ჩაწერილი, ამიტომ ვაბრუნებთ.

```cpp
int l = 0, r = num.size() - 1;

while(l <= r){
    swap(num[l], num[r]);
    l++;
    r--;
}
```

მაგალითი:

```text
N = 12345

num = "54"
შემდეგ -> "45"
```

---

## 4. სტრიქონის გადაყვანა რიცხვად

```cpp
int m;

if(num.size() == 2){
    m = (num[0] - '0') * 10 + (num[1] - '0');
}
else{
    m = num[0] - '0';
}
```

აქ ბოლო ერთი ან ორი ციფრი ნამდვილ მთელ რიცხვად გადაიქცევა.

მაგალითად:

```text
"45" -> 45
"8" -> 8
```

---

## 5. XOR-ის კანონზომიერების გამოყენება

```cpp
if(m % 4 == 0)
    cout << n;
else if(m % 4 == 1)
    cout << 1;
else if(m % 4 == 2)
    cout << op1;
else
    cout << 0;
```

აქ ვიყენებთ ცნობილ ფაქტს:

```text
XOR(1...N) =

N      , თუ N % 4 == 0
1      , თუ N % 4 == 1
N + 1  , თუ N % 4 == 2
0      , თუ N % 4 == 3
```

ამიტომ საკმარისია `N % 4` გავიგოთ და შესაბამისი პასუხი გამოვიტანოთ.

---

## რატომ გვყოფნის ბოლო ორი ციფრი?

ნებისმიერი რიცხვისთვის:

```text
123456 % 4 = 56 % 4
98765432 % 4 = 32 % 4
```

რადგან 100 თვითონ იყოფა 4-ზე, უფრო მარცხნივ მდგომი ციფრები ნაშთზე გავლენას აღარ ახდენენ.

ამიტომ უზარმაზარი რიცხვის სრული გადაყვანა საჭირო არ არის.

---

## სირთულე

### დროითი სირთულე

```text
O(|N|)
```

### მეხსიერების სირთულე

```text
O(|N|)
```

სადაც `|N|` აღნიშნავს რიცხვის ციფრების რაოდენობას.

ამ მიდგომით მარტივად ვმუშაობთ ძალიან დიდ რიცხვებზეც, რადგან არასოდეს ვცდილობთ მათ ჩატევას სტანდარტულ მთელ ტიპებში.
