# განმარტება — 470. 0-ები და 1-ები

## მიდგომა

აქ არაფერია საფიქრალი, სტანდარტული ‘implementation’ დავალებაა. სტრიქონად ვიღებ რიცხვს და უბრალოდ ვუყვები ბოლოდან ბოლომდე. როგორც კი ‘1’-ს გადავაწყდები, counter-ს ვზრდი ერთით და ეგაა. არც რიცხვად გარდაქმნა გჭირდება და არც არაფერი, პირდაპირ სტრიქონზე მუშაობა ყველაზე სწრაფი და მარტივია.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- სტრიქონად წაკითხვა ჯობია, რადგან ასე თითოეულ სიმბოლოზე წვდომა თავისუფალია და არც integer-ის overflow-ის გვეშინია.
- თუ გინდა, შეგიძლია count ფუნქციაც გამოიყენო C++-ის, მაგრამ `for` ციკლით ხელით დათვლა უფრო გამჭვირვალეა დამწყებისთვის.

## ხშირი შეცდომები

- რიცხვითი ტიპების გამოყენება, რაც ჩაიჭრება, თუ სტრიქონი ძალიან გრძელი იქნება და 'long long'-შიც არ ჩაეტევა.
