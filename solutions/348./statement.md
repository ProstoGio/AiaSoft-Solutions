# Problem Statement — 348. ინდექსები

> Copied from [AiaSoft.ge #348](https://aiasoft.ge/problem/348)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია მთელი რიცხვებისგან შედგენილი $N$ წევრიანი მასივი. დაადგინეთ მასივის ყველა $x$-ის ტოლი ელემენტის ინდექსი და გამობეჭდეთ ზრდადობით, ხოლო თუ ასეთი არცერთი ელემენტი არ მოიძებნა, მაშინ გამობეჭდეთ -1.

* `iმასივის ელემენტების დანომრვა 0-დან იწყება`

## Input

პირველ ხაზზე მოცემულია ორი მთელი რიცხვი: $N$ და $x$ - მასივის ელემენტების რაოდენობა და საძიებელი ელემენტი.
მომდევნო ხაზზე მოცემულია $N$ რაოდენობის მთელი რიცხვი - $a_0, a_1, ..., a_(n-1)$

$1 <= N, x, a_i <= 100$

## Output

გამობეჭდეთ $x$-ის ტოლი ელემენტების ინდექსები თუ ასეთი ერთი ელემენტი მაინც მოიძებნება, წინააღმდეგ შემთხვევაში -1.
