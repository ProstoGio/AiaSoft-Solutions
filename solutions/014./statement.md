# Problem Statement — 014. ლუწი ხე

> Copied from [AiaSoft.ge #14](https://aiasoft.ge/problem/14)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია ბმული არაორიენტირებული გრაფი, რომელშიც არ არის ციკლი(ანუ მოცემულია ხე).
ცხადია ხეში ნებისმიერი წიბოს ამოშლით გრაფი დაიშლება ორ კომპონენტად. ორი წიბოს ამოშლით სამ კომპონენტად და ზოგადად k წიბოს ამოშლით ხე დაიშლება k+1 კომპონენტად(ასეთ გრაფს ტყე ეწოდება).
დავადგინოთ მაქსიმუმ რამდენი წიბოს ამოშლა შეიძლება ისე, რომ თითოეული მიღებული კომპონენტი შედგებოდეს ლუწი რაოდენობის წვეროსგან.
გარანტირებულია, რომ პასუხი არსებობს.

## Input

პირველ ხაზზე შემოდის ხის წვეროების რაოდენობა $n$ (გარანტირებულია რომ n არის ლუწი) მომდევნო $n-1$ ხაზზე შემოდის ორ-ორი რიცხვი $u, v$ რომელიც ნიშნავს რომ ხეში არის წიბო $(u, v)$
$2 \leq n \leq 100$

## Output

გამოიტანეთ ერთადერთი რიცხვი, მაქსიმალური წიბოების რაოდენობა რომელთა ამოშლის შედეგად მიღებული კომპონენტები შედგება ლუწი რაოდენობის წვეროებისაგან.
