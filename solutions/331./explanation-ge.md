# განმარტება — 331. გამყოფები

## მიდგომა

ამ ამოცანის პირველივე შეხედვით მარტივი გამოსავალი ისაა, რომ 1-დან N-მდე ყველა რიცხვი გადავამოწმოთ. უბრალოდ ვუყურებთ, იყოფა თუ არა N-ი უნაშთოდ ჩვენს მიმდინარე `i`-ზე. მე `while` ციკლი გამოვიყენე, რომელიც ყოველ ჯერზე ამოწმებს `n % i == 0` პირობას და თუ სიმართლეა, ეკრანზეც გამოაქვს რიცხვი. სიმართლე გითხრათ, `O(n)` სრულიად საკმარისია ამ ლიმიტებში, თუმცა უფრო დიდ რიცხვებზე უკვე `sqrt(n)`-მდე ჩამოსვლა მოგვიწევდა.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- თუ `n % i` ნულია, ე.ი. `i` არის გამყოფი და უნდა დაიბეჭდოს.
- ციკლის სტრუქტურა აქ ყველაზე მარტივია, პირდაპირ 1-დან ვიწყებთ და ვმოძრაობთ N-მდე.

## ხშირი შეცდომები

- ფორმატირების დარღვევა, ანუ რიცხვებს შორის ჰარის დავიწყება, რაც ავტომატურ სისტემებში შეცდომას იწვევს.
