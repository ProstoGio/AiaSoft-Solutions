# Problem Statement — 607. რიცხვების XOR - 1

> Copied from [AiaSoft.ge #607](https://aiasoft.ge/problem/607)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

გამოთვალეთ 1-დან N-ის ჩათვლით რიცხვების XOR: ანუ 1oplus2oplus...oplusn გამოსახულების შედეგი, სადაც oplus - აღნიშნავს ბიტურ XOR ოპერაციას.

## Input

მოცემულია ერთი ნატურალური რიცხვი: N (1 <= N <= 10⁵)

## Output

გამობეჭდეთ ერთი რიცხვი: 1-დან N-ის ჩათვლით რიცხვების XOR.
