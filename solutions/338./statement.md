# Problem Statement — 338. საბავშვო მოედანი 1

> Copied from [AiaSoft.ge #338](https://aiasoft.ge/problem/338)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

საბავშვო მოედნის გარშემო (წრეზე) $N$ რაოდენობის სკამია განლაგებული და, რა თქმა უნდა, გადანომრილია 1-დან $N$-ის ჩათვლით.
ორმა მეგობარმა, გიგიკომ და გუგიკომ სხვადასხვა დროს შეიძინა ბილეთები საბავშვო კონცერტზე დასასწრებად. თითოეულ ბილეთს სკამის ნომერი აწერია.
მეგობრებს ძალიან უნდათ ერთმანეთის გვერდით ჯდომა.
დაწერეთ პროგრამა, რომელიც დაბეჭდავს სიტყვას YES, თუ გიგიკო და გუგიკო ერთმანეთის გვერდით აღმოჩნდებიან და სიტყვას NO - წინააღმდეგ შემთხვევაში.

## Input

პირველ სტრიქონზე შემოდის ერთი მთელი დადებითი რიცხვი $N, 1<N<=100$ - მოედნის გარშემო სკამების რაოდენობა.
მეორე სტრიქონზე შემოდის ერთმანეთისაგან ერთი ჰარით დაშორებული ორი მთელი დადებითი, ერთმანეთისაგან განსხვავებული რიცხვი - გიგიკოს და გუგიკოს სკამების ნომრები.

## Output

გამოიტანეთ ერთი სიტყვა YES ან NO. შესაბამისად, აღმოჩნდნენ თუ არა მეგობრები ერთმანეთის გვერდით.
