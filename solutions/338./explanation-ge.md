# განმარტება — 338. საბავშვო მოედანი 1

## მიდგომა

სკამები წრეზეა დალაგებული, რაც იმას ნიშნავს, რომ მეზობლობის შემოწმება მარტივად შეიძლება. ზოგად შემთხვევაში, თუ სხვაობა ერთია, ე.ი. გვერდიგვერდ სხედან, მაგრამ წრიული განლაგების გამო 1 და N-იც მეზობლები არიან. ამიტომ, უბრალოდ ვამოწმებ ან სხვაობა იყოს 1, ან გვეყოს ის განსაკუთრებული შემთხვევა, როცა ერთი 1-ია და მეორე N.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- წრიული განლაგებისას 'ბოლო' და 'პირველი' სკამი ერთმანეთს ებჯინება, რაც კოდში ცალკე უნდა გაითვალისწინო.
- საკმარისია მარტივი პირობითი ოპერატორი, არანაირი მასივები არ გჭირდება.

## ხშირი შეცდომები

- მხოლოდ |a-b| == 1-ის შემოწმება და 1-სა და N-ის კავშირის დავიწყება.
