# განმარტება — 146. სამკუთხედის ფართობი (გვერდებით)

## მიდგომა

სამკუთხედის ფართობის გამოსათვლელად აქ ყველაზე კლასიკური გზაა ჰერონის ფორმულა. მთავარია, თავიდანვე შევამოწმოთ სამკუთხედის უტოლობა: ნებისმიერი ორი გვერდის ჯამი მეტის უნდა იყოს მესამეზე. თუ ეს პირობა არ სრულდება, მაშინ სამკუთხედი ფიზიკურად არ არსებობს და პირდაპირ -1 უნდა დავაბრუნოთ. struct-ის გამოყენება იმიტომ გადავწყვიტე, რომ კოდი უფრო წაკითხვადი ყოფილიყო, თუმცა რეალურად ლოგიკა მარტივია.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- აუცილებლად უნდა შეამოწმო სამივე კომბინაცია: a+b>c, a+c>b და b+c>a, თორემ ჩავარდები.
- ნახევარპერიმეტრის (p) გამოსათვლელად აუცილებლად double გამოიყენე, თორემ მთელი რიცხვებით დათვლისას სიზუსტეს დაკარგავ.

## ხშირი შეცდომები

- fixed და setprecision(5)-ის დავიწყება, რაც output-ის ფორმატის გაფუჭებას იწვევს.
- იმის დავიწყება, რომ სამკუთხედი არ იარსებებს, თუ ჯამი ტოლია მესამე გვერდის (დეგენერატული სამკუთხედი).
