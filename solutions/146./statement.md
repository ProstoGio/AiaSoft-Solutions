# Problem Statement — 146. სამკუთხედის ფართობი (გვერდებით)

> Copied from [AiaSoft.ge #146](https://aiasoft.ge/problem/146)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია სამი მთელი რიცხვი a, b და c. თუ არსებობს სამკუთხედი, რომლის გვერდები აღნიშნული რიცხვებია მაშინ გამოიტანეთ მისი ფართობი თუ ასეთი სამკუთხედი არ არსებობს გამოიტანეთ -1.

## Input

პირველ ხაზზე შემოდის სამი მთელი რიცხვი a, b და c
1<=a, b, c <= 1000

## Output

თუ არსებობს სამკუთხედი რომლის გვერდები მოცემული რიცხვებია გამობეჭდეთ მისი ფართობი, წინააღმდეგ შემთხვევაში გამობეჭდეთ -1

ფართობის სიზუსტე უნდა იყოს არანაკლებ 5 ციფრი მძიმის შემდეგ.
