# განმარტება — 790. ბევრად ბევრი

## მიდგომა

ეს არის ტიპური ამოცანა, სადაც ორი ციკლი გვჭირდება. გარე ციკლით მივყვებით რიცხვებს 1-დან N-მდე, შიგნით კი კიდევ ერთი ციკლი გვაქვს, რომელიც ამ რიცხვს იმდენჯერ ბეჭდავს, რა მნიშვნელობაც აქვს თავად რიცხვს. ნუ ეცდები აქ რაიმე ეშმაკობას, N მხოლოდ 100-ია და ეს მიდგომა უპრობლემოდ ატრიალებს ყველაფერს.

## სირთულე

| | |
|-|--|
| დრო | O(n²) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- გარე 'for' ციკლის ცვლადი 'i' გვაძლევს იმ რიცხვს, რომელიც უნდა დაიბეჭდოს, შიდა ციკლი კი უბრალოდ 'i'-ჯერ ტრიალებს.
- რადგან N ძალიან პატარაა, O(n²) სირთულე სრულიად მისაღებია და დროის ლიმიტში ადვილად ვეტევით.

## ხშირი შეცდომები

- პროცენტულად ხშირად ავიწყდებათ, რომ თითოეულ რიცხვს შორის ჰარი უნდა იყოს, წინააღმდეგ შემთხვევაში 'Wrong Answer'-ს ამოგიგდებს.
