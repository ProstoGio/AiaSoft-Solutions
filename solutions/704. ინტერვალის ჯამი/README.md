# ამოხსნის ახსნა

ამოცანაში მოცემულია `N` ელემენტიანი მასივი და `Q` შეკითხვა.

თითოეული შეკითხვა შეიცავს ორ ინდექსს:

```text
l, r
```

და საჭიროა ვიპოვოთ:

```text
a[l] + a[l+1] + ... + a[r]
```

ანუ მასივის `[l, r]` მონაკვეთში არსებული ელემენტების ჯამი.

---

# მაგალითი

მოცემულია:

```text
1 3 5 2 4
```

თუ გვკითხავენ:

```text
l = 2
r = 4
```

უნდა დავითვალოთ:

```text
3 + 5 + 2 = 10
```

პასუხია:

```text
10
```

---

# პირდაპირი მიდგომა

ყოველი შეკითხვისთვის შეგვეძლო გაგვევლო:

```text
l-დან r-მდე
```

და დაგვეთვალა ჯამი.

---

მაგრამ:

```text
N,Q ≤ 200000
```

---

უარეს შემთხვევაში თითო შეკითხვა მოითხოვს:

```text
O(N)
```

დროს.

---

ხოლო ყველა შეკითხვა ერთად:

```text
O(N·Q)
```

იქნება.

---

ეს შეიძლება გახდეს:

```text
200000 × 200000
=
40 000 000 000
```

ოპერაცია.

---

რაც ძალიან ნელია.

---

# მთავარი იდეა - Prefix Sum

წინასწარ დავითვალოთ მასივის პრეფიქსური ჯამები.

---

განვსაზღვროთ:

```text
dp[i]
```

როგორც:

```text
მასივის პირველი i+1 ელემენტის ჯამი
```

---

მაგალითად:

```text
a = [1, 3, 5, 2, 4]
```

---

მაშინ:

```text
dp[0] = 1
dp[1] = 1+3 = 4
dp[2] = 1+3+5 = 9
dp[3] = 1+3+5+2 = 11
dp[4] = 1+3+5+2+4 = 15
```

---

მივიღებთ:

```text
dp = [1, 4, 9, 11, 15]
```

---

# როგორ ვიპოვოთ [l,r] ინტერვალის ჯამი?

```text

```

---

პრეფიქსური ჯამებიდან ვიცით:

```text

=

```

---

ასევე:

```text

მივიღებთ:

```

რაც ზუსტად გვინდოდა.

# კოდის განხილვა

## მასივების შექმნა

```cpp
vector<long long> a(n);
vector<long long> dp(n);
```

---

`a` ინახავს ელემენტებს.

---

`dp` ინახავს პრეფიქსურ ჯამებს.

---

# პრეფიქსური ჯამების აგება

```cpp
long long sum = 0;
```

---

შემდეგ:

```cpp
for(int i=0; i<n; i++){
    cin >> a[i];
    sum += a[i];
    dp[i] = sum;
}
```

---

ყოველ ახალ ელემენტს ვუმატებთ მიმდინარე ჯამს.

---

მაგალითად:

```text
1 3 5
```

---

პირველი ნაბიჯი:

```text
sum = 1
dp[0] = 1
```

---

მეორე:

```text
sum = 4
dp[1] = 4
```

---

მესამე:

```text
sum = 9
dp[2] = 9
```

# რატომ მუშაობს ეს?

# შეჯამება

პირდაპირი დათვლა თითო შეკითხვისთვის ძალიან ნელია.

ამის ნაცვლად წინასწარ ვითვლით პრეფიქსურ ჯამებს:

```text
dp[i] = a[1] + a[2] + ... + a[i]
```

შემდეგ ნებისმიერი ინტერვალის ჯამი მიიღება ფორმულით:

```text
dp[r-1] - dp[l-2]
```

(თუ `l > 1`)

ხოლო თუ `l = 1`:

```text
dp[r-1]
```

ამგვარად თითო შეკითხვას ვპასუხობთ მხოლოდ `O(1)` დროში.
