# Problem Statement — 928. მიმდევრობები

> Copied from [AiaSoft.ge #928](https://aiasoft.ge/problem/928)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია ნატურალური k და m რიცხვები. განვიხილოთ n რაოდენობის ნატურალურ რიცხვთა a1, a2, ..., an მიმდევრობა, რომლის ყველა წევრი ნაკლებია m-ზე და ყოველი მომდევნო წევრი, დაწყებული მეორედან, მიიღება წინა წევრზე k რიცხვის ან დამატებით, ან გამოკლებით. მაგალითად, თუ k = 2, m = 10, n = 9, ერთ-ერთი ასეთი მიმდევრობა იქნება: 3, 5, 3, 1, 3, 1, 3, 5, 7.
დაწერეთ პროგრამა, რომელიც მოცემული k, m და n-სათვის იპოვის ყველა ასეთ მიმდევრობათა რაოდენობას.

## Input

სტანდარტული შეტანის პირველ სტრიქონში ჩაწერილია სამი
ნატურალური k, m და n რიცხვი.
სტრიქონებში მონაცემები ერთმანეთისაგან თითო ჰარითაა გამოყოფილი.
0 < k < 50
0 < m < 50
0 < n < 50

## Output

სტანდარტული გამოტანის ერთადერთ სტრიქონში უნდა გამოიტანოთ ერთი მთელი რიცხვი - მიმდევრობათა ნაპოვნი რაოდენობა
