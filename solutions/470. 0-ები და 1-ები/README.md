# ამოხსნის ახსნა

მოცემულია სტრიქონი, რომელიც შეიცავს მხოლოდ სიმბოლოებს:

```text
0
```

და

```text
1
```

საჭიროა დავითვალოთ, რამდენჯერ გვხვდება მასში სიმბოლო `'1'`.

---

# მაგალითი

თუ შემოვიდა:

```text
1011010
```

მაშინ ერთიანები გვხვდება ოთხჯერ.

პასუხი იქნება:

```text
4
```

---

# მთავარი იდეა

გავივლით სტრიქონის ყველა სიმბოლოზე.

თუ მიმდინარე სიმბოლო არის:

```cpp
'1'
```

გავზრდით მთვლელს ერთით.

---

# კოდის განხილვა

## სტრიქონის წაკითხვა

```cpp
string s;
cin >> s;
```

---

## ერთიანების მთვლელი

```cpp
int cnt = 0;
```

ცვლადი `cnt` ინახავს ნაპოვნი `'1'` სიმბოლოების რაოდენობას.

---

## სტრიქონის გავლა

```cpp
for(int i=0; i<s.size(); i++)
```

ციკლი ამოწმებს სტრიქონის თითოეულ სიმბოლოს.

---

## შემოწმება

```cpp
if(s[i]=='1')
    cnt++;
```

თუ მიმდინარე სიმბოლო `'1'` არის, მთვლელს ერთით ვზრდით.

---

## პასუხის გამოტანა

```cpp
cout << cnt;
```

ბოლოს ვბეჭდავთ ნაპოვნი ერთიანების რაოდენობას.

---

# რატომ მუშაობს?

ციკლი სტრიქონის ყველა სიმბოლოს ერთხელ ამოწმებს.

ყოველ ჯერზე, როდესაც შეხვდება `'1'`-ს, მთვლელი იზრდება.

ამიტომ ციკლის დასრულების შემდეგ `cnt` ზუსტად უდრის სტრიქონში არსებული ერთიანების რაოდენობას.

---

# სირთულე

### დროითი სირთულე

```text
O(N)
```

სადაც `N` სტრიქონის სიგრძეა.

---

### მეხსიერების სირთულე

```text
O(1)
```

გამოიყენება მხოლოდ ერთი დამატებითი მთვლელი.

---

# შეჯამება

პროგრამა უბრალოდ გადადის სტრიქონის ყველა სიმბოლოზე და ითვლის, რამდენჯერ შეხვდება `'1'`-ს. ბოლოს სწორედ ამ რაოდენობას ბეჭდავს.
