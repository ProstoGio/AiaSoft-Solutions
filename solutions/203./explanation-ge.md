# განმარტება — 203. ბაბის უყვარს კომბოსტო

## მიდგომა

ეს ტიპური DP-ა, სადაც მთავარია სწორად გავთვალოთ, როგორ ავუაროთ გვერდი წესს, რომელიც სამი საფეხურის ზედიზედ აღებას კრძალავს. ყოველ საფეხურზე სამი არჩევანი გვაქვს: ან არ ვიღებთ ამ საფეხურს, ან ვიღებთ მარტო ამას, ან ვიღებთ მიმდევრობას (i და i-1) და i-2 საფეხურს ვტოვებთ. შედეგებს ვინახავთ მასივში და ყოველ ნაბიჯზე ვარჩევთ მაქსიმუმს ამ სამი ვარიანტიდან.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- რეკურენტული ფორმულა dp[i] = max(dp[i-1], dp[i-2]+v[i], dp[i-3]+v[i-1]+v[i]) ფარავს ყველა დასაშვებ სცენარს.
- n=1 და n=2 შემთხვევები ცალკე უნდა დამუშავდეს, რადგან წესი მაშინ ჯერ არ ირღვევა.

## ხშირი შეცდომები

- n=1 ან n=2 შემთხვევების გათვალისწინება დაგავიწყდეთ, რამაც შეიძლება ციკლის დროს ინდექსების არევა გამოიწვიოს.

## ნაბიჯ-ნაბიჯ ახსნა

ვთქვათ გვაქვს [10, 20, 30, 40]. dp[1]=10, dp[2]=30. i=3-ისთვის: ან ვტოვებთ 30-ს (dp[2]=30), ან ვიღებთ მარტო 30-ს და dp[1]+30=40, ან ვიღებთ 20-ს და 30-ს (20+30=50). ვირჩევთ 50-ს.
