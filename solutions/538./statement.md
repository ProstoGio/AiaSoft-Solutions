# Problem Statement — 538. ჭადრაკის დაფა და დედოფლები

## Task

თქვენი დავალებაა განათავსოთ ჭადრაკის დაფაზე რვა დედოფალი ისე, რომ არცერთი დედოფალი არ ემუქრებოდეს ერთმანეთს. მაგრამ სირთულეს ის წარმოადგენს, რომ დაფაზე ადგილები დაკავებულია, ამიტომ დაკავებულ ადგილებში დედოფლის დაყენება არ შეიძლება, დაკავებული უჯრები ხელს არ უშლის დედოფლებს, რომ დაემუქრონ ერთმანეთს.

თქვენი დავალებაა, იპოვოთ რამდენი განსხვავებული ვარიანტი არსებობს დაფაზე რვა დედოფლის განთავსებისა ისე, რომ არცერთი დედოფალი არ ემუქრებოდეს სხვა რომელიმე დედოფალს.

სურათზე მოცემულია თუ როგორ შეუძლია დედოფალს სვლა: 

`~https://storage.googleapis.com/problems_api_photos/images/49JgZAphi3uHnUCG4H6a.png`

## Input

მოცემულია $8$x$8$ დაფა, თუ დაფის რომელიმე უჯრაში წერია `bფიფქი` სიმბოლო, ეს ნიშნავს რომ მოცემული ადგილი დაკავებულია, ხოლო თუ წერია `bწერტილი` მაშინ ეს უჯრა თავისუფალია.

## Output

ერთადერთ ხაზზე გამობეჭდეთ დაფაზე რვა დედოფლის განთავსების რამდენი განსხვავებული ვარიანტი არსებობს ისე, რომ არცერთი დედოფალი ერთმანეთს არ ემუქრებოდეს.

შეგახსენებთ რომ იქ სადაც წერია `bფიფქი` დედოფლების დასმა არ შეიძლება, მაგრამ დაკავებული უჯრები ხელს არ უშლით დედოფლებს რომ ერთმანეთს დაემუქრონ.
