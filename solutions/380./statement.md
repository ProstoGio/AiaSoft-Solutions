# Problem Statement — 380. მთელი წერტილები

> Copied from [AiaSoft.ge #380](https://aiasoft.ge/problem/380)

**Time limit:** 0.5s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

პატარა ნიკუშამ უჯრედებიანი რვეულის ფურცელზე ორი, ერთმანეთისაგან განსხვავებული მთელი წერტილი დასვა.
ნიკუშამ უკვე იცის, რომ წერტილს ეწოდება მთელი, თუკი მისი ორივე კოორდინატი მთელია. მაგალითად, $A(3;8)$ წერტილი მთელია, ხოლო $B(2.6;7)$ არაა მთელი. 
ამის შემდეგ ეს წერტილები მონაკვეთით შეაერთა. ახლა აინტერესებს, კიდევ რამდენ მთელ წერტილს გაივლის მისი მონაკვეთი. დაეხმარეთ ნიკუშას.

## Input

შემომავალი ფაილის ერთადერთ სტრიქონში ჩაწერილია ოთხი მთელი რიცხვი $x_1, y_1, x_2, y_2$ - ნიკუშას ორი მთელი $A(x_1, y_1)$ და $B(x_2, y_2)$ წერტილის კოორდინატები.
$-1000<=x_1, y_1, x_2, y_2<=1000$.

## Output

გამოიტანეთ ერთი მთელი რიცხვი - $AB$ მონაკვეთის მთელი წერტილების რაოდენობა, გარდა მონაკვეთის ბოლოებისა.
