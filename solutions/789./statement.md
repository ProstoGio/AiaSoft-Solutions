# Problem Statement — 789. ლუწი მასივი

> Copied from [AiaSoft.ge #789](https://aiasoft.ge/problem/789)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მასივს ეწოდება ლუწი. თუკი მისი ნებისმიერი ელემენტი ლუწი ნატურალური რიცხვია.
მოცემულია ნატურალური რიცხვების მასივი. დაწერეთ პროგრამა, რომელიც მასივის კენტ ელემენტებს დაუმატებს ერთიანს.

## Input

შემომავალი ფაილის პირველ სტრიქონში ჩაწერილია მასივის ელემენტების რაოდენობა $n$, $1 <= n <=20$.
მეორე სტრიქონი შეიცავს მასივის ელმენტებს $a_k$, $1 <= a_k <=100$, $0<=k<n$.

## Output

ერთ სტრიქონზე გამობეჭდეთ მიღებული მასივი. ელემენტებს შორის ჩასვით ერთი ჰარი.
