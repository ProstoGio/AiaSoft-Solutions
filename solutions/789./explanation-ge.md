# განმარტება — 789. ლუწი მასივი

## მიდგომა

ეს საკმაოდ მარტივი ამოცანაა. თავიდან მასივს ვავსებ, მერე უბრალოდ თითოეულ ელემენტზე გადავირბენ `for` ციკლით და `modulo`-ს გამოყენებით ვამოწმებ კენტია თუ ლუწი. თუ რიცხვი კენტია, უბრალოდ ერთს ვუმატებ და მორჩა, ეგრევე ლუწი ხდება.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- კენტი რიცხვის გამოსაჭერად `a[i] % 2 != 0` საკმარისია, რთული არაფერია.
- მასივის ელემენტების შეცვლა ადგილზევე შეიძლება, დამატებითი მემორი არც კი დაგვჭირვებია.

## ხშირი შეცდომები

- შეიძლება აგერიოს და ლუწებს დაუმატო ერთი, რაც პირობას დაარღვევს.
