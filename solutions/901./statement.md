# Problem Statement — 901. ერთის დამატება

> Copied from [AiaSoft.ge #901](https://aiasoft.ge/problem/901)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

შემოდის ნატურალური რიცხვი. 
გამობეჭდეთ მისი მომდევნო ნატურალური რიცხვი.

## Input

პირველ სტრიქონზე შემოდის ერთი მთელი რიცხვი $a$ .
$0<=a<=10^1000$

## Output

გამობეჭდეთ $a+1$
