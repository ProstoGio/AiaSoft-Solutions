# განმარტება — 334. ბაბის სიმბოლო

## მიდგომა

აქ არაფერია რთული, უბრალოდ უნდა შევამოწმოთ შემოსული სიმბოლო ციფრია თუ არა. ASCII ცხრილში ციფრები '0'-დან '9'-მდე თანმიმდევრულად მიყვება ერთმანეთს. '0'-ის კოდი არის 48, '9'-ისი კი 57. თუ ჩვენი სიმბოლო ამ დიაპაზონში ზის, ესე იგი ციფრია.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- ASCII კოდები ისეა მოწყობილი, რომ ციფრების შემოწმება მათემატიკური უტოლობით ყველაზე სწრაფი გზაა.
- სიმბოლოს პირდაპირ int-ად აღქმა და შედარება გვაძლევს საშუალებას, ზედმეტი მანიპულაციების გარეშე ვიმუშაოთ.

## ხშირი შეცდომები

- საზღვრების არევა (მაგალითად, 48-ზე ნაკლების ან 57-ზე მეტის ჩათვლა ციფრად).
