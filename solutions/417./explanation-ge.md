# განმარტება — 417. ორი ბაჭია

## მიდგომა

თავიდან ცოტა დამაბნეველი ჩანს, მაგრამ სინამდვილეში ეს კლასიკური Game Theory-ა. რადგან ბაჭიები ერთმანეთს ვერ გადაახტებიან, მათი მოძრაობის არეალი შეზღუდულია. სინამდვილეში, ჩვენ გვაინტერესებს მხოლოდ ის ცარიელი უჯრები, რაც მათ შორისაა. რადგან ბაბი და ბუბი კიდურა უჯრებში სხედან, თავისუფალი სივრცე არის n-2. მთელი თამაში დაიყვანება მარტივ მათემატიკურ გათვლაზე, სადაც გადამწყვეტია ნაშთი k+1-ზე.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- n-2 უჯრაა თავისუფალი და ზუსტად ამ სივრცის გაყოფაზე მიდის ბრძოლა.
- თუ (n-2) იყოფა (k+1)-ზე, მაშინ პირველი მოთამაშე წაგებულ სიტუაციაშია, რადგან მეორე ყოველთვის შეძლებს ბალანსის შენარჩუნებას.

## ხშირი შეცდომები

- n-ის პირდაპირ გამოყენება n-2-ის ნაცვლად, რაც იწვევს არასწორ გათვლას.

## ნაბიჯ-ნაბიჯ ახსნა

წარმოიდგინე n=4, k=1. თავისუფალი უჯრებია 4-2=2. კენტია თუ ლუწი, ამას არა აქვს მნიშვნელობა, მთავარია 2 % (1+1) == 0. ამიტომ აქ ბუბი იგებს.
