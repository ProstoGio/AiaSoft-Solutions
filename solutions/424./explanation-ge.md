# განმარტება — 424. ვინ არის მარტო?

## მიდგომა

თავიდან შეიძლება მოგინდეს `map`-ის გამოყენება ან დალაგება, მაგრამ ეს მარტივი ამოცანა ბიტური ოპერაციებით წამებში გვარდება. აზრი იმაშია, რომ XOR ოპერატორს აქვს ჯადოსნური თვისება: თუ ერთსა და იმავე რიცხვს XOR-ს გაუკეთებ, შედეგი ნული იქნება. რადგან ყველა ელემენტი ორჯერაა, წყვილები ერთმანეთს 'ჭამენ' და ბოლოს მხოლოდ ის რიცხვი რჩება, რომელიც მარტოა.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- XOR-ის კომუტატიურობა ნიშნავს, რომ სულ ერთია რა თანმიმდევრობით დავა XOR-ებთ რიცხვებს.
- ნებისმიერი რიცხვის საკუთარ თავთან XOR იძლევა 0-ს, რაც წყვილების განადგურების გასაღებია.

## ხშირი შეცდომები

- ზედმეტი მეხსიერების ხარჯვა frequency array-ების ან map-ების გამოყენებით.
