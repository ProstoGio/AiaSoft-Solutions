# Problem Statement — 424. ვინ არის მარტო?

> Copied from [AiaSoft.ge #424](https://aiasoft.ge/problem/424)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია მასივი, რომელშიც თითოეული ელემენტი შედის ორჯერ, გარდა მხოლოდ ერთი რომელიღაც ელემენტისა. ეს ელემენტი მასივში შედის მხოლოდ ერთხელ.
იპოვეთ ეს მარტოდ მყოფი ელემენტი.

## Input

პირველ სტრიქონზე შემოდის ერთი ნატურალური რიცხვი N, 0<N<=100 - მასივის სიგრძე. მეორე სტრიქონზე შემოდის ერთმანეთისაგან ჰარით გამოყოფილი N ცალი მთელი რიცხვი - მასივის ელემენტები.

## Output

გამოიტანეთ ერთი მთელი რიცხვი - იმ ელემენტის მნიშვნელობა, რომელიც მასივში გვხვდება მხოლოდ ერთხელ.
