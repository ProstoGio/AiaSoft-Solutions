# განმარტება — 048. პირველკლასელი

## მიდგომა

სტრიქონი სულ სამი სიმბოლოა, ამიტომ არაფერი ზედმეტი არ სჭირდება. პირდაპირ ინდექსებით ვიღებთ რიცხვებს: ნულოვან და მეორე პოზიციაზეა ციფრები. სიმბოლოს მთელ რიცხვად გადაქცევა ყველაზე მარტივად '0'-ის ან 48-ის გამოკლებით ხდება. მერე უბრალოდ if-ით ვამოწმებთ შუა სიმბოლო პლუსია თუ მინუსი და ვბეჭდავთ პასუხს.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- ASCII ცხრილში ციფრები მიყოლებითაა, ამიტომ s[i] - '0' ყველაზე სწრაფი მეთოდია char-ის int-ად გადასაქცევად.
- სტრიქონის სიგრძე ფიქსირებულია, ასე რომ არც ციკლები გვჭირდება და არც სტრიქონის გაყოფა.

## ხშირი შეცდომები

- თუ პირდაპირ დავწერთ s[0] + s[2], კოდი მიუმატებს მათ ASCII კოდებს და არა ციფრებს.
