# Problem Statement — 138. მასივი 1

> Copied from [AiaSoft.ge #138](https://aiasoft.ge/problem/138)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია მთელი რიცხვების მასივი. დაწერეთ პროგრამა, რომელიც გამობეჭდავს ამ მასივს ისე, რომ პირველ და ბოლო ელემენტებს ადგილები გაცვლილი ექნებათ.

## Input

შემომავალი ფაილის პირველ სტრიქონში ჩაწერილია მასივის ელემენტების რაოდენობა $n$, $1 <= n <=20$.
მეორე სტრიქონი შეიცავს მასივის ელმენტებს $a_k$, $-100 <= a_k <=100$, $0<=k<n$.

## Output

გამობეჭდეთ მასივი ისე, რომ პირველ და ბოლო ელემენტებს ადგილები გაცვლილი ჰქონდეთ.
