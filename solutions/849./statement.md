# Problem Statement — 849. სად არის მინიმუმი?

> Copied from [AiaSoft.ge #849](https://aiasoft.ge/problem/849)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია მთელი რიცხვების მასივი, რომელიც შეიცავს ერთადერთ მინიმალურ ელემენტს. საჭიროა მისი ადგილის პოვნა.

## Input

პირველ სტრიქონზე ჩაწერილია მასივის ელემენტების რაოდენობა.
მეორე სტრიქონი შეიცავს ერთმანეთისაგან ჰარით გამოყოფილ მასივის ელემენტებს.

## Output

გამოიტანეთ ჰარით გამოყოფილი ორი მთელი რიცხვი - მასივის მინიმალური ელემენტის მნიშვნელობა და მისი ინდექსი მასივში.
