# Problem Statement — 188. ჩქარა, ბაბი, ჩქარა!

> Copied from [AiaSoft.ge #188](https://aiasoft.ge/problem/188)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ბაჭია ბაბის დაპროგრამების გაკვეთილზე აგვიანდება. მან რაც შეიძლება სწრაფად უნდა აირბინოს n საფეხურიანი კიბე, თორემ დაპროგრამების მასწავლებლები აკა და იკა, რომლებიც ბოლო საფეხურზე დგანან, სტაფილოს აღარ მისცემენ.  ბაჭიამ აუცილებლად უნდა დააბიჯოს ბოლო საფეხურს, თორემ სტაფილო აღარ შეხვდება.

ბაბის ისე უყვარს სტაფილო და თავისი მასწავლებლები, რომ კიბეზე სირბილისას ძალიან ინდომებს და როცა უნდა,  ერთი საფეხურის გამოტოვებაც შეუძლია. 

სკოლის კიბეზე ერთი A ტიპის ჯადოსნური საფეხურია.  ცნობილია, რომ A ტიპის ჯადოსნურ საფეხურზე თუ დააბიჯებ, უფრო სწრაფი ხდები და  როცა გინდა, ერთს ან ორი საფეხურსაც კი გამოტოვებ.

კიბეზე ასვლის რამდენი ვარიანტი აქვს ბაბის?

## Input

შემოდის ორი რიცხვი n და m, კიბის საფეხურების რაოდენობა და  ჯადოსნური საფეხურის რიგითი ნომერი. 1<=n<=20, 1<=m<=n.

## Output

გამოიტანეთ ერთი რიცხვი, ბაბის ვარიანტების რაოდენობა.
