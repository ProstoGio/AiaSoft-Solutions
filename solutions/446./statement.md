# Problem Statement — 446. დაკარგული რიცხვი

> Copied from [AiaSoft.ge #446](https://aiasoft.ge/problem/446)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

პატარა ნინიკომ ლოტოს ტომარა იპოვა. რომელშიც $n-1$ რაოდენობის ლოტოს კოჭი აღმოჩნდა. როგორც გაირკვა, ერთი კოჭი დაიკარგა. დაეხმარეთ ნინიკოს დაკარგული კოჭის ნომრის დადგენაში.

## Input

პირველ სტრიქონზე შემოდის ერთი ნატურალური რიცხვი $n, 2<=n<=1000$, ლოტოს კოჭების რაოდენობა სრულყოფილ ტომარაში.
მომდევნო სტრიქონზე შემოდის ერთმანეთისაგან ჰარით დაშორებული $n-1$ ცალი ნატურალური რიცხვი - იმ ლოტოს კოჭების ნომრები, რომლებიც ტამარაში აღმოჩნდა.

## Output

გამოიტანეთ ერთი ნატურალური რიცხვი - დაკარგული კოჭის ნომერი.
