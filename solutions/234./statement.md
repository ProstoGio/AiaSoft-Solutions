# Problem Statement — 234. მოზაიკა

> Copied from [AiaSoft.ge #234](https://aiasoft.ge/problem/234)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია $N\times N$ ზომის დაფა, რომელზეც დამაგრებულია სხვადასხვა ფერის კენჭები. თქვენი მიზანია გაუცვალოთ ორ მეზობელ განსხვავებული ფერის კენჭს ადგილი ისე, რომ მიიღოთ მაქსიმალური  რაოდენობის ერთი ფერის კენჭი რომელიმე სვეტში ან სტრიქონში. გაცვლა შეიძლება მოხდეს მხოლოდ ვერტიკალურად ან ჰორიზონტალურად.

## Input

პირველ სტრიქონში მოცემულია ერთი მთელი რიცხვი $N (3 ≤ N ≤ 50)$, მომდევნო $N$ სტრიქონი განსაზღვრავს დაფას. $i$ - ური სტრიქონის $j$ - ურ პოზიციაში მოცემულია შესაბამისი კენჭის ფერი: $C$ (წითელი), $P$ (ლურჯი), $Z$ (მწვანე), $Y$ (ყვითელი).

## Output

ერთადერთ სტრიქონში გამოიტანეთ ერთი ფერის კენჭების მაქსიმალური რაოდენობა, რომელიც შეიძლება მივიღოთ რომელიმე სვეტში ან სტრიქონში.
