# განმარტება — 234. მოზაიკა

## მიდგომა

N მაქსიმუმ 50-ია, ასე რომ O(n³) სრულიად საკმარისია. იდეა მარტივია: გადავუაროთ ყველა მეზობელ წყვილს ჰორიზონტალურად და ვერტიკალურად, შევუცვალოთ ადგილი, გადავთვალოთ მაქსიმალური თანმიმდევრობა შესაბამის სტრიქონსა და სვეტში, და ბოლოს დავაბრუნოთ თავის ადგილზე. ამას ვაკეთებთ ყველა შესაძლო წყვილისთვის და ვინახავთ მიღებულ მაქსიმუმს.

## სირთულე

| | |
|-|--|
| დრო | O(n³) |
| მეხსიერება | O(n²) |

## მთავარი დაკვირვებები

- N=50 იძლევა იმის საშუალებას, რომ უბრალოდ 'ბრუტფორსით' დავთვალოთ ყველაფერი.
- ყოველი swap-ის შემდეგ მთელი დაფის შემოწმება არ გვჭირდება, მხოლოდ იმ სვეტს და სტრიქონს ვამოწმებთ, სადაც ცვლილება მოხდა.

## ხშირი შეცდომები

- დაგავიწყდეს ქვების თავდაპირველ პოზიციაზე დაბრუნება მას შემდეგ, რაც მაქსიმალურ სიგრძეს დაითვლი.
- მხოლოდ ერთი მიმართულების (მაგალითად, ჰორიზონტალურის) შემოწმება, როცა ცვლილებამ შესაძლოა ვერტიკალური 'სტრიკიც' გაზარდოს.

## ნაბიჯ-ნაბიჯ ახსნა

ვთქვათ, გვაქვს 'C P C C' სტრიქონი. პირველ 'C'-ს და 'P'-ს ვუცვლით ადგილს, მივიღებთ 'P C C C'-ს. ამ მომენტში ვამოწმებთ ამ სტრიქონს და იმ ორ სვეტს, რომელშიც ეს ორი ქვა იდგა. მაქსიმუმს ვიმახსოვრებთ, მერე წყვილს ისევ თავის ადგილზე ვაბრუნებთ და გადავდივართ შემდეგ წყვილზე.
