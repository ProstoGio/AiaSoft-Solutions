# Problem Statement — 616. ბიტებით თამაში

> Copied from [AiaSoft.ge #616](https://aiasoft.ge/problem/616)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია Q რაოდენობის მოთხოვნა. თითოეული მოთხოვნა სამ რიცხვს შეიცავს: x, k და t-ს.

მოთხოვნებში t განსაზღვრავს მოთოვნის ტიპს, კერძოდ:
	
	თუ t=1, მაშინ x-ის ბიტურ ჩანაწერში k-ური ბიტი უნდა ჩაირთოს (თუ გამორთულია)
	თუ t=2, მაშინ x-ის ბიტურ ჩანაწერში k-ური ბიტი უნდა გამოირთოს (თუ ჩართულია)
	თუ t=3, მაშინ x-ის ბიტურ ჩანაწერში k-ური ბიტი უნდა შეიცვალოს (ნული ერთით, ხოლო ერთი - ნულით)

თქვენი დავალებაა შეასრულოთ მოცემული მოთხოვნები და დაბეჭდოთ შედეგები.

* ბიტის ჩართვა ნიშნავს, რომ მოცემული ბიტი გავუტოლოთ 1-ს, ხოლო გამორთვა - გავუტოლოთ 0-ს.

## Input

პირველ ხაზზე მოცემულია ერთი ნატურალური რიცხვი: Q 
მომდევნო Q ხაზიდან თითოეული შეიცავს სამ მთელი რიცხვს: x, k, t

1 <= Q <= 10⁵
0 <= x <= 10⁹
0 <= k <= 31

## Output

ყოველი მოთხოვნისთვის ცალკე ხაზზე გამობეჭდეთ ერთი რიცხვი: მოთხოვნის შესრულების შედეგად მიღებული ათობითი რიცხვი.
