# განმარტება — 734. შებრუნებული რიცხვი

## მიდგომა

რადგან ამოცანაში პირდაპირ გვეუბნებიან, რომ რიცხვი ორნიშნაა, არ არის საჭირო სტრინგებთან წვალება. უბრალოდ ვიყენებ მათემატიკურ ოპერაციებს: 10-ზე გაყოფით ვიღებ პირველ ციფრს, ხოლო მოდულით (a % 10) მეორეს. მერე უბრალოდ ვატრიალებ და ვკრებ - მეორე ციფრს ვამრავლებ ათზე და ვუმატებ პირველს.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- a / 10-ით ვშორდებით მეორე ციფრს და გვრჩება მხოლოდ ათეულების ადგილი.
- a % 10 არის ყველაზე სწრაფი გზა ბოლო ციფრის გამოსატანად, რაც შებრუნებულ რიცხვში პირველ ადგილზე უნდა გვეწეროს.

## ხშირი შეცდომები

- სტრინგად გადაყვანა და მასივის ინდექსებით თამაში, რაც ამ დონის ამოცანისთვის ზედმეტი 'overhead'-ია.
