# განმარტება — 085. გამრავლება

## მიდგომა

ეს პრობლემა იმაზე მარტივია, ვიდრე ერთი შეხედვით ჩანს. უბრალოდ ვიღებთ ორ მთელ რიცხვს და ვაბრუნებთ მათ ნამრავლს. ცალკე ფუნქციაში გადატანა მხოლოდ იმიტომ გავაკეთე, რომ კოდი სუფთად გამოიყურებოდეს, თორემ პირდაპირ `cout`-შიც შეიძლებოდა გაკეთება. რადგან რიცხვები მცირეა, `int`-ის ლიმიტებს არც ვუახლოვდებით.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- სრულიად საკმარისია სტანდარტული არითმეტიკული ოპერატორის გამოყენება.
- შემოსული მონაცემების ზომა იმდენად პატარაა, overflow-ის შიში არ უნდა გქონდეთ.

## ხშირი შეცდომები

- ზედმეტი შემოწმებების ან ციკლების დამატება, რაც აქ საერთოდ არ გვჭირდება.
