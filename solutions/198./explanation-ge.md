# განმარტება — 198. რეაქტიული ბაბი

## მიდგომა

მოკლედ, ეს ამოცანა ჩვეულებრივი ფიბონაჩია, ოღონდ ოდნავ სხვა პირობებით. ბაბი როცა ხტება, საფეხურებს შორის ან ერთს ტოვებს, ან ორს, რაც იმას ნიშნავს, რომ i-ურ საფეხურზე მოსახვედრად უნდა შევხედოთ (i-2) და (i-3) საფეხურებს. DP-ის მასივი იდეალურად მუშაობს, უბრალოდ ინდექსაციაში არ უნდა აგვერიოს არაფერი.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- განტოლება არის dp[i] = dp[i-2] + dp[i-3], რადგან ბაბი 'ახტება' ერთ ან ორ საფეხურს.
- საწყისი მნიშვნელობების (base case) სწორად დასმა გადამწყვეტია, თორემ მერე მთელი რიგი აირევა.

## ხშირი შეცდომები

- ინდექსების არევა, როცა n-ს ვანიჭებთ მნიშვნელობას და მერე dp[n-1]-ს ვბეჭდავთ.
