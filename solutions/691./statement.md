# Problem Statement — 691. უცნაური დალაგება 

> Copied from [AiaSoft.ge #691](https://aiasoft.ge/problem/691)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

პატარა თეომ გადაწყვიტა N რიცხვი გაანაწილოს N უჯრაში ოღონდ ისე, რომ მოცემული რიცხვებიდან ყველაზე პატარა აღმოჩნდეს პირველ უჯრაში, მეორე უჯრაში დარჩენილი რიცხვებიდან ყველაზე დიდი, მესამეში დარჩენილებიდან ისევ ყველაზე მცირე და ა. შ.

დაეხმარეთ თეოს რიცხვების დალაგებაში.

## Input

პირველ სტრიქონზე შემოდის ერთი მთელი დადებითი რიცხვი N (1<=N<=10⁵)
მეორე სტრიქონზე შემოდის ერთმანეთისგან ჰარით გამოყოფილი N რაოდენობის მთელი რიცხვი a1, a2, ..., aN (1<=ai<=10⁹)

## Output

ერთ სტრიქონზე გამოიტანეთ ერთმანეთისგან ჰარით გამოყოფილი N ცალი რიცხვი პირობაში მოცემული თანმიმდევრობით.
