# განმარტება — 691. უცნაური დალაგება

## მიდგომა

ამ ამოცანის მთავარი მუღამი იმაშია, რომ დალაგების შემდეგ რიცხვებს ვუყუროთ როგორც წყვილებს. თუ მასივს დავასორტირებთ, ყველაზე პატარა იქნება დასაწყისში (l ინდექსი), ხოლო ყველაზე დიდი — ბოლოში (r ინდექსი). ამიტომ, უბრალოდ ვბეჭდავთ a[l]-ს, მერე a[r]-ს, მერე ვამოძრავებთ ინდექსებს ცენტრისკენ, სანამ არ შეხვდებიან. საკმაოდ მარტივი ლოგიკაა, მთავარია 'two pointers' მიდგომა გამოიყენო და არ აირიო ინდექსებში.

## სირთულე

| | |
|-|--|
| დრო | O(n log n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- მასივის დალაგება 0-დან n-1-მდე ზრდადობით გვაძლევს ყველაფერს, რაც გვჭირდება.
- ორი პოინტერით მუშაობა საუკეთესო ვარიანტია, რადგან არ გვჭირდება დამატებითი მასივის შექმნა შედეგის შესანახად.

## ხშირი შეცდომები

- თუ n კენტია, ბოლო იტერაციაზე l და r ერთმანეთს ემთხვევა; თუ ეგ არ გაითვალისწინე და ორივეს დაბეჭდავ, ზედმეტ რიცხვს მიიღებ.
