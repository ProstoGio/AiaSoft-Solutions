# Problem Statement — 376. კომპონენტები

> Copied from [AiaSoft.ge #376](https://aiasoft.ge/problem/376)

**Time limit:** 2.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია არაორიენტირებული N წვეროიანი გრაფი, რომელიც თავდაპირველად არ შეიცავს წიბოებს.

თქვენი დავალებაა ყოველი წიბოს შემოსვლის შემდეგ გამობეჭდოთ გრაფში არსებული კომპონენტების ზომები ზრდადობით.

`i* შესაძლოა გრაფის საბოლოო სახე შეიცავდეს ჯერად წიბოებს. ანუ შესაძლოა ორი წვერო ერთზე მეტი წიბოთი უკავშირდებოდეს ერთმანეთს.`

## Input

პირველ ხაზზე მოცემულია ორი მთელი რიცხვი: N და M - შესაბამისად გრაფის წვეროებისა და წიბოების რაოდენობა.
მომდევნო M ხაზზე მოცემულია ორ-ორი რიცხვი: vi და ui, რაც ნიშნავს რომ i წიბო აკავშირებს vi და ui წვეროებს.

2 <= N <= 10³
1 <= M <= 10⁴
1 <= vi, ui <= N

## Output

ყოველი წიბოს შემოსვლის შემდეგ ერთ ცალკეულ ხაზზე გამობეჭდეთ გრაფში არსებული ყველა კომპონენტის ზომა ზრდადობით.
ერთ ხაზზე დაწერილი რიცხვები გამოყავით თითო ჰარით.
