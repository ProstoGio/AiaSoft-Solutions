# Problem Statement — 867. დაბეჭდე კენტები

> Copied from [AiaSoft.ge #867](https://aiasoft.ge/problem/867)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია N ელემენტია მასივი. თქვენი მიზანია დაბეჭდოთ მოცემულ მასივში კენტი რიცხვები იგივე თანმიმდევრობით, როგორი თანმიმდევრობითაც ისინი მოცემულია საწყის მასივში.

## Input

პირველ სტრიქონზე შემოდის ერთი მთელი დადებითი რიცხვი N, შემდეგ სტრიქონზე შემოდის N ცალი მთელი დადებითი რიცხვი.
$1<=N<=100$
$1<=a_i<=100$

## Output

ერთ სტრიქონზე გამოიტანეთ ყველა კენტი რიცხვი, თითოეული მათგანი დაშორებული უნდა იყოს ჰარებით.
