# Problem Statement — 903. ქალაქ ბაიას მერი

> Copied from [AiaSoft.ge #903](https://aiasoft.ge/problem/903)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ქალაქი ბაია ყველაზე ლამაზი ქალაქია მთელს ქვეყანაში. ერთხელ ბაიას მერმა გადაწყვიტა აეშენებინა განსაკუთრებული რესტორანი, რომელშიც განსაკუთრებული პურმარილი მომზადდებოდა და მთელი ქვეყანაში სახელს მოუტანდა. 
გადაწყდა, რესტორანი აშენებულიყო ბაიას ისეთ მოედანზე, რომელშიც ყველაზე მეტი ქუჩა შედიოდა (ბაიას ქუჩებზე ორმხრივი მოძრაობაა).
ბაიას მერმა თქვენ დაგიდოთ ხელშეკრულება და გთხოვათ, გეპოვათ მოედანი, რომელზეც ყველაზე მეტი ქუჩა შედის.
მოცემული გაქვთ იმ მოედნების წყვილები, რომლებიც ერთმანეთს უკავშირდებიან ქუჩებით. დაადგინეთ სარესტორნე მოედნის რიგითი ნომერი (ბაიაში მოედნებს ნომრები აქვთ). თუ ასეთი მოედანი ერთზე მეტია, მაშინ დაადგინეთ ისეთი ნომერი, რომელიც დანარჩენებზე მეტია.

## Input

შემომავალი მონაცემების პირველ სტრიქონზე შემოდია ორი ნატურალური რიცხვი N, (1<=N<=100), ქალაქ ბაიას მოედნების რაოდენობა და M, 0<=M<=N*(N-1)/2, ქუჩების რაოდენობა ბაიაში.

## Output

გამოიტანეთ ერთი ნატურალური რიცხვი, იმ მოედნის რიგითი ნომერი, რომელსაც ყველაზე მეტი ქუჩა უერთდება. თუ ასეთი ქუჩა რამდენიმეა, მაშინ გამოიტანეთ მათგან უდიდესი ნომერი.
