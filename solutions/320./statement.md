# Problem Statement — 320. ტალღა მატრიცაში

> Copied from [AiaSoft.ge #320](https://aiasoft.ge/problem/320)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 128 MB

## Task

მოცემულია NxM ზომის მატრიცა. მატრიცის (x, y) წერტილში აფეთქება მოხდა და იგი ტალღისებურად ვრცელდება. თქვენი მიზანია გამობეჭდოთ ტალღის გავრცელება ისე, როგორც მაგალითშია ნაჩვენები.

## Input

ერთ ხაზზე მოცემულია ოთხი მთელი რიცხვი: N, M, x და y

1 <= N, M <= 10³
1 <= x <= N
1 <= y <= M

## Output

შეავსეთ და დაბეჭდეთ NxM ზომის მატრიცა.
