# Problem Statement — 850. მინიმალური კენტი

> Copied from [AiaSoft.ge #850](https://aiasoft.ge/problem/850)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია ნატურალური რიცხვების მასივი. საჭიროა კენტ რიცხვებს შორის ყველაზე პატარა ელემენტის მნიშვნელობის პოვნა.

## Input

პირველ სტრიქონზე ჩაწერილია მასივის ელემენტების რაოდენობა.
მეორე სტრიქონზე მოთავსებულია ერთმანეთისაგან ჰარით გამოყოფილი მასივის ელემენტები.

## Output

გამოიტანეთ ერთი მთელი რიცხვი  - მასივის ყველაზე პატარა კენტი ელემენტის მნიშვნელობა.
თუ მასივი არ შეიცავს კენტ ელემენტებს, მაშინ გამოიტანეთ -1.
