# Problem Statement — 374. რიცხვთა ნაერთი

> Copied from [AiaSoft.ge #374](https://aiasoft.ge/problem/374)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

ორი რიცხვის ერთმანეთზე მიწებების შედეგად მიღებულ რიცხვს ვუწოდოთ "რიცხვთა ნაერთი". მაგალითად თუ გვაქვს 5 და 2, მაშინ მათი ნაერთი შეიძლება იყოს 52 ან 25.

მოცემულია ორი დადებითი მთელი რიცხვი. გამოთვალეთ მაქსიმალური მნიშვნელობის ნაერთი, რომელიც შეიძლება რომ მოცემული ორი რიცხვით მივიღოთ.

## Input

ერთ ხაზზე შემოდის ორი დადებითი მთელი რიცხვი: $a$ და $b$,  $(1 <= a, b <= 10^9)$

## Output

გამობეჭდეთ ერთი მთელი რიცხვი: მაქსიმალური ნაერთის მნიშვნელობა.
