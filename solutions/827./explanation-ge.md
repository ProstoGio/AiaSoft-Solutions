# განმარტება — 827. პალინდრომია?

## მიდგომა

პალინდრომის შემოწმების ყველაზე მარტივი გზა ისაა, რომ სტრიქონი შევატრიალოთ და შევადაროთ ორიგინალს. თავიდანვე შევინახე სტრიქონის ასლი `word2`-ში, რომ მერე reverse-ის მერე არ დამეკარგა საწყისი მნიშვნელობა. თუ შებრუნებული ვერსია და ორიგინალი ერთმანეთს ემთხვევა, ესე იგი პალინდრომია. თან ამ ამოცანაში სიგრძე 20-ს არ სცდება, ასე რომ ეს მიდგომა იდეალურია.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(n) |

## მთავარი დაკვირვებები

- C++-ის სტანდარტული reverse ფუნქცია ამარტივებს ყველაფერს, არაფერია საჭირო ხელით ატრიალო.
- ორიგინალი სტრიქონის ასლის შენახვა აუცილებელია, რადგან reverse ცვლის თავად სტრიქონს.

## ხშირი შეცდომები

- დაგავიწყდეს ორიგინალის ასლის გაკეთება და reverse-ით ორიგინალის გადაწერა, რის გამოც შედარება ვეღარ ხერხდება.
