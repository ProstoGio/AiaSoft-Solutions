# განმარტება — 622. სანაყინე

## მიდგომა

მოკლედ რომ ვთქვათ, გვაქვს განტოლება 3x + 5y = N და უნდა ვიპოვოთ ნებისმიერი არაუარყოფითი მთელი x და y. მათემატიკური ფორმულების წერას ისევ მარტივი ლუპი ჯობდა. დავიწყე მაქსიმალური რაოდენობის 5-ლარიანი ნაყინებით და ვამოწმებდი: თუ დარჩენილი თანხა 3-ზე იყოფა, ესე იგი ვიპოვე პასუხი. თუ არა, მაშინ 5-ლარიანების რაოდენობას ვაკლებდი ერთს და თავიდან ვამოწმებდი.

## სირთულე

| | |
|-|--|
| დრო | O(n) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- N მაქსიმუმ 200-ია, ამიტომ O(n) სრულიად საკმარისია და დროში ვერანაირად ვერ ჩავვარდებით.
- ხრიკი იმაშია, რომ როგორც კი (N - 5*y) გაიყოფა 3-ზე, ეგრევე გავდივართ ლუპიდან (break), რადგან ერთი სწორი წყვილიც საკმარისია.

## ხშირი შეცდომები

- მათემატიკურად რთული გზების ძებნა, როცა უბრალო brute-force-ი უფრო სწრაფად იწერება.
