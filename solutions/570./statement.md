# Problem Statement — 570. უახლოესი რიცხვი

> Copied from [AiaSoft.ge #570](https://aiasoft.ge/problem/570)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემულია არაკლებადი თანმიმდევრობით დალაგებული N ელემენტიანი a მიმდევრობა. თქვენი დავალებაა მოცემულ მიმდევრობაში იპოვოთ მოცემული x რიცხვის უახლოესი ელემენტი (თუ ასეთი რამდენიმეა, აირჩიეთ მათგან მინიმალური).

## Input

პირველ ხაზზე მოცემულია ორი მთელი რიცხვი: N და Q - შესაბამისად მიმდევრობაში ელენეტებისა და მოსაძებნი რიცხვების რაოდენობა.
მეორე ხაზზე მოცემულია N რაოდენობის მთელი რიცხვი a1,a2,..., aN.
მესამე ხაზზე მოცემულია Q რაოდენობის მთელი რიცხვი x1,x2,..., xQ.

1<=N,Q<=10⁵
-10⁹<=ai, xi<=10⁹

## Output

ერთ ხაზზე გამობეჭდეთ თითო ჰარით დაშორებული Q რაოდენობის რიცხვი: b1, b2, ..., bQ სადაც bi წარმოადგენს xi-სთან უახლოეს რიცხვს a მიმდევრობაში.
