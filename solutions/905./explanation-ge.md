# განმარტება — 905. ფიფქების სამკუთხედი

## მიდგომა

ეს არის კლასიკური შემთხვევა, სადაც nested loop-ები უნდა გამოვიყენო. გარე ციკლი ზრუნავს სტრიქონებზე, 1-დან N-მდე, ხოლო შიდა ციკლი აბეჭდინებს იმდენ ვარსკვლავს, რა ნომერ სტრიქონზეც ვარ. რეალურად, თუ პირველ სტრიქონზე ვარ, ერთი ვარსკვლავი მინდა, თუ მეორეზე — ორი. ამიტომაც შიდა ციკლის პირობა `j <= i` პირდაპირ ამ კავშირს მიჰყვება.

## სირთულე

| | |
|-|--|
| დრო | O(n²) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- რიგის ნომერი და ვარსკვლავების რაოდენობა ერთმანეთს ემთხვევა, რაც ამოცანას ძალიან ამარტივებს.
- ორმაგი ციკლი აქ ყველაზე ოპტიმალური და გასაგები მიდგომაა.

## ხშირი შეცდომები

- ციკლის ინდექსების არევა, როცა 0-დან ვიწყებთ და ვარსკვლავების რაოდენობა ერთი ცალით აცდენილი გამოდის.
