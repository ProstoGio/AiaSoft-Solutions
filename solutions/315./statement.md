# Problem Statement — 315. ირაკლი და მიმდევრობა

> Copied from [AiaSoft.ge #315](https://aiasoft.ge/problem/315)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 64 MB

## Task

ორაკლის ძალიან უყვარს მიმდევრობები, რომლებიც მხოლოდ 0-ს და 1-ს შეიცავს. დღეს ის ცდილობს დაადგინოს რამდენი განსხვავებული N წევრიანი მიმდევრობა არსებობს, სადაც სამი 1-იანი გვერდიგვერდ არ იქნება ჩაწერილი. ბევრი წვალების შემდეგ მიხვდა, რომ ეს საქმე არც თუ ისე მარტივია და დახმარებისთვის თქვენ მოგმართათ, მის ერთგულ პროგრამისტ მეგობარს. დაეხმარეთ ირაკლის ამოცანის ამოხსნაში.

## Input

მოცემულია ერთი მთელი რიცხვი: N  (1 <= N <= 32)

## Output

გამობეჭდეთ ერთი მთელი რიცხვი: N წევრიანი მიმდევრობების რაოდენობა, სადაც სამი ერთიანი გვერდიგვერდ არ დგას.
