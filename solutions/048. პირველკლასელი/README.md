# ამოხსნის ახსნა

მოცემულია 3 სიმბოლოსგან შემდგარი სტრიქონი.

პირველი და ბოლო სიმბოლო ყოველთვის ციფრია, ხოლო შუა სიმბოლო არის:

```text
+
```

ან

```text
-
```

საჭიროა გამოვთვალოთ ამ გამოსახულების მნიშვნელობა.

---

# მაგალითი

თუ შემოვიდა:

```text
2+3
```

პასუხია:

```text
5
```

თუ შემოვიდა:

```text
7-4
```

პასუხია:

```text
3
```

---

# მთავარი იდეა

რადგან გამოსახულება ყოველთვის სამი სიმბოლოსგან შედგება, შეგვიძლია პირდაპირ მივწვდეთ თითოეულ მათგანს.

```cpp
s[0]
```

პირველი რიცხვია.

```cpp
s[1]
```

ოპერაციის ნიშანია.

```cpp
s[2]
```

მეორე რიცხვია.

---

# სიმბოლოს რიცხვად გადაქცევა

სტრიქონში ციფრები ინახება როგორც სიმბოლოები.

მაგალითად:

```text
'5'
```

ჯერ კიდევ სიმბოლოა და არა რიცხვი.

მის რიცხვად გადასაქცევად ვაკლებთ `'0'`-ს (ან ASCII კოდით `48`).

```cpp
int a = s[0] - 48;
int b = s[2] - 48;
```

მაგალითად:

```text
'7' - '0' = 7
```

ასე ვიღებთ ნამდვილ მთელ რიცხვს.

---

# ოპერაციის შემოწმება

თუ შუა სიმბოლო არის:

```cpp
'+'
```

ვბეჭდავთ:

```cpp
a + b
```

ხოლო თუ არის:

```cpp
'-'
```

ვბეჭდავთ:

```cpp
a - b
```

# კოდის განხილვა

## სტრიქონის წაკითხვა

```cpp
string s;
cin >> s;
```

---

## რიცხვების მიღება

```cpp
int a = s[0] - 48;
int b = s[2] - 48;
```

---

## ოპერაციის შესრულება

```cpp
if(s[1] == '+'){
    cout << a + b;
}
else if(s[1] == '-'){
    cout << a - b;
}
```

# შეჯამება

რადგან გამოსახულების ფორმატი ყოველთვის ერთნაირია, პირდაპირ ვიღებთ პირველ რიცხვს, ოპერაციის ნიშანს და მეორე რიცხვს.

შემდეგ ვამოწმებთ რომელი ოპერაციაა მოცემული და ვბეჭდავთ შესაბამის შედეგს.
