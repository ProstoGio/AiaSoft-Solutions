# განმარტება — 885. წრის ფართობი

## მიდგომა

აქ არაფერია რთული, უბრალოდ ფორმულის შებრუნება გვჭირდება. თუ S = 3.14 * R², მაშინ R-ის საპოვნელად S უნდა გავყოთ 3.14-ზე და ფესვი ამოვიღოთ. რადგან რადიუსი მთელი რიცხვია, sqrt ფუნქცია ზუსტად იმას დაგვიბრუნებს, რაც გვინდა.

## სირთულე

| | |
|-|--|
| დრო | O(1) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- პრობლემის პირობით pi=3.14-ია, ამიტომ არ გამოიყენოთ M_PI ან სხვა უფრო ზუსტი მნიშვნელობები, თორემ პასუხი აირევა.
- მარტივი მათემატიკაა, უბრალოდ S/3.14-ის ფესვი უნდა ავიღოთ.

## ხშირი შეცდომები

- სტანდარტული PI-ს გამოყენება 3.14-ის მაგივრად, რაც პასუხს დააზიანებს.
