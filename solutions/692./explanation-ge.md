# განმარტება — 692. ორადი მეზობლები

## მიდგომა

მოკლედ, ამოცანა ითხოვს ორის ისეთი ხარისხების პოვნას, რომლებიც მოცემულ რიცხვზე მკაცრად ნაკლები და მკაცრად მეტია. მარტივი ლოგიკაა: ვატრიალებთ ციკლს, სანამ 2-ის ხარისხი რიცხვს არ გადააჭარბებს. მას მერე, რაც ვიპოვით პირველ ხარისხს, რომელიც x-ზე დიდია, მარტივია წინა ხარისხის გამოთვლა. მთავარია სწორად გავითვალისწინოთ მკაცრი უტოლობა, რომ შემთხვევით თვითონ რიცხვი არ ჩაგვეთვალოს მეზობლად.

## სირთულე

| | |
|-|--|
| დრო | O(n log x) |
| მეხსიერება | O(1) |

## მთავარი დაკვირვებები

- 2-ის ხარისხები ძალიან სწრაფად იზრდება, ამიტომ while loop-ით 30-40 იტერაციაზე მეტი არ დასჭირდება.
- long long აუცილებელია, რადგან x 2*10^9-მდეა და 32-ბიტიანი int გადაივსება.

## ხშირი შეცდომები

- x=2-ის შემთხვევაში არასწორი ლოგიკა, რადგან 1 ნაკლებია და 4 მეტია, რაც ბევრმა შეიძლება გამორჩეს.
