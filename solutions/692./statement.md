# Problem Statement — 692. ორადი მეზობლები

> Copied from [AiaSoft.ge #692](https://aiasoft.ge/problem/692)

**Time limit:** 1.0s &nbsp;|&nbsp; **Memory limit:** 256 MB

## Task

მოცემული რიცხვის ორად ნაკლები და ორად მეტი მეზობელი ვუწოდოთ, მასზე მკაცრად ნაკლებ და მასზე მკაცრად მეტ უახლოეს 2-ის ხარისხს.
მოცემული რიცხვებიდან თითოეულისთვის დაადგინეთ ორადი მეზობლები.

## Input

პირველ სტრიქონზე შემოდის ერთი ნატურალური რიცხვი N (1<=N<=10⁵)
შემდეგი N სტრიქონიდან თითოეულზე შემოდის ერთი ნატურალური რიცხვი x (2<=x<=2*10⁹)

## Output

თითოეული რიცხვისთვის განსხვავებულ სტრიქონზე გამოიტანეთ ჰარით გამოყოფილი ორი ნატურალური რიცხვი, შესაბამისად - მასზე მკაცრად ნაკლები და მკაცრად მეტი ორის ხარისხები.
