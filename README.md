# AiaSoft Solutions
## About This Repository

Clean and well-documented solutions to programming problems from AiaSoft.

## Structure

- `solutions/` — Contains solutions of problems
- Each problem folder contains:
  - `sol.cpp`
  - `expl.md` with explanation, complexity, and insights

## Languages

- C++

## How to Use

1. Browse solutions by difficulty or topic
2. Read the per-problem README for approach and complexity
3. Run the code locally

Contributions, suggestions, and feedback are welcome! 🚀
